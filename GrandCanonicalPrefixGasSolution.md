# grand canonical prefix gas解

この解では、停止前prefixを粒子数可変の気体の分配関数として読む。

長さ`n`まで停止しない条件は

```text
u_i>=0,  u_1+...+u_n<=1.
```

この領域の体積を`Z_n`とする。尾和公式より

```text
E[tau]=sum_{n>=0}Z_n.
```

累積和

```text
t_k=u_1+...+u_k
```

へ移ると、領域は順序付き配置

```text
0<=t_1<=...<=t_n<=1
```

になる。これは区間`[0,1]`上に`n`個の同種粒子を置くgrand canonical sectorであり、体積は

```text
Z_n=1/n!.
```

したがって活動度`z`の分配関数は

```text
Z(z)=sum_{n>=0} z^n Z_n = exp(z).
```

元の期待停止時刻は活動度`z=1`での全分配関数なので

```text
E[tau]=Z(1)=e.
```

停止前prefix全体を粒子数可変の順序配置として束ねることで、`e`が正規化定数として現れる。
