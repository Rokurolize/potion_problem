# 典型余白Palm解

この解では、停止前prefix全体の総量を先に正規化し、その中から「典型的な生存prefix」を一つ選んだときの余白を調べる。期待値`e`は、余白分布が自分自身を再生する正規化定数として出る。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

であり、求めたい量は

```text
A = E[tau].
```

## prefix測度を作る

区間`[0,1]`上の測度`M`を

```text
M(B) = sum_{n >= 0} P(S_n in B, S_n <= 1)
```

で定める。これは、停止前prefixの位置を全部重ねた測度である。全質量は

```text
M([0,1]) = A.
```

したがって、`M/A`でprefix位置`X`を一つ選ぶと、これは「典型的な停止前prefix」の位置になる。その余白を

```text
L = 1 - X
```

と呼ぶ。

## 典型余白に点を差し込む

ある停止前prefixが位置`x`にあるとき、その後ろには長さ`1-x`の余白がある。この余白の中に追加点`v`を置くと、

```text
x + v <= 1
```

なので、新しい停止前prefixが一つできる。逆に、空prefix以外の停止前prefixは、直前のprefixと最後の増分に一意に分解できる。

したがって、全ての非空停止前prefixの総量は、全ての停止前prefixの余白長を足したものに等しい。

```text
A - 1 = int_{[0,1]} (1 - x) M(dx).
```

典型余白`L`で書けば

```text
A - 1 = A E[L].
```

つまり

```text
E[L] = 1 - 1/A.        (1)
```

## 余白の尾をもう一度prefix測度として読む

次に、典型余白の尾分布を測る。`0 <= y <= 1`について

```text
P(L >= y)
= (1/A) M([0, 1-y]).
```

右辺は、境界を`1-y`に縮めたときの停止前prefix総量である。これを

```text
A(t) = M([0,t])
```

と書けば

```text
P(L >= y) = A(1-y) / A(1).
```

一方、非空prefixを直前prefixと最後の増分へ分解する議論は、境界`x`でも同じなので

```text
A(x) - 1 = int_0^x A(t) dt.        (2)
```

ここで(2)を`x=1`に代入すると

```text
A - 1 = int_0^1 A(t) dt.
```

しかし左辺は(1)より`A E[L]`であり、右辺も尾積分公式で

```text
int_0^1 A(t) dt
= A int_0^1 P(L >= y) dy
= A E[L].
```

つまり、典型余白を介すると、全質量の欠けた`1`は「選んだprefix自身」、残りの`A-1`は「典型余白の中に差し込める次prefixの量」として見える。

## 正規化定数を決める

(2)は全ての`x`で成り立つ。したがって微分して

```text
A'(x) = A(x),
A(0) = 1.
```

よって

```text
A(x) = exp(x).
```

初期境界`1`では

```text
E[tau] = A(1) = e.
```

## この見方の芯

通常は全prefix総量`A(x)`をそのまま積分方程式で解く。この解では一度`A(1)`で正規化し、典型的な停止前prefixの余白`L`を見る。余白に点を差し込む操作は、典型prefixを非空prefixへ写すPalm変換であり、非空部分の総量`A-1`を`A E[L]`として読む。

`e`は、典型余白の自己再生を可能にするprefix測度の正規化定数として現れる。
