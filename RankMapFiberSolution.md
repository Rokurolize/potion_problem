# 順位写像の繊維解

この解では、生存prefixの単体を、独立点配置から順位写像で切り出される一つの繊維として読む。

長さ`n`のprefixで

```text
S_i = X_1 + ... + X_i
```

と置く。`S_n <= 1`なら

```text
0 <= S_1 <= S_2 <= ... <= S_n <= 1
```

である。増分列`(X_1,...,X_n)`から累積時刻列`(S_1,...,S_n)`への変換はJacobian`1`なので、求める体積はこの順序領域の体積である。

## 順位写像で見る

独立に選んだ`n`個の点

```text
Y_1,...,Y_n in [0,1]
```

を考える。対角集合を除けば、各配置には一意な順位

```text
rank(Y_1,...,Y_n) in S_n
```

がある。対称性により、`n!`個の順位はすべて同じ体積を持つ。

全立方体`[0,1]^n`の体積は`1`である。したがって、順位写像の一つの繊維、つまり

```text
Y_1 <= Y_2 <= ... <= Y_n
```

の領域の体積は

```text
1/n!
```

である。この領域は上の累積時刻列の領域と同じなので

```text
P(S_n <= 1) = 1/n!.
```

尾和公式から

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
       = sum_{n >= 0} 1/n!
       = e.
```

この証明では、単体を直接積分しない。独立点配置の順位写像を見て、生存prefixが`n!`個の同じ繊維のうち一つに対応することだけを使う。
