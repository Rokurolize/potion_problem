# graded semigroup prefix解

この解では、停止前のprefixを、合計値で次数付けされた可換半群の要素として扱う。

独立な一様乱数のprefix

```text
(x_1,...,x_n)
```

は、非負実数の加法半群の`n`個の元を並べたものと見なせる。次数を

```text
deg(x_1,...,x_n)=x_1+...+x_n
```

と置くと、停止前条件は

```text
deg(x_1,...,x_n)<=1
```

である。

固定した長さ`n`で次数が`r`のprefixは、`n-1`次元単体の層を作る。その層の体積密度は

```text
r^{n-1}/(n-1)!
```

であるから、次数1以下の部分の体積は

```text
int_0^1 r^{n-1}/(n-1)! dr=1/n!.
```

これはそのまま

```text
P(tau>n)=1/n!
```

である。尾和を取ると

```text
E[tau]=sum_{n>=0}P(tau>n)=sum_{n>=0}1/n!=e.
```

prefixの長さを外側の和、合計値を内側の次数として分離すると、停止時刻の期待値は次数付き半群の単位球体積を全長さで足したものになる。
