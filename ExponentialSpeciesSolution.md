# 有限列の分配関数として解く

この解では、停止前prefix全体を「有限個の順序付き点配置」としてまとめ、その全質量を一つの分配関数として読む。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前に生きているprefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).        (1)
```

## 増分列を点配置へ送る

固定した`n`について、

```text
t_i = S_i
```

と置く。`S_n <= 1`は

```text
0 < t_1 < t_2 < ... < t_n <= 1
```

と同じである。逆変換は

```text
U_1 = t_1,
U_2 = t_2 - t_1,
...
U_n = t_n - t_{n-1}.
```

この変換は三角行列で対角成分がすべて`1`なので、体積を変えない。

よって(1)は

```text
E[tau]
= sum_{n >= 0} volume { 0 < t_1 < ... < t_n <= 1 }.        (2)
```

ここまでは、各`n`を個別に数えているように見える。次に、それらを一つの対象として束ねる。

## 順序を忘れた有限集合として数える

`[0,1]`の中の`n`個の相異なる点の集合を考える。点に順序を付けずに集合として見れば、ほとんどすべての配置はただ一つの昇順表示

```text
t_1 < ... < t_n
```

を持つ。

一方、順序付きの`n`個組

```text
(x_1,...,x_n) in [0,1]^n
```

は、同じ点集合を`n!`通りに並べ替えて表す。したがって

```text
volume { 0 < t_1 < ... < t_n <= 1 } = 1 / n!.
```

これを(2)に代入すれば

```text
E[tau] = sum_{n >= 0} 1 / n!.        (3)
```

しかし、ここで重要なのは`1/n!`を個別に得たことではない。(3)の右辺は、区間`[0,1]`上の有限集合全体の分配関数である。

## 分配関数としての`e`

有限集合の分配関数は、各点が重み`1`を持つとき

```text
Z = sum_{n >= 0} 1/n!.
```

これは指数型母関数の最も基本的な形であり、

```text
Z = exp(1).
```

したがって

```text
E[tau] = Z = exp(1) = e.
```

## 見方の要点

一口ごとの増分列を、停止前の到着時刻の有限集合へ変える。すると、求める期待値は「停止時刻の平均」から「区間上の有限集合全体の分配関数」へ変わる。

答えの`e`は、各回の確率を最後に足した偶然の値ではない。有限集合という構造の分配関数が、最初から指数関数だから現れる。
