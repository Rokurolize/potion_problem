# colimit simplex volume解

この解では、停止前履歴を長さごとの単体の直和的なcolimitとして見る。

長さ`n`で止まっていない履歴は

```text
D_n={x_i>=0, x_1+...+x_n<=1}
```

である。期待値の尾和公式は、停止前である各長さの空間を互いに混ぜず、長さ方向に足し合わせる。

```text
E[tau]=sum_{n>=0}P(tau>n).
```

各`D_n`は標準`n`単体であり、その体積は

```text
vol(D_n)=1/n!
```

である。したがって、長さごとの停止前単体をcolimit的に集めた全体の測度は

```text
sum_{n>=0}vol(D_n)=sum_{n>=0}1/n!=e.
```

停止時刻の期待値は、単一の大きな空間を積分するというより、長さごとの停止前単体を余積として並べ、その体積を合計したものとして現れる。
