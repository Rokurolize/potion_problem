# Wiener-Hopf renewal因子解

この解では、停止前prefixの総量を、正の増分だけを持つランダムウォークのWiener-Hopf因子が退化したrenewal因子として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 正の増分では下降因子が消える

一般のランダムウォークでは、境界通過の解析は上昇側と下降側のWiener-Hopf因子に分かれる。しかしこの問題の増分は常に非負である。したがって下降側の因子は自明で、停止前prefixの測度は上昇側のrenewal測度だけになる。

一様密度を`f=1_{[0,1]}`と書くと、renewal測度の密度側は

```text
R = delta_0 + f + f*f + f*f*f + ...
```

である。境界`1`までしか見ないので、各畳み込みで`f`の右端はまだ先頭部分に影響しない。したがって`0 <= x <= 1`では、renewal因子は半直線上の単位密度`h=1_{x>=0}`に対する

```text
R = delta_0 + h + h*h + h*h*h + ...
```

と同じである。

Laplace側で見れば、これはWiener-Hopfの上昇因子

```text
1 / (1 - 1/s)
```

の局所展開であり、時間側では

```text
R(dx) = delta_0(dx) + exp(x) dx
```

になる。よって、境界`1`までのrenewal質量は

```text
R([0,1]) = 1 + int_0^1 exp(x) dx = e.
```

これは停止前prefixの期待個数そのものなので

```text
E[tau] = e.
```

この証明では、停止時刻の分布を直接求めない。非負増分のためWiener-Hopf分解の下降因子が消え、上昇renewal因子だけが残る。`e`は、その上昇因子を境界`1`まで積んだ総質量として現れる。
