# Mellin transform tail解

この解では、停止前の尾確率を合計値のMellin型モーメントとして読む。

長さ`n`で停止前にある条件は

```text
S_n=x_1+...+x_n<=1
```

である。合計値`S_n=r`で単体を切ると、断面の体積密度は

```text
r^{n-1}/(n-1)!
```

になる。これは区間`0<r<1`上のMellin核`r^{s-1}`を、`s=n`で評価したものと同じ形である。

したがって尾確率は

```text
P(tau>n)=1/(n-1)! int_0^1 r^{n-1} dr
        =1/(n-1)! * 1/n
        =1/n!.
```

あとは停止時刻の尾和公式だけで

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

合計値の断面密度をMellin核として見れば、`1/n`という積分値と`1/(n-1)!`という断面係数が噛み合って、ちょうど`1/n!`が出る。
