# beta function slice解

この解では、停止前単体を一つの座標と残りの単体に分け、Beta関数で評価する。

長さ`n`で停止前の領域を

```text
Delta_n={x_i>=0, x_1+...+x_n<=1}
```

とする。最後の座標を`t=x_n`で固定すると、残りの`n-1`個は

```text
x_1+...+x_{n-1}<=1-t
```

を満たす。したがって帰納的に断面体積は

```text
(1-t)^{n-1}/(n-1)!
```

である。よって

```text
vol(Delta_n)=int_0^1 (1-t)^{n-1}/(n-1)! dt
            =B(1,n)/(n-1)!
            =1/n!.
```

これはそのまま

```text
P(tau>n)=1/n!
```

を意味する。尾和を取れば

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

一つの座標を剥がすたびにBeta積分が現れ、停止前確率の階乗分母を作っている。
