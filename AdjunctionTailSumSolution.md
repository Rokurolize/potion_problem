# adjunction tail sum解

この解では、停止時刻を「点ごとの履歴」と「合計値」の間の随伴で読む。

長さ`n`の履歴

```text
(x_1,...,x_n)
```

から合計値

```text
s=x_1+...+x_n
```

を取る写像を考える。停止判定は履歴全体ではなく、この合計値が`1`以下かどうかだけで決まる。つまり、履歴空間から合計値空間へ押し出し、そこから区間`[0,1]`へ引き戻す随伴的な読み替えである。

合計値が`s`のファイバーは`n-1`単体で、体積密度は

```text
s^{n-1}/(n-1)!
```

である。したがって

```text
P(tau>n)=int_0^1 s^{n-1}/(n-1)! ds=1/n!.
```

尾和公式から

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

履歴を合計値へ送る操作と、しきい値以下へ戻す操作の間で見ると、期待値は各ファイバー体積の尾和になる。
