# Padé有限地平線解

この解では、無限に飲み続ける問題をいきなり解かず、まず最大`N`口までしか数えない有限地平線問題を作り、その境界値を極限で読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 有限地平線

境界を`x`にしたとき、停止前prefixを長さ`N`までだけ数える量を

```text
F_N(x) = sum_{n=0}^N P(S_n <= x)
```

と置く。`0 <= x <= 1`では一口追加は積分なので

```text
F_N(x) = 1 + int_0^x F_{N-1}(t) dt,
F_0(x) = 1.
```

したがって`F_N`は指数関数の有限Taylor多項式

```text
F_N(x) = sum_{n=0}^N x^n/n!
```

である。

ここで有限地平線を「指数関数のPadé近似へ向かう対角列の片側」として見る。有限段の再帰は、分母側の補正をまだ入れていないPadé表の最初の列であり、`N`を増やすと境界値`F_N(1)`は単調に真の評価汎関数へ近づく。

非負項だけなので

```text
lim_{N -> infinity} F_N(1)
  = sum_{n >= 0} 1/n!
  = exp(1).
```

したがって

```text
E[tau] = e.
```

この証明では、停止時刻の分布を求めない。有限地平線の値関数を作ると、各段は前段の原始関数になり、境界`1`での値は指数関数の有限近似列になる。`e`は、有限地平線を外した極限の境界値として現れる。
