# テンソル余代数余単位解

この解では、生存prefix全体をテンソル余代数の全次数成分として束ね、余単位で総質量を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefix数から

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

累積時刻

```text
t_k=S_k
```

へ移ると、長さ`n`の生存prefixは

```text
0<t_1<...<t_n<1
```

という順序付きテンソル成分である。対称化してテンソル余代数の`n`次成分へ送ると、順序付き成分の体積は

```text
1/n!
```

になる。全次数を足した群的元は、区間`[0,1]`の定数関数`1`から作る指数型の元であり、余単位評価は

```text
sum_{n>=0} 1/n! = e.
```

したがって

```text
E[tau]=e.
```

ここでは、停止前履歴を長さごとにばらして計算するかわりに、全prefixをテンソル余代数の一つの群的元としてまとめている。
