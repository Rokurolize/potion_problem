# isometry volume prefix解

この解では、停止前単体を座標の等長写像で標準位置に送って測る。

長さ`n`で停止前にある集合は

```text
D_n={x_i>=0, x_1+...+x_n<=1}
```

である。座標基底をそのまま標準基底へ送る写像は等長で、Lebesgue体積を保つ。したがって`D_n`は標準単体

```text
conv(0,e_1,...,e_n)
```

として測ってよい。

標準単体の体積は基底平行体の`1/n!`だから

```text
P(tau>n)=vol(D_n)=1/n!.
```

よって

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

等長写像で標準座標に置き直すと、停止前prefix空間はただの標準単体になる。
