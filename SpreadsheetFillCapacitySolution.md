# spreadsheet fill capacity解

この解では、残り容量ごとの値をspreadsheetの列として埋める再帰で見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。残り容量が`x`のとき、停止前prefixの期待個数を`F(x)`と書く。空prefixを一つ数え、次の増分`u`が`0<=u<=x`なら残りは`x-u`である。

```text
F(x)=1+int_0^x F(x-u) du
    =1+int_0^x F(t) dt.
```

spreadsheetで左から右へ容量列を埋めるなら、各セルは「1」と左側までの累積和で決まる。連続極限では

```text
F'(x)=F(x),  F(0)=1.
```

したがって

```text
F(x)=e^x.
```

初期容量は`1`なので

```text
E[tau]=F(1)=e.
```

表計算の比喩で言えば、答えは最後のセルを読むだけであり、列全体を支配する累積和ルールが指数関数を作る。
