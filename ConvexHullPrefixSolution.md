# convex hull prefix解

この解では、停止前prefixを原点と標準基底の凸包として見る。

長さ`n`で止まらない条件は

```text
x_1+...+x_n<=1,  x_i>=0
```

である。これはちょうど

```text
conv(0,e_1,...,e_n)
```

という標準単体である。したがって停止前確率は、この凸包の`n`次元体積に等しい。

標準基底を並べた行列の行列式は1なので、この単体の体積は平行体の体積を`n!`で割って

```text
P(tau>n)=vol conv(0,e_1,...,e_n)=1/n!
```

となる。

尾和公式から

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

停止前領域は、確率の問題というより、原点と標準基底が張る凸包の体積そのものとして現れる。
