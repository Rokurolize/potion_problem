# Monge map prefix volume解

この解では、停止前prefixを単位立方体から単体へのMonge型の押し出しで測る。

長さ`n`で止まらない履歴は

```text
D_n={x_i>=0, x_1+...+x_n<=1}
```

である。単位立方体`[0,1]^n`上の一様測度を、順序統計量の差分

```text
0<y_1<...<y_n<1,
x_1=y_1,
x_2=y_2-y_1,
...,
x_n=y_n-y_{n-1}
```

で`D_n`へ送る。この三角写像のJacobianは1で、順序領域の体積は立方体の`1/n!`である。

したがって

```text
P(tau>n)=vol(D_n)=1/n!.
```

尾和公式により

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

停止前単体は、順序付けた一様点から隣接差分を取るMonge型写像で得られ、その体積は順序領域の`1/n!`として現れる。
