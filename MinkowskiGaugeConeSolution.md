# Minkowskiゲージ錐解

この解では、生存条件を正錐上のMinkowskiゲージで切った部分として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

`n`口の生存領域は正錐

```text
C_n = { u_i >= 0 }
```

を線形汎関数

```text
g(u) = u_1 + ... + u_n
```

のゲージ球

```text
{ g(u) <= 1 }
```

で切ったものである。したがって`P(S_n <= 1)`は、このゲージ球の体積である。

## ゲージ半径で分解する

正錐上で

```text
u = r p,
g(p) = 1,
r >= 0
```

と分解する。断面`g(p)=1`は標準単体であり、ゲージ半径`r`で拡大すると体積要素は`r^{n-1}`倍になる。

断面の正規化を直接体積公式で決めず、指数重みで決める。独立な指数積分から

```text
int_{C_n} exp(-g(u)) du = 1
```

である。一方、ゲージ半径で分解すると

```text
int_0^infty exp(-r) r^{n-1} dr * A_n = 1,
```

ここで`A_n`は断面側の定数である。Gamma積分により

```text
A_n = 1/(n-1)!.
```

したがってゲージ球`g(u) <= 1`の体積は

```text
int_0^1 r^{n-1}/(n-1)! dr = 1/n!.
```

空prefixは`1`である。よって

```text
E[tau]
  = sum_{n >= 0} 1/n!
  = e.
```

この証明では、単体体積を最初から使わない。正錐をMinkowskiゲージ半径とゲージ断面に分け、断面の正規化を指数積分で決める。`e`は、ゲージ球の体積を全次元で足した値として現れる。
