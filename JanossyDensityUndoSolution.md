# Janossy density undo解

この解では、停止前prefixの体積をPoisson過程のJanossy密度から空白因子だけ外して読む。

単位区間上の強度`1`のPoisson過程を考える。ちょうど`n`個の点があり、それを順序付きに

```text
0<=T_1<=...<=T_n<=1
```

と並べたとき、Janossy密度はこの順序単体上で一定の

```text
e^{-1}
```

である。従って

```text
P(N_1=n)=e^{-1} vol{0<=T_1<=...<=T_n<=1}.
```

一方、元の問題の停止前条件

```text
U_1+...+U_n<=1
```

に対し

```text
T_k=U_1+...+U_k
```

と置けば、ヤコビアン`1`で同じ順序単体になる。よって

```text
P(U_1+...+U_n<=1)
= vol{0<=T_1<=...<=T_n<=1}
= e P(N_1=n).
```

尾和公式から

```text
E[tau]
=sum_{n>=0}P(U_1+...+U_n<=1)
=e sum_{n>=0}P(N_1=n)
=e.
```

Poisson化で新しい確率を入れるのではなく、Janossy密度の共通因子`e^{-1}`を戻すだけで、停止前prefix全体の質量が正規化定数の逆数として出る。
