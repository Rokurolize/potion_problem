# entire function survival解

この解では、生存確率列を容量変数の整関数としてまとめる。

容量`x`に対して

```text
F(x)=sum_{n>=0}P(U_1+...+U_n<=x)
```

とおく。元の期待停止時刻は`F(1)`である。`0<=x<=1`では、各項を最後の変数で切れば

```text
P(U_1+...+U_n<=x)=int_0^x P(U_1+...+U_{n-1}<=x-u)du.
```

従って項別微分により

```text
F'(x)=F(x).
```

また`x=0`では空prefixだけが残るので

```text
F(0)=1.
```

この初期値問題を満たす整関数は一意であり

```text
F(x)=e^x.
```

したがって

```text
E[tau]=F(1)=e.
```

ここでは、階数ごとの単体体積を計算する代わりに、生存確率列の総和が整関数として指数関数に固定されることを使う。
