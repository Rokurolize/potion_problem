# SSA phi capacity解

この解では、残り容量をSSA形式の値として引き継ぐ再帰で読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。残り容量が`x`のブロックで、停止前prefix期待個数を`F(x)`と書く。ブロック先頭では空prefixを一つ数え、次の入力`u`が`0<=u<=x`ならphiで次ブロックの容量`x-u`を受け取る。

```text
F(x)=1+int_0^x F(x-u) du.
```

変数を替えると

```text
F(x)=1+int_0^x F(t) dt.
```

したがって

```text
F'(x)=F(x),  F(0)=1,
```

より

```text
F(x)=e^x.
```

初期容量は`1`なので

```text
E[tau]=F(1)=e.
```

SSAの言葉では、残り容量の値が各反復でphiを通って更新され、その不動点方程式が指数関数になる。
