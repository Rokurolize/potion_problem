# Erlang順序時計解

この解では、生存prefixの領域をErlang時計の到着時刻の順序密度から読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数える標準恒等式は

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

`S_n<=1`の領域で

```text
T_k=U_1+...+U_k
```

と置くと、Jacobianは`1`で

```text
0<T_1<...<T_n<1
```

を得る。

単位率Poisson過程の第`n`到着時刻はErlang時計であり、`N(1)=n`で条件付けた到着時刻はこの順序領域上で密度`n!`を持つ。条件付き密度の全質量は`1`なので、この順序領域の体積は

```text
1/n!.
```

したがって

```text
P(S_n<=1)=1/n!,
E[tau]=sum_{n>=0}1/n!=e.
```

この証明では、Erlang時計が持つ「到着順序の正規化」を使う。累積和は到着時刻、元の一様増分は隣接到着間隔として読まれる。
