# Dirichlet series volume解

この解では、尾確率列をDirichlet級数風に重み付けしてから、重みを外す。

長さ`n`の停止前領域を

```text
D_n={x_i>=0, x_1+...+x_n<=1}
```

と置く。体積は

```text
vol(D_n)=1/n!
```

である。ここで確認用にDirichlet型の母関数

```text
B(s)=sum_{n>=1} vol(D_n)/n^s
```

を考えると、係数として入っているのはまさに停止前確率である。期待値に必要なのは重みなしの値なので、`s=0`に戻せば

```text
B(0)=sum_{n>=1}1/n!.
```

`n=0`の項`P(tau>0)=1`を加えて

```text
E[tau]=1+B(0)=sum_{n>=0}1/n!=e.
```

Dirichlet級数の形で係数列を保持しても、重みを外した瞬間に停止前単体体積の尾和がそのまま現れる。
