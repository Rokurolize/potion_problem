# renewal測度の微分方程式解

この解では、各`n`の確率を個別に扱わず、部分和全体が作るrenewal測度をそのまま未知量にする。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

`S_0=0`とすれば、停止時刻は境界`1`以下にある部分和の個数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって求めたい値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## renewal測度を置く

`S_n`の分布を`mu_n`と書き、測度

```text
R = sum_{n >= 0} mu_n
```

を考える。これは部分和が各集合を平均何回訪れるかを数える測度である。

上の期待値は、この測度の区間質量そのものになる。

```text
E[tau] = R([0,1]).
```

一様分布の測度を

```text
lambda_1(dt) = 1_{0<=t<=1} dt
```

と書くと、`mu_{n+1}=mu_n * lambda_1`なので

```text
R = delta_0 + R * lambda_1.
```

ここまではrenewal方程式そのものである。

## `0<x<1`では微分方程式になる

分布関数

```text
F(x) = R([0,x])
```

を`0<=x<=1`で見る。`R`には`0`に原子`1`があり、その後の連続部分は`R*lambda_1`から来る。

`0<x<1`では、一様分布の台の上端はまだ効かないので

```text
R((x,x+h])
= (R * lambda_1)((x,x+h]) + o(h)
```

と薄い幅で見るのが明瞭である。区間`(x,x+h]`へ新しく入るrenewal質量は、直前のrenewal点`y`が`[0,x]`にあり、次の一様増分が`x-y`から`x+h-y`の間に来る分である。`0<h`を小さく取れば、この幅は`h`であり、増分密度は`1`である。

したがって、`h`で割って`h↓0`とすると、流入密度は

```text
R([0,x]) = F(x)
```

である。つまり連続部分について

```text
F'(x) = F(x)        (0<x<1)
```

が成り立つ。

初期条件は、`S_0=0`の原子を含むので

```text
F(0) = R([0,0]) = 1.
```

したがって

```text
F(x) = exp(x)       (0<=x<=1).
```

最後に

```text
E[tau] = R([0,1]) = F(1) = e.
```

## この見方の芯

この証明では、生存確率列`P(S_n<=1)`も、停止時刻の分布も、単体体積も主役にしない。

部分和全体を一つのrenewal測度`R`としてまとめると、一様増分の密度`1`により「境界`x`でのrenewal流入密度」が「そこまでに蓄積したrenewal質量」そのものになる。境界までの総質量が自分自身を増殖率にするため、指数関数が現れる。
