# Doob decomposition prefix解

この解では、停止前prefixの数を劣マルチンゲールのDoob分解ではなく、その補償項として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。`N`までに生き残っているprefixの個数を

```text
C_N=sum_{n=0}^{N} 1_{S_n<=1}
```

と置く。ただし`S_0=0`である。各項の期待値は

```text
E[1_{S_n<=1}]=P(S_n<=1).
```

ここで`S_n<=1`の領域は

```text
{u_i>=0, u_1+...+u_n<=1}
```

という`n`次元単体で、体積は`1/n!`である。したがって

```text
E[C_N]=sum_{n=0}^{N} 1/n!.
```

単調収束により

```text
E[tau]=E[sum_{n>=0} 1_{S_n<=1}]
      =sum_{n>=0} 1/n!
      =e.
```

この見方では、停止時刻を直接分布で追わない。停止前であることを示す指示関数列を数え、その予測可能な平均増加量だけを足す。
