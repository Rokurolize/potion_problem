# 分布差分影解

この解では、Irwin-Hall分布そのものを求めず、境界`1`で有限差分表示がどう潰れるかだけを見る。

`X_i`を独立な`[0,1]`一様乱数、`S_n=X_1+\cdots+X_n`とする。`S_n`の分布関数は、一様密度を畳み込んだものの原始関数である。

一様密度はHeaviside関数`H(x)=1_{x>=0}`を使って

```text
1_{0 <= x <= 1} = H(x) - H(x-1)
```

と書ける。したがって`n`回畳み込むと、形式的には

```text
(H - shift_1 H)^{*n}
```

という有限差分になる。

## 境界1では最初の影だけが残る

分布関数まで一回積分すると、`S_n`の分布関数は

```text
P(S_n <= t)
  = 1/n! * sum_{k=0}^{floor(t)} (-1)^k binom(n,k) (t-k)^n
```

で表される。

ここで欲しいのは`0 <= t <= 1`の端点`t=1`だけである。この点では、`k>=2`の項は範囲外であり、`k=1`の項も

```text
(1-1)^n = 0
```

で消える。残るのは`k=0`の項だけなので

```text
P(S_n <= 1) = 1/n!.
```

したがって尾和公式から

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
       = sum_{n >= 0} 1/n!
       = e.
```

この証明では、Irwin-Hall分布全体を扱わない。上端`1`を持つ一様分布は有限差分として見えるが、評価点も同じ`1`なので、差分の影は端点でほとんど消え、最初の半直線項だけが残る。
