# 指数型生成測度解

独立一様乱数を `X_1, X_2, ...` とし、

```text
S_n = X_1 + ... + X_n
```

とおく。停止前prefixの個数として

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので、

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

`n >= 1` について、累積時刻

```text
t_i = S_i
```

に変数変換する。差分で元に戻れるためヤコビアンは `1` であり、`S_n <= 1` は

```text
0 <= t_1 <= ... <= t_n <= 1
```

という順序付き配置空間になる。

## ラベルを忘れた生成測度

順序付き配置を、ラベル付きの `n` 点配置 `[0,1]^n` から「時刻順に並べ直す」ことで得たものと見る。対角集合は体積 `0` なので、ほとんどすべての配置は `n!` 通りのラベル順を持つ。

したがって、順序付き配置空間の測度は

```text
1 / n!
```

である。

ここで重要なのは、個々の `n` を積分で処理することではない。全ての有限配置を同時に持つ生成測度

```text
Z = sum_{n >= 0} vol([0,1]^n) / n!
```

を考えると、1点配置の体積が `1` なので、指数型生成関数の基本公式から

```text
Z = exp(1)
```

である。

停止前prefix測度は、まさにこの生成測度の各次数成分に一致する。よって

```text
E[tau]
= sum_{n >= 0} P(S_n <= 1)
= Z
= e
```

となる。

この証明では、`1/n!` を単体積分の答えとしてではなく、ラベル付き有限配置から時刻順を忘れる指数型生成測度の正規化として読む。`e` は、区間 `[0,1]` 上の有限配置全体の指数型総質量である。
