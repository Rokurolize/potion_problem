# 対数残量時計解

この解では、残り距離の割合をさらに対数時計へ変える。停止前である条件は時計が有限であることに吸収され、`1/n!`は残量の掛け算を対数で直線化したときの指数積分として現れる。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 残量割合から対数時計へ

長さ`n`の停止前prefixを固定する。残り距離を

```text
R_0 = 1,
R_i = 1 - S_i
```

と書く。停止前では`0 <= U_i <= R_{i-1}`なので、

```text
V_i = U_i / R_{i-1}
```

は`[0,1]`に入る。ここで

```text
W_i = -log(1 - V_i)
```

と置く。すると`W_i >= 0`であり、

```text
1 - V_i = exp(-W_i),
R_i = R_{i-1} exp(-W_i).
```

したがって

```text
R_i = exp(-(W_1 + ... + W_i)).
```

残量の掛け算は、対数時計ではただの足し算になる。

## Jacobianを読む

`U_i`は

```text
U_i = R_{i-1}(1 - exp(-W_i))
```

である。`U_i`は`W_1,...,W_i`だけに依存するので、変換は下三角である。対角成分は

```text
partial U_i / partial W_i
= R_{i-1} exp(-W_i)
= R_i
= exp(-(W_1 + ... + W_i)).
```

よってJacobianは

```text
prod_{i=1}^n exp(-(W_1 + ... + W_i)).
```

同じ`W_j`が何回現れるかを数えると、

```text
prod_{i=1}^n exp(-(W_1 + ... + W_i))
= exp(-sum_{j=1}^n (n-j+1) W_j).
```

停止前prefixの領域は、`W_i >= 0`の正直交錐全体に写る。したがって

```text
P(S_n <= 1)
= int_{[0,infty)^n} exp(-sum_{j=1}^n (n-j+1) w_j) dw_1...dw_n.
```

積分は分離して、

```text
P(S_n <= 1)
= prod_{j=1}^n 1/(n-j+1)
= 1/n!.
```

`n=0`は空prefixなので寄与は`1`である。

## 期待値

よって

```text
E[tau]
= sum_{n >= 0} P(S_n <= 1)
= sum_{n >= 0} 1/n!
= e.
```

## この見方の芯

停止前の履歴を、飲んだ絶対量ではなく「残量を何倍に縮めたか」の対数時計として見る。すると停止条件は斜めの単体ではなく正直交錐になり、各時計`W_j`はその後の全ての残量に効くため、係数`n-j+1`を持つ指数重みになる。

`e`は単体体積の和としてではなく、残量を掛け算で縮める履歴を対数化したときの、分離した指数時計積分の総和として現れる。
