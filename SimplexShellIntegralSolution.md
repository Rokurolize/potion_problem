# simplex shell integral解

この解では、停止前単体を合計値のshellに分解する。

長さ`n`の停止前領域は

```text
D_n={u_i>=0, u_1+...+u_n<=1}.
```

合計値

```text
r=u_1+...+u_n
```

で切る。`r`のshellは、`n-1`次元単体を倍率`r`で拡大したもので、その密度は

```text
r^{n-1}/(n-1)!
```

である。したがって

```text
vol(D_n)=int_0^1 r^{n-1}/(n-1)! dr=1/n!.
```

尾和公式により

```text
E[tau]=sum_{n>=0}vol(D_n)=sum_{n>=0}1/n!=e.
```

この見方では、停止境界までの単体を同じ合計値を持つshellの連続和として測っている。
