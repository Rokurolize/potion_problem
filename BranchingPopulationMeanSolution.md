# 分枝個体群の平均数解

独立一様乱数を `X_1, X_2, ...` とし、

```text
S_n = X_1 + ... + X_n
```

とおく。停止前prefixを個体と見る。空prefix `S_0=0` は時刻 `0` にいる一つの祖先であり、prefix `S_n=s` は境界 `1` の手前にいる限り、次の一口 `x` によって時刻 `s+x` に子prefixを作る。

境界を `t` に動かし、時刻 `t` までに存在しうるprefix個体の期待総数を

```text
B(t) = sum_{n >= 0} P(S_n <= t)
```

と書く。求める値は `B(1)` である。

## 個体が作る出生密度

あるprefix個体が時刻 `s <= t` にいるとする。一様性により、その個体が次の一口で薄い区間

```text
(t, t + h]
```

へ子を作る確率は、`h` の一次まで

```text
h
```

である。これは `s` に依存しない。必要なのは `X = u - s` が長さ `h` の区間に入ることだけで、`0 <= t-s <= 1` は `0 <= t <= 1` の範囲では境界内のprefixに対して問題にならない。

したがって、時刻 `t` の薄い層へ流れ込む子個体の期待密度は、その時点までに存在していた全prefix個体の期待数そのもの、

```text
B(t)
```

である。

## 平均個体数の方程式

空prefixが初期個体として一つあるので

```text
B(0) = 1
```

である。上の出生密度から

```text
B'(t) = B(t)      (0 <= t <= 1)
```

を得る。よって

```text
B(t) = exp(t)
```

である。

最後に

```text
E[tau]
= sum_{n >= 0} P(S_n <= 1)
= B(1)
= e
```

となる。

この証明では、`n`段目の単体体積を個別に測らない。停止前prefix全体を、時刻方向に子を生む分枝個体群として見て、平均出生率が現在の平均個体数そのものになることだけを見る。`e` は、単位出生率の平均個体数として現れる。
