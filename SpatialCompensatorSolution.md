# 空間補償子で読む解

この解では、飲む回数を時間方向の停止問題としてではなく、境界`x`を右へ動かしたときの点過程として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

`S_0=0`も含め、境界`x`までに現れたprefixの個数を

```text
N(x) = sum_{n >= 0} 1_{S_n <= x}
```

と置く。求めたい値は

```text
E[tau] = E[N(1)].
```

## 薄い層へ入る期待数

`0<=x<x+h<=1`を考える。`N(x+h)-N(x)`は、薄い層

```text
(x, x+h]
```

に入ったprefixの個数である。

層に入るprefix`S_{n+1}`は、必ず一つ前のprefix`S_n`を持つ。もし`S_n=s<=x`なら、

```text
S_{n+1}=s+U_{n+1}
```

が`(x,x+h]`に入る条件は

```text
U_{n+1} in (x-s, x+h-s].
```

ここで`0<=s<=x`かつ`x+h<=1`なので、この区間は`[0,1]`内にあり、長さはちょうど`h`である。一様性により、その条件付き確率は`h`である。

したがって、境界`x`までに既に存在する各prefixは、次の一口で薄い層へ子prefixを置く機会を、条件付き期待で`h`だけ持つ。和を取ると

```text
E[ N(x+h)-N(x) | S_n<=x で見えている全prefix ]
= h N(x) + o(h).
```

`o(h)`は、同じ幅の中でさらに次のprefixも続けて入るような二重以上の寄与であり、幅`h`に対して高次で消える。

期待値を取って

```text
m(x) = E[N(x)]
```

と書けば

```text
m(x+h)-m(x) = h m(x) + o(h).
```

よって

```text
m'(x) = m(x)        (0<x<1).
```

初期条件は、空prefix`S_0=0`が最初から1つあるので

```text
m(0)=1.
```

したがって

```text
m(x)=exp(x).
```

最後に

```text
E[tau] = E[N(1)] = m(1) = e.
```

## この見方の芯

この証明では、`S_n`の密度や単体体積を先に計算しない。境界を動かす空間方向に沿って、prefixの点過程`N(x)`の補償子を見る。

一様性は「既存prefixごとに薄層へ入る確率が層の幅そのもの」として現れ、独立性は各prefixの次の一口を同じ規則で評価できる点に現れる。期待個数の増加率が現在の期待個数に等しいため、答えは`e`になる。
