# Tauberian局所窓解

この解では、境界`1`までの局所窓だけを見れば十分であることを、Tauberian風に読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めると

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

境界`x`までのprefix総量を

```text
T(x)=sum_{n>=0} P(S_n<=x)
```

と置く。半直線上のrenewal級数なら変換側に全体の極や遅延項が現れる。しかし欲しいのは`0<=x<=1`の局所窓だけである。この窓では、一様分布の右端による返り波はまだ境界内の薄層計算を変えない。

したがって局所窓で見えるTauberianな主項は、既存prefixが一様密度`1`で次のprefixを生む寄与だけである。境界を`h`だけ動かすと

```text
T(x+h)-T(x)=hT(x)+o(h).
```

よって

```text
T'(x)=T(x),
T(0)=1.
```

したがって

```text
T(x)=exp(x),
E[tau]=T(1)=e.
```

この証明では、変換公式を全半直線で反転しない。境界`1`までの局所窓では主項だけが働くというTauberianな読みで、薄層再帰を取り出す。`e`は、その局所窓の主項が積み上がった値である。
