# 正規言語prefix解

この解では、停止前prefixを「容量を超えない語」からなる重み付き言語として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空語を含めた受理語数の期待値は

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

境界`x`までに受理される語の総重みを

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

と置く。語は

```text
受理語 = 空語 + 受理語に最後の一文字を足したもの
```

と分解できる。最後の一文字の長さが境界の薄い層`dx`へ入る確率密度は一様性により`dx`なので

```text
F'(x)=F(x),  F(0)=1.
```

したがって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

重み付き正規言語の見方では、受理語の生成関数が一文字追加の自己再帰で閉じる。`e`はその言語の総重みである。
