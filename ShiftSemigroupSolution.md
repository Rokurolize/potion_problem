# シフト半群解

この解では、生存確率を1本ずつ足すのではなく、全生存確率列を無限ベクトルとして動かす。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止時刻は、生きているprefixを数えたものだから

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 生存確率列をベクトルにする

`0 <= x <= 1`で

```text
a_n(x) = P(S_n <= x)
```

とおく。`a_0(x)=1`である。

一様性と独立性により、`n >= 0`について

```text
a_{n+1}(x) = int_0^x a_n(t) dt.
```

よって

```text
a_{n+1}'(x) = a_n(x).
```

ここで列ベクトル

```text
a(x) = (a_0(x), a_1(x), a_2(x), ...)^T
```

を考える。微分は

```text
a'(x) = L a(x)
```

という線形系になる。ただし`L`は成分を1つ下へ送る作用素

```text
(L a)_0 = 0,
(L a)_{n+1} = a_n
```

である。初期値は

```text
a(0) = (1, 0, 0, ...)^T.
```

したがって

```text
a(x) = exp(xL) a(0).
```

## 期待値は全成分和である

求めたい境界版の期待値は、全成分和

```text
A(x) = <1, a(x)> = sum_{n >= 0} a_n(x)
```

である。ここで`<1, ->`は全成分を足す線形汎関数である。

作用素`L`は、全成分和に対して固有値`1`を持つ。実際、和が収束する範囲では

```text
<1, L a> = sum_{n >= 1} a_{n-1} = sum_{n >= 0} a_n = <1, a>.
```

したがって

```text
A'(x)
= <1, a'(x)>
= <1, L a(x)>
= <1, a(x)>
= A(x).
```

初期値は

```text
A(0) = 1.
```

よって

```text
A(x) = exp(x).
```

求める期待値は

```text
E[tau] = A(1) = e.
```

## どこが効いているか

この証明では、`1/n!`の各項を個別に計算してから足さない。境界`x`を動かすと、生存確率列そのものがシフト生成子`L`で流れ、その全成分和が左固有ベクトルとして`e^x`を読む。

一様性は`a_{n+1}'=a_n`を作り、独立性は同じシフトを全段で反復できることを保証する。
