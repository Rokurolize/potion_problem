# sufficient statistic sum解

この解では、停止判定に必要な十分統計量が部分和だけであることを使う。

長さ`n`までの履歴

```text
(X_1,...,X_n)
```

について、停止前かどうかは

```text
S_n=X_1+...+X_n
```

だけで決まる。つまり`S_n`は、この判定に関する十分統計量である。

`0<=s<=1`で`S_n=s`に近い断面は`n-1`次元単体で、その密度は

```text
s^{n-1}/(n-1)!
```

である。したがって

```text
P(tau>n)=P(S_n<=1)
        =int_0^1 s^{n-1}/(n-1)! ds
        =1/n!.
```

尾和公式から

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

履歴全体を追う必要はなく、十分統計量である部分和の断面密度だけで期待値が決まる。
