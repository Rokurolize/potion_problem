# モーメントシフトresolvent解

この解では、生存確率列をモーメント列として見て、添字シフトのresolventを評価する。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`までの総prefix質量を

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

と置く。`0<=x<=1`では

```text
P(S_n<=x)=x^n/n!.
```

係数列

```text
a_n(x)=x^n/n!
```

は

```text
a_{n+1}(x)=int_0^x a_n(t) dt
```

を満たす。つまり一口追加は、モーメント列の添字を一つ進める積分シフト作用素`J`であり、

```text
F(x)=((I-J)^{-1} 1)(x)
```

である。微分するとresolvent方程式は

```text
F'(x)=F(x),  F(0)=1
```

へ落ちる。したがって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

この証明では、生存列をモーメント列として束ね、積分シフトのresolventが指数関数を返すことだけを使っている。
