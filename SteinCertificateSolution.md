# Stein証明書解

この解では、停止までの期待回数を直接展開せず、残り容量に対するStein型の証明関数を作る。

残り容量が`r`である状態から始めた期待停止回数を`m(r)`とする。求めたいのは`m(1)`である。

状態`r`で一口`U`を飲む。`U <= r`なら状態は`r-U`へ移り、`U > r`なら停止する。ここで関数

```text
f(r) = exp(r)
```

を置く。

## Stein作用素

停止前の状態`r`に対して、次の一口の後に残る関数値の期待値は

```text
E[f(r-U) 1_{U <= r}]
  = int_0^r exp(r-u) du
  = exp(r) - 1.
```

したがって

```text
f(r) - E[f(r-U) 1_{U <= r}] = 1.
```

これは、殺される一様ジャンプ核`K`に対して

```text
(I - K) f = 1
```

となるStein方程式である。

## 証明書から期待値を読む

停止するまでの各時刻で、上の恒等式は「一回飲むこと」をちょうど一単位として数える。`N`回で切って和を取ると、望ましい望遠和は

```text
E[min(tau,N)] = f(1) - E[f(R_N) 1_{tau > N}]
```

となる。ただし`R_N`は`N`回後の残り容量である。

右端の残り項は非負で、`N`を大きくすると停止後は消える。単調収束で

```text
E[tau] = f(1).
```

よって

```text
E[tau] = exp(1) = e.
```

この証明では、`P(S_n <= 1)`も`P(tau=n)`も計算しない。指数関数が殺される一様ジャンプ核のStein方程式を解くことを確認し、その証明書を停止まで足し合わせるだけで期待値を読む。
