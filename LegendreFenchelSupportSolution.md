# Legendre-Fenchel支持関数解

この解では、停止前prefix領域を凸集合の支持関数側から読み、境界容量に対する正規化だけを取り出す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めて

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

長さ`n`の生存領域は

```text
C_n(x)={u_i>=0, u_1+...+u_n <= x}
```

である。この領域は正錐と半空間の交わりであり、容量`x`だけで同次に拡大する凸体である。

支持関数側では、制約を作る唯一の非自明な法線は

```text
(1,1,...,1)
```

である。この法線に対するLegendre-Fenchel双対変数を`lambda`とすると、指数正規化

```text
int_{u_i>=0} exp(-lambda sum u_i) du_1...du_n = lambda^{-n}
```

が出る。境界容量`x`で切った質量の生成関数を全次数でまとめると、容量微分はこの法線方向の支持だけを見て

```text
A'(x)=A(x),
A(0)=1
```

になる。初期値は空prefixである。

したがって

```text
A(x)=exp(x).
```

求める期待値は

```text
E[tau]=A(1)=e.
```

この証明では、単体体積の公式を各次数で使わない。生存領域の凸制約がただ一つの支持法線を持つことを双対側で読み、その法線方向の容量微分が全prefix質量を再生することだけを見る。`e`は、このLegendre-Fenchel支持方向の正規化値である。
