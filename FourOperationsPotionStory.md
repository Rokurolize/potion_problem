# 四則演算だけの媚薬問題

この読み物では、媚薬問題を次のルールで解く。

使ってよい計算:

```text
足す
引く
掛ける
割る
```

使わないもの:

```text
微分
積分
行列式
指数関数の公式
測度論
```

最後に出る数は、数学者が`e`と呼ぶ数になる。けれど、この物語では最初から`e`を使わない。まず、四則演算だけでその数を作る。

## 1. 魔女の厨房

雨の夜、塔の厨房には甘い匂いが満ちていた。

薬師リサは、銀の匙で赤い液をかき混ぜていた。見習いのノアは、棚の上の羊皮紙を見て眉を寄せた。

「先生、この媚薬、どれだけ飲むことになるんですか」

「効く量ではなく、止める量が問題だよ」

リサは小さな水晶杯を置いた。

「杯には目盛りがある。限界を`1`とする。媚薬を一口ずつ注ぐ。一口の量は毎回ばらばらで、`0`から`1`までのどこか。合計が`1`を超えたら、その時点で止める」

ノアは指で目盛りをなぞった。

「何口で止まるか、という問題ですね」

「そう。正確には、何口飲むことになるかの平均を知りたい」

厨房の隅で帳簿をつけていた書記のミラが顔を上げた。

「平均なら、何度も試して、口数を足して、試した回数で割ればよいのでは」

「その通り。ただし、それを無限に正確に知りたい。しかも今日は、四則演算だけでやる」

「四則演算だけで」

ノアは不安そうに繰り返した。

リサはうなずいた。

「足す、引く、掛ける、割る。それだけ」

## 2. 口数を直接数えない

ミラは帳簿に表を書いた。

```text
0口後にまだ残っているか
1口後にまだ残っているか
2口後にまだ残っているか
3口後にまだ残っているか
...
```

ノアが首をかしげた。

「止まる口数を知りたいのに、なぜ残っているかを見るんですか」

リサは空の小皿を四枚並べた。

「たとえば、ある人が4口目で限界を超えたとする。その人は、0口後、1口後、2口後、3口後にはまだ残っている。4口後にはもう残っていない」

リサは小皿に豆を置いた。

```text
0口後: 豆1つ
1口後: 豆1つ
2口後: 豆1つ
3口後: 豆1つ
```

「豆は4つ。これは飲んだ口数と同じ」

ミラが続けた。

「2口目で止まった人なら、豆は0口後と1口後の2つ。6口目で止まった人なら、豆は6つ」

ノアの目が少し明るくなった。

「つまり、口数を直接数えなくても、各段でまだ残っている人を数えればよい」

「そう。だから平均口数は、各段の『まだ残っている確率』を足したものになる」

ミラは書いた。

```text
平均口数
= 0口後に残っている確率
+ 1口後に残っている確率
+ 2口後に残っている確率
+ 3口後に残っている確率
+ ...
```

リサは笑った。

「ここまでの計算は、足し算だけだ」

## 3. 最初の二つは簡単

ノアは表の先頭を埋めた。

```text
0口後に残っている確率 = 1
```

「まだ飲んでいないのだから、全員残っています」

次に、1口後。

```text
1口後に残っている確率 = 1
```

ミラが確認した。

「一口の量は`0`から`1`まで。限界を超えるのは合計が`1`より大きくなった時。1口だけでは、普通はまだ超えない」

リサが補足した。

「ちょうど`1`のような境目は、実際の連続したくじでは確率0として無視してよい。ここでは計算を乱さない」

帳簿はこうなった。

```text
平均口数 = 1 + 1 + その先
```

ノアは首をひねった。

「だから答えは少なくとも2より大きい」

「よい」

## 4. 二口後は半分

リサは正方形の板を出した。

「横を1口目の量、縦を2口目の量とする。どちらも`0`から`1`まで」

ノアは板に点を打った。

「たとえば、1口目が`0.2`、2口目が`0.7`なら、この場所」

ミラが斜めの線を引いた。

「合計が`1`以下なら、まだ残っている。合計が`1`を超えたら止まる。境目はこの斜めの線」

正方形は、斜めの線で二つの同じ大きさの三角形に分かれた。

ノアが言った。

「まだ残る側は半分ですね」

リサは帳簿に書いた。

```text
2口後に残っている確率 = 1 / 2
```

「割り算が出た。まだ四則演算だけだ」

ここまでで、

```text
平均口数 = 1 + 1 + 1 / 2 + その先
```

## 5. 三口後の部屋

三口目になると、板ではなく箱が必要になった。

横が1口目、奥行きが2口目、高さが3口目。

ノアは木箱を見てため息をついた。

「三次元は面倒です」

「だから、少し見方を変える」

リサは三つの数を並べた。

```text
u1 = 1口目の量
u2 = 2口目の量
u3 = 3口目の量
```

そして、その途中合計を書いた。

```text
s1 = u1
s2 = u1 + u2
s3 = u1 + u2 + u3
```

「3口後も残っているとは、`s3`が`1`以下ということだ」

ミラがうなずいた。

「しかも、途中合計は順に大きくなります。だから」

```text
0 < s1 < s2 < s3 <= 1
```

ノアは考え込んだ。

「三つの点`s1,s2,s3`が、`0`から`1`の間で小さい順に並んでいる」

「そう」

ノアはまだ納得しきれない顔をした。

「でも先生、もとのくじは`u1,u2,u3`です。途中合計`s1,s2,s3`に変えても、数え方は変わらないのですか」

リサは別の羊皮紙に逆向きの式を書いた。

```text
u1 = s1
u2 = s2 - s1
u3 = s3 - s2
```

「戻せる。しかも使うのは引き算だけだ。細かい格子で考えると、もとの小さな箱が斜めの小さな箱にずれるだけで、箱の数は変わらない」

ミラが言い添えた。

「だから、`u`で数えても`s`で数えても、同じ分だけの場所を数えていることになります」

リサは三枚のカードを出した。

```text
A, B, C
```

「三つの点に名前をつけると、並び方は何通りある」

ノアは指で数えた。

```text
ABC
ACB
BAC
BCA
CAB
CBA
```

「6通り」

「そのうち、小さい順に並んでいるのは1通り」

ミラが帳簿に書いた。

```text
3口後に残っている確率 = 1 / 6
```

ノアはうなずいた。

「6で割るだけ」

「掛け算で書けば、6は`3 * 2 * 1`だ」

```text
3口後に残っている確率 = 1 / (3 * 2 * 1)
```

## 6. 四口、五口、その先

ノアは自分で次を書いた。

「四つの途中合計なら、並び方は」

```text
4 * 3 * 2 * 1 = 24
```

「だから、四口後に残っている確率は」

```text
1 / 24
```

ミラが続けた。

「五口後なら」

```text
5 * 4 * 3 * 2 * 1 = 120
```

「だから」

```text
1 / 120
```

リサは満足そうに杯を揺らした。

「一般に、`n`口後にまだ残っている確率は」

```text
1 / (n * (n - 1) * ... * 2 * 1)
```

ノアは言った。

「使っているのは、掛け算と割り算だけですね」

「そして平均口数は、それらを足す」

## 7. 帳簿の最後の行

ミラはここまでの結果を一枚にまとめた。

```text
0口後に残っている確率 = 1
1口後に残っている確率 = 1
2口後に残っている確率 = 1 / 2
3口後に残っている確率 = 1 / (3 * 2 * 1)
4口後に残っている確率 = 1 / (4 * 3 * 2 * 1)
5口後に残っている確率 = 1 / (5 * 4 * 3 * 2 * 1)
...
```

したがって、

```text
平均口数
= 1
+ 1
+ 1 / 2
+ 1 / (3 * 2 * 1)
+ 1 / (4 * 3 * 2 * 1)
+ 1 / (5 * 4 * 3 * 2 * 1)
+ ...
```

ノアは両手で顔を覆った。

「まだ終わっていません。永遠に足すのですか」

リサは笑った。

「では、少しだけ足してみよう」

ミラが四則演算だけで計算した。

```text
1 + 1 = 2
2 + 1 / 2 = 2.5
2.5 + 1 / 6 = 2.666...
2.666... + 1 / 24 = 2.70833...
2.70833... + 1 / 120 = 2.71666...
```

ノアが表を見つめた。

「2より大きくて、3より小さい。だんだん`2.7`あたりに落ち着いてきます」

ミラはさらに足した。

```text
1 / (6 * 5 * 4 * 3 * 2 * 1) = 1 / 720
2.71666... + 1 / 720 = 2.71805...
```

「次は」

```text
1 / (7 * 6 * 5 * 4 * 3 * 2 * 1) = 1 / 5040
2.71805... + 1 / 5040 = 2.71825...
```

ノアは小さく声を上げた。

「ほとんど動かなくなった」

## 8. 媚薬の答え

リサは最後の行を指した。

```text
平均口数
= 1
+ 1
+ 1 / 2
+ 1 / (3 * 2 * 1)
+ 1 / (4 * 3 * 2 * 1)
+ 1 / (5 * 4 * 3 * 2 * 1)
+ ...
```

「この数に、昔の数学者は名前をつけた」

ノアが先に言った。

「`e`」

「そう」

ミラは帳簿の最後に書いた。

```text
媚薬が限界1を超えるまでに飲む平均口数 = e
```

その下に、ノアが自分の字で書き足した。

```text
ただし今日のeは、指数関数ではなく、
四則演算だけで作った数。
```

リサは水晶杯を棚に戻した。

「媚薬は人の心を乱す。だが、用量の帳簿は乱してはいけない」

ミラが笑った。

「恋の薬も、最後は割り算に従うのですね」

ノアはまだ帳簿を見ていた。

```text
1
+ 1
+ 1 / 2
+ 1 / 6
+ 1 / 24
+ 1 / 120
+ ...
```

それは呪文のように見えた。

けれど、呪文ではなかった。

足して、掛けて、割っただけの、厨房の答えだった。

## 四則演算だけの骨格

物語の計算だけを抜き出すと、次の通り。

1. `k`口で止まる人は、`0`口後から`k-1`口後までの`k`段で「まだ残っている」と数えられる。
2. だから平均口数は、各段の「まだ残っている確率」を全部足したもの。
3. `n`口後に残るには、途中合計が`0`から`1`の間で小さい順に並ぶ必要がある。
4. `n`個の点の並び方は`n * (n - 1) * ... * 2 * 1`通り。
5. 小さい順はそのうち1通り。
6. したがって`n`口後に残る確率は`1 / (n * (n - 1) * ... * 2 * 1)`。
7. よって平均口数は

```text
1
+ 1
+ 1 / 2
+ 1 / (3 * 2 * 1)
+ 1 / (4 * 3 * 2 * 1)
+ 1 / (5 * 4 * 3 * 2 * 1)
+ ...
```

この数を`e`と呼ぶ。
