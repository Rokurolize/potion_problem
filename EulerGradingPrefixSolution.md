# Euler次数prefix解

この解では、停止前prefixを深さで次数付けし、境界容量を動かすEuler型の次数作用素で読む。

容量`r`までで生き残る長さ`n`のprefix質量を

```text
A_n(r) = P(X_1 + ... + X_n <= r)
```

と置く。`A_0(r)=1`である。求めたい期待値は

```text
sum_{n >= 0} A_n(1)
```

である。

## 容量微分は次数を一つ下げる

`n >= 1`について、最後の一口を境界近くで剥がすと

```text
A_n(r) = int_0^r A_{n-1}(r-u) du.
```

したがって

```text
d A_n(r) / dr = A_{n-1}(r).
```

ここで形式級数

```text
F(r,y) = sum_{n >= 0} A_n(r) y^n
```

を作る。上の式は、容量微分が`y`を一つ掛ける操作になることを意味する。

```text
partial_r F(r,y)
  = sum_{n >= 1} A_{n-1}(r) y^n
  = y F(r,y).
```

初期条件は

```text
F(0,y)=1
```

である。よって

```text
F(r,y)=exp(yr).
```

全次数を数えるには`y=1`とすればよいので

```text
E[tau] = F(1,1) = e.
```

この証明では、各`A_n(1)`を単体体積として直接求めない。容量微分がprefixの次数を一つ下げるという一点だけを使い、その次数下げ作用を形式変数`y`で対角化する。`e`は全次数を同じ重みで評価したときのEuler型生成関数の値である。
