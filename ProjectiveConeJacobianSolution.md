# 射影錐ヤコビアン解

`S_n = X_1 + ... + X_n` とする。`S_n <= 1` の範囲では一様密度は1なので、

```text
P(S_n <= 1) = vol{x_i >= 0, x_1 + ... + x_n <= 1}
```

である。

正錐の点 `x = (x_1, ..., x_n)` を、総量と比率に分ける。

```text
r = x_1 + ... + x_n
p_i = x_i / r
```

ただし `r > 0` とする。すると `p` は確率単体

```text
p_i >= 0,  p_1 + ... + p_n = 1
```

上の点であり、

```text
x_i = r p_i
```

と戻せる。原点は体積0なので無視してよい。

独立変数を `r, p_1, ..., p_{n-1}` とし、

```text
p_n = 1 - p_1 - ... - p_{n-1}
```

と置く。この変換のヤコビアンは

```text
r^{n-1}
```

である。したがって

```text
vol{x_i >= 0, sum x_i <= 1}
  = int_0^1 r^{n-1} dr
    * vol{p_1, ..., p_{n-1} >= 0, sum_{i<n} p_i <= 1}
```

となる。右の `p` 側は標準的な `n - 1` 次元単体で、同じ射影を次元を落として繰り返すと体積は

```text
1/(n-1)!
```

である。よって

```text
P(S_n <= 1)
  = (1/n) * 1/(n-1)!
  = 1/n!
```

を得る。`n = 0` は空和なので寄与は1である。

到着数は

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので、単調収束により

```text
E[tau]
  = sum_{n >= 0} P(S_n <= 1)
  = sum_{n >= 0} 1/n!
  = e
```

となる。待ち時間の列を、総量 `r` と比率 `p` に分解すると、`1/n!` は「総量方向の `1/n`」と「比率方向の `1/(n-1)!`」に割れて現れる。
