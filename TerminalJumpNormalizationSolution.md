# 終端ジャンプ正規化解

この解では、停止前のprefixだけでなく、その直後に境界を越える最後の一口まで合わせて数える。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixの個数は

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので、

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 終端一口を加える

`S_n <= 1`で止まっていないprefixを固定する。このprefixの直後に停止するためには、次の一口が

```text
U_{n+1} > 1 - S_n
```

であればよい。条件付き確率は

```text
S_n
```

ではなく、正しくは

```text
1 - (1 - S_n) = S_n
```

である。ただし、これをそのまま足すと停止時刻ではなく「最後の一口で止まる重み」になる。

そこで、終端一口の長さそのものを積分変数として残す。`S_n=s`のとき、終端一口は区間

```text
1 - s < u <= 1
```

を走る。この長さは`s`である。

## 失われた量は末尾ギャップ

prefix側の測度を

```text
dU_1 ... dU_n,   S_n <= 1
```

と見る。ここに終端一口`u`を加えると、領域は

```text
U_i >= 0,
S_n <= 1,
1 - S_n < u <= 1
```

である。全ての`n`についてこの体積を足すと、停止がどの段で起きたかを分けた全確率なので、総量は`1`である。

一方、prefixだけを数える測度の総量を

```text
M = sum_{n >= 0} P(S_n <= 1)
```

と書く。終端一口を加える操作は、各prefixに対して末尾ギャップ

```text
1 - S_n
```

を除いた長さ`s`を掛けるので、直接には`M`を保たない。

このままだと`M`は出ない。そこで、終端一口ではなく、境界を越えるために必要な余剰

```text
v = u - (1 - S_n)
```

を使う。`0 < v <= S_n`であり、変換のJacobianは`1`である。すると全停止測度は、prefix測度に「利用可能な余剰長」を載せたものになる。

## 余剰長の累積を微分する

閾値を`x`に動かし、

```text
M(x) = sum_{n >= 0} P(S_n <= x)
```

を置く。終端ジャンプの分解を`x`で見ると、`x`から`x+dx`へ境界を動かしたときに新しくprefixとして復活する量は、すでに`x`以下にいたprefixから密度`1`で流れ込む量である。したがって

```text
M'(x) = M(x),
M(0) = 1.
```

よって

```text
M(x) = exp(x)
```

であり、

```text
E[tau] = M(1) = e.
```

この証明の要点は、停止後の一口まで含めると全確率は`1`に正規化され、境界を少し動かしたときの復活量がその時点までのprefix総量そのものになる、という終端側からの保存則である。
