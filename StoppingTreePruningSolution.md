# stopping tree pruning解

この解では、無限分岐木を停止境界で刈り込んだときの期待節点数として解く。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。残り容量が`x`の節点からは、次の枝長`u`を`0<=u<=x`の範囲で選んだ枝だけが生き残る。刈り込み後の部分木の期待節点質量を`T(x)`と書くと、根を一つ数えた上で、残り容量`x-u`の部分木を全部足すから

```text
T(x)=1+int_0^x T(x-u) du.
```

変数を替えて

```text
T(x)=1+int_0^x T(t) dt.
```

したがって

```text
T'(x)=T(x),  T(0)=1,
```

なので

```text
T(x)=e^x.
```

最初の容量は`1`であり、刈り込み木の節点数は停止前prefix数、つまり`tau`である。よって

```text
E[tau]=T(1)=e.
```

この証明では、停止した枝を数えない。境界で切られてなお残る木の節点質量だけを見る。
