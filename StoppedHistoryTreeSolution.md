# 停止前履歴木の総質量解

この解では、停止時刻を確率変数として先に解析せず、境界を越える前にあり得る全ての有限履歴を一本の木として数える。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前のprefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 履歴の木を作る

残り容量が`r`ある節点を考える。この節点自身は、そこまでの履歴がまだ停止前であることを表すので、質量`1`を持つ。

次の一口を`u`とすると、停止前に次の節点へ進めるのは

```text
0 <= u <= r
```

のときだけである。その先の残り容量は`r-u`になる。

ここで、残り容量`r`から下に伸びる停止前履歴木の総Lebesgue質量を

```text
T(r)
```

と書く。節点自身の質量と、その子孫全体を足すと

```text
T(r) = 1 + int_0^r T(r-u) du.
```

変数を替えれば

```text
T(r) = 1 + int_0^r T(v) dv.        (1)
```

この式は期待値の再帰ではなく、停止前履歴全体を木として展開したときの総質量分解である。各節点の子の集合は長さ`r`の連続体で、子ごとに同じ規則の部分木がぶら下がる。

## 木の総質量は指数になる

(1)を微分すると

```text
T'(r) = T(r),
T(0) = 1.
```

したがって

```text
T(r) = exp(r).
```

初期状態では残り容量が`1`なので、停止前履歴木の総質量は

```text
T(1) = e.
```

## 期待値との対応

深さ`n`の節点全体は

```text
u_i >= 0,
u_1 + ... + u_n <= 1
```

を満たす履歴の集合である。この集合のLebesgue体積は、まさに

```text
P(S_n <= 1)
```

である。したがって、全深さの節点質量を足した`T(1)`は

```text
sum_{n >= 0} P(S_n <= 1)
```

に等しい。

よって

```text
E[tau] = T(1) = e.
```

この証明では、`n`段ごとの体積を個別に評価しない。停止前に存在する履歴を一本の連続分岐木として見て、節点ひとつとその下の相似な部分木への分解だけで総質量を決める。
