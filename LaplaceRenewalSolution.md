# Laplace再生測度解

この解では、停止時刻を生存prefixの個数として数え、その期待値を再生測度のLaplace変換から読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止時刻は、累積和がまだ`1`以下である時点の数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 再生測度として見る

`U_i`の分布を`mu`とする。`S_n`の分布は畳み込み冪`mu^{*n}`であり、`S_0=0`に対応する`mu^{*0}`は`0`の単位質量である。

再生測度を

```text
R = sum_{n >= 0} mu^{*n}
```

とおくと、求める期待値はこの測度の区間質量

```text
E[tau] = R([0,1])
```

である。

ここで`mu`は`[0,1]`上一様分布なので、`0 <= x <= 1`では`mu`の密度は`1`である。したがって、`[0,1]`の内側での再生測度の密度`r(x)`は

```text
r(x) = sum_{n >= 1} f_n(x)
```

であり、`f_n`は`n`個の一様乱数の和の密度である。

## Laplace変換で閉じる

一様分布のLaplace変換は

```text
hat_mu(s) = int_0^1 e^{-sx} dx = (1 - e^{-s}) / s.
```

再生測度のLaplace変換は幾何級数として

```text
hat_R(s) = sum_{n >= 0} hat_mu(s)^n = 1 / (1 - hat_mu(s)).
```

ただし、求めたい`R([0,1])`には`x <= 1`の部分だけが効く。この範囲では、各畳み込み密度は上端`1`の切れ目をまだ見ていない。つまり`0 <= x <= 1`では、一様分布の密度`1_{0<=x<=1}`は単に定数`1`として働く。

そのため、`[0,1]`内の正の部分の再生密度は、定数核`1`の畳み込み級数

```text
q(x) = 1 + x + x^2/2! + x^3/3! + ...
```

に等しい。これはLaplace側では

```text
hat_q(s) = 1/s + 1/s^2 + 1/s^3 + ... = 1 / (s - 1)
```

に対応し、逆変換で

```text
q(x) = e^x
```

となる。

最後に、`R([0,1])`は`0`の原子`1`と正の部分の密度`q`を合わせたものとして

```text
R([0,1])
= 1 + int_0^1 e^x dx
= e.
```

よって

```text
E[tau] = e.
```

## 境界の扱い

Laplace変換を全実軸の公式として振り回すのではなく、`x <= 1`だけを切り出す。そこでは一様分布の上端制約がまだ畳み込みの積分領域を削らないため、定数核の再生級数として閉じる。停止時刻の期待値は、その再生測度が`[0,1]`に置く質量である。
