# coarea断面流量解

この解では、停止前prefixを各`n`ごとに積分せず、容量`r`で切った全次元の断面量としてまとめる。

独立な一様乱数を`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

と置く。`S_0=0`を含めると、

```text
tau = sum_{n>=0} 1_{S_n <= 1}
```

である。したがって

```text
E[tau] = sum_{n>=0} P(S_n <= 1).        (1)
```

## 全次元の容量関数

境界を`1`に固定せず、容量`r`まで生きているprefix全体の量を

```text
F(r) = sum_{n>=0} volume { u_1,...,u_n >= 0 : u_1+...+u_n <= r }
```

と置く。空prefixの体積は`1`とする。

元の期待値は

```text
E[tau] = F(1)
```

である。

## 薄い断面を見る

容量を`r`から`r+dr`へ増やしたときに増える量は、

```text
r < u_1+...+u_n <= r+dr
```

に入る全ての語の体積である。

非空語を最後の文字`u`と前の語`w`に分ける。最後の文字を増やして総和が薄い層に入るためには、前の語の総和が`r`以下にあればよい。`u`の密度はLebesgue測度そのものなので、薄い幅`dr`に入る自由度は各`w`につき`dr`だけである。

したがって、coarea的に薄い断面の流量を取ると

```text
F'(r) = F(r).        (2)
```

直感だけでなく、差分で書けば

```text
F(r+h) - F(r)
= int_0^h F(r+h-v) dv
```

となる。右辺は、最後の文字が境界から`v`だけ内側に着地し、前の語が残り容量`r+h-v`以下である寄与を足したものだ。`h`で割って`h`を`0`に送ると(2)になる。

## 初期断面

容量`0`では空prefixだけが残るので

```text
F(0) = 1.
```

(2)と合わせて

```text
F(r) = exp(r).
```

したがって

```text
E[tau] = F(1) = exp(1) = e.
```

## この見方の芯

長さ`n`の単体を個別に評価しない。全ての次元をまとめた容量関数`F`を置き、境界を少し動かしたときに新しく入る薄い断面の流量だけを見る。

一様増分の密度が`1`であるため、断面に流れ込む量は、それ以前に蓄積していた全prefix量`F(r)`そのものになる。この自己流量が指数関数を強制する。
