# ordinary母関数対角解

この解では、長さと境界容量の二変数ordinary母関数を作り、その対角評価で期待値を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めて

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

`a_n(x)=P(S_n<=x)`と置き、

```text
G(x,z)=sum_{n>=0} a_n(x) z^n
```

を考える。`0<=x<=1`では

```text
a_n'(x)=a_{n-1}(x)
```

なので

```text
partial_x G(x,z)=zG(x,z),
G(0,z)=1.
```

したがって

```text
G(x,z)=exp(zx).
```

求める値は長さ重みを外した`z=1`、境界`x=1`の対角評価で

```text
E[tau]=G(1,1)=e.
```

この証明では、各係数を先に計算しない。二変数ordinary母関数を作り、境界微分が長さ変数の掛け算になることだけを見る。`e`は、その対角評価である。
