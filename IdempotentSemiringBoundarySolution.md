# 冪等半環境界解

この解では、境界容量の増加を冪等半環の重み付き経路和としていったん抽象化し、最後に通常の和へ戻す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含む標準恒等式は

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

境界`x`までの総重み

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

を見る。冪等半環の言葉では、一口追加は「現在のprefixから境界方向へ伸びる経路」の重みを足す操作である。`0<=x<=1`では、最後の一口が薄い層`dx`に入る重みは一様密度により`dx`であり、全経路重みは

```text
F(x+dx)=F(x)+F(x)dx+o(dx).
```

通常の和へ戻すと

```text
F'(x)=F(x),  F(0)=1.
```

したがって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

ここで冪等半環は、境界に向かう経路の組み合わせを抽象化するだけで、実際の値は通常の加法へ戻して評価している。
