# 印付き円周ギャップ解

この解では、停止前prefixを線分上の順序単体として直接測らない。飲んだ量と最後に残った余白をまとめて、長さ`1`の円周を切ったギャップ列として読む。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

`S_0=0`を含めれば

```text
tau = sum_{n>=0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n>=0} P(S_n <= 1).
```

長さ`n`の停止前prefixでは

```text
u_1+...+u_n <= 1
```

であり、余白

```text
g_0 = 1 - (u_1+...+u_n)
```

を足すと

```text
g_0 + u_1 + ... + u_n = 1
```

になる。

## 線分ではなく円周で見る

長さ`1`の円周を考え、そこに`n+1`個のギャップを時計回りに並べる。

```text
g_0, u_1, u_2, ..., u_n
```

ここで`g_0`を「印付きギャップ」と呼ぶ。印付きギャップの直後から読み始めると、残りのギャップ列が飲んだ量

```text
u_1,...,u_n
```

になる。したがって、停止前prefixの領域は「円周の全長を`n+1`個の非負ギャップに分け、そのうち一つを印として選んだもの」と同じである。

重要なのは、線分の端点`0`を先に固定していない点である。円周上では、印付きギャップを選んで初めて、どこからprefixを読むかが決まる。

## 印を外して正規化する

円周上に独立な一様点を`n`個置く。ほとんど確実に相異なるので、円周は`n`個の点によって`n`個の点間ギャップに分かれる。さらに、基準点を一つ独立に一様に置き、それが入ったギャップを印付きギャップとする。

基準点と`n`個の点を合わせると、円周上の`n+1`個の点である。これらを基準点から時計回りに読むと、`n+1`個のギャップ

```text
g_0,u_1,...,u_n
```

が得られる。円周の回転対称性により、このギャップ列の全確率質量は`1`で正規化されている。

一方、同じギャップ列を線分上の座標で書くと、密度は一定である。なぜなら元の`n`個の点は独立一様で、順序を読む操作はヤコビアン`1`の区分に分かれるだけだからである。密度の正規化だけを見れば、

```text
1
= n! * volume { g_0,u_i>=0, g_0+u_1+...+u_n=1 }
```

と読める。ここで`n!`は、印付きギャップ以外の`n`個の切断点が円周上でどの順に現れるかのラベル順の数である。

したがって、印付きギャップを含むギャップ単体の質量は`1/n!`である。`g_0`を消して

```text
g_0 = 1 - (u_1+...+u_n)
```

と戻せば、これはちょうど

```text
P(S_n <= 1) = 1/n!
```

を意味する。

## 期待値へ戻す

よって

```text
E[tau]
= sum_{n>=0} P(S_n <= 1)
= sum_{n>=0} 1/n!
= e.
```

## この見方の芯

この証明では、停止時刻の分布を求めない。生存prefixを、飲んだ量だけでなく「まだ飲めた余白」も含む円周ギャップ列として見る。

一様性は、円周上の切断点が回転対称な一定密度を持つ点に使われる。独立性は、切断点のラベル順が等しく測られる点に使われる。答えの`e`は、線分の単体体積を直接積分した値ではなく、円周をどのギャップで切ってprefixとして読むかという正規化から現れる。
