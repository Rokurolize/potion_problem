# Bellman価値反復解

この解では、停止時刻を分布として求めず、残り容量を状態にした価値反復で期待回数を読む。

残り容量が`r`の状態から始めたとき、これから飲む期待回数を

```text
V(r)
```

とする。求めたい値は`V(1)`である。

## 有限段の価値反復

まず、あと高々`N`回までしか数えない有限段価値を`V_N(r)`と置く。`V_0(r)=0`である。

`N+1`段では、まず一口飲むので`1`を得る。その後、`U <= r`なら残り容量`r-U`であと`N`段の価値が残り、`U > r`なら終わる。したがって

```text
V_{N+1}(r) = 1 + int_0^r V_N(r-u) du.
```

変数を入れ替えると

```text
V_{N+1}(r) = 1 + int_0^r V_N(x) dx.
```

これは停止問題のBellman反復である。政策を選ぶ問題ではないが、状態価値を一段先の状態価値で閉じている。

## 反復の極限

`V_N(r)`は単調に増え、極限は期待停止回数`V(r)`である。単調収束により

```text
V(r) = 1 + int_0^r V(x) dx.
```

微分して

```text
V'(r) = V(r),   V(0)=1.
```

ここで`V(0)=1`は、残り容量が`0`なら次の一口で必ず停止し、その一口だけを数えるからである。

よって

```text
V(r)=exp(r).
```

したがって

```text
E[tau]=V(1)=e.
```

この証明では、`P(tau=n)`や`P(S_n<=1)`を個別に出さない。有限段のBellman価値反復を先に作り、その単調極限が同じ積分方程式を満たすことだけで答えを読む。
