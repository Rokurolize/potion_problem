# 二項変換生存列解

この解では、生存prefix列に二項変換をかけ、境界`1`で変換が指数型の全和へ戻ることを見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めて

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

`a_n(x)=P(S_n<=x)`と置く。`0<=x<=1`では

```text
a_n'(x)=a_{n-1}(x),
a_0(x)=1.
```

二項変換の母関数

```text
B(x,z)=sum_{n>=0} z^n a_n(x)
```

を考えると、上のシフト関係から

```text
partial_x B(x,z)=zB(x,z),
B(0,z)=1.
```

したがって

```text
B(x,z)=exp(zx).
```

変換を外して全生存列を足すには`z=1`とすればよいので

```text
E[tau]=B(1,1)=e.
```

この証明では、個々の`a_n(1)`を求めない。生存列を二項変換の母関数に入れ、境界微分が添字シフトになることだけを見る。`e`は、その変換を`z=1`で戻した値である。
