# 一人ゲーム価値解

この解では、媚薬問題を「一人ゲーム」として見て、生存中に得る単位報酬の価値を計算する。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。残り容量`r`で、今いる生存状態を一回数え、その後も生存すれば続く価値を`G(r)`とする。ゲームには選択肢がなく、次の遷移だけが一様乱数で決まるので

```text
G(r)=1+int_0^r G(r-u) du.
```

これは一人ゲームの価値反復方程式である。変数変換により

```text
G(r)=1+int_0^r G(v) dv,
```

したがって

```text
G'(r)=G(r),  G(0)=1.
```

よって

```text
G(r)=e^r.
```

初期状態`r=1`での価値が期待停止時刻なので

```text
E[tau]=G(1)=e.
```

この証明では、分布を先に求めない。生きている各prefixを単位報酬として数える一人ゲームの価値だけを解く。
