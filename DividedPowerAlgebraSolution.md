# divided power代数解

この解では、`1/n!`を単体積分の答えとして後から出すのではなく、停止前prefixをdivided power基底に乗せてから全次数を足す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 容量を変数にする

境界を`x`までにしたときの`n`口生存質量を

```text
A_n(x) = P(S_n <= x)
```

と置く。`0 <= x <= 1`では、最後の一口の上限`1`はまだ切断として効かないので

```text
A_0(x) = 1,
A_{n+1}(x) = int_0^x A_n(t) dt.
```

ここで通常の冪基底ではなく

```text
x^{[n]} = x^n/n!
```

というdivided power基底を使う。この基底では積分作用素は

```text
int_0^x t^{[n]} dt = x^{[n+1]}
```

と、係数を変えずに次数を一つ上げるだけである。

したがって

```text
A_n(x) = x^{[n]}.
```

求める境界`x=1`では

```text
P(S_n <= 1) = A_n(1) = 1^{[n]} = 1/n!.
```

## 全次数の和

全prefix総量はdivided power指数

```text
sum_{n >= 0} x^{[n]}
```

であり、これは通常の指数関数`exp(x)`である。よって

```text
E[tau]
  = sum_{n >= 0} A_n(1)
  = sum_{n >= 0} 1^{[n]}
  = exp(1)
  = e.
```

この証明では、各次元の体積をその都度計算しない。積分で一段進む操作を、divided power基底では「係数を変えないシフト」として扱う。`e`は、生存prefix列がそのままdivided power指数になることから現れる。
