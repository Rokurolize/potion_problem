# 終点renewal強度解

独立一様乱数を `X_1, X_2, ...` とし、

```text
S_n = X_1 + ... + X_n
```

とおく。停止前prefixの個数として

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので、

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

空prefix `S_0=0` は別に数え、正の長さのprefixが区間内の点 `x` に到着する密度を考える。長さ1のprefixの終点密度は

```text
q_1(x) = 1      (0 <= x <= 1)
```

である。長さ `n+1` のprefixが `x` に到着するには、直前のprefixがどこか `0 <= y <= x` にあり、最後の一口が `x-y` であればよい。一様密度は `1` なので、

```text
q_{n+1}(x) = int_0^x q_n(y) dy
```

となる。

全ての正の長さを重ねた終点強度を

```text
q(x) = sum_{n >= 1} q_n(x)
```

と置くと、

```text
q(x)
= 1 + int_0^x q(y) dy
```

である。両辺を微分して

```text
q'(x) = q(x),   q(0) = 1
```

を得る。したがって

```text
q(x) = exp(x)
```

である。

停止前prefixの期待個数は、空prefixの `1` と、正の長さのprefix終点強度の区間内総量なので、

```text
E[tau]
= 1 + int_0^1 q(x) dx
= 1 + int_0^1 exp(x) dx
= e
```

となる。

この証明では、長さごとの単体体積を求めない。各prefixがどの終点へ流れ込むかの強度を重ねると、総強度が自分自身の原始関数で再生する。`e` は、空prefixと区間内に流れ込む全終点強度の総量として現れる。
