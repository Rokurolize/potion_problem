# 最大エントロピー正規化解

この解では、停止前prefix全体を有限配置の非正規化測度として見て、それを確率測度へ直す正規化定数を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## prefix測度

`S_n <= 1`のprefixを累積時刻

```text
0 <= t_1 <= ... <= t_n <= 1
```

へ移す。増分から累積時刻への変換はJacobian`1`である。したがって、長さ`n`のprefix測度は、区間`[0,1]`上の順序付き`n`点配置のLebesgue測度である。

全長さをまとめた非正規化測度を

```text
mu = sum_{n >= 0} dt_1 ... dt_n on {0 <= t_1 <= ... <= t_n <= 1}
```

と書く。この全質量が求める期待値である。

一方、区間`[0,1]`上の有限点配置で、点数以外に情報を入れない確率測度を考える。平均点数を1に固定し、点の位置をLebesgue測度に対して一様に保つなら、最大エントロピーの点数分布はPoisson(1)である。つまり順序付き配置側の確率測度は、各階層で

```text
exp(-1) dt_1 ... dt_n
```

という共通の空白因子を持つ。

この確率測度の全質量は`1`である。したがって、空白因子を外したprefix測度の全質量は

```text
exp(1)
```

である。

よって

```text
E[tau] = mu(total) = e.
```

この証明では、各`P(S_n <= 1)`を個別に積分しない。停止前prefix測度を、平均点数1で位置に余計な情報を入れない最大エントロピー配置測度から共通因子`exp(-1)`を外したものとして読む。`e`はその正規化定数である。
