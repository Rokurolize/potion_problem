# Fredholm空白展開解

この解では、停止前prefix全体を、区間上の空白確率のFredholm型展開から逆向きに読む。

長さ`n`の生存prefixは、累積時刻

```text
0 <= t_1 <= ... <= t_n <= 1
```

に移すと、区間`[0,1]`内の順序付き`n`点配置である。増分から累積時刻への変換はJacobian`1`なので、この順序付き領域の体積が

```text
P(S_n <= 1)
```

である。

## 空白確率の展開

単位強度のPoisson点過程を`[0,1]`上に置く。区間に点が無い確率は

```text
P(N([0,1])=0) = exp(-1).
```

これはLaplace汎関数、あるいは空白確率のFredholm型展開

```text
exp(-int_0^1 1 dx)
```

として読める。ここで指数の中に入っている`int_0^1 1 dx`は、1点を置く強度測度の全質量である。

空白確率の逆数は

```text
exp(1)
```

であり、その指数展開は

```text
exp(1) = sum_{n >= 0} 1/n!.
```

各`1/n!`は、`n`個の非ラベル点配置を時刻順に並べた順序付きprefix領域の体積である。Poisson確率側では全ての階層に共通して空白因子`exp(-1)`が掛かっているので、それを外すと停止前prefix測度の総質量になる。

したがって

```text
E[tau]
  = sum_{n >= 0} P(S_n <= 1)
  = exp(1)
  = e.
```

この証明では、各単体体積を個別に積分しない。Poisson空白確率をFredholm型の指数汎関数として見て、その空白正規化を外した全配置展開が、そのまま停止前prefixの総量になることだけを見る。
