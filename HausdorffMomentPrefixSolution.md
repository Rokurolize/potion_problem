# Hausdorffモーメントprefix解

この解では、生存prefix列を`[0,1]`上のHausdorffモーメント列として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めて

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

境界`x`までのprefix総量を

```text
M(x)=sum_{n>=0} P(S_n<=x)
```

と置く。長さ方向をモーメント次数と見れば、prefixを一つ伸ばす操作は、直前のprefixに対して許される一様幅を掛けるHausdorff型のモーメント更新である。

境界`x`から`x+h`へ移ると、直前のprefix和が`y<=x`の各点は、最後の一口を

```text
x-y < u <= x+h-y
```

に選ぶことで新しい薄層へ入る。この幅は`h+o(h)`であり、一様密度は`1`である。したがって全モーメント列を畳んだ総量は

```text
M(x+h)-M(x)=hM(x)+o(h).
```

よって

```text
M'(x)=M(x),
M(0)=1.
```

したがって

```text
M(x)=exp(x),
E[tau]=M(1)=e.
```

この証明では、`1/n!`を先に計算しない。生存prefix列をHausdorffモーメント列としてまとめ、境界薄層でモーメント更新が総量をそのまま再生することだけを見る。`e`は、そのモーメント列の全和である。
