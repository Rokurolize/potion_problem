# 生存ハザード連鎖解

この解では、各段の生存確率を直接求めず、次に死なない条件付き確率だけを連鎖させる。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。`p_n=P(S_n<=1)`と書けば

```text
E[tau]=sum_{n>=0} p_n.
```

`S_{n-1}<=1`のもとで、残り容量を

```text
R_{n-1}=1-S_{n-1}
```

とする。次の一口は独立一様なので

```text
P(S_n<=1 | U_1,...,U_{n-1}) = R_{n-1}.
```

したがって条件付き生存率は

```text
q_n=P(S_n<=1 | S_{n-1}<=1)
    =E[R_{n-1} | S_{n-1}<=1].
```

生存している`n-1`個の一口に余白を足すと、`n`個の非負成分の和が`1`である対称な単体になる。対称性から余白の平均は

```text
1/n.
```

ゆえに

```text
q_n=1/n.
```

生存確率はハザード連鎖として

```text
p_n=q_1 q_2 ... q_n
    =1/(1*2*...*n)
    =1/n!.
```

よって

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

この証明の主役は`p_n`の体積ではなく、死なずに一段進む条件付き確率である。余白の平均が毎回`1/n`になるため、生存ハザードの積がそのまま階乗を作る。
