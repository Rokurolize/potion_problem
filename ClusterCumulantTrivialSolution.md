# クラスターキュムラント自明化解

この解では、停止前prefixをクラスター展開で数え、連結キュムラントが一次項だけになることを見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含む停止前prefixの活動度つき分配関数を

```text
Z(x,z)=sum_{n>=0} z^n P(S_n <= x)
```

と置く。境界`x`で許される次の一口は、一様密度により幅`dx`だけ増える。相互作用は停止境界だけで、内部のprefix同士には結合がないので

```text
partial_x Z(x,z)=z Z(x,z),
Z(0,z)=1.
```

クラスターキュムラント、つまり

```text
K(x,z)=log Z(x,z)
```

を取ると、全ての高次連結寄与は消えて

```text
partial_x K(x,z)=z,
K(0,z)=0.
```

したがって

```text
K(x,z)=zx,
Z(x,z)=exp(zx).
```

欲しい期待値は`x=1,z=1`での停止前prefix総量だから

```text
E[tau]=Z(1,1)=e.
```

この証明の要点は、クラスター展開を始めても非自明な連結クラスターが存在しないことにある。停止境界の薄層だけが一次キュムラントとして残る。
