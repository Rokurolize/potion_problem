# 生成消滅prefix解

この解では、停止前prefixを体積として数える代わりに、「一口を末尾に足す作用」と「末尾を消す作用」の関係を見る。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

だから

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## prefixを生成する作用素

境界を`x`まで動かし、

```text
a_n(x) = P(S_n <= x)
```

と置く。`0<=x<=1`だけを考える。空prefixについて

```text
a_0(x) = 1.
```

一様性と独立性により、末尾に一口を足す操作は

```text
a_{n+1}(x) = int_0^x a_n(t) dt
```

になる。つまり、積分作用素

```text
(C f)(x) = int_0^x f(t) dt
```

は、停止前prefixを一段生成する作用素である。

## 末尾を消す

一方、微分作用素

```text
A = d/dx
```

をかけると

```text
A a_{n+1}(x) = a_n(x)
```

となる。これは、長さ`n+1`のprefixの末尾を消して、長さ`n`のprefixへ戻す操作に対応する。境界`x`を少し動かしたとき、最後の一口がその薄い層に入る密度が`1`だからである。

したがって、`C`は生成、`A`は消滅であり、

```text
A C = I
```

がこの問題の一様性を表している。

## 全prefix和を真空から作る

全ての停止前prefixをまとめて

```text
G(x) = sum_{n >= 0} a_n(x)
```

と置く。非負項なので、`0<=x<=1`では単調収束で積分や微分の議論を部分和から通せる。

生成作用素で見ると

```text
G = 1 + C G.
```

ここで両辺に消滅作用素`A`をかけると、

```text
A G = A(1) + A C G = 0 + G.
```

つまり

```text
G'(x) = G(x).
```

また

```text
G(0) = 1
```

である。容量`0`では空prefixだけが残るからだ。したがって

```text
G(x) = exp(x).
```

求める値は境界`1`での全prefix和なので

```text
E[tau] = G(1) = e.
```

## この見方の芯

この証明では、各`P(S_n<=1)`を`1/n!`と計算しない。prefixを一段伸ばす積分作用素`C`と、境界微分で末尾を消す作用素`A`を先に見る。

全prefix和`G`は、空prefixという真空から生成作用素を任意回かけたもの

```text
G = (I + C + C^2 + ...) 1
```

である。消滅作用素をかけると生成された末尾が一つ消え、全体がそのまま戻る。だから`G`は`A`の固有関数になり、初期値`1`から指数関数が強制される。
