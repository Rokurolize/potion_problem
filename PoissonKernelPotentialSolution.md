# Poisson核ポテンシャル解

この解では、残容量上の殺されるジャンプ過程をPoisson核のポテンシャル方程式として解く。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。残容量が`x`のとき、停止前prefixの期待個数を`V(x)`と置く。現在のprefixを一つ数え、次の一口が`u<=x`なら残容量は`x-u`になるので

```text
V(x)=1+int_0^x V(x-u) du.
```

これは区間で殺される積分核

```text
K(x,dy)=1_{0<=y<=x} dy
```

のPoissonポテンシャル方程式

```text
(I-K)V=1
```

である。微分すれば

```text
V'(x)=V(x),  V(0)=1.
```

したがって

```text
V(x)=e^x,
E[tau]=V(1)=e.
```

この見方では、分布を長さごとに求めず、定数報酬`1`のポテンシャルをPoisson核で解いている。
