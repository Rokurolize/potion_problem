# q類似極限解

この解では、境界容量をq格子で近似し、q指数の極限として答えを読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めて

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

境界`x`までの総量を

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

と置く。q格子で境界を細かく進めると、一段の相対幅はq差分として読める。`q -> 1`の極限では、既存prefixが新しい薄層へ送る質量は通常の幅`dx`に戻るので

```text
D_q F(x) -> F'(x)=F(x).
```

初期値は空prefixだけなので

```text
F(0)=1.
```

したがってq指数の極限として

```text
F(x)=exp(x),
E[tau]=F(1)=e.
```

この証明では、単体体積を直接計算しない。q格子上の境界差分が、`q -> 1`で自己増殖方程式へ戻ることを見る。`e`は、そのq指数極限である。
