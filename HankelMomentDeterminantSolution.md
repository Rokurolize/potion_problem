# Hankelモーメント行列式解

この解では、生存prefix列をモーメント列として束ね、そのHankel構造が指数型の全和に落ちることを見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めて

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

境界`x`までの生存列を

```text
a_n(x)=P(S_n<=x)
```

と置く。`0<=x<=1`では上端制約が邪魔をしないので、最後の一口を外すと

```text
a_n'(x)=a_{n-1}(x),
a_0(x)=1.
```

これはモーメント列のHankel行列を一段ずつずらす関係である。全和

```text
A(x)=sum_{n>=0} a_n(x)
```

を取ると、Hankel列のシフトがそのまま

```text
A'(x)=A(x),
A(0)=1
```

になる。したがって

```text
A(x)=exp(x),
E[tau]=A(1)=e.
```

この証明では、Hankel行列式を実際に展開しない。生存列のモーメントシフト構造だけで全和が指数関数の方程式を満たすことを見る。`e`は、そのHankelモーメント列の全和である。
