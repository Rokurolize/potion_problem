# 対称冪体積解

待ち時間を `X_1, X_2, ...` とし、部分和を `S_n = X_1 + ... + X_n` とする。到着数は

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので、期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

ここで `S_n <= 1` という条件を、到着時刻

```text
t_i = S_i
```

で書き直す。すると領域は

```text
0 <= t_1 <= t_2 <= ... <= t_n <= 1
```

という標準単体になる。写像 `x_i = t_i - t_{i-1}` のヤコビアンは1なので、この領域の体積がそのまま `P(S_n <= 1)` である。

この単体は、区間 `[0, 1]` の `n` 個の点からなる配置を順序を忘れて見た空間、つまり対称冪

```text
Sym^n([0, 1]) = [0, 1]^n / S_n
```

そのものと見られる。対角集合は体積0なので、ほとんどすべての軌道は `n!` 個のラベル付き点を持つ。したがって

```text
vol(Sym^n([0, 1])) = 1 / n!
```

である。

よって

```text
E[tau] = sum_{n >= 0} vol(Sym^n([0, 1]))
       = sum_{n >= 0} 1 / n!
       = e
```

これは停止時刻の分布を直接求めているのではなく、区間内の有限個の非ラベル付き配置全体の体積を足し上げているだけである。単位区間の対称冪を全次数で集めると、指数級数がそのまま現れ、答えは `e` になる。
