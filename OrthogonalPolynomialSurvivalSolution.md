# orthogonal polynomial survival解

この解では、生存多項式列を直交多項式ではなく、その前段階のAppell型基底として扱う。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。生存多項式

```text
p_n(x)=P(S_n<=x)
```

は`0<=x<=1`で

```text
p_0(x)=1,
p_n'(x)=p_{n-1}(x),
p_n(0)=0
```

を満たす。これは直交化する前の単項式列を積分で正規化した基底であり、

```text
p_n(x)=x^n/n!
```

である。

停止前prefixの期待個数は、この基底を`x=1`で全次数足した値である。

```text
E[tau]=sum_{n>=0} p_n(1)
      =sum_{n>=0} 1/n!
      =e.
```

直交多項式論のように三項漸化式へ進む必要はない。停止境界内では、積分で一段ずつ上げる最も単純な多項式基底だけで全てが決まる。
