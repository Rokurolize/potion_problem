# antiderivative volume解

この解では、停止前単体の体積を原始関数で再帰的に計算する。

長さ`n`で停止前の体積を

```text
V_n(c)=vol{x_i>=0, x_1+...+x_n<=c}
```

と置く。最後の座標を`t`で固定すると

```text
V_n(c)=int_0^c V_{n-1}(c-t) dt.
```

これは

```text
V_n'(c)=V_{n-1}(c),  V_n(0)=0
```

という原始関数の再帰である。`V_0(c)=1`から始めると

```text
V_n(c)=c^n/n!
```

となる。元の問題では`c=1`なので

```text
P(tau>n)=V_n(1)=1/n!.
```

尾和公式により

```text
E[tau]=sum_{n>=0}1/n!=e.
```

停止前確率は、定数関数1を`n`回原始関数にした値として現れる。
