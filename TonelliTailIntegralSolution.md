# Tonelli tail integral解

この解では、停止時刻の期待値を非負関数のTonelli交換として読む。

停止時刻について

```text
tau=sum_{n>=0} 1_{tau>n}
```

が成り立つ。右辺は非負なので、Tonelliの定理により期待値と和を交換できる。

```text
E[tau]=sum_{n>=0}P(tau>n).
```

長さ`n`で停止前にある事象は

```text
{X_1+...+X_n<=1}
```

であり、独立一様分布の密度は1だから

```text
P(tau>n)=int_{x_i>=0, x_1+...+x_n<=1}1 dx=1/n!.
```

したがって

```text
E[tau]=sum_{n>=0}1/n!=e.
```

非負性だけで和と積分を入れ替え、残った各項を停止前単体の体積として評価する。
