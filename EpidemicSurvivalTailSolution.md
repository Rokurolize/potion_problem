# epidemic survival tail解

この解では、流行が時刻`n`まで生存している尾確率を足す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。流行の残り資源を`1-S_n`と見れば、時刻`n`まで生存している条件は

```text
tau>n  <=>  S_n<=1.
```

したがって整数値変数の尾和から

```text
E[tau]=sum_{n>=0} P(tau>n)
      =sum_{n>=0} P(S_n<=1).
```

`S_n<=1`の有限次元領域は標準単体であり、

```text
P(S_n<=1)=1/n!.
```

よって

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

この見方では、流行が消えた後のovershootではなく、生存している全時刻の尾確率だけを数える。
