# Noether境界電荷解

この解では、境界を平行移動する対称性に対応する保存量を、prefix総量の境界電荷として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`以下にあるprefixの総量を

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

と置く。境界を微小量`dx`だけ平行移動すると、既存prefixは最後の一口が新しい境界層に入ることで新しいprefixを一つ作る。一様密度が定数なので、この境界電荷は現在の総量そのものに等しい。

```text
dF = F dx.
```

したがってNoether型の境界保存式は

```text
F'(x)=F(x)
```

である。初期境界`0`では空prefixだけが存在するので

```text
F(0)=1.
```

よって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

ここで保存されているのは通常の力学的エネルギーではなく、境界を動かしたときに薄層へ流入するprefix質量である。
