# 鎖測度の正規化解

この解では、期待値を「正規化されていない確率測度の全質量」として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

生存しているprefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって

```text
E[tau]
= sum_{n >= 0} P(S_n <= 1).
```

ここまではよくある形だが、ここから各項を確率としてではなく、同じ空間上の鎖の質量としてまとめて見る。

## 生存prefixは区間内の鎖である

`n >= 1`について、累積和

```text
t_i = S_i
```

へ変数変換する。この変換のJacobianは`1`で、事象`S_n <= 1`は

```text
0 < t_1 < t_2 < ... < t_n <= 1
```

という長さ`n`の鎖そのものになる。`n=0`は空の鎖である。

よって`E[tau]`は、すべての有限鎖

```text
empty chain,
0 < t_1 <= 1,
0 < t_1 < t_2 <= 1,
...
```

に対して、Lebesgue密度`1`を置いたときの全質量である。

この未正規化測度を`mu`と書くと、

```text
mu(total)
= E[tau].
```

## 正規化定数をPoisson過程から読む

一方、単位強度のPoisson点過程を区間`[0,1]`に置く。点数が`n`で、その点を昇順に

```text
0 < T_1 < ... < T_n <= 1
```

と書くと、その同時密度は

```text
exp(-1)
```

である。つまりPoisson過程の法は、上の鎖測度`mu`に一つの定数`exp(-1)`を掛けたものになっている。

Poisson過程の法は確率測度なので、全質量は`1`である。したがって

```text
1 = exp(-1) * mu(total).
```

ゆえに

```text
E[tau]
= mu(total)
= exp(1)
= e.
```

## 何が起きているか

この証明では、`1/n!`を個別に計算してから足すのではない。

「生存prefix全体」を、長さが自由な有限鎖の空間としてまとめる。その鎖測度は、Poisson点過程のJanossy密度から見ると、正規化定数だけが欠けた確率測度である。

欠けている正規化定数が`exp(-1)`なので、全質量としての期待値はその逆数`e`になる。
