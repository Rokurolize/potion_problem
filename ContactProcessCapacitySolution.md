# contact process capacity解

この解では、容量`1`の接触資源が枯れるまでのprefix数として解く。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。接触のたびに資源を`U_i`だけ消費し、累積消費が`1`を越えたら接触過程は止まる。残り容量が`x`のとき、停止前prefixの期待個数を`C(x)`と書くと

```text
C(x)=1+int_0^x C(x-u) du
    =1+int_0^x C(t) dt.
```

したがって

```text
C'(x)=C(x),  C(0)=1,
```

なので

```text
C(x)=e^x.
```

初期容量は`1`だから

```text
E[tau]=C(1)=e.
```

contact processと呼んでいるが、ここでの本質は空間的な感染ではなく、接触資源の残量に閉じた一段再帰である。
