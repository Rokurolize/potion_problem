# sum pushforward density解

この解では、全ての停止前prefix空間を和写像で`[0,1]`へ押し出してから数える。

長さ`n`の入力列に対し

```text
q_n(u_1,...,u_n)=u_1+...+u_n
```

とおく。元の停止時刻

```text
tau=min { n>=1 : q_n(U_1,...,U_n)>1 }
```

について、尾和公式は

```text
E[tau]=sum_{n>=0} vol{q_n<=1}
```

である。

`n>=1`について、正の直交象限のLebesgue測度を`q_n`で押し出す。和が`x`であるfiberの測度密度は

```text
x^{n-1}/(n-1)!
```

である。これは

```text
vol{q_n<=x}=x^n/n!
```

を微分しても得られる。

全ての次数を一つに束ねた押し出し測度は、原点の空prefix質量`delta_0`と、密度

```text
sum_{n>=1} x^{n-1}/(n-1)! = e^x
```

を持つ測度である。従って

```text
E[tau]
= delta_0([0,1]) + int_0^1 e^x dx
= 1 + (e-1)
= e.
```

ここでは、停止前prefixを次数ごとに残さず、和だけを見る測度へ押し出すことで、指数関数が密度として現れる。
