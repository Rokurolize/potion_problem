# Radonレベル変換解

この解では、生存prefixの質量を、和写像

```text
h_n(u_1,...,u_n) = u_1 + ... + u_n
```

に沿ったRadon型のレベル変換として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は尾和で

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## レベル質量をまとめる

`n`口後の和がちょうど`x`付近にある生存prefixの密度を

```text
a_n(x)
```

と書く。これは、正の直交象限上のルベーグ測度を`h_n`で押し出した密度を、`0 <= x <= 1`で見たものである。空prefixは

```text
a_0 = delta_0
```

と考える。

一様増分を一つ足すことは、レベル密度を区間`[0,1]`の一様密度で畳み込むことだが、`x <= 1`だけを見る限り右端の切れ目は現れない。したがって

```text
a_{n+1}(x) = int_0^x a_n(y) dy
```

である。

正の長さの全レベル密度を

```text
B(x) = sum_{n >= 1} a_n(x)
```

とまとめる。`a_1(x)=1`であり、`x > 0`では

```text
B(x) = 1 + int_0^x B(y) dy
```

となる。微分して

```text
B'(x) = B(x),
B(0+) = 1.
```

よって

```text
B(x) = exp(x).
```

## 境界値として読む

容量`1`までに入る全prefixの総量は、空prefixの`1`と、正の長さのレベル密度の累積である。

```text
sum_{n >= 0} P(S_n <= 1)
  = 1 + int_0^1 B(x) dx
  = 1 + int_0^1 exp(x) dx
  = e.
```

この証明では、個々の単体体積を先に計算しない。和写像のレベルへ測度を押し出し、境界`1`より手前では一様密度の畳み込みが単なる積分作用になることだけを使う。答えの`e`は、Radon型のレベル変換が境界の手前で自己積分方程式に閉じるために現れる。
