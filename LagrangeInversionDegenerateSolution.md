# Lagrange反転退化解

この解では、停止前prefixの総量を、自己増殖方程式の退化したLagrange反転として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`までの停止前prefix総量を

```text
F(x)=sum_{n>=0} P(S_n <= x)
```

と置く。`0 <= x <= 1`では、一様分布の上端はまだ次のprefix生成を削らない。境界を`dx`だけ進めると、既存prefixの総量`F(x)`がそのまま薄層へ一段伸びるので

```text
F'(x)=F(x),  F(0)=1.
```

これを形式的に

```text
F(x)=1+int_0^x F(t) dt
```

という固定点方程式として見る。通常のLagrange反転では合成の木が現れるが、この方程式では枝分かれが一列に退化しているため、反復積分だけが残る。

```text
F(x)=sum_{n>=0} x^n/n!.
```

したがって

```text
E[tau]=F(1)=e.
```

反転公式を本格的に使う必要はない。自己参照が単なる積分作用素に退化しているので、Lagrange反転の木係数が一本鎖の階乗だけに潰れる。
