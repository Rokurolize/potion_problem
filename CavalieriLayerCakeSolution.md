# Cavalieri層積分解

この解では、各次元の単体体積を直接出さず、生存prefix全体を境界高さで層に切って足す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。正整数値の尾和公式より

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

境界を`r`までにしたときの停止前prefix総量を

```text
M(r) = sum_{n >= 0} P(S_n <= r)
```

とおく。求めたい値は`M(1)`である。

## 薄い層のCavalieri分解

`M(r+h)-M(r)`は、和が薄い層

```text
(r, r+h]
```

に入るprefixの総量である。非空prefixを、直前のprefixと最後の一口に分ける。

直前のprefixの和を`y <= r`とする。このprefixが薄い層へ入るには、最後の一口`u`が

```text
r - y < u <= r + h - y
```

に入ればよい。`0 <= r < r+h <= 1`なら、この区間の長さは`h`であり、一様密度は`1`である。したがって、境界`r`までにある各prefixは、薄い層へちょうど`h + o(h)`だけ寄与する。

Cavalieriの原理で全prefixを層ごとに足すと

```text
M(r+h) - M(r) = h M(r) + o(h).
```

よって

```text
M'(r) = M(r),
M(0) = 1.
```

初期値`M(0)=1`は空prefixだけである。したがって

```text
M(r) = exp(r).
```

特に

```text
E[tau] = M(1) = e.
```

この証明では、`P(S_n <= 1)`を長さごとに計算しない。高さ方向の層を見て、薄い層へ入る総量が直前までの総量そのものになることだけを使う。`e`は、Cavalieri層積分で得られる自己増殖方程式の境界値である。
