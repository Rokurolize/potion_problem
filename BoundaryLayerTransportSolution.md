# 境界層輸送解

独立一様乱数を `X_1, X_2, ...` とし、

```text
S_n = X_1 + ... + X_n
```

とおく。境界を `t` に動かして、

```text
M(t) = sum_{n >= 0} P(S_n <= t)
```

を考える。求める量は `M(1)` である。

`M(t)` は、境界 `t` の内側に残っている全prefixの総質量である。`n=0` の空prefixは常に1つあるので、

```text
M(0) = 1
```

である。

## 境界を少し動かす

境界を `t` から `t+h` に広げたとき、新しく現れる長さ `n+1` のprefixは、長さ `n` の生存prefixに最後の一杯を足して、和が新しい薄い層

```text
t < S_n + x <= t+h
```

へ入ったものだけである。

長さ `n` のprefixを固定すると、最後の一杯 `x` の許される幅は

```text
max(0, t+h-S_n) - max(0, t-S_n)
```

である。`S_n <= t` のprefixでは、この幅は一次の項として `h` であり、境界ぴったりの集合は体積0なので、全体では

```text
P(S_{n+1} <= t+h) - P(S_{n+1} <= t)
  = h P(S_n <= t) + o(h)
```

となる。

したがって

```text
d/dt P(S_{n+1} <= t) = P(S_n <= t)
```

である。

## 全prefix質量だけを見る

上の式を `n >= 0` で足すと、

```text
M'(t)
= sum_{n >= 1} d/dt P(S_n <= t)
= sum_{n >= 0} P(S_n <= t)
= M(t)
```

を得る。初期値は `M(0)=1` だから、

```text
M(t) = exp(t)
```

である。

よって

```text
E[tau]
= sum_{n >= 0} P(S_n <= 1)
= M(1)
= e
```

となる。

この証明では、各次元の単体体積を個別に求めない。境界を押し出したとき、薄い境界層が「既存の全prefix質量」をそのまま次のprefixへ輸送することだけを見る。`e` は、全prefix質量が自分自身を増加率として持つことから出る。
