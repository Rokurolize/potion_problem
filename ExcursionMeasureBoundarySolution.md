# excursion measure boundary解

この解では、`1`を越える直前までの軌道片を境界へ向かうexcursionの集まりとして測る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。長さ`n`の停止前軌道片は

```text
0<=u_i<=1,  u_1+...+u_n<=1
```

を満たす増分列である。和の条件があるため、`u_i<=1`は自動的に従う。したがって軌道片全体の測度は標準単体

```text
{u_i>=0, u_1+...+u_n<=1}
```

の体積であり、

```text
1/n!
```

である。

停止時刻`tau`は、空軌道片を含む停止前軌道片の数である。よって軌道片測度を全長で足し上げれば

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

境界を越えた後のovershootは、この期待値には入らない。境界に達する前の全ての軌道片だけを測れば十分である。
