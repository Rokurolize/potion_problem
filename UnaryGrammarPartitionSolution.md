# unary grammar partition解

この解では、停止前prefixを一つの再帰文法が生成する語の分配関数として扱う。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。残り容量`x`を持つ非終端記号`A_x`を考える。`A_x`は空語を一つ生成するか、長さ`u`の文字を一つ出して`A_{x-u}`へ戻る。

```text
A_x -> empty
A_x -> u A_{x-u}   (0<=u<=x)
```

この文法の分配関数を`Z(x)`とすると、

```text
Z(x)=1+int_0^x Z(x-u) du
    =1+int_0^x Z(t) dt.
```

よって

```text
Z'(x)=Z(x),  Z(0)=1,
```

から

```text
Z(x)=e^x.
```

停止前prefixはこの文法が容量`1`で生成できる有限語そのものであり、各語の重みは対応する増分列の体積である。したがって

```text
E[tau]=Z(1)=e.
```

文法の規則は一つだけだが、容量を添字に持たせることで停止境界が自然に埋め込まれる。
