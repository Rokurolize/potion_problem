# 根付き経路の質量輸送解

この解では、停止前のprefixを「根から境界`1`まで伸びる経路」と見て、各経路が次の一口で新しい層へ送る質量を数える。

独立な一様乱数を`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

と置く。`S_0=0`を含めると、停止時刻は境界内にあるprefixの個数である。

```text
tau = sum_{n>=0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n>=0} P(S_n <= 1).        (1)
```

## 境界までの全経路質量

境界を`r`に変え、そこまでに存在するprefixの期待個数を

```text
M(r) = sum_{n>=0} P(S_n <= r)
```

と置く。求める値は`M(1)`である。

`S_0=0`が必ずあるので

```text
M(0) = 1.
```

## 各prefixは同じ幅だけ質量を送る

境界を`r`から`r+h`へ少し動かす。新たに増えるprefixは、どこか`x<=r+h`にいた古いprefixから、次の一口で薄い層

```text
(r, r+h]
```

へ着地したものである。

一様増分の密度は`1`なので、古いprefixが位置`x<=r`にあるとき、次の一口が`(r-x, r+h-x]`に入る確率はちょうど`h`である。ただし`x`が`r`に近いか遠いかには依存しない。境界の左にある各prefixが、同じ幅`h`だけ新しい層へ質量を送る。

したがって、薄い層に入る期待質量は

```text
h M(r) + o(h).
```

つまり

```text
M'(r) = M(r).        (2)
```

これは、根から伸びた各prefix点が、境界を少し先へ押したとき同じ量の子を生む、という質量輸送の式である。

## 初期質量から解く

(2)と`M(0)=1`より

```text
M(r) = exp(r).
```

したがって

```text
E[tau] = M(1) = exp(1) = e.
```

## この見方の芯

ここでは、`P(S_n<=1)=1/n!`を先に出さない。境界内のprefixを全て根付き経路上の質量として集める。

一様性は、「どのprefixから見ても、薄い次層へ送る確率が同じ幅になる」という一点で使われる。既にある全prefix質量がそのまま次の層の生成率になり、自己増殖の方程式`M'=M`から`e`が出る。
