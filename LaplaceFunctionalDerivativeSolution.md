# Laplace汎関数微分解

この解では、停止前prefix全体をPoisson過程のLaplace汎関数を活動度方向に動かした配置展開として読む。

長さ`n`の生存prefixは、累積時刻

```text
0 <= t_1 <= ... <= t_n <= 1
```

に移すと、区間`[0,1]`上の順序付き`n`点配置である。増分から累積時刻への変換はJacobian`1`なので、この順序付き領域の体積が

```text
P(S_n <= 1)
```

である。

## 活動度を入れたLaplace汎関数

単位強度のPoisson過程で、区間`[0,1]`上の点に一様な印`z`を掛ける。生成汎関数は

```text
E[z^{N([0,1])}] = exp(z-1).
```

この式はLaplace汎関数

```text
E[prod_{x in N} z]
  = exp(int_0^1 (z-1) dx)
```

そのものである。

Poisson確率では、各`n`点配置に共通して空白因子`exp(-1)`が掛かっている。したがって、その空白因子を外した停止前prefix測度の活動度付き総量は

```text
exp(1) E[z^{N([0,1])}]
  = exp(z).
```

ここで`z^n`の係数は、長さ`n`の停止前prefix全体の質量である。期待停止回数は活動度を`z=1`にした総量なので

```text
E[tau]
  = exp(1)
  = e.
```

この証明では、各`n`の単体体積を個別に計算しない。Poisson過程のLaplace汎関数に活動度を入れ、空白因子を外した配置展開をそのまま停止前prefixの総量として読む。
