# Grothendieck residue tail解

この解では、停止前単体の体積を係数抽出、つまり留数として取り出す。

長さ`n`で止まらない確率は

```text
P(tau>n)=vol{x_i>=0, x_1+...+x_n<=1}
```

である。合計値`r=x_1+...+x_n`で切ると、その断面密度は`r^{n-1}/(n-1)!`である。したがって

```text
P(tau>n)=int_0^1 r^{n-1}/(n-1)! dr.
```

この積分は、指数関数の係数を留数で読む形に書ける。すなわち

```text
1/n! = [z^n] exp(z)
     = Res_{z=0} exp(z) dz/z^{n+1}.
```

ここで係数`[z^n]exp(z)`は、長さ`n`の停止前領域の体積そのものを表している。よって

```text
P(tau>n)=Res_{z=0} exp(z) dz/z^{n+1}.
```

尾和を取ると

```text
E[tau]=sum_{n>=0} Res_{z=0} exp(z) dz/z^{n+1}
      =sum_{n>=0}1/n!
      =e.
```

確率計算の中身は、停止前単体の体積を指数関数の係数として留数で抜くことに尽きる。
