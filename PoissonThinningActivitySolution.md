# Poisson薄化活動度解

この解では、停止前prefixをPoisson点過程の薄化パラメータで数える。

長さ`n`の生存prefixは、累積時刻

```text
0 <= t_1 <= ... <= t_n <= 1
```

に移すと、区間`[0,1]`上の順序付き`n`点配置である。増分から累積時刻への変換はJacobian`1`である。

## 薄化パラメータを入れる

単位強度Poisson点過程を`[0,1]`上に置き、各点を確率`p`で残す。薄化後の点過程は強度`p`のPoisson過程なので、点数の全確率は

```text
sum_{n >= 0} exp(-p) p^n / n! = 1.
```

ここで、確率側の各`n`点配置には共通の空白因子`exp(-p)`が付いている。これを外すと、薄化後に残った有限配置の非正規化総量は

```text
sum_{n >= 0} p^n / n! = exp(p).
```

この`p^n`は、停止前prefixの長さを記録する活動度である。活動度を`p=1`に戻すと、全ての停止前prefixを同じ重みで数える。

したがって

```text
E[tau]
  = sum_{n >= 0} P(S_n <= 1)
  = exp(1)
  = e.
```

この証明では、各`P(S_n<=1)`を個別に積分しない。Poisson点過程の薄化で点数を活動度として記録し、薄化後Poisson測度の空白正規化を外した総量として`e`を読む。
