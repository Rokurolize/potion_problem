# 一単位ずつ燃えるポテンシャル解

ここでは、媚薬問題を「残り距離を資産とする償却解析」として解く。

設定は通常通りとする。各回の増分を独立な一様乱数`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

`n`回飲んだ後にまだ止まっていない事象を

```text
A_n = { S_n <= 1 }
```

と書く。`A_n`上では、境界までの残り距離は

```text
r_n = 1 - S_n
```

である。

## 証明書を先に出す

残り距離`r`に対して、資産

```text
Phi(r) = exp(r)
```

を置く。ここで重要なのは、`exp`という名前ではなく、次の等式である。

```text
Phi(r) - int_0^r Phi(r-u) du = 1.        (1)
```

実際、`Phi(r)=exp(r)`なら

```text
int_0^r exp(r-u) du
= int_0^r exp(t) dt
= exp(r) - 1.
```

したがって(1)が成り立つ。

## 一口飲むたびに、期待値で一単位燃える

生存中、残り距離が`r`であるとする。次の一口の量を`U`とする。

`U > r`なら境界を超えて終了するので、次の資産は`0`である。`U <= r`なら生存し、次の残り距離は`r-U`である。よって、次回の資産の条件付き期待値は

```text
E[ Phi(r-U) * 1_{U <= r} ]
= int_0^r Phi(r-u) du
= Phi(r) - 1.
```

つまり、生きて一口飲ませるたびに、資産`Phi`は条件付き期待値でちょうど`1`だけ減る。

`n`回後の生存資産を

```text
V_n = Phi(1 - S_n) * 1_{A_n}
```

とする。上の計算は、任意の`n`について

```text
E[V_{n+1} | F_n] = V_n - 1_{A_n}
```

を意味する。したがって

```text
M_n = V_n + sum_{k=0}^{n-1} 1_{A_k}
```

は平均が変わらない量である。

## 資産を使い切るまでの平均回数

初期資産は

```text
V_0 = Phi(1) = exp(1).
```

上の平均保存を`n`回目まで見ると、

```text
E[V_n] + sum_{k=0}^{n-1} P(A_k) = exp(1).        (2)
```

左の和は単調に増え、右辺以下なので収束する。したがって`P(A_n)`は`0`に向かう。また

```text
0 <= V_n <= exp(1) * 1_{A_n}
```

なので`E[V_n]`も`0`に向かう。

よって(2)で`n`を無限に送ると、

```text
sum_{k=0}^infty P(A_k) = exp(1).
```

一方、停止時刻は「まだ生きていた段数」を数えたものだから

```text
tau = sum_{k=0}^infty 1_{A_k}.
```

期待値を取って

```text
E[tau]
= sum_{k=0}^infty P(A_k)
= exp(1)
= e.
```

## 何を数えなかったか

この解では、`P(tau=n)`を求めない。`P(S_n<=1)=1/n!`も使わない。かわりに、境界までの残り距離に`exp(r)`という資産を載せる。

一口ごとに、この資産は期待値で正確に一単位ずつ燃える。だから、最初に持っていた資産`exp(1)`が、そのまま平均飲用回数になる。
