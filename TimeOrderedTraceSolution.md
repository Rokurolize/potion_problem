# 時間順序跡解

この解では、生存prefixを単体の体積として個別に数えず、「時間順序積分作用素」の全階層の跡としてまとめる。

独立な一様乱数`U_1,U_2,...`に対して

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixの個数として

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので、

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 順序付き積分作用素

区間`[0,1]`上の関数に対して、Volterra型の積分作用素

```text
(Tf)(t) = int_0^t f(s) ds
```

を考える。`T`を一回かけることは、時刻`t`の手前に一つ切断点を置くことに対応する。

実際、`T^n 1`を展開すると

```text
(T^n 1)(t)
```

は

```text
0 <= x_1 <= x_2 <= ... <= x_n <= t
```

という時間順序付き配置の体積である。ここで`x_i`は部分和`S_i`に対応し、増分から部分和への変換はJacobian`1`である。

したがって

```text
P(S_n <= 1) = (T^n 1)(1).
```

## 全階層を一つの resolvent にする

求める量は

```text
sum_{n >= 0} (T^n 1)(1).
```

これを

```text
F = sum_{n >= 0} T^n 1
```

と置けば、

```text
F = 1 + TF.
```

つまり

```text
F(t) = 1 + int_0^t F(s) ds.
```

よって

```text
F'(t) = F(t),
F(0) = 1.
```

したがって

```text
F(t) = exp(t)
```

であり、

```text
E[tau] = F(1) = e.
```

## この見方の芯

この証明は、`1/n!`を先に計算してから足すのではない。停止前prefix全体を、時間順序積分作用素`T`のNeumann級数

```text
(I - T)^{-1} 1
```

として一度に読む。`e`は、全ての時間順序付きprefixをまとめて解いたresolvent方程式の値として現れる。
