# Doob正規化停止木解

この解では、停止前prefix全体をそのまま足す代わりに、別の確率過程へ正規化する。正規化された過程では全ての停止履歴の確率が合計`1`になるので、元の停止前prefix総量はその正規化定数として読める。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 停止前履歴木を作る

長さ`n`の停止前prefixは

```text
u_i >= 0,
u_1 + ... + u_n <= 1
```

を満たす履歴である。この領域のLebesgue体積が`P(S_n<=1)`である。したがって、全ての深さの停止前履歴の総体積は`E[tau]`である。

ここから各深さの体積を個別に計算しない。代わりに、残り距離

```text
r = 1 - (u_1 + ... + u_n)
```

を持つ節点から、次のような別の確率過程を走らせる。

- `0<=u<=r`の生存枝へ進む密度を`exp(-u) du`にする。
- その場で停止する確率を`exp(-r)`にする。

これは本当に確率分岐である。なぜなら

```text
int_0^r exp(-u) du + exp(-r)
= (1 - exp(-r)) + exp(-r)
= 1
```

だからである。

## どの停止履歴も同じ重みになる

初期残り距離は`1`である。停止前に`u_1,...,u_n`だけ進み、残り距離

```text
r_n = 1 - (u_1 + ... + u_n)
```

で停止する履歴を考える。この正規化過程でのその履歴の密度は

```text
exp(-u_1) ... exp(-u_n) * exp(-r_n).
```

指数をまとめると

```text
exp(-(u_1 + ... + u_n + r_n)) = exp(-1).
```

つまり、深さがいくつであっても、停止前履歴領域のLebesgue測度に対して、終端停止まで含めた正規化後の密度は常に`exp(-1)`である。

## 正規化定数を読む

正規化過程は必ずどこかの停止履歴で止まるので、全停止履歴の確率の和は`1`である。一方、上で見た通り、その確率測度は元の停止前履歴総体積に共通因子`exp(-1)`を掛けたものになっている。

したがって

```text
1
= exp(-1) * sum_{n >= 0} P(S_n <= 1)
= exp(-1) * E[tau].
```

よって

```text
E[tau] = exp(1) = e.
```

## この見方の芯

この証明では、`1/n!`も`P(tau=n)`も出さない。停止前履歴木そのものに、指数関数で作った確率分岐を入れる。

生存枝の重み`exp(-u)`と、最後に止まる重み`exp(-r)`が、どの履歴でも合計距離`1`をちょうど埋める。そのため、全停止履歴は一様に`exp(-1)`倍される。答えの`e`は、停止前履歴の非正規化測度を確率測度へ直すための正規化定数として現れる。
