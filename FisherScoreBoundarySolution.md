# Fisher score境界解

この解では、境界を動かしたときのprefix総量の変化を、定数密度のscoreが境界に残す流量として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`以下で生きているprefixの総量を

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

と置く。

一様密度は内部でscoreが`0`であり、境界を動かしたときの寄与は境界面に出る流量だけである。`x`から`x+dx`へ境界を動かすと、既存prefixのそれぞれに対して最後の一口が薄い層に入る確率は`dx`である。したがって

```text
F(x+dx)-F(x)=F(x)dx+o(dx).
```

つまり

```text
F'(x)=F(x),  F(0)=1.
```

よって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

Fisher scoreの見方では、内部の密度変化ではなく、境界移動で生じる定数流量だけが期待停止時刻を決めている。
