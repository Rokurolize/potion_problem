# vacuum expectation prefix解

この解では、停止前prefixの総質量を生成消滅演算子の真空期待値として読む。

長さ`n`まで停止しない事象は

```text
U_1+...+U_n<=1
```

である。累積和

```text
t_k=U_1+...+U_k
```

へ移すと、停止前領域は

```text
0<=t_1<=...<=t_n<=1
```

という順序領域になる。

点を一つ生成する演算子`a^*`を、区間`[0,1]`上の順序付き配置へ点を挿入する演算子と見る。真空`Omega`から`n`回生成した成分の質量は

```text
<1, (a^*)^n Omega> = int_{0<=t_1<=...<=t_n<=1} dt_1...dt_n = 1/n!.
```

したがって空prefixを含む全停止前prefixの質量は

```text
<1, exp(a^*) Omega>
=sum_{n>=0} 1/n!
=e.
```

尾和公式によりこの量が元の停止時刻の期待値だから

```text
E[tau]=e.
```

この見方では、停止前履歴を順序付き粒子配置に移し、指数ベクトルの真空期待値として`e`を得る。
