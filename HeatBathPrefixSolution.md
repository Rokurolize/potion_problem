# 熱浴prefix解

この解では、一口の乱数を熱浴から受ける入力として見て、生存prefixの総応答を計算する。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。残り容量`r`の系が熱浴から一口`u`を受け取り、`u<=r`なら生存して残り`r-u`へ移る。空prefixを含む生存応答を`H(r)`と置くと

```text
H(r)=1+int_0^r H(r-u) du.
```

熱浴の密度が一様で、`0<=r<=1`では密度が定数`1`として働くためである。変数を替えると

```text
H(r)=1+int_0^r H(v) dv.
```

よって

```text
H'(r)=H(r),  H(0)=1.
```

したがって

```text
H(r)=e^r,
E[tau]=H(1)=e.
```

この証明では、熱浴という語は外部入力の一様性を示す。全ての生存prefixは、同じ密度の入力を受けるので指数応答になる。
