# nerve of prefix chain解

この解では、停止前prefixを累積和の鎖のnerveとして読む。

長さ`n`まで停止しない条件は

```text
U_1+...+U_n<=1.
```

累積和

```text
t_k=U_1+...+U_k
```

へ写すと、これは

```text
0<=t_1<=...<=t_n<=1
```

という`[0,1]`内の鎖になる。ヤコビアンは`1`である。

したがって、長さ`n`の停止前prefixの質量は、区間順序集合`[0,1]`の`n`-simplex nerveの体積

```text
vol{0<=t_1<=...<=t_n<=1}=1/n!
```

に等しい。尾和公式から

```text
E[tau]=sum_{n>=0}P(S_n<=1)
       =sum_{n>=0}1/n!
       =e.
```

この見方では、乱数列そのものではなく、停止前までの累積和が作る順序鎖の全nerve質量を数えている。
