# 越境辺Palm解

この解では、停止前prefixを先に数えない。各標本路で一度だけ現れる「境界を越えた最後の一口」をPalm点として固定し、その直前にあり得るprefixの総量を逆向きに読む。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。求めたい量は

```text
E[tau] = E[sum_{n >= 0} 1_{S_n <= 1}]
```

である。

## 越境辺は確率測度である

停止が起きる直前のprefix位置を`X`、最後の一口を`U`と書く。すると

```text
0 <= X <= 1,
X + U > 1,
0 <= U <= 1.
```

各標本路にはこの越境辺`(X,U)`がちょうど一つだけある。したがって、全ての深さ、全てのprefixを通して越境辺を足した測度は、全質量`1`の確率測度である。

いま、停止前prefix全体の位置測度を`M(dx)`と置く。

```text
M(B) = sum_{n >= 0} P(S_n in B, S_n <= 1).
```

位置`x`にあるprefixが次の一口で越境する確率は

```text
P(U > 1 - x) = x.
```

したがって越境辺の正規化は

```text
int_0^1 x M(dx) = 1.        (1)
```

である。

## 越境辺から直前prefixを削る

越境辺`(x,u)`を固定する。越境条件は

```text
u > 1 - x.
```

ここで最後の一口`u`を削ると、直前prefix位置`x`だけが残る。逆に、任意の停止前prefix位置`x`からは、長さ`x`の区間

```text
1 - x < u < 1
```

に一口を選ぶことで越境辺ができる。

この「削る/足す」の対応は、全ての深さで一対一である。越境辺側では全質量が`1`に正規化され、prefix側では重み`x`が掛かる。これが(1)である。

## 境界内に流れ込む密度を決める

原点の空prefixを除いた`M`の密度を`m(x)`と書く。点`x>0`に停止前prefixが来るには、直前prefixがどこか`y<x`にあり、最後の一口が`x-y`でなければならない。一様密度は`1`なので

```text
m(x) = 1 + int_0^x m(y) dy.        (2)
```

ここで右辺の`1`は、空prefix`0`から一口で直接`x`へ来る流入である。

通常なら(2)をすぐ解く。しかしこの解では、越境辺の正規化(1)と組にして読む。(2)を微分すると

```text
m'(x) = m(x),
m(0) = 1.
```

したがって

```text
m(x) = C exp(x)
```

で、初期流入から`C=1`である。さらに(1)は

```text
int_0^1 x m(x) dx = 1
```

を要求し、実際

```text
int_0^1 x exp(x) dx = 1.
```

越境辺の全質量が`1`であることと、境界内の自己流入が同じ指数密度を指している。

## 期待値へ戻す

停止前prefixの期待個数は、空prefixの`1`と、`(0,1]`内に流れ込んだ全prefix密度の質量である。

```text
E[tau]
= 1 + int_0^1 m(x) dx
= 1 + int_0^1 exp(x) dx
= e.
```

## この見方の芯

主役は生存確率ではなく、各標本路に一度だけ存在する越境辺である。越境辺をPalm点として見ると、その全質量は初めから`1`に正規化される。直前prefixへ削ると、prefix位置`x`には越境可能な一口の長さ`x`が重みとして付く。

`e`は、境界内で増えるprefix密度と、境界外へ一度だけ出る越境辺測度が同時に満たす正規化から現れる。
