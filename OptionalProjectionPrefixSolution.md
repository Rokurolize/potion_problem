# 任意射影prefix解

この解では、無限列全体の標本空間から「長さ`n`まで境界内にある」という事象へ射影し、その射影の期待値を足す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。各標本路について

```text
tau = sum_{n>=0} 1_{S_n<=1}
```

である。Tonelliの定理から

```text
E[tau]=sum_{n>=0} E[1_{S_n<=1}]
      =sum_{n>=0} P(S_n<=1).
```

ここで`n`番目の項は、無限積空間`[0,1]^N`から最初の`n`座標だけへ射影したときの、単体

```text
u_1+...+u_n<=1
```

の測度である。射影後の残りの座標は測度`1`の全空間として消えるため、計算すべき量は有限次元の単体測度だけになる。

累積和座標

```text
T_k=u_1+...+u_k
```

へ移るとJacobianは`1`で

```text
0<T_1<...<T_n<1.
```

この順序領域は立方体`[0,1]^n`の`n!`個の同体積な順序領域の一つだから

```text
P(S_n<=1)=1/n!.
```

したがって

```text
E[tau]=sum_{n>=0}1/n!=e.
```

無限列から有限prefixへ落とす射影で停止時刻を分解し、各射影像を順序領域として数えるだけで答えが出る。
