# probability integral transform tail解

この解では、一様乱数であることを確率積分変換の終点として使う。

任意の連続分布変数`Y_i`に対して

```text
U_i=F(Y_i)
```

とすれば`U_i`は一様分布になる。元問題はすでにこの変換後の座標で書かれているので、停止前条件はそのまま

```text
U_1+...+U_n<=1
```

である。つまり、長さ`n`で止まらない確率は、単位立方体内の標準単体の体積

```text
P(tau>n)=vol{u_i>=0, u_1+...+u_n<=1}
```

に等しい。

この単体の体積は

```text
1/n!
```

なので、尾和公式から

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

確率積分変換の座標では、独立性と一様性がLebesgue体積そのものになるため、停止時刻の期待値は単体体積の総和として読める。
