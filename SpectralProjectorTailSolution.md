# spectral projector tail解

この解では、停止前判定を部分和作用素のスペクトル射影として見る。

長さ`n`の履歴に対して、部分和を返す線形作用素

```text
S(x_1,...,x_n)=x_1+...+x_n
```

を考える。停止前事象は

```text
S<=1
```

という閾値以下の射影をかけた部分である。独立一様分布の同時密度は1なので、この射影が拾う質量は単に

```text
{x_i>=0, x_1+...+x_n<=1}
```

の体積になる。

この領域は標準`n`単体だから

```text
P(tau>n)=1/n!.
```

尾和公式により

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

部分和作用素の閾値射影が切り出すスペクトル側の質量が、そのまま停止前単体の体積になっている。
