# 順序単体への累積和写像解

この解では、`S_n<=1`の領域を「順序づけられた点配置」そのものとして見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

境界を越えるまでに見えているprefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

あとは`P(S_n<=1)`をどう読むかである。

## 累積和は順序点になる

`S_n<=1`が起きているとき、

```text
0 < S_1 < S_2 < ... < S_n <= 1
```

である。逆に、任意の

```text
0 < t_1 < t_2 < ... < t_n <= 1
```

に対して

```text
u_1 = t_1,
u_2 = t_2 - t_1,
...
u_n = t_n - t_{n-1}
```

と置けば、`u_i>0`かつ

```text
u_1 + ... + u_n = t_n <= 1
```

である。つまり

```text
(u_1,...,u_n) with u_i>=0, u_1+...+u_n<=1
```

と

```text
0<=t_1<=...<=t_n<=1
```

は、累積和

```text
t_k = u_1 + ... + u_k
```

で互いに移り合う。同じ領域を、増分で見るか、順序点で見るかの違いだけである。

## 順序点の体積は`1/n!`

`[0,1]^n`は、座標の大小関係によって`n!`個の領域に分かれる。

```text
0 <= t_{\sigma(1)} <= ... <= t_{\sigma(n)} <= 1
```

の各領域は座標置換で互いに同じ体積を持ち、境界の重なりは体積`0`である。したがって、そのうち一つである

```text
0 <= t_1 <= ... <= t_n <= 1
```

の体積は`1/n!`である。

累積和写像のJacobianは三角行列で対角がすべて`1`なので、体積を変えない。よって

```text
P(S_n <= 1) = 1/n!.
```

## 期待値

以上から

```text
E[tau]
= sum_{n >= 0} P(S_n <= 1)
= sum_{n >= 0} 1/n!
= e.
```

この見方では、`e`は単に級数として現れるのではなく、部分和列を順序点に変えたときの「全ての順序のうち、指定された一つの順序だけを選ぶ比率」として現れる。乱数の加算で作った停止時刻が、区間上の点の並び替え対称性にそのまま接続される。
