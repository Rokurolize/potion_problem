# Mayerクラスター自明化解

この解では、停止前prefixを有限粒子系の分配関数として読み、Mayerクラスター展開が相互作用なしで一次だけに潰れることを見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixも数えれば

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

増分列から累積時刻

```text
t_i = S_i
```

へ移す。Jacobianは`1`であり、`S_n <= 1`は

```text
0 <= t_1 <= ... <= t_n <= 1
```

である。したがって長さ`n`の停止前prefixは、区間`[0,1]`上の非ラベル`n`粒子配置と同じ測度を持つ。

ここで活動度`z`の非相互作用粒子系を考える。ラベル付き配置の重みは`z^n dt_1...dt_n`で、非ラベル化により

```text
Z(z)=sum_{n>=0} z^n P(S_n <= 1)
```

は有限粒子全体の大分配関数になる。

Mayer展開では

```text
log Z(z)
```

を連結クラスターで書く。粒子間相互作用がないので、二点以上の連結クラスター重みはすべて`0`である。残るのは一点クラスターだけで、その重みは区間長

```text
int_0^1 dt = 1
```

である。したがって

```text
log Z(z)=z,
Z(z)=exp(z).
```

求める値は活動度`z=1`で

```text
E[tau]=Z(1)=e.
```

この証明では、各単体体積を先に出さない。停止前prefixを非相互作用粒子系の全粒子数分配関数として読み、Mayerクラスター展開が一点クラスターだけに自明化することを使う。`e`は、その自明なクラスター指数化である。
