# Hilbert射影prefix解

この解では、停止前prefixをHilbert空間上の指示関数へ射影して、そのノルムではなく積分質量を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。長さ`n`では、Lebesgue空間`L^2([0,1]^n)`上の部分空間

```text
0 <= t_1 <= ... <= t_n <= 1
```

への射影を考える。増分から累積時刻` t_k=S_k `への変換はJacobian`1`であり、停止前条件`S_n<=1`はこの順序室への射影と同じである。

立方体は`n!`個の同じ大きさの順序室に分かれるので、射影された指示関数の積分質量は

```text
P(S_n<=1)=1/n!.
```

空prefixを含めて全次数を足すと

```text
E[tau]=sum_{n>=0}P(S_n<=1)=sum_{n>=0}1/n!=e.
```

この証明では、量子力学的な内積構造は補助である。必要なのは、停止条件が順序室への射影として表せることだけである。
