# Palm空白prefix解

この解では、停止前prefixを「Poisson点過程を一点から見たときの、右側の空白が保つprefix」として読む。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixの個数は

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

である。

## prefixをPalm側へ移す

`S_n <= 1`であるprefixは、区間`[0,1]`の中に

```text
0 = S_0 <= S_1 <= ... <= S_n <= 1
```

という左からの点列を置いたものと見られる。増分から部分和への変換のJacobianは`1`なので、このprefix測度は順序付き点列のLebesgue測度である。

ここで、強度`1`のPoisson過程を半直線`[0,infty)`上に置き、原点にPalm点を固定して見る。原点から右へ進んだとき、`[0,1]`内に見えるPoisson点列を

```text
0 < T_1 < ... < T_n <= 1
```

とする。

この有限列が指定された位置にある密度には、右端`1`までに余分な条件は要らない。ただし、Poisson過程の確率測度として見ると、`[0,1]`全体の点配置には空白因子

```text
exp(-1)
```

が常に付いている。

つまり、停止前prefixの順序付き測度は、Palmで見た`[0,1]`内のPoisson配置測度から、この全階層共通の空白因子`exp(-1)`を外したものになっている。

## 確率測度に戻す

Palm側のPoisson配置は確率測度である。すべての点数、すべての配置を足した総質量は

```text
1
```

である。

停止前prefix測度は各階層でその`exp(1)`倍なので、全prefixの総質量は

```text
exp(1) * 1 = e.
```

一方、この総質量は

```text
sum_{n >= 0} P(S_n <= 1) = E[tau]
```

そのものである。よって

```text
E[tau] = e.
```

この見方では、`e`は`1/n!`を足して出るのではない。Palm点から右に見えるPoisson配置を確率測度へ正規化するとき、区間`[0,1]`の空白因子`exp(-1)`が一括で掛かっており、その逆数が平均停止前prefix数として現れる。
