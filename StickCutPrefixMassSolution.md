# 棒切断prefix質量解

この解では、停止前の列を「長さ1の棒を左から切っていく途中経過」として見る。

独立な一様乱数`U_1,U_2,...`に対して

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前であることは

```text
S_n <= 1
```

であり、`tau`はこの条件を満たすprefixの個数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

よって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 切断点へ移す

`S_n <= 1`の上では、増分列`(U_1,...,U_n)`は切断点列

```text
0 <= S_1 <= S_2 <= ... <= S_n <= 1
```

を決める。逆に、このような切断点列を与えれば

```text
U_1 = S_1,
U_2 = S_2 - S_1,
...
U_n = S_n - S_{n-1}
```

で増分列が一意に戻る。この三角変換のJacobianは`1`である。

したがって、`P(S_n <= 1)`は「長さ1の棒の中に、順序付き切断点を`n`個置く」空間の体積である。

## 順序を外す

順序付き切断点の空間は、`[0,1]^n`を座標の大小順で分けた`n!`個の同じ体積の部屋のうち一つである。対角集合は体積`0`なので、

```text
P(S_n <= 1) = 1 / n!.
```

よって

```text
E[tau]
  = sum_{n >= 0} 1/n!
  = e.
```

この見方では、`e`は停止時刻の分布を先に求めて出るのではない。停止前prefixを「棒の切断点配置」と見て、順序を忘れた`n`次元立方体との比が`1/n!`になることから直接現れる。
