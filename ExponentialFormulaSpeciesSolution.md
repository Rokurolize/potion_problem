# 指数公式による有限集合解

この解では、停止前のprefixを「区間`[0,1]`上の有限集合を順序で読むもの」として数える。

独立な一様乱数を`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

と置く。

停止時刻は、境界を越える前に存在するprefixの数なので、

```text
tau = sum_{n>=0} 1_{S_n <= 1}
```

したがって

```text
E[tau] = sum_{n>=0} P(S_n <= 1).        (1)
```

## prefixを有限集合に忘れる

`S_n <= 1`なら、到着時刻

```text
t_i = S_i
```

は

```text
0 <= t_1 <= ... <= t_n <= 1
```

を満たす。増分`(U_1,...,U_n)`から到着時刻`(t_1,...,t_n)`への変換は三角行列で、Jacobianは`1`である。

つまり`P(S_n <= 1)`は、区間`[0,1]`から`n`個の点を選び、それを小さい順に並べた空間の体積である。

ここで順序をいったん忘れる。ほとんど至る所、相異なる`n`点の集合は`n!`通りの順序付けを持つ。したがって、順序付き単体の体積を`n!`倍すると、ラベル付きの`n`点配置の体積

```text
[0,1]^n
```

の体積`1`になる。

よって

```text
P(S_n <= 1) = 1/n!.
```

この式は、単体を直接積分して出したのではない。有限集合を順序付けると`n!`通りになる、という対称性だけで出ている。

## 全ての有限集合を足す

(1)に代入すると

```text
E[tau] = sum_{n>=0} 1/n!.
```

この和は、区間`[0,1]`上の有限集合全体の指数型母関数である。組合せ種の言葉で言えば、有限集合`SET`の分配関数は、台の大きさを`x`としたとき

```text
Z_SET(x) = exp(x)
```

である。

ここでは台の体積が`1`なので

```text
E[tau] = Z_SET(1) = exp(1) = e.
```

## この見方の芯

境界内に残るprefix列は、増分で見ると確率過程だが、到着時刻で見ると有限集合を順序に沿って読んだものになる。

`e`は、個々の停止確率の偶然の和ではない。有限集合を作る指数公式そのものが、停止前prefixの総量として現れている。
