# 畳み込み指数の質量解

この解では、各`n`の体積を別々に計算するかわりに、畳み込み代数の指数関数として一度に読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止時刻は、境界`1`をまだ越えていない部分和の個数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 一様分布を畳み込む

区間`[0,1]`上のLebesgue測度を`mu`と書く。これは一様乱数1個の分布である。すると`S_n`の分布は`mu^{*n}`であり、

```text
P(S_n <= 1) = mu^{*n}([0,1]).
```

ここで重要なのは、`[0,1]`上の質量だけを見るなら、各変数の上限`U_i<=1`は余分になることである。

実際、`u_i >= 0`かつ

```text
u_1 + ... + u_n <= 1
```

なら自動的に各`u_i <= 1`である。よって

```text
mu^{*n}([0,1])
= volume { (u_1,...,u_n) in R_{\ge 0}^n :
            u_1 + ... + u_n <= 1 }.
```

この領域は標準`n`単体なので、体積は`1/n!`である。

## 指数関数としてまとめる

畳み込み代数で形式的な指数を

```text
Exp_*(mu) = sum_{n >= 0} mu^{*n} / n!
```

と置く、という話ではない。ここで期待値に現れる係数はすでに

```text
mu^{*n}([0,1]) = 1/n!
```

であり、畳み込みを`[0,1]`で評価した瞬間に指数級数の係数が出ている。

つまり

```text
E[tau]
= sum_{n >= 0} mu^{*n}([0,1])
= sum_{n >= 0} 1/n!
= e.
```

## 見方

一様分布を何度も畳み込むと、普通はIrwin-Hall分布の折れ線多項式が現れる。しかしこの問題では境界がちょうど`1`なので、密度全体を見る必要がない。

境界以下の質量だけを読むと、畳み込みの複雑さは非負単体の体積に潰れ、全ての生存確率が指数関数の級数係数として並ぶ。停止時刻の期待値は、その係数列を足し上げたものとして`e`になる。
