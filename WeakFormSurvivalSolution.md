# weak form survival解

この解では、生存確率列を微分方程式の弱形式としてまとめる。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。`0<=x<=1`で

```text
A_n(x)=P(S_n<=x)
```

と置く。最後の増分で分けると

```text
A_n(x)=int_0^x A_{n-1}(t) dt,
A_0(x)=1.
```

したがって滑らかなテスト関数`phi`に対して

```text
int_0^1 A_n'(x) phi(x) dx = int_0^1 A_{n-1}(x) phi(x) dx
```

が成り立つ。境界値`A_n(0)=0`と合わせれば、これは強形式の

```text
A_n'=A_{n-1}
```

と同じであり、

```text
A_n(x)=x^n/n!
```

を与える。よって

```text
E[tau]=sum_{n>=0} A_n(1)=sum_{n>=0} 1/n! = e.
```

弱形式で見ると、必要なのは密度を点ごとに追うことではなく、停止境界内の生存質量が一段前の質量を微分として持つという関係である。
