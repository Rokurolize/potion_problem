# monomial ideal Hilbert解

この解では、停止条件を単項式イデアルの補集合のHilbert級数のように読む。

長さ`n`の履歴`(x_1,...,x_n)`が停止前であることは

```text
x_1+...+x_n<=1
```

である。したがって、停止していない履歴全体の大きさは、次数を合計値で測ったときの「次数1以下の単項式領域」の体積である。

通常のHilbert級数では、次数`m`の単項式数が係数になる。連続版では、次数`r`の層

```text
x_i>=0, x_1+...+x_n=r
```

の体積密度が係数の役をする。この層は`(n-1)`単体なので密度は

```text
r^{n-1}/(n-1)!
```

である。次数`0<=r<=1`を積分すると

```text
P(tau>n)=int_0^1 r^{n-1}/(n-1)! dr=1/n!
```

を得る。

よって

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

単項式を数えるHilbert級数の視点を連続体積に移すと、停止前履歴の各長さが指数級数の係数として並ぶ。
