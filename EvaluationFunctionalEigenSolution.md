# 評価汎関数の固有関係解

この解では、停止時刻の期待値そのものではなく、「残り距離`x`でまだ生きている列を合計する汎関数」を見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

いつも通り

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

ここから各項を直接計算しない。

## 生存列を作る作用素

`0<=x<=1`で

```text
a_n(x) = P(S_n <= x)
```

と置く。`a_0(x)=1`である。

一様性と独立性から

```text
a_{n+1}(x) = int_0^x a_n(x-u) du.
```

すなわち、作用素

```text
(Tf)(x) = int_0^x f(t) dt
```

を使えば

```text
a_n = T^n 1.
```

求めたい量は、点`1`で評価する汎関数`ev_1(f)=f(1)`を使って

```text
E[tau] = sum_{n >= 0} ev_1(T^n 1).
```

である。

## 評価を後ろへ押し戻す

ここで`ev_1 T`を計算する。

```text
ev_1(Tf) = (Tf)(1) = int_0^1 f(t) dt.
```

つまり、`T`を1回通してから点`1`で見ることは、区間全体を積分することに等しい。

さらに`T^n 1`は、`n`回積分された定数関数である。ここで係数を個別に追う代わりに、和

```text
g(x) = sum_{n >= 0} T^n 1(x)
```

を先に作る。`0<=x<=1`では単調な正項級数なので、積分との交換は単調収束で正当化できる。

この`g`は

```text
g = 1 + Tg
```

を満たす。両辺を微分すると

```text
g'(x) = g(x),
```

また`g(0)=1`である。したがって

```text
g(x) = exp(x).
```

最後に

```text
E[tau] = g(1) = e.
```

## 何を見ているか

この解の主役は、単体体積でも密度でもなく、`T`が「生存prefixを1段伸ばす作用素」になっている点である。

停止時刻の期待値は、`1, T1, T^2 1, ...`を点`1`で読む評価汎関数の総和である。その総和を先に閉じると、Resolvent方程式

```text
(I-T)g = 1
```

になり、点評価の値がそのまま指数関数の値として出る。
