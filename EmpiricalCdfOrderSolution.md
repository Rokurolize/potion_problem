# 経験分布関数順序解

この解では、停止前prefixを経験分布関数の昇順標本として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。`t_k=S_k`へ移すと、Jacobianは`1`であり、`S_n<=1`は

```text
0 <= t_1 <= ... <= t_n <= 1
```

である。これは区間`[0,1]`上の標本点を経験分布関数のジャンプ位置として昇順に並べたものに等しい。

ラベル付き標本`(x_1,...,x_n)`の経験分布関数は、点の順序を忘れる。対角を除けば同じ経験分布関数を持つラベル付き標本は`n!`個あるので、昇順ジャンプ位置の領域の体積は

```text
1/n!.
```

したがって

```text
P(S_n<=1)=1/n!,
E[tau]=sum_{n>=0}1/n! = e.
```

ここではIrwin-Hall分布を出さない。停止前prefixを経験分布関数のジャンプ列として見て、ラベルを忘れたときの`n!`だけを読む。
