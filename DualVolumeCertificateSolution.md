# dual volume certificate解

この解では、単体体積を直接積分せず、境界上で証明する双対certificateとして求める。

固定した`n`について

```text
A_n(x)=vol{u_i>=0, u_1+...+u_n<=x}
```

とおく。停止時刻には

```text
E[tau]=sum_{n>=0} A_n(1)
```

が必要である。

領域

```text
Delta_n(x)={u_i>=0, u_1+...+u_n<=x}
```

上で線形関数

```text
F(u)=u_1+...+u_n
```

を考える。ベクトル場`u/n`の発散は`1`なので、発散定理から体積は境界流束で表せる。座標面`u_i=0`では流束が消え、残るのは斜面`F(u)=x`だけである。斜面上の法線成分は`x/n`に等しいので

```text
A_n(x)=x/n * area_slope(Delta_n(x)).
```

同じ斜面を`u_n=x-u_1-...-u_{n-1}`で媒介すると、法線面積と法線成分の積はちょうど

```text
x/n * area_slope = x/n * n A_{n-1}(x)/x = x A_{n-1}(x)/n
```

を与える。つまり

```text
A_n(x)=x A_{n-1}(x)/n,  A_0(x)=1.
```

したがって

```text
A_n(1)=1/n!
```

であり

```text
E[tau]=sum_{n>=0}A_n(1)=sum_{n>=0}1/n!=e.
```

この証明は、体積そのものではなく「発散が1で、座標面では消え、斜面だけが寄与する」双対certificateで階数を一つ落としている。
