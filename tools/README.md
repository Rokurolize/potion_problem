# Potion Problem - Tools

Support tools and utilities for the Potion Problem project.

## Contents

### api_database/
SQLite database for efficient API discovery and management:
- `mathlib_apis.db` - Pre-verified mathlib4 APIs
- `api_tools.sh` - Shell interface for API operations
- Migration scripts for database updates

### python/
Python configuration and potential future scripts:
- `pyproject.toml` - Python project configuration
- `uv.lock` - Dependency lock file

### logs/
Build logs and debugging outputs

## Usage

See `api_database/README.md` for detailed API database usage instructions.
