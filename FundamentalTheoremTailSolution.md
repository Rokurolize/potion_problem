# fundamental theorem tail解

この解では、微積分の基本定理で尾確率の再帰を解く。

残容量`c`から長さ`n`だけ停止しない確率を

```text
F_n(c)=P(X_1+...+X_n<=c)
```

とする。`0<=c<=1`では

```text
F_n(c)=int_0^c F_{n-1}(c-u) du.
```

変数変換により

```text
F_n(c)=int_0^c F_{n-1}(v) dv.
```

したがって微積分の基本定理から

```text
F_n'(c)=F_{n-1}(c),  F_n(0)=0.
```

`F_0(c)=1`なので帰納的に

```text
F_n(c)=c^n/n!.
```

よって

```text
P(tau>n)=F_n(1)=1/n!,
E[tau]=sum_{n>=0}1/n!=e.
```

尾確率を残容量の関数にすると、階乗分母は微積分の基本定理を繰り返しただけで出る。
