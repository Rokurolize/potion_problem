# 因果芽畳み込み解

待ち時間の密度を

```text
f(t) = 1_{0 <= t <= 1}
```

とする。`n` 回目までの和 `S_n` の密度は `f^{*n}` であり、

```text
P(S_n <= 1) = int_0^1 f^{*n}(t) dt
```

である。

ここで重要なのは、`f` の全体ではなく、時刻1以前から見える「因果的な芽」だけである。`0 <= t <= 1` の範囲では

```text
f(t) = H(t)
```

と同じである。ただし `H(t) = 1_{t >= 0}` とする。

さらに、畳み込みは因果的である。つまり、`f` と `H` が `[0, 1]` で一致していれば、`f^{*n}` と `H^{*n}` も `[0, 1]` で一致する。実際、`t <= 1` に寄与する畳み込み積分では、各待ち時間は非負で合計が `t` 以下なので、どの成分も自動的に1以下である。したがって上端の切断 `t <= 1` はまだ一度も働かない。

よって

```text
f^{*n}(t) = H^{*n}(t) = t^{n-1}/(n-1)!      (0 <= t <= 1, n >= 1)
```

である。したがって

```text
P(S_n <= 1)
  = int_0^1 t^{n-1}/(n-1)! dt
  = 1/n!
```

を得る。`n = 0` は空和なので寄与1である。

到着数は

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

だから、単調収束により

```text
E[tau]
  = sum_{n >= 0} P(S_n <= 1)
  = sum_{n >= 0} 1/n!
  = e
```

となる。

この見方では、一様分布の右端1は「時刻1までの計算」には影を落とさない。上端の切断が効く直前まで、密度は半直線の単位密度と同じ因果芽を持ち、その畳み込み代数だけで指数級数が現れる。
