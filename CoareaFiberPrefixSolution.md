# coarea fiber prefix解

この解では、和写像のfiberで停止前prefix領域を分解する。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。固定した`n`で、和写像

```text
q(u_1,...,u_n)=u_1+...+u_n
```

を考える。停止前領域は`0<=q<=1`の部分である。`q=s`のfiberは一つ低い単体で、その測度は

```text
s^{n-1}/(n-1)!
```

である。coareaの形で積分すると

```text
P(S_n<=1)=int_0^1 s^{n-1}/(n-1)! ds = 1/n!
```

ただし`n=0`は空prefixで重み`1`である。よって

```text
E[tau]=sum_{n>=0} P(S_n<=1)
      =sum_{n>=0} 1/n!
      =e.
```

この見方では、単体そのものではなく、和写像のfiberを積み上げて停止前領域を再構成している。
