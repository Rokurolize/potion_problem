# 生存列のBorel変換解

この解では、`P(S_n<=1)`を個別に積分せず、部分和が作る生存列をBorel変換として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止時刻は、境界`1`以下に残っているprefixの個数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 生存条件をBorel係数にする

`S_n<=1`は

```text
u_1 + ... + u_n <= 1,
0 <= u_i <= 1
```

である。和が`1`以下なら各`u_i<=1`は自動的に満たされるので、この確率は非負単体

```text
Delta_n = { (u_1,...,u_n) : u_i>=0, u_1+...+u_n<=1 }
```

の体積である。

ここで、`Delta_n`を直接計算するかわりに、単体の全次元を同時に含む級数

```text
B(z) = sum_{n >= 0} volume(Delta_n) z^n
```

を考える。求めたい期待値は`B(1)`である。

## Borel変換として閉じる

`n`次元単体の体積は、次の再帰だけで決まる。

```text
volume(Delta_0) = 1,
volume(Delta_{n+1}) = int_0^1 volume({u_1+...+u_n <= 1-t}) dt.
```

右辺で残り容量を`x`にすると、同じ単体を大きさ`x`に縮めたものになる。`n`次元体積は縮尺`x^n`で変わるので

```text
volume({u_1+...+u_n <= x}) = x^n volume(Delta_n).
```

したがって

```text
volume(Delta_{n+1})
= volume(Delta_n) int_0^1 x^n dx
= volume(Delta_n)/(n+1).
```

帰納的に

```text
volume(Delta_n) = 1/n!.
```

ここで出た`1/n!`は、通常の指数級数の係数である。つまり生存確率列は、定数列`1,1,1,...`のBorel変換

```text
sum_{n >= 0} z^n  ->  sum_{n >= 0} z^n/n!
```

として現れている。

ゆえに

```text
B(z) = sum_{n >= 0} z^n/n! = exp(z).
```

求める値は

```text
E[tau] = B(1) = e.
```

## この見方の芯

この証明は、停止時刻の分布を求めない。生存prefixの列全体を「次元を1つ上げるたびに`1/(n+1)`が掛かる」Borel変換の係数列として見る。

一様性は非負単体の体積として現れ、境界`1`以下では上限制約が消える。すると、各次元の生存確率が階乗で割られるだけの列になり、その全和が指数関数として`e`になる。
