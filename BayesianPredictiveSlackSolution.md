# Bayesian予測余白解

この解では、生存しているprefixに残る余白を、対称なDirichlet事後分布の予測平均として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。`p_n=P(S_n<=1)`と書けば

```text
E[tau]=sum_{n>=0}p_n.
```

`S_{n-1}<=1`で条件付け、余白

```text
R=1-S_{n-1}
```

を加えると、`n`個の非負成分

```text
U_1,...,U_{n-1},R
```

は和`1`の単体上で対称である。これはDirichlet`(1,...,1)`の一様事後分布として読める。次の一口`U_n`が生き残る予測確率は余白の事後平均なので

```text
P(S_n<=1 | S_{n-1}<=1)=E[R | S_{n-1}<=1]=1/n.
```

したがって

```text
p_n=p_{n-1}/n,  p_0=1,
```

であり

```text
p_n=1/n!.
```

よって

```text
E[tau]=sum_{n>=0}1/n!=e.
```

ここでは階乗を単体体積から直接出さず、余白成分の予測平均が毎段`1/n`になることだけを使う。
