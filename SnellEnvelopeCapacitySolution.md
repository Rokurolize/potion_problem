# Snell envelope capacity解

この解では、残り容量に対する停止前報酬の包絡関数として期待値を出す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。残り容量が`x`の状態から、停止前に得られる単位報酬の期待総量を`V(x)`と書く。いま最適停止問題を解くわけではないが、停止境界で切られる報酬過程の包絡関数として`V`を見る。

空prefixの報酬を一つ数え、次の増分`u`が`0<=u<=x`なら残り容量は`x-u`なので

```text
V(x)=1+int_0^x V(x-u) du.
```

変数を替えると

```text
V(x)=1+int_0^x V(t) dt.
```

したがって

```text
V'(x)=V(x),  V(0)=1,
```

より

```text
V(x)=e^x.
```

初期容量は`1`だから

```text
E[tau]=V(1)=e.
```

Snell envelopeの語を借りるなら、ここでの包絡は選択の最適性ではなく、停止境界内に残るprefix報酬を全部包む関数である。
