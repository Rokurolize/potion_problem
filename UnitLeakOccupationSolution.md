# 単位漏れ占有量解

この解では、停止前のprefixを「残り容量を状態とする鎖の占有量」として見て、境界からの漏れ率を正規化する。

独立な一様乱数`U_1,U_2,...`について

```text
R_0 = 1,
R_n = 1 - (U_1 + ... + U_n),
tau = min { n >= 1 : R_n < 0 }
```

とする。`R_n >= 0`である段数が、停止前prefixの個数である。

```text
tau = sum_{n >= 0} 1_{R_n >= 0}.
```

残り容量が`r`のとき、次の一口でまだ生きているためには`0 <= U <= r`でなければならない。したがって、非負領域上の遷移作用素は

```text
(Kf)(r) = int_0^r f(r-u) du = int_0^r f(s) ds.
```

である。

## 占有量を未知量にする

初期状態`1`から出発したときの総占有測度を`G`とする。任意のテスト関数`f`に対して

```text
int f(r) G(dr)
  = E[ sum_{n >= 0} f(R_n) 1_{R_n >= 0} ].
```

特に求めたい値は

```text
E[tau] = int 1 G(dr).
```

占有測度は、初期点の質量と、生きている状態から一回ジャンプして入ってくる質量の和である。

```text
G = delta_1 + K^* G.
```

ここで`K^*G`は、各点`r`から`[0,r]`へ一様密度`1`で流す測度である。

## 右から見た密度

点`1`の原子を除き、`0 <= x < 1`で`G`の密度を`g(x)`と書く。状態`x`に流れ込むには、直前の残り容量`r`が`x <= r <= 1`にあり、かつ`U=r-x`であればよい。一様密度は`1`なので、

```text
g(x) = G([x,1]).
```

右辺には`x`以上にある総占有質量がそのまま出る。

そこで

```text
H(x) = G([x,1])
```

と置く。`H`は右側に残る占有量であり、上の式から

```text
H'(x) = -g(x) = -H(x)        (0 <= x < 1)
```

を満たす。

一方、`x=1`では初期点`R_0=1`の原子が一つだけあるので、

```text
H(1) = 1.
```

したがって

```text
H(x) = exp(1 - x).
```

求める総占有量は

```text
E[tau] = G([0,1]) = H(0) = e.
```

この証明では、`P(S_n <= 1)`を個別に求めない。残り容量の鎖がどれだけ長く非負領域に滞在するかを測度として集め、右側に残っている占有量がその点への流入密度そのものになる、という保存則だけで指数関数を得る。
