# Feynman図式真空解

この解では、停止前prefixを一点挿入の図式展開として読み、真空図式が非相互作用で指数化することを見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めれば

```text
E[tau]=sum_{n>=0} P(S_n <= 1).
```

増分列を累積時刻

```text
t_i=S_i
```

へ移す。Jacobianは`1`で、`S_n <= 1`は

```text
0 <= t_1 <= ... <= t_n <= 1
```

になる。したがって長さ`n`のprefixは、区間`[0,1]`に`n`個の順序付き挿入点を置く図式である。

活動度`z`を付けた真空生成汎関数を

```text
Z(z)=sum_{n>=0} z^n int_{0<=t_1<=...<=t_n<=1} dt_1...dt_n
```

とする。相互作用線はなく、許される頂点は一点挿入だけである。連結真空図式は一点図式だけなので、連結生成汎関数は

```text
W(z)=z int_0^1 dt=z.
```

非連結真空図式全体は連結図式の指数で

```text
Z(z)=exp(W(z))=exp(z).
```

欲しい期待値は`z=1`で

```text
E[tau]=Z(1)=e.
```

この証明では、各`n`の単体体積を個別に出さない。停止前prefixを挿入点図式としてまとめ、相互作用のない真空図式が一点連結図式の指数になることを使う。`e`は、その真空汎関数の値である。
