# Hawkesクラスター平均解

この解では、停止前prefixを自己励起点過程のクラスターとして読む。

時刻`0`に祖先点を1つ置く。各点は、自分の発生後`0 <= u <= 1`の範囲で、密度`1`の子点を生むとする。子点も同じ規則で子点を生む。これは再生核

```text
phi(u) = 1_{0 <= u <= 1}
```

を持つHawkesクラスターの枝分かれ表示である。

## prefixをクラスターの系譜にする

祖先からある子孫までの辺の長さを

```text
X_1, X_2, ..., X_n
```

と書く。各辺の長さは、期待測度として区間`[0,1]`上のLebesgue測度を持つ。したがって、深さ`n`の子孫が時刻`1`以前に現れる期待数は

```text
int_{x_i >= 0, x_i <= 1, x_1+...+x_n <= 1} dx_1 ... dx_n.
```

これは、元の問題の

```text
P(X_1 + ... + X_n <= 1)
```

と同じ量である。よって、時刻`1`までに見えるクラスターの期待総点数は

```text
sum_{n >= 0} P(S_n <= 1).
```

右辺は停止時刻の尾和なので、

```text
E[tau] = E[# {クラスター点 at time <= 1}].
```

## 平均クラスター方程式

`M(t)`を、祖先を含めて時刻`t`までに現れる期待クラスター点数とする。`0 <= t <= 1`では、各既存点は未来の微小区間`(s, s+ds]`へ密度`1`で子を送れる。したがって、境界`t`を少し右へ動かしたときの新しい期待点数は、そこまでの期待点数そのものに比例する。

積分形で書くと

```text
M(t) = 1 + int_0^t M(t-u) du.
```

変数を入れ替えて

```text
M(t) = 1 + int_0^t M(v) dv.
```

微分すれば

```text
M'(t) = M(t),   M(0)=1.
```

よって

```text
M(t) = exp(t).
```

したがって

```text
E[tau] = M(1) = e.
```

この証明では、各深さの単体体積を別々に計算しない。停止前の飲用履歴を、Hawkesクラスターの祖先から子孫への一本の系譜として読み、全系譜の期待個数が自己励起の平均方程式で指数増殖することだけを見る。
