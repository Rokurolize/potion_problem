# Potion Problem - Visualization

Interactive TypeScript/WebGL visualizations for the Potion Problem mathematical concepts.

## Features

- 3D simplex visualization of probability distributions
- Interactive potion simulation
- PMF landscape rendering
- Telescoping series animation
- Emergence visualization of e

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Architecture

- `src/core/` - Core math engine and WebGL renderer
- `src/math/` - Linear algebra utilities
- `src/scenes/` - Individual visualization scenes

For mathematical background, see the [docs](../docs) directory.
