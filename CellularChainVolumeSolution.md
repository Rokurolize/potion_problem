# セル鎖体積解

この解では、停止前prefixを単体のセル鎖として並べ、境界写像ではなくセル体積の全和を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めると

```text
E[tau] = sum_{n>=0} P(S_n <= 1)
```

である。`S_n <= 1`は`n`次元標準単体

```text
Delta_n = { (u_1,...,u_n) : u_i >= 0, u_1+...+u_n <= 1 }
```

そのものなので、各次数のセルの重みは

```text
vol(Delta_n)=1/n!.
```

停止前prefix全体をセル鎖

```text
C = sum_{n>=0} Delta_n
```

として見れば、期待値はその体積増大列の全質量である。

```text
E[tau]=sum_{n>=0} vol(Delta_n)
      =sum_{n>=0} 1/n!
      =e.
```

この見方では、隣り合う単体の境界キャンセルは使わない。停止条件が作る全次数のセルをそのまま足すと指数級数になる、というだけで十分である。
