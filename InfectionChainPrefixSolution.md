# infection chain prefix解

この解では、停止前prefixを感染がまだ消えていない連鎖として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。初期感染力を`1`とし、各感染段階で`U_i`だけ感染力を消費すると考える。長さ`n`の連鎖がまだ続く条件は

```text
S_n<=1
```

である。停止時刻は、感染力が尽きる前に存在するprefixの個数なので

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

固定した`n`では、感染連鎖が続く増分列の領域は

```text
{u_i>=0, u_1+...+u_n<=1}
```

であり、体積は`1/n!`である。したがって

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

感染連鎖の比喩では、必要なのは枝分かれではなく、一本の連鎖が容量内で続く全prefixの質量である。
