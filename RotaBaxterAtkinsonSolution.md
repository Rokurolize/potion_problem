# Rota-Baxter再帰解

この解では、媚薬問題を「積分作用素が作る再帰」として解く。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

残り距離を`x`として、そこから境界を越えるまでの平均飲用回数を`m(x)`と書く。求めたいのは`m(1)`である。

## 一歩後の再帰

残り距離が`x`のとき、次の一口は必ず1回数える。その後、飲んだ量`u`が`x`以下なら残り距離は`x-u`になり、`u>x`なら終了する。

よって

```text
m(x) = 1 + int_0^x m(x-u) du.
```

変数を入れ替えると

```text
m(x) = 1 + int_0^x m(t) dt.        (1)
```

ここで積分作用素

```text
(R f)(x) = int_0^x f(t) dt
```

を置けば、(1)は

```text
m = 1 + R(m).                       (2)
```

である。

## Atkinson型に解く

作用素`R`はVolterra型なので、(2)を反復しても過去方向へしか戻らない。

```text
m = 1 + R(1) + R^2(1) + R^3(1) + ...
```

ただし、この展開を最後まで足し切る必要はない。両辺を微分すれば、`R`の定義から

```text
m'(x) = m(x),
m(0) = 1.
```

したがって

```text
m(x) = exp(x).
```

とくに

```text
E[tau] = m(1) = exp(1) = e.
```

## 何を使っているか

この証明の核は、分布を先に求めることではない。停止するまでの平均回数そのものを未知関数にして、積分作用素`R`の固定点

```text
m = 1 + R(m)
```

として解く。

`R`は積分の上端だけを見る作用素なので、再帰は微分方程式`m'=m`に潰れる。答えの`e`は、単体体積の級数を後で足したものではなく、このRota-Baxter型の再帰が持つ指数解として現れる。
