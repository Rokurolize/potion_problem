# Beta余白剥がし解

この解では、生存条件

```text
U_1 + ... + U_n <= 1
```

を、最後の余白を一枚ずつ剥がすBeta積分として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。正整数値の停止時刻なので

```text
E[tau] = sum_{n >= 0} P(tau > n)
       = sum_{n >= 0} P(S_n <= 1)
```

である。空和は`S_0=0`とする。

## 余白を一成分として加える

`S_n <= 1`のとき、余白

```text
R_n = 1 - S_n
```

を加えると

```text
U_1 + ... + U_n + R_n = 1,
U_i >= 0,
R_n >= 0
```

となる。つまり、生存している`n`口は、`n+1`個の非負成分が合計`1`になる単体の、最後の余白成分を忘れた姿である。

この単体の大きさを直接覚えておくかわりに、余白を最後から剥がす。`n+1`成分のうち最後の余白が`r`である断面では、残り`n`成分の合計は`1-r`である。大きさは同次性により

```text
(1-r)^{n-1}
```

倍になる。したがって、正規化定数はBeta積分

```text
int_0^1 (1-r)^{n-1} dr = 1/n
```

で一段落ちる。

同じ剥がしを繰り返すと

```text
P(S_n <= 1)
  = int_{r_n >= 0} int_{r_{n-1} >= 0} ... int_{r_1 >= 0}
      1_{r_1 + ... + r_n <= 1} dr_1 ... dr_n
  = 1/n!.
```

ここで各段の係数は

```text
1/n, 1/(n-1), ..., 1/1
```

であり、余白を剥がすたびにBeta積分が一つ現れる。

## 期待値

よって

```text
E[tau]
  = sum_{n >= 0} P(S_n <= 1)
  = sum_{n >= 0} 1/n!
  = e.
```

この証明では、`n`次元単体の体積公式をそのまま引用しない。余白を一成分として足し、最後の余白を剥がすBeta積分を繰り返すことで、階乗係数を一段ずつ出す。
