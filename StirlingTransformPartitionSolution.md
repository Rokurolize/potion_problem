# Stirling変換分割解

この解では、停止前prefixを有限配置の分割で見て、Stirling変換が一点ブロックだけに潰れることを使う。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めて

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

累積時刻へ移すと、長さ`n`のprefixは`[0,1]`上の順序付き`n`点配置である。分割のStirling変換で全配置を連結ブロックへ分解すると、この問題には相互作用がないため、連結ブロックは一点だけで十分である。

したがって活動度`z`つきの全配置母関数は

```text
Z(z)=exp(z).
```

`z=1`で

```text
E[tau]=Z(1)=e.
```

この証明では、単体体積を個別に求めない。Stirling変換で配置分割を連結ブロックへ移し、一点ブロックだけが残ることを見る。`e`は、その逆変換で戻った全配置量である。
