# test function prefix functional解

この解では、停止前prefixをテスト関数に作用する線形汎関数としてまとめる。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。容量`x`までの停止前prefix総量を

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

と書く。最後の増分で一段剥がすと、任意の`x<=1`について

```text
F(x)=1+int_0^x F(t) dt.
```

この等式をテスト関数で評価する線形汎関数として見ても、同じ関係が成り立つ。すなわち「一文字を足す」作用は積分作用素であり、停止前prefix全体は

```text
(I - integral)^{-1} 1
```

の値である。微分すれば

```text
F'(x)=F(x),  F(0)=1,
```

だから

```text
F(x)=e^x.
```

したがって

```text
E[tau]=F(1)=e.
```

この見方では、prefixを個別に列挙するのではなく、積分作用素の逆を定数テスト関数に作用させた値として一括で読む。
