# 時間順序指数解

この解では、媚薬問題の生存prefixを、時間順序指数のDyson展開として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止時刻は、境界`1`をまだ越えていないprefixの数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 生存prefixは時間順序列である

`n >= 1`について累積和

```text
t_i = S_i
```

へ変数変換する。Jacobianは`1`であり、`S_n <= 1`は

```text
0 < t_1 < ... < t_n <= 1
```

という時間順序列になる。よって

```text
P(S_n <= 1)
= int_{0<t_1<...<t_n<=1} dt_1 ... dt_n.
```

これは、定数Hamiltonian`A(t)=1`の時間順序積分の`n`次項である。

## Dyson展開を一度に閉じる

時間順序指数

```text
T exp( int_0^x A(t) dt )
```

は、Dyson展開により

```text
sum_{n >= 0} int_{0<t_1<...<t_n<=x} A(t_n) ... A(t_1) dt_1 ... dt_n
```

である。ここでは`A(t)=1`なので、この展開は

```text
Y(x) = sum_{n >= 0} int_{0<t_1<...<t_n<=x} dt_1 ... dt_n
```

そのものになる。

時間順序指数は方程式

```text
Y'(x) = A(x) Y(x),
Y(0) = 1
```

を満たす。今回`A(x)=1`だから

```text
Y'(x) = Y(x),
Y(0) = 1.
```

したがって

```text
Y(x) = exp(x).
```

求める期待値は`x=1`での全Dyson項の和なので、

```text
E[tau] = Y(1) = exp(1) = e.
```

## 何が見えているか

この証明では、`P(tau=n)`を先に求めない。飲んだ量の累積和を時刻列へ変換し、生存prefix全体を時間順序積分の全次数としてまとめる。

答えの`e`は、単体体積の級数を最後に足したものではなく、定数生成子`A=1`の時間順序指数が満たす微分方程式から出る。
