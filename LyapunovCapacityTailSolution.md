# Lyapunov capacity tail解

この解では、残り容量をLyapunov関数として、非負に保たれる尾確率を足す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。Lyapunov関数として

```text
V_n=1-S_n
```

を取る。停止前である条件は

```text
V_n>=0.
```

したがって

```text
E[tau]=sum_{n>=0} P(V_n>=0).
```

固定した`n`で`V_n>=0`となる領域は標準単体

```text
{u_i>=0, u_1+...+u_n<=1}
```

だから

```text
P(V_n>=0)=1/n!.
```

よって

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

ここでLyapunov関数は安定性を証明するためではなく、停止境界までの残り容量を一つの非負性条件として表すために使っている。
