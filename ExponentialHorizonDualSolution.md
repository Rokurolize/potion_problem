# 指数地平線双対解

この解では、境界`1`の手前にあるprefixを直接数えず、同じrenewal測度を指数地平線で試験してから、随伴方程式で境界値へ戻す。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = R([0,1])
```

である。ただし

```text
R = sum_{n >= 0} law(S_n)
```

は部分和全体のrenewal測度である。

## 半直線では指数地平線で一気に読める

独立な指数分布`T`を一つ置き、平均個数

```text
E[ sum_{n >= 0} 1_{S_n <= T} ]
```

を考える。`T`は独立なので

```text
P(S_n <= T | S_n) = exp(-S_n).
```

したがってこの平均個数は

```text
sum_{n >= 0} E[exp(-S_n)].
```

一様乱数`U`について

```text
E[exp(-U)] = int_0^1 exp(-u) du = 1 - exp(-1).
```

独立性より

```text
sum_{n >= 0} E[exp(-S_n)]
= sum_{n >= 0} (1 - exp(-1))^n
= exp(1).
```

ここまでで分かったのは、renewal測度の指数試験

```text
int_0^infty exp(-x) R(dx)
```

が`e`だということだ。

## なぜ境界`1`の質量になるか

残る問題は、指数地平線で測った量が、欲しい`R([0,1])`と同じ値を持つ理由である。ここでrenewal方程式

```text
R = delta_0 + R * 1_{0<=u<=1}du
```

を、区間`[0,1]`へ戻す。

分布関数

```text
F(x) = R([0,x])
```

は`0<=x<=1`で

```text
F(x) = 1 + int_0^x F(t) dt.
```

この式を解く必要はない。両辺を

```text
psi(x) = exp(-x)
```

で重み付けして`[0,1]`に積分する。右辺の二重積分を入れ替えると

```text
int_0^1 exp(-x) F(x) dx
= 1 - exp(-1)
  + int_0^1 F(t) int_t^1 exp(-x) dx dt.
```

内側の積分は

```text
int_t^1 exp(-x) dx = exp(-t) - exp(-1).
```

したがって両辺の`int exp(-x)F(x)dx`が消えて

```text
exp(-1) int_0^1 F(t) dt = 1 - exp(-1).
```

また`F(1)=1+int_0^1 F(t)dt`だから

```text
F(1)=exp(1).
```

つまり、半直線で指数地平線が読んだ総量と、境界`1`までのrenewal質量が同じ値に戻る。

## 期待値に戻す

求める期待値は

```text
E[tau] = R([0,1]) = F(1).
```

上で`F(1)=exp(1)`を得たので

```text
E[tau] = e.
```

## この見方の芯

この証明では、`P(S_n<=1)`を個別に測らない。まず指数地平線で半直線上のrenewal測度を一気に試験し、独立性だけで幾何級数を読む。

その後、renewal方程式の随伴側で指数重みを使うと、内部の積分が相殺され、指数地平線の値が境界`1`の質量として戻る。`e`は、固定境界と指数地平線が一様増分の長さ`1`で双対になる正規化値として現れる。
