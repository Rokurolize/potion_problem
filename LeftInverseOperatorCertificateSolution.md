# 左逆作用素証明解

この解では、各長さの生存確率を出さない。prefixを一つ伸ばす積分作用素と、それを一つ戻す微分作用素を置き、全prefix和が微分作用素の固有関数であることを証明する。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

境界を`x`に動かし、

```text
A_n(x) = P(S_n <= x)        (0 <= x <= 1)
```

と置く。`A_0(x)=1`である。元の期待値は

```text
E[tau] = sum_{n>=0} A_n(1).
```

## prefixを伸ばす作用素

`0<=x<=1`では、生存している限り各増分の上端`1`はまだ効かない。したがって、一口追加する操作は

```text
(Cf)(x) = int_0^x f(t) dt
```

で表せる。実際、

```text
A_{n+1} = C A_n.
```

よって

```text
A_n = C^n 1.
```

全prefix和を

```text
G(x) = sum_{n>=0} C^n 1 (x)
```

と書く。求めたい値は`G(1)`である。

## 一つ戻す作用素

微分作用素

```text
D = d/dx
```

を考える。積分作用素`C`に対して

```text
D C f = f
```

が成り立つ。つまり`D`は「一口追加」を左から消す作用素である。

一方、`C f`は必ず`x=0`で`0`になる。したがって、全prefix和は

```text
G = 1 + C G.        (1)
```

を満たすだけでなく、`G(0)=1`も持つ。

(1)から

```text
G - 1 = C G.
```

両辺に`D`をかけると、左逆性`D C = I`により

```text
G' = G.
```

初期値は空prefixだけが残るので

```text
G(0)=1.
```

したがって

```text
G(x)=exp(x).
```

元の境界では

```text
E[tau] = G(1) = e.
```

## この見方の芯

この証明は、`P(S_n<=1)`を計算しない。prefixを一つ伸ばす操作を`C`として先に固定し、全prefix和を

```text
G = 1 + C G
```

という作用素方程式に閉じ込める。

一様性は、追加操作の核が単なる積分`C`になる点に現れる。独立性は、何段進んでも同じ`C`を反復できる点に現れる。答えの`e`は、微分作用素`D`が`C`を左から消すため、全prefix和が`D G = G`という固有関数方程式へ落ちることから出る。
