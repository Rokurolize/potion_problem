# 積分作用素レゾルベント解

この解では、生存prefixの期待個数を、積分作用素のNeumann級数として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止時刻は、境界`1`を越える前に存在するprefixの個数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 1歩足す操作を作用素にする

`0 <= x <= 1`で、残り容量`x`の中に`n`個の一様乱数を収められる確率を

```text
p_n(x) = P(U_1 + ... + U_n <= x)
```

とおく。`p_0(x)=1`である。

`n+1`個目へ進むには、最初の飲量`u`を選び、残り`x-u`の中に`n`個が収まればよい。`U_1`の密度は`[0,1]`上で`1`なので、`0 <= x <= 1`では

```text
p_{n+1}(x) = int_0^x p_n(x-u) du.
```

変数を入れ替えると

```text
p_{n+1}(x) = int_0^x p_n(t) dt.
```

ここで積分作用素`J`を

```text
(Jf)(x) = int_0^x f(t) dt
```

と書けば、

```text
p_n = J^n 1.
```

## 期待値はレゾルベントである

残り容量`x`から見た生存prefixの期待個数は

```text
G(x) = sum_{n >= 0} p_n(x)
```

だから

```text
G = sum_{n >= 0} J^n 1 = (I - J)^{-1} 1.
```

この等式は形式的な名前だけではない。実際に

```text
J^n 1 = x^n / n!
```

が帰納的に成り立つ。よって

```text
G(x) = sum_{n >= 0} x^n / n! = exp(x).
```

求める期待値は`x=1`で

```text
E[tau] = G(1) = e.
```

## 何を圧縮しているか

この証明は、各`P(S_n<=1)`を個別に単体体積として計算してから足すのではない。乱数を1個追加する操作そのものを`J`として切り出し、停止時刻の期待値を「作用素の全反復を足したレゾルベント」として一度に読む。

一様性は`J`の核が定数`1`になる点に、独立性は同じ作用素を何度も反復できる点に現れる。
