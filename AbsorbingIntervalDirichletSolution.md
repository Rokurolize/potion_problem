# 吸収区間の境界値解

この解では、問題を「長さ`r`の区間から一様な歩幅で右端を越えるまでの平均歩数」として見る。

独立な一様乱数を`U_1,U_2,...`とし、残り距離`r`から始めたときの停止時刻を

```text
T(r) = min { n >= 1 : U_1 + ... + U_n > r }
```

と置く。元の問題は`r=1`の場合なので、求めたい値は

```text
m(1),   m(r) = E[T(r)]
```

である。

## 端での値

`r=0`なら、一口目で必ず`0`を越える。したがって

```text
m(0) = 1.
```

これは境界条件である。

## 内部での方程式

`0 < r <= 1`とする。最初の一口`U`は必ず1回として数える。

- `U > r`なら、その一口で吸収される。
- `U <= r`なら、残り距離は`r-U`になり、そこから同じ問題が続く。

よって

```text
m(r) = 1 + int_0^r m(r-u) du.        (1)
```

右辺の積分範囲が`0`から`r`で止まるのは、`U>r`のときは追加の未来がないからである。一様分布の密度が`1`なので、重みは付かない。

変数を入れ替えると

```text
m(r) = 1 + int_0^r m(v) dv.
```

したがって`m`は境界値問題

```text
m'(r) = m(r),   m(0) = 1
```

を満たす。

## 解

この境界値問題の解は一意に

```text
m(r) = exp(r)
```

である。ゆえに元の問題では

```text
E[tau] = m(1) = exp(1) = e.
```

## なぜこれで止まる回数を数えているか

この方程式は、`S_n <= 1`の体積を個別に求めていない。残り距離`r`ごとに、吸収までの平均歩数を未知関数として置き、最初の一口だけで方程式を閉じている。

境界`r=0`では平均歩数が`1`。内部では、最初の一口の後に残った区間の平均を足す。この二つだけから指数関数が強制され、`r=1`で`e`が出る。
