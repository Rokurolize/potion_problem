# shuffle指数解

この解では、生存prefixの総量を、テンソル代数のshuffle指数として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止時刻は、まだ境界を越えていないprefixの個数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

よって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 生存prefixを語にする

`n >= 1`で累積和

```text
t_i = S_i
```

へ変数変換する。Jacobianは`1`で、`S_n <= 1`は

```text
0 < t_1 < ... < t_n <= 1
```

という順序付きの語になる。したがって

```text
P(S_n <= 1)
= int_{0<t_1<...<t_n<=1} dt_1 ... dt_n.
```

これは区間`[0,1]`上の1文字形式`a = dt`の反復積分

```text
int a^n
```

である。

## shuffle指数を評価する

反復積分にはshuffle積がある。1文字`a`について、同じ積分区間の中で

```text
(int a)^n = n! * int a^n
```

となる。理由は、`n`個の独立な時刻を並べ替える`n!`通りの順序領域が、単位立方体を分割するからである。

ここで

```text
int_0^1 a = int_0^1 dt = 1.
```

したがって

```text
int a^n = 1 / n!.
```

しかし、この証明で本当に使いたいのは各次数の値ではなく、全次数をまとめたshuffle指数である。

```text
Exp_shuffle(a)
= sum_{n >= 0} a^n
```

を反復積分で評価すると、shuffle指数の通常の指数法則により

```text
int Exp_shuffle(a)
= sum_{n >= 0} int a^n
= sum_{n >= 0} (int_0^1 a)^n / n!
= exp(1).
```

上で`int a^n`はちょうど`P(S_n <= 1)`だったので、

```text
E[tau]
= int Exp_shuffle(a)
= exp(1)
= e.
```

## 見方

この解法では、飲む量の列をそのまま足し続けるのではなく、累積時刻の順序語として見る。生存しているすべての長さの語を足すと、1文字形式`dt`のshuffle指数になる。

答えの`e`は、単に最後に級数を足した結果ではなく、反復積分をshuffle指数として評価したときの指数法則として現れる。
