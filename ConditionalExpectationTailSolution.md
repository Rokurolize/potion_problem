# conditional expectation tail解

この解では、停止時刻を自然なフィルトレーションで条件付き期待値に分解する。

`F_n=sigma(X_1,...,X_n)`とする。事象`{tau>n}`は`F_n`可測で、

```text
1_{tau>n}=1_{X_1+...+X_n<=1}
```

である。したがって

```text
P(tau>n)=E[1_{tau>n}]
        =E[E[1_{X_1+...+X_n<=1} | F_n]].
```

内側の条件付き期待値は指示関数そのものなので、独立一様分布の密度1で積分すればよい。

```text
P(tau>n)=int_{x_i>=0, x_1+...+x_n<=1}1 dx=1/n!.
```

尾和公式により

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

条件付き期待値で見ても、停止前事象はその時点のprefixだけで完全に決まり、残る計算は単体体積になる。
