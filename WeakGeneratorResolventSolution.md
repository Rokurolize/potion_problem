# 弱生成子resolvent解

この解では、残り容量過程の生成子を弱形式で書き、停止前報酬のresolventを読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。残り容量`r`から始めて停止前prefixの期待個数を

```text
V(r)
```

と書く。求める値は`V(1)`である。

一口飲む前の空prefixが一つあり、まだ生きているなら同じ問題が残り容量で再開するので

```text
V(r)=1+int_0^r V(r-u) du.
```

ここで弱形式に移す。滑らかな試験関数`phi`に対して、積分項の生成子は一様密度`1`のVolterra作用素であり、部分積分すると境界項だけが残る。弱形式は

```text
V'(r)=V(r)
```

を意味する。初期値は

```text
V(0)=1
```

である。これは残り容量が`0`なら空prefixだけが数えられるからである。

したがって

```text
V(r)=exp(r)
```

となり、

```text
E[tau]=V(1)=e.
```

この証明では、停止時刻の分布を求めない。残り容量の弱生成子に対するresolvent方程式だけを使い、生成子の弱形式が`V'=V`へ落ちることを見る。`e`は、そのresolventの境界値である。
