# Wiener chaos真空成分解

この解では、停止前prefix全体をPoisson-Fock型のカオス展開として束ね、真空から生成される全次数の質量を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 順序付き配置をカオス次数にする

`S_n <= 1`を累積時刻

```text
0 <= t_1 <= ... <= t_n <= 1
```

へ移す。Jacobianは`1`であり、長さ`n`の生存prefixは区間`[0,1]`上の順序付き`n`点配置である。

Fock空間の言葉では、これは`n`次カオスの対称テンソル成分を順序付き代表で一つ選んだものに対応する。区間`[0,1]`の定数関数`1`から作る指数ベクトルは

```text
Omega + 1 + 1^{otimes 2}/2! + 1^{otimes 3}/3! + ...
```

という全カオス次数の和を持つ。各次数の係数`1/n!`は、対称化で`n!`個の順序を一つに割った正規化である。

停止前prefixの期待個数は、この指数ベクトルを全次数で真空期待値として評価したものになる。

```text
E[tau]
  = sum_{n >= 0} ||1_{[0,1]}||_1^n / n!
  = sum_{n >= 0} 1/n!
  = e.
```

この証明では、各単体体積を個別に積分しない。生存prefixの長さをカオス次数と見て、区間`[0,1]`の定数関数が作る指数ベクトルの真空成分を読む。`e`は全カオス次数を足した指数ベクトルの正規化として現れる。
