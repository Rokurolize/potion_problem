# Perron-Frobeniusランク1境界解

この解では、長さごとの単体体積を計算しない。境界`x`までに生きているprefix総量を、薄い境界層へ写す正作用素のPerron-Frobenius固有量として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止時刻は、境界`1`以下にいるprefixの個数に`1`を足したものではなく、空prefixを含めて

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

とそのまま数えられる。したがって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 境界までの総量を状態にする

`0 <= x <= 1`に対して

```text
M(x) = sum_{n >= 0} P(S_n <= x)
```

を置く。`M(x)`は境界`x`までに存在する全prefixの質量である。

境界を`x`から`x+h`へ広げる。新しく境界層`(x,x+h]`に入るprefixは、既存prefixの和を`y <= x`として、最後に一様増分`u`を足したものだけである。条件は

```text
x - y < u <= x + h - y.
```

`0 <= y <= x <= 1`かつ`h`が小さいとき、この区間は一様分布の支持`[0,1]`の内部にあり、幅は`h`である。したがって、どの既存prefixも同じ係数`h + o(h)`で新しい境界層へ写る。

つまり薄い境界層への更新は、既存prefix空間上の関数を総和だけで読むランク1の正作用素である。総質量だけを見ると

```text
M(x+h) - M(x) = h M(x) + o(h).
```

このランク1作用素の正固有量は総質量であり、Perron-Frobenius固有値は薄い時間あたり`1`である。したがって

```text
M'(x) = M(x).
```

初期値は空prefixだけなので

```text
M(0) = 1.
```

よって

```text
M(x) = exp(x).
```

求める期待値は

```text
E[tau] = M(1) = e.
```

この証明では、各`n`の確率を別々に出さない。境界層へ入る作用素が、総質量方向だけを増やすランク1の正更新になることを見る。`e`は、そのPerron-Frobenius正固有量を境界`0`から`1`まで流した値である。
