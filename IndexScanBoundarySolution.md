# index scan境界解

この解では、境界`1`以下のprefixを、累積和indexで範囲scanするものとして読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。各長さ`n`について、行`(u_1,...,u_n)`に累積和index

```text
s = u_1+...+u_n
```

を付ける。停止前prefixは範囲条件

```text
s <= 1
```

でscanされる行である。より細かいindexとして部分和列

```text
0 <= S_1 <= ... <= S_n <= 1
```

を持てば、範囲scanの対象は順序室そのものになる。増分から部分和列への変換はJacobian`1`なので、このscan結果の質量は

```text
1/n!.
```

全長さのscan結果を足すと

```text
E[tau]=sum_{n>=0}1/n! = e.
```

この証明では、各分布を計算しない。累積和indexで境界以下を範囲scanすると、順序室一つ分だけが返る。
