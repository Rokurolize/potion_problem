# 超幾何退化解

この解では、生存prefixの母関数を超幾何関数の退化として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めれば

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

`0<=x<=1`で

```text
a_n(x)=P(S_n<=x)
```

と置く。最後の一口を外すと

```text
a_n'(x)=a_{n-1}(x),
a_0(x)=1.
```

したがって全和

```text
H(x)=sum_{n>=0} a_n(x)
```

は

```text
H'(x)=H(x),
H(0)=1
```

を満たす。これは超幾何級数

```text
_0F_0(x)=sum_{n>=0} x^n/n!
```

への退化である。よって

```text
H(x)=_0F_0(x)=exp(x).
```

求める値は

```text
E[tau]=H(1)=e.
```

この証明では、各単体体積を直接計算しない。生存列の境界微分が、超幾何関数`_0F_0`の退化方程式そのものになることを見る。`e`は、その退化超幾何関数の値である。
