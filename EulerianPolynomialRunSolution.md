# Eulerian多項式run解

この解では、累積時刻の順序条件をrun構造で見て、Eulerian多項式の端の係数だけを読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めれば

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

長さ`n`について、増分から累積時刻`t_i=S_i`へ移す。生存条件は

```text
0<=t_1<=...<=t_n<=1.
```

`n`個の独立点にラベルを付けた全順序をEulerian多項式で分類すると、完全に上昇しているrunは端のただ一つの順序である。したがってこの順序領域の比率は`1/n!`であり、全prefix量は

```text
sum_{n>=0} 1/n! = e.
```

この証明では、Irwin-Hall分布を求めない。順序のEulerian多項式の端係数、つまり全上昇runが一つだけであることを使う。`e`は、その端係数を全長さで足した値である。
