# Poisson分配関数解

この解では、停止前prefix全体のLebesgue質量を、Poisson配置確率の分配関数として読む。各長さの単体体積を個別に出すのではなく、「確率の総和は1である」という正規化だけを使う。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止前prefixを数えると

```text
tau = sum_{n>=0} 1_{S_n <= 1}
```

なので

```text
E[tau]
= sum_{n>=0} volume { u_i>=0, u_1+...+u_n <= 1 }.
```

この右辺を`Z`と書く。

## prefixを点配置へ変える

長さ`n`の停止前prefix

```text
u_1, ..., u_n
```

から、累積時刻

```text
t_i = u_1 + ... + u_i
```

を作る。すると

```text
0 < t_1 < ... < t_n <= 1
```

であり、逆に隣り合う差分を取れば元の`u_i`に戻る。この変換のJacobianは`1`である。

したがって`Z`は、区間`[0,1]`上の有限な順序付き点配置全体のLebesgue質量

```text
Z = sum_{n>=0} volume { 0 < t_1 < ... < t_n <= 1 }
```

である。

## Poisson確率で正規化する

強度`1`のPoisson過程を`[0,1]`上に置く。点数が`n`で、点が

```text
0 < t_1 < ... < t_n <= 1
```

にある確率密度は

```text
exp(-1)
```

である。これは点数`n`に依存しない。Poisson過程の全ての有限配置の確率を足せば`1`なので

```text
1
= sum_{n>=0} int_{0<t_1<...<t_n<=1} exp(-1) dt_1...dt_n
= exp(-1) Z.
```

よって

```text
Z = exp(1) = e.
```

したがって

```text
E[tau] = e.
```

## この見方の芯

この証明では、`volume {S_n<=1}`を一つずつ計算しない。停止前prefix全体を、Poisson配置空間の非正規化測度として見る。

Poisson確率は全配置に共通の空白因子`exp(-1)`を掛けたものなので、確率測度の全質量`1`を外すだけで、停止前prefix測度の分配関数が

```text
exp(1)
```

だと分かる。`e`は階乗級数の計算結果ではなく、Poisson配置測度の正規化定数として現れる。
