# upper triangular operator解

この解では、停止前prefixを次数付き空間上の上三角作用素で数える。

長さ`n`の停止前領域を

```text
D_n(x)={u_i>=0, u_1+...+u_n<=x}
```

とし、`A_n(x)=vol(D_n(x))`と書く。尾和公式より

```text
E[tau]=sum_{n>=0}A_n(1).
```

次数付きベクトル

```text
A(x)=(A_0(x),A_1(x),A_2(x),...)
```

を考える。最後の変数で切ると

```text
A_n'(x)=A_{n-1}(x),  A_0(x)=1.
```

これは、左に一段ずらす上三角作用素`N`を用いて

```text
A'(x)=N A(x),  A(0)=(1,0,0,...)
```

と書ける。したがって

```text
A(x)=exp(xN)A(0).
```

初期ベクトルに`N`を`n`回かけると次数`n`だけが残るので

```text
A_n(x)=x^n/n!.
```

ゆえに

```text
E[tau]=sum_{n>=0}A_n(1)=sum_{n>=0}1/n!=e.
```

ここでは、単体体積の列が上三角の微分システムとして指数化されることを使う。
