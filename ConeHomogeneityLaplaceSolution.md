# 錐同次ラプラス解

待ち時間を `X_1, X_2, ...` とし、`S_n = X_1 + ... + X_n` とする。`S_n <= 1` の範囲では一様密度は1なので、

```text
P(S_n <= 1) = vol{x_i >= 0, x_1 + ... + x_n <= 1}
```

である。

ここで正錐

```text
C_n = {x_i >= 0}
```

と高さ

```text
h(x) = x_1 + ... + x_n
```

を考える。`h` で体積測度を押し出した測度を `mu_n` と置く。

```text
mu_n([0, t]) = vol{x in C_n | h(x) <= t}
```

正錐と高さは同時に拡大するので、`t > 0` について

```text
mu_n([0, t]) = t^n mu_n([0, 1])
```

が成り立つ。したがって、ある定数 `a_n = mu_n([0, 1])` を用いて

```text
d mu_n(s) = n a_n s^{n-1} ds
```

である。

この `a_n` は、正錐上のラプラス積分で決まる。

```text
int_{C_n} exp(-h(x)) dx
  = prod_{i=1}^n int_0^infty exp(-x_i) dx_i
  = 1
```

一方で押し出し測度を使えば

```text
int_{C_n} exp(-h(x)) dx
  = int_0^infty exp(-s) d mu_n(s)
  = n a_n int_0^infty exp(-s) s^{n-1} ds
  = n a_n (n-1)!
  = n! a_n
```

である。よって

```text
a_n = 1/n!
```

すなわち

```text
P(S_n <= 1) = 1/n!
```

である。`n = 0` は空和なので同じく寄与1とする。

到着数は

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

だから、単調収束により

```text
E[tau]
  = sum_{n >= 0} P(S_n <= 1)
  = sum_{n >= 0} 1/n!
  = e
```

となる。単体の体積を先に知っている必要はない。正錐の拡大同次性が形を決め、独立な指数積分の積が正規化定数だけを決める。
