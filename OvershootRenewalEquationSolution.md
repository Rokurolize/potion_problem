# 超過量renewal方程式解

この解では、`1`までの残り距離そのものではなく、「あと`x`だけ超えるまでに必要な回数」を関数として見る。階乗級数を先に数えるのではなく、最後の一口を足す前後で同じ問題が再生することから、期待値が微分方程式に閉じる。

独立な一様乱数`U_1,U_2,...`について

```text
T(x) = min { n >= 1 : U_1 + ... + U_n > x }
```

と置く。求めたい値は`E[T(1)]`である。

## 一口目で分ける

`x>=0`に対して

```text
F(x) = E[T(x)]
```

と書く。`x=0`なら、一口目は必ず正なので

```text
F(0) = 1
```

である。

`0<x<=1`を考える。一口目`U`が`x`を超えればそこで終わる。`U<=x`なら一回飲んだあと、残る問題はしきい値`x-U`の同じ問題である。したがって

```text
F(x)
= int_0^x (1 + F(x-u)) du + int_x^1 1 du.
```

右辺の二つ目は、一口で超える場合の寄与である。整理すると

```text
F(x)
= x + int_0^x F(x-u) du + 1 - x
= 1 + int_0^x F(t) dt.
```

ここで変数変換`t=x-u`をした。

## 微分方程式に閉じる

得られた式は

```text
F(x) = 1 + int_0^x F(t) dt.        (1)
```

である。これは`F`が連続であることも同時に与えるので、微分して

```text
F'(x) = F(x),
F(0) = 1.
```

よって

```text
F(x) = exp(x).
```

したがって

```text
E[T(1)] = F(1) = e.
```

## この見方の芯

尾確率

```text
P(U_1 + ... + U_n <= 1)
```

を深さごとに測るかわりに、しきい値`x`を動かした期待停止時刻そのものを見る。一口目を条件にすると、「まだ超えていない場合」は同じ形の問題が短い残り距離で現れる。

その再生構造が

```text
F = 1 + ∫F
```

というrenewal方程式になり、`e`は階乗体積の和としてではなく、停止期待値の境界値として出る。
