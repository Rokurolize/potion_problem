# Poisson測度変換解

この解では、停止前prefixの体積測度を、Poisson過程の確率測度へ測度変換して読む。

独立な一様乱数を`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

と置く。`S_0=0`を含めれば、

```text
tau = sum_{n>=0} 1_{S_n <= 1}
```

なので、

```text
E[tau] = sum_{n>=0} P(S_n <= 1).        (1)
```

## 停止前prefixの基準測度

`S_n<=1`のとき、到着時刻

```text
t_i = S_i
```

は

```text
0 <= t_1 <= ... <= t_n <= 1
```

を満たす。増分から到着時刻への変換のJacobianは`1`である。

したがって、(1)の右辺は、全ての有限順序配置に対する基準測度

```text
mu = sum_{n>=0} dt_1 ... dt_n
```

の全質量である。ただし各`n`では

```text
0 <= t_1 <= ... <= t_n <= 1
```

上で積分する。

つまり

```text
E[tau] = mu(全配置空間).        (2)
```

## 確率測度へ傾ける

区間`[0,1]`上の強度`1`のPoisson過程を考える。その配置確率を`P_Pois`と書く。

Poisson過程が`n`点を持ち、その順序統計量が`(t_1,...,t_n)`であるとき、配置空間上の密度は

```text
dP_Pois = exp(-1) dmu
```

である。

したがって、基準測度`mu`はPoisson配置確率に対して

```text
dmu = exp(1) dP_Pois
```

というRadon-Nikodym密度を持つ。密度が配置に依存せず、全ての点数で同じ定数`exp(1)`になるところが、この問題の要点である。

## 全質量を戻す

`P_Pois`は確率測度だから

```text
P_Pois(全配置空間) = 1.
```

(2)と測度変換から

```text
E[tau]
= mu(全配置空間)
= exp(1) P_Pois(全配置空間)
= exp(1)
= e.
```

## この見方の芯

個々の`P(S_n<=1)`も、単体体積`1/n!`も計算しない。停止前prefixが作る全配置空間の測度`mu`を先に置き、それをPoisson配置確率へ一度正規化する。

答えの`e`は、正規化済みのPoisson配置確率から、停止前prefixの基準測度へ戻すRadon-Nikodym密度そのものである。
