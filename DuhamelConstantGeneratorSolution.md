# Duhamel定数生成子解

この解では、停止前prefixの総量を「境界幅を増やす発展」として見て、Duhamel公式の積分項が定数生成子になることを使う。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。求めたい値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。ここで残り容量を`r`とし、容量`r`以内に入る停止前prefixの総量を

```text
F(r) = sum_{n >= 0} P(S_n <= r)
```

とおく。空prefixがあるので`F(0)=1`である。

## 発展方程式として見る

容量を`r`から`r+h`へ広げる。新しく許されるprefixは、すでに容量`r`以内にいたprefixの末尾に一つ一様変数を足し、その終点が薄い区間

```text
(r, r+h]
```

に入ったものである。

終点が`y <= r`のprefixを固定すると、末尾に足す値`u`がこの薄い区間へ入る条件は

```text
r-y < u <= r+h-y
```

である。`h`が十分小さいと、この区間の長さは`h`であり、一様密度が`1`なので、このprefixが新しい層へ寄与する量は`h + o(h)`である。

すべての既存prefixについて足し合わせると

```text
F(r+h) = F(r) + h F(r) + o(h).
```

したがって

```text
F'(r) = F(r),
F(0) = 1.
```

これは、境界幅`r`を時刻と見たとき、停止前prefix総量が生成子`A = 1`の発展

```text
dF/dr = A F
```

に従う、ということに等しい。

## Duhamel公式の形

同じことを積分形で書くと

```text
F(r) = 1 + int_0^r F(s) ds.
```

右辺の第1項は空prefixであり、積分項は「最後の一口がどの境界層で生まれたか」で停止前prefixを分類している。これは非同次項が空prefix、相互作用項が定数生成子であるDuhamel公式である。

反復すると

```text
F(r) = 1 + r + r^2/2! + r^3/3! + ...
     = exp(r).
```

よって

```text
E[tau] = F(1) = e.
```

この見方では、各`P(S_n <= 1)`を個別の単体体積として先に計算しない。境界を広げる発展のDuhamel項が、既存prefix総量を同じ重みで再投入することだけを見る。一様性はその再投入率を定数`1`にし、独立性は再投入後も同じ`F`を使えることに現れる。
