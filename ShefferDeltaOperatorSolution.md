# Sheffer差分作用素解

この解では、各長さの単体体積を直接出さない。境界`x`を動かしたときの生存prefix列を、差分作用素で一段下がるSheffer列として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めて数えると

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

`0 <= x <= 1`で

```text
a_n(x) = P(S_n <= x)
```

と置く。`a_0(x)=1`であり、`n>=1`では最後の一様増分を外すと

```text
a_n(x) = int_0^x a_{n-1}(t) dt.
```

ここで通常の微分`D`をデルタ作用素として見ると

```text
D a_n = a_{n-1},
a_n(0)=0  (n>=1)
```

である。つまり`a_0,a_1,a_2,...`は、`D`で一段ずつ下がるSheffer型の基底になっている。

全prefixをまとめた母関数

```text
A(x)=sum_{n>=0} a_n(x)
```

を取る。`0<=x<=1`では単調収束により微分と和を交換できるので

```text
D A(x) = sum_{n>=1} D a_n(x)
       = sum_{n>=1} a_{n-1}(x)
       = A(x).
```

また`A(0)=1`である。したがって

```text
A(x)=exp(x).
```

求める期待値は

```text
E[tau]=A(1)=e.
```

この証明では、`a_n(1)`を個別に求めてから足さない。停止前prefix列がデルタ作用素で一段下がるSheffer列であり、その全和がデルタ作用素の固有関数になることだけを見る。`e`は、このSheffer列の全和を境界`1`で評価した値である。
