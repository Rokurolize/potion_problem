# Newton級数境界解

この解では、生存prefix列を境界容量`x`のNewton級数として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めれば

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

境界`x`までの総量を

```text
N(x)=sum_{n>=0} P(S_n<=x)
```

と置く。`0<=x<=1`では一様分布の上端はまだ薄層計算を邪魔しない。境界を`x`から`x+h`へ動かすと、既存prefixが次のprefixを新しい層へ送る幅は`h+o(h)`である。

したがってNewton級数の前進差分は

```text
N(x+h)-N(x)=hN(x)+o(h).
```

連続極限で

```text
N'(x)=N(x),
N(0)=1.
```

よって

```text
N(x)=exp(x),
E[tau]=N(1)=e.
```

この証明では、各係数を単体体積として求めない。境界容量のNewton差分が総量自身を返すことだけを見る。`e`は、そのNewton級数を境界`1`で評価した値である。
