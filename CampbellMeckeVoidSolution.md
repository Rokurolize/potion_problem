# Campbell-Meckeの空事象正規化解

この解では、停止前のprefix全体を、Poisson点過程のCampbell-Mecke公式に入る「配置空間の総質量」として読む。

独立な一様乱数を`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

と置く。`S_0=0`を含めれば、停止時刻は境界内にいるprefixの個数である。

```text
tau = sum_{n>=0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n>=0} P(S_n <= 1).        (1)
```

## 生存prefixを配置空間に移す

`S_n <= 1`の上では、到着時刻

```text
t_i = S_i
```

が

```text
0 <= t_1 <= ... <= t_n <= 1
```

を満たす。増分から到着時刻への変換はJacobian`1`なので、

```text
P(S_n <= 1)
```

は、この順序付き配置空間のLebesgue体積である。

よって(1)の右辺は

```text
sum_{n>=0} int_{0<=t_1<=...<=t_n<=1} dt_1 ... dt_n.        (2)
```

ここでは各項を`1/n!`と計算しない。

## Poisson過程の全確率だけを使う

強度`1`のPoisson点過程`N`を区間`[0,1]`上に置く。`N`がちょうど`n`点を持ち、その順序統計量が

```text
0 <= T_1 <= ... <= T_n <= 1
```

であるとき、その密度は順序付き配置空間上で

```text
exp(-1) dt_1 ... dt_n
```

である。これはCampbell-Mecke公式の有限配置版であり、点を1個ずつ足す強度測度がLebesgue測度、最後に全体の空隙確率`exp(-1)`が掛かる。

Poisson過程は必ず何らかの有限点数を取るので、全ての`n`について足せば総確率は`1`である。

```text
1
= sum_{n>=0} exp(-1)
   int_{0<=t_1<=...<=t_n<=1} dt_1 ... dt_n.
```

したがって

```text
sum_{n>=0} int_{0<=t_1<=...<=t_n<=1} dt_1 ... dt_n
= exp(1).        (3)
```

## 期待値

(2)と(3)を合わせて

```text
E[tau] = exp(1) = e.
```

## この見方の芯

この証明は、単体体積を個別に計算しない。停止前prefixの全体を、Poisson点過程の配置空間から空隙確率`exp(-1)`を外した非正規化測度として見る。

Campbell-Mecke公式は「全ての有限配置の確率を足すと1」という当たり前の正規化を、配置空間のLebesgue総量へ変換している。その正規化を外した瞬間、平均停止回数として`e`が現れる。
