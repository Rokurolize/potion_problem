# 余白Palm正規化解

この解では、停止前prefixをそのまま数える代わりに、各prefixの後ろに残っている余白を「次の一手を差し込める量」として測る。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## prefixの余白で次のprefixを量る

境界を`x`まで動かし、

```text
A(x) = sum_{n >= 0} P(S_n <= x)
```

とおく。`A(x)`は、容量`x`の中に入る停止前prefix全体の質量である。

あるprefixの和が`s <= x`なら、その後ろには

```text
x - s
```

だけ余白がある。この余白の中に次の一様増分`u`を選ぶと、新しいprefixはまだ境界内に残る。逆に、長さ`n+1`の生存prefix

```text
u_1 + ... + u_{n+1} <= x
```

は、長さ`n`の生存prefixと、そのprefixの余白内に置いた最後の増分`u_{n+1}`に一意に分解される。

したがって、空prefixを除いた全prefix質量は

```text
A(x) - 1
```

であり、これは全prefixの余白長を足したものに等しい。

## 余白長の総和を断面で読む

全prefixの余白長を直接足す代わりに、余白の中に補助点`y`を1つ置く。prefixの和を`s`とすると、条件は

```text
0 <= y <= x - s
```

である。`y`を余白の深さとして見ると、これは

```text
s <= x - y
```

と言い換えられる。

つまり、固定した`y`に対する断面の質量は、容量`x-y`での全prefix質量

```text
A(x-y)
```

そのものになる。余白全体を`y`で積分すれば

```text
A(x) - 1 = int_0^x A(x-y) dy.
```

変数を替えると

```text
A(x) = 1 + int_0^x A(t) dt.
```

したがって

```text
A'(x) = A(x),
A(0) = 1.
```

よって

```text
A(x) = exp(x).
```

求める値は

```text
E[tau] = A(1) = e.
```

## この見方の芯

この証明では、`P(S_n<=1)=1/n!`を先に出さない。長さを1つ増やす操作を、「既存のprefixの後ろに残った余白に点を置くこと」として数える。

余白に置いた補助点で断面を切ると、断面がまた同じprefix総質量`A`に戻る。`e`は単体体積の列を足した結果ではなく、余白Palm測度が自分自身を断面に再生する正規化定数として現れる。
