# continuous Ehrhart simplex解

この解では、停止前の列を「格子点数の連続版」として数える。

長さ`n`でまだ止まっていない条件は

```text
x_1+...+x_n<=1,  0<=x_i<=1
```

である。上限`x_i<=1`は合計条件から自動的に従うので、領域は標準`n`単体

```text
Delta_n={x_i>=0, x_1+...+x_n<=1}
```

そのものになる。

Ehrhart多項式は整数格子点を拡大率で数えるが、ここでは拡大率を固定して、格子点数の代わりにLebesgue体積を数える。単体を座標軸方向に一つずつ積み上げると

```text
vol(Delta_n)=int_0^1 vol({x_1+...+x_{n-1}<=1-s}) ds
```

であり、帰納的に

```text
vol(Delta_n)=1/n!
```

となる。

尾和公式を使えば

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}vol(Delta_n)
      =sum_{n>=0}1/n!
      =e.
```

離散の格子点数を数える代わりに、停止前単体の連続的な数え上げを行うと、指数関数の階乗係数がそのまま現れる。
