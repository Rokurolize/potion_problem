# incidence Möbius prefix解

この解では、停止前prefixを有限集合のincidence代数で数え、Möbius側ではなくzeta側の全質量として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。累積時刻へ移すと、長さ`n`の生存prefixは区間`[0,1]`内の順序付き`n`点配置である。

有限集合の包含束で見ると、`n`点配置のzeta側の重みは

```text
1/n!
```

である。これは、ラベル付きの`n`点を順序で割った非ラベル配置の質量であり、incidence代数のzeta関数が全ての有限配置を符号なしに足すことに対応する。

停止前prefixの期待値は全次数のzeta質量なので

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

Poisson空白事象をMöbius側で見ると符号付きの`e^{-1}`が現れるが、ここでは最初から符号を外したzeta側の全有限配置を数えている。その全質量が`e`である。
