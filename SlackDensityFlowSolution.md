# 残量密度流解

この解では、累積和`S_n`ではなく、境界までの残り距離

```text
R_n = 1 - S_n
```

を状態として見る。停止前prefixの総量は、残量空間`[0,1]`に流れ込む期待密度の全質量である。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止前prefixを数えると

```text
tau = sum_{n>=0} 1_{S_n <= 1}.
```

よって求めたい値は、全ての停止前prefixが残量空間に置く期待質量である。

## 残量側の訪問測度

停止前prefixの残量の期待測度を`Q`とする。

```text
Q(A) = E[sum_{n>=0} 1_{R_n in A, R_n >= 0}]
```

初期prefixは`R_0=1`なので、`Q`は点`1`に質量`1`を持つ。その他の残量`r<1`は、以前の残量`y`から一口`u=y-r`を飲んで来る。

一様分布の密度は`1`だから、ある親残量`y`が`r<y`へ子を送る密度は常に`1`である。したがって、`0<=r<1`における`Q`の密度を`q(r)`と書けば

```text
q(r) = 1 + int_r^1 q(y) dy.        (1)
```

右辺の`1`は、初期残量`1`から直接`r`へ一口で来る寄与である。積分項は、密度`q(y)`を持つ全ての親残量`y>r`からの流入である。

## 流入方程式を解く

(1)を微分すると

```text
q'(r) = -q(r),
q(1-) = 1.
```

よって

```text
q(r) = exp(1-r).
```

停止前prefixの総期待個数は、初期prefixの質量`1`と、`0<=r<1`にある密度部分の質量を足したものだから

```text
E[tau]
= Q([0,1])
= 1 + int_0^1 exp(1-r) dr
= e.
```

## この見方の芯

通常は飲んだ量の累積`S_n`を右向きに追う。この証明では、残り距離`R_n`を左向きに流れる質量として見る。

一様性は「どの親残量`y`からも、下側の残量`r<y`へ密度`1`で流れる」という形で現れる。全ての停止前prefixを残量空間の流入密度としてまとめると、方程式は

```text
q = 1 + int q
```

に閉じ、`e`はその流れの全質量として出る。
