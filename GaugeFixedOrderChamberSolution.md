# gauge固定順序室解

この解では、ラベル付き時刻配置の置換自由度をgaugeとみなし、昇順gaugeを固定して停止前prefixを読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。部分和

```text
t_k=S_k
```

への変換は三角行列でJacobian`1`である。したがって長さ`n`の生存prefixは

```text
0 <= t_1 <= ... <= t_n <= 1
```

という順序室である。

いまラベル付き時刻`(x_1,...,x_n) in [0,1]^n`を考える。置換群`S_n`がラベルを入れ替える自由度を持ち、ほとんど全ての点で安定化群は自明である。gauge条件

```text
x_1 <= x_2 <= ... <= x_n
```

を課すと、各軌道から代表を一つだけ選ぶ。したがってFaddeev-Popov係数に相当するものは単に`1/n!`であり、

```text
vol{0 <= t_1 <= ... <= t_n <= 1}=1/n!.
```

よって

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

ここでgaugeという語は飾りではなく、順序条件が「置換自由度を固定した代表選択」であることを示す。とはいえ計算そのものは、基本領域の体積を読むだけで閉じている。
