# 余帰納ストリーム解

この解では、停止前prefixの列を、残り容量から生成されるストリームの最終余代数として見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は停止前prefixの個数なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 残り容量が作るストリーム

残り容量`r`から始めたとき、停止前prefixの期待個数を

```text
M(r)
```

と書く。空prefixは必ず一つある。非空prefixは、最初の一口`u`が`0 <= u <= r`に入った場合だけ残り容量`r-u`の同じ問題へ進む。

したがって、状態`r`は

```text
head = 1,
tail = (u in [0,r] に沿った M(r-u) の積分)
```

というストリームを生成する。この余代数を期待値で畳み込むと

```text
M(r) = 1 + int_0^r M(r-u) du.
```

変数を替えれば

```text
M(r) = 1 + int_0^r M(s) ds.
```

この式は、空prefixをhead、残りの全prefixをtailとして分ける余帰納的な定義である。微分すると

```text
M'(r) = M(r),
M(0) = 1.
```

よって

```text
M(r) = exp(r).
```

求める初期状態は`r=1`なので

```text
E[tau] = M(1) = e.
```

この証明では、各段の確率を先に計算しない。停止前prefix列を、空prefixと一口後の同じストリームに分解する余帰納的対象として扱う。`e`は、そのストリームの期待総和を初期残量`1`で評価した値である。
