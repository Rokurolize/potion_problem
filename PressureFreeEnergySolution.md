# 圧力自由エネルギー解

この解では、停止前prefix全体を活動度つきの分配関数としてまとめ、その対数である圧力を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めると

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

境界`x`までの停止前prefixに活動度`z`を付けて

```text
Z(x,z)=sum_{n>=0} z^n P(S_n <= x)
```

を置く。境界を`x`から`x+h`へ動かすと、既存prefixは一様密度`1`により幅`h+o(h)`だけ次のprefixを新しい薄層へ送る。活動度が一つ増えるので

```text
Z(x+h,z)-Z(x,z)=h z Z(x,z)+o(h).
```

したがって

```text
partial_x Z(x,z)=z Z(x,z),
Z(0,z)=1.
```

圧力、または自由エネルギーを

```text
P(x,z)=log Z(x,z)
```

と書くと

```text
partial_x P(x,z)=z,
P(0,z)=0.
```

よって

```text
P(x,z)=zx,
Z(x,z)=exp(zx).
```

欲しい期待値は活動度`z=1`、境界`x=1`で

```text
E[tau]=Z(1,1)=e.
```

この証明では、各次数の体積を個別に求めない。活動度つき停止前prefixの圧力が、非相互作用のため一次関数に潰れることを見る。`e`は、その自由エネルギーを指数に戻した値である。
