# Born則順序室解

この解では、順序室への射影確率をBorn則風に読み、全次数で足す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。長さ`n`の一様な振幅を立方体`[0,1]^n`上に置き、観測として「累積時刻が昇順で境界内にある」という射影を考える。部分和座標ではこの観測事象は

```text
0<=t_1<=...<=t_n<=1
```

であり、Jacobianは`1`である。Born則風に射影事象の確率をその領域の質量として読むと、

```text
P(S_n<=1)=1/n!.
```

よって期待停止時刻は生存射影確率の尾和で

```text
E[tau]=sum_{n>=0}P(S_n<=1)
      =sum_{n>=0}1/n!
      =e.
```

この証明では、量子性は退化している。射影確率という言い方で、順序室一つ分の体積を読むだけである。
