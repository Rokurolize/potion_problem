# Bernoulli differential recursion解

この解では、停止前prefix数の母関数を微分再帰として閉じる。

初期容量を`x`とし、長さ`n`まで停止しない確率を

```text
a_n(x)=P(U_1+...+U_n<=x)
```

とする。`a_0(x)=1`で、`n>=1`では最後の入力で切って

```text
a_n(x)=int_0^x a_{n-1}(x-u)du.
```

したがって

```text
a_n'(x)=a_{n-1}(x),  a_n(0)=0.
```

ここで、期待prefix数そのものを

```text
B(x)=sum_{n>=0} a_n(x)
```

とおく。項別微分により

```text
B'(x)=sum_{n>=1} a_n'(x)
     =sum_{n>=1} a_{n-1}(x)
     =B(x).
```

また`x=0`では空prefixだけが残るので

```text
B(0)=1.
```

ゆえに

```text
B(x)=e^x.
```

元の問題は`x=1`であり、尾和公式により

```text
E[tau]=sum_{n>=0}P(U_1+...+U_n<=1)=B(1)=e.
```

この証明は、各項の値を先に求めず、停止前確率列全体が微分で一つ左へずれることだけから指数関数を強制する。
