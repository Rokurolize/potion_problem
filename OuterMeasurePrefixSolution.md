# outer measure prefix解

この解では、停止前prefix領域を外測度から生成される可測集合として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。長さ`n`のprefixが停止前である集合は

```text
E_n={u_i>=0, u_1+...+u_n<=1}
```

である。これは半空間と正錐の交わりなので、立方体上の外測度から見ても可測であり、外測度は通常のLebesgue体積に一致する。

部分和座標

```text
t_k=u_1+...+u_k
```

に移すと

```text
0<=t_1<=...<=t_n<=1
```

で、Jacobianは`1`である。よって

```text
mu*(E_n)=1/n!.
```

停止時刻は停止前prefixの個数だから

```text
E[tau]=sum_{n>=0} mu*(E_n)
      =sum_{n>=0} 1/n!
      =e.
```

外測度から始めても、停止境界で切ったprefix領域が可測単体であるため、計算はそのまま体積の総和へ落ちる。
