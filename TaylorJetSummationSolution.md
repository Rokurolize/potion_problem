# Taylor jet総和解

独立一様乱数を `X_1, X_2, ...` とし、

```text
S_n = X_1 + ... + X_n
```

とおく。境界を `x` に動かした生存prefix総量を

```text
F(x) = sum_{n >= 0} P(S_n <= x)
```

と書けば、求めたい値は `F(1)` である。

ここでは `F` を方程式として解くのではなく、境界 `0` におけるTaylor jetを直接読む。

`n=0` の項は常に `1` である。一方、`n>=1` では

```text
P(S_n <= x)
```

は `x=0` で `n` 次以上にしか立ち上がらない。さらに一様密度が `1` なので、境界を微分すると一段短いprefixが現れる。

```text
d/dx P(S_n <= x) = P(S_{n-1} <= x)
```

したがって、`x=0` での導関数は

```text
F(0) = 1,
F'(0) = 1,
F''(0) = 1,
...
```

と全部 `1` になる。なぜなら、`k` 回微分すると長さが `k` だけ下がり、最後に空prefixの値 `1` へ着地するからである。

よって、`F` のTaylor展開は

```text
F(x) = sum_{k >= 0} F^(k)(0) x^k / k!
     = sum_{k >= 0} x^k / k!
     = exp(x)
```

である。したがって

```text
E[tau] = F(1) = e
```

となる。

この解では、各長さの単体体積を直接求めない。境界 `0` にある空prefixの値 `1` が、微分で一段ずつ上の生存prefix列へ伝播し、全てのTaylor係数を `1` にすることだけを見る。`e` は、境界jetを総和した値として現れる。
