# distributional tail pairing解

この解では、停止時刻の尾和を分布と定数テスト関数の pairing として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。整数値変数の尾和により

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

ここで`S_n`の分布を`mu_n`と書けば、各項は

```text
<mu_n, 1_{[0,1]}>
```

である。`0<=x<=1`では、`mu_n`の密度は正錐の和写像から来る

```text
x^{n-1}/(n-1)!
```

である。したがって`n>=1`で

```text
<mu_n, 1_{[0,1]}> = int_0^1 x^{n-1}/(n-1)! dx = 1/n!,
```

また`n=0`は空和で`1`を与える。よって

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

この表現では、期待値は停止時刻の分布そのものではなく、各部分和分布を同じテスト関数`1_{[0,1]}`に当てた総和として現れる。
