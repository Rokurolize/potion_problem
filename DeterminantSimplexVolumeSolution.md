# determinant simplex volume解

この解では、停止前単体の体積を行列式で計算する。

長さ`n`で停止前にある集合は

```text
D_n=conv(0,e_1,...,e_n)
```

である。これは条件

```text
x_i>=0, x_1+...+x_n<=1
```

と同じである。

単体の体積公式より、辺ベクトル`e_1,...,e_n`を列に持つ行列の行列式を使って

```text
vol(D_n)=|det I_n|/n! = 1/n!.
```

独立一様分布の密度は1なので

```text
P(tau>n)=vol(D_n)=1/n!.
```

したがって

```text
E[tau]=sum_{n>=0}P(tau>n)=sum_{n>=0}1/n!=e.
```

停止前領域を頂点表示に直すと、答えは単位行列の行列式を`n!`で割るだけになる。
