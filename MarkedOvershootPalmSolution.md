# 印付き超過点のPalm解

この解では、止まる一口を「境界`1`をまたぐ印付き点」として扱い、停止時刻の平均を印付き配置の質量から読む。

独立な一様乱数を`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

と置く。`S_0=0`を含めると、`tau`は境界内に残っていたprefixの数である。

```text
tau = sum_{k>=0} 1_{S_k <= 1}.
```

したがって

```text
E[tau] = sum_{k>=0} P(S_k <= 1).        (1)
```

## 生存prefixに停止の印を足す

`S_k=x<=1`であるprefixを一つ固定する。そこから次の一口`U_{k+1}`で停止するには

```text
U_{k+1} > 1 - x
```

が必要である。この条件の確率は

```text
x
```

である。

つまり、境界内のprefix点`x`は、次の一口で停止印を付けられる重み`x`を持つ。

しかしここで数えたいのは「停止するprefix」ではなく「全ての生存prefix」である。そこで、点`x`に重み`x`を掛けた後、Palm的に印の位置で割り戻す。`x`の密度を先に積むと、この割り戻しがちょうど指数関数の積分方程式になる。

## 印の直前までの祖先配置

境界内のprefixのうち、最後の位置を`x`に固定する。そこまでにある祖先prefixは

```text
0 < s_1 < ... < s_{k-1} < x
```

であり、その全ての長さを足した総量は

```text
H(x) = sum_{m>=0} x^m/m! = exp(x).
```

これは、`[0,x]`内の有限個の祖先点を順序で読む総量である。

したがって、境界内prefix全体の総量は

```text
1 + int_0^1 H(x) dx.        (2)
```

先頭の`1`は空prefix`S_0=0`である。

一方、停止印を付けると、位置`x`のprefixは重み`x`を得る。印付きprefixの総量は

```text
int_0^1 x H(x) dx.
```

部分積分で

```text
int_0^1 x exp(x) dx = 1.
```

これは「実際にどこかで一度だけ停止する」ことの質量である。停止印を外して生存prefixへ戻すと、(2)が得られる。

## 期待値を読む

`H(x)=exp(x)`なので、(2)は

```text
E[tau]
= 1 + int_0^1 exp(x) dx
= exp(1)
= e.
```

## この見方の芯

停止時刻そのものを先に分布へ落とさない。境界内にあるprefixを一つ選び、その直後の一口が境界をまたぐ可能性を「停止印」として載せる。

印を付けた側では停止は一度だけ起きる。印を外してprefix配置の総量へ戻すと、祖先配置の指数関数`H(x)=exp(x)`が現れ、その全質量が平均停止回数`e`になる。
