# exponential species tail解

この解では、停止前履歴を「ラベル付き構造」の指数型母関数として読む。

長さ`n`で停止前にある条件は

```text
x_1+...+x_n<=1
```

である。この領域は標準`n`単体で、座標の順序をラベルと見ると、ラベル付き`n`構造1個ぶんの連続体積が

```text
1/n!
```

になる。これは指数型母関数で、ラベル付き集合構造の係数が`1/n!`で現れるのと同じ正規化である。

実際、

```text
P(tau>n)=vol{x_i>=0, x_1+...+x_n<=1}=1/n!.
```

停止時刻の期待値は尾和なので

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

各prefix長さをラベル数、停止前単体の体積をそのラベル付き構造の重みと見ると、問題全体が指数型母関数`exp(1)`の評価になる。
