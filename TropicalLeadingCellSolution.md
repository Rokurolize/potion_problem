# tropical先頭セル解

この解では、畳み込みで現れる区分多項式全体を見ず、境界`1`の手前で残る先頭セルだけを見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

`S_n`の密度は、一様密度の`n`回畳み込みであり、区間ごとに違う多項式片を持つ。tropicalに言えば、折れ目は整数格子点

```text
0,1,2,...,n
```

にあり、各区間が一つのセルである。

しかし`P(S_n <= 1)`で積分する範囲は

```text
0 <= x <= 1
```

だけである。この範囲では、右端の切断から来る後続セルはまだ現れない。残るのは原点から伸びる先頭セルだけであり、その密度は半直線定数密度を`n`回積分したものと同じである。

先頭セルの密度を`b_n(x)`とすれば

```text
b_1(x) = 1,
b_{n+1}(x) = int_0^x b_n(t) dt.
```

したがって

```text
b_n(x) = x^{n-1}/(n-1)!.
```

よって

```text
P(S_n <= 1)
  = int_0^1 b_n(x) dx
  = 1/n!.
```

空prefixの項を加えると

```text
E[tau]
  = sum_{n >= 0} 1/n!
  = e.
```

この証明では、Irwin-Hall分布の全セルを展開しない。境界`1`がちょうど最初の折れ目にあるため、tropical分割の先頭セルだけが期待値に必要な情報を持つ。`e`は、その先頭セルの全次数和として現れる。
