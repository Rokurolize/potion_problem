# 割引resolvent解

この解では、`P(S_n<=1)`を一つずつ計算しない。まず停止前prefixを割引つきで数える問題に拡張し、その割引resolventを閉じた形で解く。最後に割引を外すと`e`が出る。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。欲しい量は

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

の期待値である。

## 割引つきで数える

残り容量が`x`あるとき、割引率`q`で停止前prefixを数える値を

```text
H_q(x) = E_x[ sum_{n >= 0} q^n 1_{S_n <= x} ]
```

と置く。`0 <= q < 1`ならこの和は自動的に有限であり、後で`q`を`1`へ上げる。

最初のprefix自身が一つあるので、まず`1`を得る。次の一口`u`が`0 <= u <= x`なら残り容量は`x-u`になり、その後の全てのprefixは一段深くなるので割引`q`を一つ受ける。したがって

```text
H_q(x) = 1 + q int_0^x H_q(x-u) du.
```

変数を入れ替えると

```text
H_q(x) = 1 + q int_0^x H_q(y) dy.
```

これは確率分布を求める式ではなく、停止前報酬のresolvent方程式である。

## `q`を入れたまま解く

上の式を微分すると

```text
H_q'(x) = q H_q(x),
H_q(0) = 1.
```

よって

```text
H_q(x) = exp(q x).
```

特に境界`1`では

```text
H_q(1) = exp(q).
```

左辺は定義から

```text
H_q(1) = E[ sum_{n >= 0} q^n 1_{S_n <= 1} ].
```

つまり、停止前prefix数の全ての段を、割引変数`q`で一つの解析関数にまとめると、答えはただ`exp(q)`になる。

## 割引を外す

`0 <= q < 1`では

```text
sum_{n >= 0} q^n 1_{S_n <= 1}
```

は`q`について単調に増える。`q`を`1`へ上げると単調収束により

```text
E[tau]
= E[ sum_{n >= 0} 1_{S_n <= 1} ]
= lim_{q -> 1-} H_q(1)
= lim_{q -> 1-} exp(q)
= e.
```

## この見方の芯

この証明は、段ごとの生存確率を最後まで展開しない。停止前の各prefixに深さによる割引を置き、割引つき報酬のresolventを解く。割引率`q`は、何口目かを覚えるための印であり、方程式の中では積分作用素の係数としてだけ働く。

その結果、全ての候補prefixを足した母関数が`exp(q)`に圧縮される。求める`e`は、割引を外した`q=1`の境界値として現れる。
