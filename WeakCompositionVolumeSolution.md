# weak composition volume解

この解では、停止前履歴を合計1以下の弱合成の連続版として見る。

離散では、非負整数の弱合成

```text
k_1+...+k_n<=m
```

を数える。連続版では、`k_i/m`を`x_i`に置き換え、`m`を無限大に送ることで

```text
x_i>=0, x_1+...+x_n<=1
```

という停止前領域が得られる。

弱合成の個数は

```text
binom(m+n,n)
```

なので、正規化体積は

```text
lim_{m->infty} binom(m+n,n)/m^n=1/n!.
```

したがって

```text
P(tau>n)=1/n!
```

であり、尾和により

```text
E[tau]=sum_{n>=0}P(tau>n)=sum_{n>=0}1/n!=e.
```

停止前確率は、弱合成の数え上げを連続体積へ移したものとして読める。
