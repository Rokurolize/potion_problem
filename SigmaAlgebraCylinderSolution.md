# sigma代数cylinder解

この解では、停止前prefixを無限列空間のcylinder集合として見て、その有限次元射影の測度を足す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。事象

```text
S_n <= 1
```

は、無限列空間上では最初の`n`座標だけで決まるcylinder集合である。その有限次元射影は

```text
{u_i>=0, u_1+...+u_n<=1}
```

である。累積和座標

```text
t_k=u_1+...+u_k
```

へ移すとJacobianは`1`で、射影は順序cylinder

```text
0<=t_1<=...<=t_n<=1
```

になる。これは`[0,1]^n`の`n!`個の順序室の一つなので

```text
P(S_n<=1)=1/n!.
```

よって

```text
E[tau]=sum_{n>=0}P(S_n<=1)=sum_{n>=0}1/n!=e.
```

この見方では、停止条件は無限過程の複雑な事象ではなく、各段で有限cylinderへ落ちる。
