# de Rham単体原始形式解

この解では、生存領域の体積を、単体上の体積形式を原始形式で一段ずつ境界へ送る操作として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えれば

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

長さ`n`の生存領域は

```text
Delta_n={u_i>=0, u_1+...+u_n<=1}
```

である。体積形式

```text
du_1...du_n
```

に対してEulerベクトル場

```text
E=sum_i u_i partial_{u_i}
```

を縮約すると、標準の原始形式`i_E(du_1...du_n)/n`が得られる。Stokesの定理で境界へ押し出すと、座標面`u_i=0`は向き付きで低次元単体に落ち、斜面

```text
u_1+...+u_n=1
```

だけが正味の寄与を持つ。したがって

```text
vol(Delta_n)=vol(Delta_{n-1})/n.
```

初期値は`vol(Delta_0)=1`なので

```text
P(S_n<=1)=vol(Delta_n)=1/n!.
```

ゆえに

```text
E[tau]=sum_{n>=0}1/n!=e.
```

ここでは単体体積公式を引用せず、体積形式をde Rham複体の原始形式で一段ずつ境界へ送って階乗を得ている。
