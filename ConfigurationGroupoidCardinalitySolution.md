# 配置群oid濃度解

この解では、停止前prefixを、区間`[0,1]`上の有限配置の群oid濃度として読む。

まず、長さ`n`の飲用列が生存している条件は

```text
S_n = X_1 + ... + X_n <= 1
```

である。累積時刻

```text
T_i = S_i
```

へ移すと、

```text
0 <= T_1 <= ... <= T_n <= 1
```

となる。対角集合は測度`0`なので、ほとんど至る所では不等号を厳密にしてよい。増分列と時刻列の変換は三角行列でJacobianが`1`である。

したがって`P(S_n <= 1)`は、区間内の`n`点順序配置のLebesgue質量である。

## ラベルを忘れた配置群oid

区間`[0,1]`上の`n`点ラベル付き配置の空間は、測度零の対角を除けば

```text
[0,1]^n
```

で、体積は`1`である。

ラベルを忘れた配置を対象とし、その自己同型としてラベルの置換を持つ群oidで割って見る。ほとんど全ての配置の自己同型は自明で、同じ非ラベル配置には`n!`通りのラベル付けがある。したがって、非ラベル`n`点配置の群oid濃度は

```text
1/n!
```

である。

これは順序付き時刻列の体積と一致する。なぜなら、非ラベル配置を時刻順に並べると、ほとんど一意に順序付きprefixになるからである。

## 全次数の群oid濃度

停止前prefix全体は、点数を固定しない有限配置全体である。よって期待停止回数は

```text
E[tau]
  = sum_{n >= 0} P(S_n <= 1)
  = sum_{n >= 0} 1/n!
  = e.
```

この証明では、単体体積を積分で出すのではなく、停止前prefixを有限配置群oidの濃度として読む。`1/n!`は単体計算の結果ではなく、ラベルを忘れるときの対称群の濃度として現れ、`e`は全有限配置の群oid濃度である。
