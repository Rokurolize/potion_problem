# dynamic programming tail解

この解では、停止前で残っている容量を状態変数にして動的計画として読む。

残容量を

```text
c=1-(x_1+...+x_k)
```

とする。まだ停止していないときだけ`c>=0`であり、次の一様乱数が`c`を超えれば停止する。

長さ`n`まで停止しない確率を、残容量`c`から始めた値

```text
v_n(c)
```

と書く。初期値は`v_0(c)=1`で、再帰は

```text
v_n(c)=int_0^c v_{n-1}(c-u) du
```

である。帰納的に

```text
v_n(c)=c^n/n!
```

となるので、元の初期状態`c=1`では

```text
P(tau>n)=v_n(1)=1/n!.
```

尾和公式より

```text
E[tau]=sum_{n>=0}P(tau>n)=sum_{n>=0}1/n!=e.
```

停止前確率は、残容量を状態とする動的計画の値関数を`c=1`で評価したものになっている。
