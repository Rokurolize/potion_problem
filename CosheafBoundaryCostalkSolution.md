# 余層の境界costalk解

この解では、境界`x`までに存在する停止前prefixの総量を、容量方向の余層として見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 容量区間上の余層

各`x in [0,1]`に対して、`S_n <= x`で生きている全prefixの総質量を

```text
M(x) = sum_{n >= 0} P(S_n <= x)
```

と置く。これは容量区間`[0,x]`に含まれるprefixを足し合わせる量であり、境界を広げると制限ではなく包含で進むので、層というより余層である。

境界を`x`から`x+h`へ広げる。新しく入るcostalkは、和が薄い層

```text
(x, x+h]
```

に入るprefixの総量である。

非空prefixを、直前のprefixと最後の一口に分ける。直前の和が`y <= x`なら、最後の一口が新しいcostalkへ入る条件は

```text
x-y < u <= x+h-y
```

であり、その幅は`h`である。一様密度は`1`なので、各既存prefixは新しいcostalkへ`h + o(h)`だけ寄与する。

したがって余層の構造写像が作る境界costalkは

```text
M(x+h)-M(x) = h M(x) + o(h).
```

よって

```text
M'(x) = M(x),
M(0) = 1.
```

初期値は空prefixだけである。したがって

```text
M(x)=exp(x).
```

求める値は

```text
E[tau] = M(1) = e.
```

この証明では、長さごとの単体体積を求めない。容量方向にprefix質量を貼り合わせる余層を作り、境界のcostalkが既存の全切片の総量そのものになることだけを見る。`e`は、この余層の境界値である。
