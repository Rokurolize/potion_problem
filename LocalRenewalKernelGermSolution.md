# 局所更新核芽解

待ち時間の密度を

```text
f(t) = 1_{0 <= t <= 1}
```

とする。更新核を

```text
R = delta_0 + f + f^{*2} + f^{*3} + ...
```

と置くと、到着数の期待値は

```text
E[tau] = R([0, 1])
```

である。

ここで `R` 全体を求める必要はない。区間 `[0, 1]` 上の局所的な芽だけを見ればよい。この区間に限れば、`f` は半直線の単位密度

```text
g(t) = 1_{t >= 0}
```

と同じである。さらに畳み込みは因果的なので、`[0, 1]` の値を作るとき、各成分は非負で合計が1以下になる。したがって、各成分は自動的に1以下であり、`f` の右端の切断はまだ見えない。

つまり `[0, 1]` 上では

```text
R = delta_0 + g + g^{*2} + g^{*3} + ...
```

と同じである。

この右辺は半直線上の最も単純なGreen核である。`g^{*n}` は

```text
g^{*n}(t) = t^{n-1}/(n-1)!   (n >= 1)
```

だから、`[0, 1]` 上の密度部分は

```text
sum_{n >= 1} t^{n-1}/(n-1)! = exp(t)
```

である。原点の `delta_0` は空和、つまり `n = 0` の到着数に対応する。

したがって

```text
E[tau]
  = R([0, 1])
  = 1 + int_0^1 exp(t) dt
  = e
```

となる。

この解では、停止時刻の分布も単体体積も先に求めない。長さ1の窓から見える更新核の芽では、一様分布の右端がまだ観測されず、半直線核のGreen関数だけが残る。
