# 自由モノイド体積の分配関数解

この解では、停止前の飲み方を「合計長さが`1`以下の有限語」として数える。

独立な一様乱数を`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

と置く。`S_0=0`を含めると、`tau`は境界`1`以下に残っているprefixの数である。

```text
tau = sum_{n>=0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n>=0} P(S_n <= 1).        (1)
```

## 有限語の体積として見る

長さ`n`の語を

```text
(u_1,...,u_n)
```

とし、各文字の重みをその値`u_i>=0`とする。語の総重みを

```text
|u| = u_1 + ... + u_n
```

と書く。

`S_n<=1`は、長さ`n`の語で総重みが`1`以下であるという条件である。和が`1`以下なら各`u_i<=1`は自動的に成り立つ。

よって(1)の右辺は、

```text
Z(1) = sum_{n>=0} volume { u_i>=0, |u| <= 1 }
```

という分配関数である。

## 容量を変数にする

容量を`r`に変えて

```text
Z(r) = sum_{n>=0} volume { u_i>=0, u_1+...+u_n <= r }
```

と置く。空語の寄与は常に`1`である。

非空語は、最後の文字`u`と、その前にある語`w`に一意に分解できる。

```text
wu,   |w| + u <= r.
```

最後の文字を`u`で固定すると、前半の語は容量`r-u`以下の任意の語でよい。したがって

```text
Z(r) = 1 + int_0^r Z(r-u) du.        (2)
```

これは、自由モノイドの「語は空語または語に一文字を足したもの」という分解を、体積で書いた式である。

## 指数関数が強制される

(2)を微分すると

```text
Z'(r) = Z(r),   Z(0) = 1.
```

したがって

```text
Z(r) = exp(r).
```

求める期待値は`r=1`の分配関数なので、

```text
E[tau] = Z(1) = exp(1) = e.
```

## この見方の芯

この証明は、長さごとの単体体積を先に求めない。停止前prefixを「合計重みが境界内にある有限語」全体としてまとめ、その分配関数が自由モノイドの再帰

```text
語 = 空語 + 語 * 一文字
```

に従うことだけを使う。

`e`は階乗級数を思い出した結果ではなく、容量を1だけ持つ自由モノイド体積の分配関数として現れる。
