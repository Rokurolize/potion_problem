# Palm的prefix計数解

この解では、停止時刻を「境界内にあるprefix点の平均個数」として見て、その点を1つ選ぶPalm的な数え方に移す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

`S_0=0`を含めると、停止時刻は境界`1`以下にあるprefix点の個数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## prefix点を1つ数える

右辺の`n`番目の項は、「長さ`n`のprefix点が境界内にある」寄与である。

`S_n<=1`の領域では

```text
u_1 + ... + u_n <= 1,
u_i >= 0.
```

ここでprefix点そのものを選ぶかわりに、その点までの経路

```text
0 = s_0 < s_1 < ... < s_n <= 1
```

を選ぶ。変数変換

```text
u_i = s_i - s_{i-1}
```

のJacobianは`1`なので、寄与の大きさは

```text
int_{0<s_1<...<s_n<=1} ds_1...ds_n
```

である。

## 選ばれたprefixの直前時刻を消す

この積分を、最後のprefix位置`x=s_n`で分解する。`x`を固定すると、その前にある`n-1`個の時刻

```text
0<s_1<...<s_{n-1}<x
```

の体積は、区間`[0,x]`に`n-1`個の点を順序つきで置く体積である。

順序を外して考えると、`[0,x]^{n-1}`の体積は`x^{n-1}`で、順序は`(n-1)!`通りある。したがって

```text
volume {0<s_1<...<s_{n-1}<x} = x^{n-1}/(n-1)!.
```

よって`n>=1`について

```text
P(S_n<=1)
= int_0^1 x^{n-1}/(n-1)! dx
= 1/n!.
```

また`n=0`の空prefixの寄与は`1`であり、これは`1/0!`である。

## 期待値

したがって

```text
E[tau]
= sum_{n >= 0} P(S_n <= 1)
= sum_{n >= 0} 1/n!
= e.
```

## この見方の芯

この証明は、単体の体積を一度に公式で出すのではなく、境界内にあるprefix点を1つ選び、その位置`x`でPalm的に分解する。

選ばれたprefix点が`x`にあるとき、そこまでの祖先prefixは`[0,x]`内の順序点である。最後の点を先に見ることで、`x^{n-1}/(n-1)!`という「祖先配置の数」が現れ、全ての長さを足すと指数関数になる。
