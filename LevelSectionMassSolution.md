# レベル断面の質量保存解

ここでは、停止する前の各段を、境界`1`の手前に積まれた断面として見る。

独立な一様乱数を`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

と置く。求めたいのは`E[tau]`である。

## 生存している段を全部重ねる

`n`口飲んだ後もまだ止まっていないことは、

```text
S_n <= 1
```

である。ここで`n=0`も含めると、`S_0=0`なので最初の段は必ず生存している。

停止時刻は「生存している段数」を数えたものだから、

```text
tau = sum_{n>=0} 1_{S_n <= 1}
```

である。したがって

```text
E[tau] = sum_{n>=0} P(S_n <= 1).        (1)
```

ここまでは単なる断面の足し合わせで、まだ各確率の値は計算していない。

## 断面量の再帰

`a_n(x)`を、`n`口後の和が`x`以下にいる確率密度の断面量として考える。厳密には、`S_n`の密度を`[0,1]`上だけ見たものとする。

`n=0`では原点に質量`1`がある。1口増やすとは、一様な長さだけ右に流すことなので、`0 < x < 1`では

```text
a_{n+1}(x) = int_0^x a_n(y) dy.        (2)
```

つまり次の断面は、前の断面の左側にある全質量を積分したものになる。境界を越えた質量は戻ってこない。

この再帰から、断面の総量

```text
b_n = P(S_n <= 1)
```

を直接足し上げる代わりに、全断面をまとめた量

```text
A(x) = sum_{n>=0} a_n(x)
```

を見る。原点の質量を除いた`0 < x < 1`では、(2)を全ての`n`で足して

```text
A(x) = int_0^x A(y) dy + 1.        (3)
```

右辺の`1`は、原点に置かれた初期質量が各`x>0`へ一度だけ届く分である。

## 保存則が指数関数を作る

(3)を微分すると

```text
A'(x) = A(x),   A(0+) = 1.
```

よって

```text
A(x) = exp(x).
```

ここで欲しい期待値は、全ての生存段の総量である。断面を`0`から`1`まで積分するかわりに、(3)を`x=1`で読むと、

```text
A(1) = 1 + int_0^1 A(y) dy.
```

一方、右辺の

```text
1 + int_0^1 A(y) dy
```

は、初期段の`1`と、以後の各生存段の総質量を足したもの、つまり(1)そのものである。したがって

```text
E[tau] = A(1) = exp(1) = e.
```

## この見方の要点

個々の`P(S_n <= 1)`を先に`1/n!`と求めてから足すのではない。生存している全ての段を一枚の断面場`A`に重ねると、左側の質量が次の断面を作るという保存則だけが残る。その保存則が`A'=A`を出し、境界`1`で読んだ値がそのまま平均飲用回数になる。
