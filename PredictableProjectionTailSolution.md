# predictable projection tail解

この解では、停止前指示関数の期待値を予測可能射影として足す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前であることを

```text
I_n=1_{S_n<=1}
```

で表すと

```text
tau=sum_{n>=0} I_n.
```

この列の期待値列

```text
E[I_n]=P(S_n<=1)
```

を予測可能射影の質量として見る。固定した`n`では、事象`S_n<=1`は最初の`n`個の増分だけで決まり、その測度は

```text
vol{u_i>=0, u_1+...+u_n<=1}=1/n!.
```

したがって

```text
E[tau]=sum_{n>=0} E[I_n]
      =sum_{n>=0} 1/n!
      =e.
```

予測可能射影という言葉で見ると、ランダムな停止前指示列を、各時刻の決定論的な生存質量列へ押し出してから足している。
