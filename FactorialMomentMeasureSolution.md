# 階乗モーメント測度解

この解では、生存prefixをPoisson過程の階乗モーメント測度の台として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めれば

```text
E[tau]=sum_{n>=0} P(S_n <= 1).
```

区間`[0,1]`上の単位強度Poisson過程を考える。その`n`次階乗モーメント測度は、互いに異なる`n`点の順序なし配置にLebesgue測度を与える。これを時刻順に並べると、順序付き領域

```text
0 <= t_1 <= ... <= t_n <= 1
```

の質量は

```text
1/n!.
```

一方、変数変換

```text
t_k = u_1 + ... + u_k
```

はJacobian`1`で、停止前prefix領域

```text
u_i >= 0,  u_1+...+u_n <= 1
```

をこの順序付き領域へ移す。したがって

```text
P(S_n <= 1)=1/n!.
```

全階層を足せば

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

ここではPoisson確率の空白因子を主役にしない。階乗モーメント測度が、順序を忘れた配置から停止前prefixの単体体積を直接返すことだけを使う。
