# normal fan単体解

この解では、生存単体を法線ファンの境界方向で切り、各次数の体積再帰を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。条件`S_n<=x`は、正直交錐を法線`(1,...,1)`の半空間で切った単体である。境界法線方向に`x`を増やすと、切断面の断面体積がそのまま体積の微分になる。

`A_n(x)=P(S_n<=x)`と書けば、`0<=x<=1`で

```text
A_0(x)=1,
A_n'(x)=A_{n-1}(x).
```

これは法線ファンのうち、停止境界に対応する一つの錐だけを見ていることに等しい。初期値から

```text
A_n(x)=x^n/n!.
```

したがって

```text
E[tau]=sum_{n>=0}A_n(1)=e.
```

ここで必要なのは法線ファン全体の分類ではない。停止境界の法線方向に動かしたとき、断面が一段低い同型単体になることだけである。
