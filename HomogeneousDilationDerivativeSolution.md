# homogeneous dilation derivative解

この解では、単体体積を同次拡大と微分の整合性だけで決める。

長さ`n`の停止前領域を

```text
D_n(x)={u_i>=0, u_1+...+u_n<=x}
```

とし、その体積を`A_n(x)`と書く。元の問題では

```text
E[tau]=sum_{n>=0}A_n(1)
```

である。

第一に、`D_n(x)`は`D_n(1)`を倍率`x`で拡大したものなので

```text
A_n(x)=x^n A_n(1).
```

第二に、最後の変数で切れば

```text
A_n(x)=int_0^x A_{n-1}(x-u) du
```

だから

```text
A_n'(x)=A_{n-1}(x).
```

同次式を微分して`x=1`を代入すると

```text
n A_n(1)=A_{n-1}(1).
```

`A_0(1)=1`なので帰納的に

```text
A_n(1)=1/n!.
```

従って

```text
E[tau]=sum_{n>=0}A_n(1)=sum_{n>=0}1/n!=e.
```

この見方では、積分値を直接計算せず、領域の同次性と境界を一枚はがす微分則が factorial を強制する。
