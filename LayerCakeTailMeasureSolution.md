# layer cake尾測度解

この解では、停止時刻を整数値変数のlayer cake表示で測る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。正整数値確率変数には

```text
E[tau]=sum_{n>=0} P(tau>n)
```

が成り立つ。ここで`tau>n`は、最初の`n`個を足してもまだ`1`を越えないこと、すなわち

```text
S_n<=1
```

である。したがって

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

`n`個の一様増分の取り得る領域は単位立方体で、そのうち`S_n<=1`の部分は

```text
{u_i>=0, u_1+...+u_n<=1}
```

という標準単体である。この体積は`1/n!`だから

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

この見方では、停止時刻そのものを分布から求めない。整数値変数の尾測度を積み上げるだけで、停止前prefixの単体体積の和がそのまま現れる。
