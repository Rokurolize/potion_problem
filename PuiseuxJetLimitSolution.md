# Puiseuxジェット極限解

この解では、境界`x=0`まわりのジェットをPuiseux級数の退化極限として見て、最終的に指数級数へ戻す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`以下のprefix総量

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

を考える。`0<=x<=1`では、一様乱数の上端制約は生存単体の中で効かないため

```text
P(S_n<=x)=x^n/n!.
```

したがって`F`の`0`まわりのジェットは

```text
1, 1, 1, ...
```

を階乗で割ったものになる。Puiseux的な分岐を許しても、この問題では境界が滑らかな一次法線だけなので、分岐指数は全て整数へ退化する。よって

```text
F(x)=sum_{n>=0} x^n/n! = e^x.
```

境界`1`で

```text
E[tau]=F(1)=e.
```

この証明では、複雑な特異境界を想定したPuiseux展開が、この問題では通常のTaylorジェットへ潰れることを使っている。
