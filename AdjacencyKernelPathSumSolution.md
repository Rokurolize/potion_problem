# 隣接核path-sum解

この解では、停止前prefixを連続状態グラフの全経路和として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めて数えれば

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

残り容量を状態`r in [0,1]`とする。状態`r`から一口`u`を飲み、まだ停止しないなら次の状態は`r-u`である。したがって連続状態グラフの隣接核は

```text
(Af)(r)=int_0^r f(r-u) du.
```

一歩の経路重みは一様密度`1`によるLebesgue測度である。

初期状態`r`から出る長さ`n`の受理経路の総重みは

```text
A^n 1(r)
```

である。したがって全停止前経路のpath-sumは

```text
P(r)=sum_{n>=0} A^n 1(r).
```

これは隣接核のresolvent方程式

```text
P(r)=1+(AP)(r)
    =1+int_0^r P(r-u) du
```

を満たす。変数を替えると

```text
P(r)=1+int_0^r P(v) dv.
```

よって

```text
P'(r)=P(r),
P(0)=1.
```

したがって

```text
P(r)=exp(r).
```

初期容量は`1`なので

```text
E[tau]=P(1)=e.
```

この証明では、長さごとの単体体積を出さない。停止前prefixを、容量グラフの隣接核が作る全受理経路の和としてまとめる。`e`は、連続隣接核のpath-sumを初期状態で評価した値である。
