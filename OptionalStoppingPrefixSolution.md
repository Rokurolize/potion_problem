# optional stopping prefix解

この解では、停止時刻を止められた過程のprefix報酬として見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。止められた過程で時刻`n`の前にまだ生きている指示関数を

```text
I_n = 1_{S_n<=1}
```

と置く。ただし`S_0=0`である。停止時刻は、停止前に集める単位報酬の総和で

```text
tau = sum_{n>=0} I_n.
```

よって単調収束から

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

各項は標準単体

```text
{u_i>=0, u_1+...+u_n<=1}
```

の体積であり、

```text
P(S_n<=1)=1/n!.
```

したがって

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

optional stoppingと呼んでいるが、必要なのはマルチンゲールの停止定理ではない。止められた過程の報酬を停止前prefixの尾和として読むだけである。
