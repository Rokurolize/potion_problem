# boundary flux recursion解

この解では、単体体積を境界流束の再帰だけで決める。

停止前領域を

```text
D_n(x)={u_i>=0, u_1+...+u_n<=x}
```

とし、`A_n(x)=vol(D_n(x))`とする。尾和公式で必要なのは

```text
E[tau]=sum_{n>=0}A_n(1).
```

ベクトル場

```text
F(u)=u/n
```

を`R^n`で考えると、`div F=1`である。発散定理から

```text
A_n(x)=int_{boundary D_n(x)} F・nu dS.
```

座標面`u_i=0`では`F・nu=0`なので流束はない。寄与するのは斜面`u_1+...+u_n=x`だけである。この斜面を最後の座標で解くと、流束測度は

```text
(x/n) dS_n = (1/n) d(u_1,...,u_{n-1})
```

に落ち、基底は`D_{n-1}(x)`である。従って

```text
A_n(x)=x A_{n-1}(x)/n.
```

`A_0(x)=1`から

```text
A_n(1)=1/n!.
```

よって

```text
E[tau]=sum_{n>=0}1/n!=e.
```

この証明では、体積積分を境界流束へ移し、座標面の消滅と斜面の寄与だけでfactorialを得る。
