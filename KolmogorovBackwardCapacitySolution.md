# Kolmogorov後退容量解

この解では、残り容量を状態とする殺されたMarkov連鎖の後退方程式として読む。

状態`r`は、あと`r`だけ飲めることを表す。`0 <= r <= 1`で一口`U`を飲むと、

```text
U <= r
```

なら状態は`r-U`へ移り、`U > r`なら吸収墓地へ落ちる。吸収される一口も停止時刻に数える。

`b(r)`を状態`r`から始めた期待残り飲用回数とする。求めたい値は`b(1)`である。

## 後退方程式

最初の一口をまず数える。その後、生き残った場合だけ将来の期待値を足すので

```text
b(r) = 1 + int_0^r b(r-u) du.
```

これは、遷移核を

```text
P_r(dx) = 1_{0 <= x <= r} dx
```

と見たときの後退方程式

```text
b = 1 + P b
```

である。右辺は現在状態`r`から見た次状態の期待値なので、前向き分布を解かずに現在状態だけで閉じている。

変数を入れ替えると

```text
b(r) = 1 + int_0^r b(x) dx.
```

したがって

```text
b'(r) = b(r).
```

境界では、`r=0`から始めると次の一口で必ず越えるため

```text
b(0)=1.
```

よって

```text
b(r)=exp(r)
```

であり、

```text
E[tau] = b(1)=e.
```

この証明では、停止時刻の分布や生存確率列を先に作らない。残り容量から見た後退方程式だけを立て、吸収される一口を単位報酬として入れる。`e`は、殺された一様ジャンプ核の後退方程式が持つ容量方向の指数解である。
