# abstract interpretation widening解

この解では、停止前prefixを抽象解釈で失われずに残る安全状態として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。具体状態を残り容量`x`、抽象状態を区間`[0,x]`と見る。入力`u`を読む一段の遷移で、`u<=x`なら残り容量は`x-u`に移る。`u>x`なら安全抽象状態から外れて停止する。

深さ`n`まで安全に残る具体入力列は

```text
u_1+...+u_n<=1
```

を満たすものだけである。したがって、抽象解釈が深さ`n`で保持する安全領域の体積は標準単体の体積

```text
1/n!
```

である。停止時刻は安全prefixの個数として

```text
tau = sum_{n>=0} 1_{S_n<=1}
```

と書けるので、期待値は

```text
E[tau] = sum_{n>=0} P(S_n<=1)
       = sum_{n>=0} 1/n!
       = e.
```

wideningの言葉では、抽象状態を荒く広げるのではなく、容量制約で正確に切った安全prefix領域を各深さで数えている。
