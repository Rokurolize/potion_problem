# 有限差分尾和解

`X_i` を独立な `[0, 1]` 一様分布とし、`S_n = X_1 + ... + X_n` とする。到着数は

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので、必要なのは各 `P(S_n <= 1)` だけである。

一様分布の密度は、ヘヴィサイド関数 `H(x) = 1_{x >= 0}` を使うと

```text
1_{0 <= x <= 1} = H(x) - H(x - 1)
```

である。畳み込みでは平行移動と差分がそのまま効くので、`n` 個の和の密度は

```text
(H - H(.-1))^{*n}
```

と書ける。ここで

```text
H^{*n}(x) = x_+^{n-1} / (n-1)!
```

だから、分布関数はさらに1回積分して

```text
P(S_n <= t)
  = 1/n! sum_{k=0}^n (-1)^k binom(n, k) (t - k)_+^n
```

となる。これはIrwin-Hall分布の公式だが、ここでは有限差分として出しているだけである。

いま欲しいのは `t = 1` の値である。すると

```text
(1 - k)_+^n
```

は `k = 0` だけが寄与し、`k = 1` は `0^n`、`k >= 2` は正部が0になる。したがって `n >= 1` で

```text
P(S_n <= 1) = 1/n!
```

である。`n = 0` は空和なので `P(S_0 <= 1) = 1` とする。

よって単調収束により

```text
E[tau]
  = sum_{n >= 0} P(S_n <= 1)
  = sum_{n >= 0} 1/n!
  = e
```

となる。分布全体を求める必要はない。有限差分表示では、境界点 `t = 1` に代入した瞬間に差分のほとんどが消え、最初の項だけが残る。
