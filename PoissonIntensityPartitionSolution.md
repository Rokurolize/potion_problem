# Poisson強度分配関数解

この解では、停止前prefixを強度パラメータつきで数え、Poisson過程の正規化から分配関数を読む。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。求めたい量は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 強度を入れて全階層を重みづける

`z >= 0`について

```text
Z(z) = sum_{n >= 0} z^n P(S_n <= 1)
```

を考える。`z=1`を代入すれば求める値になる。

`S_n <= 1`の上では、増分列から部分和列への変換

```text
(U_1,...,U_n) <-> (S_1,...,S_n)
```

のJacobianは`1`であり、

```text
0 <= S_1 <= ... <= S_n <= 1
```

の順序付き配置空間を得る。したがって`z^n P(S_n <= 1)`は、`[0,1]`上の順序付き`n`点配置に重み`z^n`を付けた総質量である。

## Poisson過程の正規化を使う

強度`z`のPoisson過程を`[0,1]`上で見る。点数が`n`で、順序付き点が

```text
0 <= t_1 <= ... <= t_n <= 1
```

にある密度は

```text
exp(-z) z^n dt_1 ... dt_n
```

である。

つまり、`Z(z)`はPoisson配置確率測度から、全階層に共通の正規化因子`exp(-z)`を外した全質量である。Poisson配置確率測度の全質量は`1`なので、

```text
exp(-z) Z(z) = 1.
```

よって

```text
Z(z) = exp(z).
```

最後に`z=1`として

```text
E[tau] = Z(1) = e.
```

この証明では、各階層の体積`1/n!`を先に出さない。強度`z`を入れた停止前prefix全体をPoisson配置の分配関数として一括で正規化し、その正規化定数そのものから`e`を読む。
