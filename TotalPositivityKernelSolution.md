# 全正値畳み込み核解

この解では、一様分布の畳み込みを全正値核の先頭部分として読み、境界`1`までに必要な成分だけを取り出す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 先頭の畳み込み核

一様密度

```text
f(x) = 1_{0 <= x <= 1}
```

を考える。`S_n`の密度は`f^{*n}`である。一般にはこれはIrwin-Hall密度で、整数点ごとに多項式片が変わる。

しかし`P(S_n <= 1)`では`0 <= x <= 1`しか積分しない。この範囲では、右端`1`で切られた影はまだ出ない。したがって必要な核は、半直線上の定数密度

```text
h(x) = 1_{x >= 0}
```

の畳み込みの先頭部分と一致する。

`h`は畳み込み順序に対して完全に単調な先頭核を作り、各畳み込みは

```text
h^{*n}(x) = x^{n-1}/(n-1)!  (x >= 0)
```

となる。これは、全正値核の先頭セルでは畳み込みが一回積分の反復に落ちる、ということでもある。

よって

```text
P(S_n <= 1)
  = int_0^1 h^{*n}(x) dx
  = int_0^1 x^{n-1}/(n-1)! dx
  = 1/n!.
```

空prefixの項は`1`である。したがって

```text
E[tau]
  = sum_{n >= 0} 1/n!
  = e.
```

この証明では、畳み込み密度全体を求めない。全正値な畳み込み核のうち、境界`1`までに見える先頭部分だけを使う。`e`は、その先頭核の全畳み込み次数を足した値として現れる。
