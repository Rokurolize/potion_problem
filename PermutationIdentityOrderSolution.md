# 恒等順列順序統計解

この解では、累積和が`1`以下に残る条件を、独立な点を並べたときに恒等順列の室へ入ることとして読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

長さ`n`について、変数を

```text
T_k=U_1+...+U_k
```

に替える。Jacobianは`1`であり、`S_n<=1`は

```text
0<T_1<T_2<...<T_n<1
```

に変わる。

一方、`n`個の独立一様点`X_1,...,X_n`を区間`[0,1]`に置くと、ほとんど確実に相異なる。`n!`個の順序

```text
X_{\sigma(1)}<...<X_{\sigma(n)}
```

は対称性で同じ確率を持つ。したがって、恒等順序

```text
X_1<...<X_n
```

の確率は`1/n!`である。その領域は上の`T`領域と同じなので

```text
P(S_n<=1)=1/n!.
```

よって

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

階乗は単体積分の値ではなく、独立点の順序を決める順列の個数として現れる。元の増分は、恒等順序の隣接差に戻されている。
