# 対称化射影解

この解では、生存prefixの順序単体を、立方体上の対称化射影で取り出す。

長さ`n`のprefixについて、累積時刻

```text
T_i = X_1 + ... + X_i
```

を置く。`S_n <= 1`であることは

```text
0 <= T_1 <= T_2 <= ... <= T_n <= 1
```

と同じであり、増分列から累積時刻列への変換はJacobian`1`である。

## 立方体を対称化する

立方体`[0,1]^n`上で、対称群`S_n`が座標を入れ替える。対角集合は体積`0`なので、ほとんど全ての点の軌道は`n!`個の点を持つ。

対称化射影で各軌道を一つの代表、つまり昇順に並んだ点へ送ると、基本領域は

```text
0 <= t_1 <= ... <= t_n <= 1
```

である。全立方体の体積は`1`で、`n!`個の基本領域は合同なので、この領域の体積は

```text
1/n!.
```

したがって

```text
P(S_n <= 1) = 1/n!.
```

尾和公式より

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
       = sum_{n >= 0} 1/n!
       = e.
```

この証明では、単体積分を直接計算しない。累積時刻列を立方体へ置き、対称化射影の基本領域として生存prefixを読む。`1/n!`は積分結果ではなく、対称群軌道を一つに割る規格化として現れる。
