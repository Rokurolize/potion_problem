# Volterra固有関数解

`m(t)` を、残り時間が `t` のときに数える到着数の期待値とする。最初の1回は必ず数えるので、

```text
m(t) = 1 + int_0^t m(t - x) dx
```

である。右辺の積分はVolterra作用素

```text
(Vf)(t) = int_0^t f(t - x) dx
```

だから、

```text
(I - V)m = 1
```

を解けばよい。

ここで指数関数を見ると

```text
V(exp)(t)
  = int_0^t exp(t - x) dx
  = exp(t) - 1
```

となる。したがって

```text
(I - V)exp(t) = exp(t) - (exp(t) - 1) = 1
```

である。つまりVolterra方程式の解は

```text
m(t) = exp(t)
```

である。

欲しいのは残り時間1の期待値なので

```text
E[tau] = m(1) = e
```

となる。

この解では停止時刻の分布も、`n` 次元単体の体積も先に求めない。到着後に残り時間だけが状態として残ることをVolterra作用素に押し込み、指数関数がその作用素のずれをちょうど打ち消すことだけを見る。
