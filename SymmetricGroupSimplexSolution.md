# symmetric group simplex解

この解では、標準単体を対称群の順序付けで測る。

元の停止条件は

```text
S_n=U_1+...+U_n<=1.
```

累積和座標`T_k=S_k`へ移ると、領域は順序単体

```text
0<=T_1<=...<=T_n<=1
```

である。`[0,1]`上の`n`点配置をラベル付きで選ぶと全体体積は`1`である。ほとんど全ての配置には異なる`n!`通りのラベル順序があり、そのうち一つだけが上の順序単体に入る。

よって

```text
vol{0<=T_1<=...<=T_n<=1}=1/n!.
```

尾和公式から

```text
E[tau]=sum_{n>=0}P(S_n<=1)=sum_{n>=0}1/n!=e.
```

対称群がラベル順序を忘れるときの商体積が、そのまま停止前prefixの確率になる。
