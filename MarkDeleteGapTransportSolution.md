# 印付き点削除の質量輸送解

この解では、停止前prefixを順序付き点配置に移したあと、点を一つ印付けして削除する。単体体積を積分で計算するかわりに、削除後に残るギャップの全長が常に`1`であることを使う。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止前prefixを数えると

```text
tau = sum_{n>=0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n>=0} P(S_n <= 1).
```

`n`個の累積時刻

```text
t_i = U_1 + ... + U_i
```

へ移ると、`S_n<=1`は

```text
0 < t_1 < ... < t_n <= 1
```

である。Jacobianは`1`なので、この領域の質量を`A_n`と書けば

```text
E[tau] = sum_{n>=0} A_n,
A_0 = 1.
```

## 点を一つ印付けする

`n>=1`とする。`n`点配置に点を一つ印付けした空間の質量は

```text
n A_n
```

である。印を外せば各配置に`n`通りの選び方があるからである。

この印付き配置から、印の付いた点だけを削除する。残った`n-1`点を

```text
0 < s_1 < ... < s_{n-1} <= 1
```

とする。削除した点を戻すには、`[0,1]`のうち、既存点をまたがないどこかのギャップに1点を挿入すればよい。

具体的には

```text
(0,s_1), (s_1,s_2), ..., (s_{n-1},1)
```

のどこかに新しい点を置く。これらのギャップ長の合計は、残った配置が何であっても

```text
s_1 + (s_2-s_1) + ... + (1-s_{n-1}) = 1
```

である。

したがって、各`(n-1)`点配置の上にある「削除された印付き点を戻す自由度」の全長は常に`1`であり、

```text
n A_n = A_{n-1}.        (1)
```

となる。

## 係数が決まる

`A_0=1`と(1)から帰納的に

```text
A_n = 1/n!
```

である。よって

```text
E[tau]
= sum_{n>=0} A_n
= sum_{n>=0} 1/n!
= e.
```

## この見方の芯

ここでは`n`重積分を解かない。`n`点の生存prefix配置を「点を一つ印付けして削る」側と「`n-1`点配置のギャップへ点を戻す」側で二重に数える。

一様性は、挿入位置の長さがそのまま質量になる点に使われる。独立性は、累積時刻への変換後に各prefix長で同じLebesgue質量を扱えることに使われる。答えの`e`は、ギャップ長の和が常に`1`であるために生じる再帰`nA_n=A_{n-1}`を全長さで足した値として現れる。
