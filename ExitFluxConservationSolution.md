# 出口流束保存解

この解では、停止前prefixを直接数えるのではなく、境界を越えて外へ出る最後の一口を流束として数える。停止は必ず一度だけ起きるので、出口側の総質量は最初から`1`に正規化されている。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えれば

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 境界内の終点密度

原点以外の停止前prefixが点`x`に来る総密度を

```text
q(x) dx
```

と書く。ここで`0 < x < 1`だけを見る。

点`x`へ来るには、直前のprefixが`0 <= y <= x`にあり、最後の一口が`x-y`であればよい。一様分布の密度は`1`なので、原点から直接来る流束`1`も含めて

```text
q(x) = 1 + int_0^x q(y) dy.
```

したがって

```text
q'(x) = q(x),
q(0) = 1,
```

ゆえに

```text
q(x) = exp(x).
```

ここまでは、各段の`P(S_n<=1)`を求めていない。境界内へ流れ込む全世代の密度を、流入保存則だけで求めている。

## 出口側で正規化を確認する

点`x`にいるprefixが次の一口で停止するには

```text
U > 1 - x
```

であればよい。この確率は`x`である。したがって、原点以外のprefixから外へ出る流束の総量は

```text
int_0^1 x q(x) dx.
```

`q(x)=exp(x)`を入れると

```text
int_0^1 x exp(x) dx = 1.
```

これは偶然の計算ではない。各標本路では停止を引き起こす一口はちょうど一つだけなので、全ての段と全ての直前位置に分けて出口流束を足せば総質量は必ず`1`になる。この出口側の保存則と、内側の流入密度`q'=q`が噛み合っている。

## 期待値に戻す

停止前prefixの期待個数は、空prefix`S_0=0`の`1`と、区間`(0,1]`内に入った全prefix密度の質量の和である。

```text
E[tau]
= 1 + int_0^1 q(x) dx
= 1 + int_0^1 exp(x) dx
= e.
```

## この見方の芯

この証明の主役は尾確率でも単体体積でもない。境界内では、既にあるprefixの総密度がそのまま次のprefixの流入密度になる。境界では、各prefixが外へ出る確率`x`を持ち、その出口流束を全て足すと「停止は一度だけ」という保存則で`1`になる。

`e`は、内側で自己再生する流入密度と、外側で一度だけ消える出口流束を同時に満たす正規化として現れる。
