# Lie対称性ODE解

この解では、境界容量を動かしたときの停止前prefix総量が持つ無限小対称性から指数関数を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

境界を`x`にした停止前prefix総量を

```text
F(x) = sum_{n >= 0} P(S_n <= x)
```

と置く。求める値は`F(1)`である。

## 無限小対称性

境界を`x`から`x+h`へ動かす。新しく入るprefixは、既存prefixに最後の一口を足して和が薄い層`(x,x+h]`へ入るものだけである。

既存prefixの和を`y <= x`とすると、最後の一口が薄い層へ入る幅は

```text
x-y < u <= x+h-y
```

で、長さは`h`である。一様密度は`1`なので、各既存prefixは同じ無限小量`h`だけ新しい層へ送られる。

したがって無限小変換

```text
x -> x + h
```

に対して、総量は

```text
F -> F + h F + o(h)
```

と変わる。これは、境界平行移動の生成ベクトル場が

```text
X = d/dx + F d/dF
```

として働く、というLie対称性である。積分曲線は

```text
dF/dx = F.
```

初期条件は、容量`0`では空prefixだけなので

```text
F(0)=1.
```

よって

```text
F(x)=exp(x).
```

したがって

```text
E[tau] = F(1) = e.
```

この証明では、各長さの単体体積を計算しない。境界を無限小に動かしたとき、既存prefix総量がそのまま新しい層の生成率になるという対称性だけを見る。`e`は、そのLie生成ベクトル場の流れを時刻`1`だけ進めた値である。
