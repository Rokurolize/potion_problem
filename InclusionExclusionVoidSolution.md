# 包除空白反転解

この解では、停止前prefixの総量を、Poisson空白事象の包除展開を符号反転して読む。

長さ`n`の生存prefixは、累積時刻

```text
0 <= t_1 <= ... <= t_n <= 1
```

に移すと、区間`[0,1]`上の順序付き`n`点配置である。増分から累積時刻への変換はJacobian`1`なので

```text
P(S_n <= 1)
```

はこの順序単体の体積である。

## 空白を包除で見る

単位強度のPoisson点過程を`[0,1]`上に置く。区間に点が無い確率は

```text
P(N([0,1])=0)=e^{-1}.
```

この空白確率は、区間内に点が現れる可能性を包除で消す展開として

```text
e^{-1} = sum_{n >= 0} (-1)^n / n!
```

と読める。ここで`1/n!`は、`n`個の非ラベル点配置を時刻順に並べた順序単体の体積である。符号`(-1)^n`だけが、「点が一つも無い」という事象へ向けて全配置を相殺するために入っている。

元の問題で欲しいのは、空白を作るために相殺された配置を符号なしで全部戻した総量である。したがって符号を外した全配置展開は

```text
sum_{n >= 0} 1/n! = e.
```

尾和公式より

```text
E[tau] = sum_{n >= 0} P(S_n <= 1),
```

各項が上の順序単体体積なので

```text
E[tau] = e.
```

この証明では、各単体を個別に積分しない。Poisson空白事象を作る包除展開を見て、空白のために交互符号で消された全有限配置を符号なしに戻す。`e`は、空白確率`e^{-1}`を作る包除の反転側に現れる総配置量である。
