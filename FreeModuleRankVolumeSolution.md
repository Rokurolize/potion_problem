# free module rank volume解

この解では、長さ`n`のprefixを階数`n`の自由加群の非負座標として見る。

基底を

```text
e_1,...,e_n
```

とし、prefixを

```text
x_1 e_1+...+x_n e_n
```

と書く。停止前条件は、係数和が1以下であること、つまり

```text
x_i>=0,  x_1+...+x_n<=1
```

である。これは自由基底が張る標準単体である。

基底の平行体の体積は1で、単体はその`1/n!`なので

```text
P(tau>n)=1/n!.
```

よって

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

階数`n`の自由加群の基底係数としてprefixを読むと、停止前領域は基底単体であり、階乗分母は基底を並べ替える対称性から現れる。
