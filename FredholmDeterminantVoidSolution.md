# Fredholm行列式空白解

この解では、停止前prefixの全質量を、区間`[0,1]`上の空白確率を表すFredholm行列式の逆正規化として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 配置測度へ移す

`S_n <= 1`を累積時刻

```text
0 <= t_1 <= ... <= t_n <= 1
```

に移す。Jacobianは`1`なので、長さ`n`の停止前prefixの質量は順序付き`n`点配置の体積である。

この順序付き配置を、非ラベル配置として見ると、`n!`通りの順序を割るので、全階層を足した質量は

```text
sum_{n >= 0} 1/n!
```

である。ただし、ここではこの級数を直接計算するのではなく、空白のFredholm行列式から読む。

区間`[0,1]`上で、恒等核`K=1_{[0,1]}`を「1点を置く強度」として考える。Poisson配置の空白確率は、有限階層のFredholm行列式展開として

```text
det(I - K) = exp(-Tr K) = exp(-1)
```

と読める。ここで`Tr K = int_0^1 1 dx = 1`である。

停止前prefix測度は、Poisson配置確率に掛かっている共通の空白因子`exp(-1)`を外した配置測度である。したがって全prefix質量は

```text
det(I - K)^{-1} = exp(1).
```

よって

```text
E[tau] = e.
```

この証明では、各`P(S_n <= 1)`の単体体積を個別に計算しない。空白確率をFredholm行列式として圧縮し、そこに含まれる空白正規化を外すことで停止前prefixの全質量を読む。`e`は、そのFredholm行列式の逆数として現れる。
