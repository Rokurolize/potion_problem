# quotient module tail解

この解では、prefix空間を合計値で割った商の層として停止前領域を測る。

長さ`n`の履歴空間は非負直交象限であり、線形写像

```text
q(x_1,...,x_n)=x_1+...+x_n
```

を持つ。停止前条件は

```text
q(x)<=1
```

である。各値`q(x)=r`のファイバーは、合計が`r`の`n-1`単体で、その体積密度は

```text
r^{n-1}/(n-1)!
```

である。

したがって、商方向`r`を`0`から`1`まで積分すれば

```text
P(tau>n)=int_0^1 r^{n-1}/(n-1)! dr=1/n!.
```

尾和公式から

```text
E[tau]=sum_{n>=0}P(tau>n)=sum_{n>=0}1/n!=e.
```

履歴空間を「合計値方向」と「同じ合計値を持つ商ファイバー」に分けると、停止前確率は商ファイバー体積の累積になる。
