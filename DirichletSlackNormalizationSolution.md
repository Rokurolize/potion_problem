# Dirichlet余白正規化解

この解では、`S_n<=1`の「余り」を新しい成分として加え、確率単体の正規化定数として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止時刻は、境界`1`以下にいるprefixの個数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 余りを成分にする

`S_n<=1`の領域では

```text
u_1 + ... + u_n <= 1,
u_i >= 0.
```

ここに余り

```text
u_{n+1} = 1 - (u_1 + ... + u_n)
```

を加えると、

```text
u_1 + ... + u_{n+1} = 1,
u_i >= 0
```

という`n`次元確率単体になる。逆に、この単体上の点から最後の成分`u_{n+1}`を捨てれば、元の`S_n<=1`の領域に戻る。

つまり

```text
P(S_n <= 1)
```

は、`n+1`成分の確率単体を`(u_1,...,u_n)`で見たLebesgue体積である。

## Dirichlet正規化定数

`n+1`成分のDirichlet分布`Dirichlet(1,...,1)`は、この確率単体上で一様な分布である。その密度は定数で、

```text
n!
```

でなければならない。なぜなら、一般のDirichlet正規化定数は

```text
Gamma(alpha_1+...+alpha_{n+1})
/
(Gamma(alpha_1)...Gamma(alpha_{n+1}))
```

であり、すべての`alpha_i`を`1`にすると

```text
Gamma(n+1) = n!
```

になるからである。

密度`n!`が単体全体で積分して`1`になるので、単体の体積は

```text
1/n!.
```

したがって

```text
P(S_n <= 1) = 1/n!.
```

## 期待値

これを生存prefixの和に戻すと

```text
E[tau]
= sum_{n >= 0} P(S_n <= 1)
= sum_{n >= 0} 1/n!
= e.
```

## この見方の芯

この証明では、`n`個の一様乱数の和を、最後に残った余白を含む`n+1`個の割合として見る。

境界`1`は「総量が1の割合ベクトル」という制約になり、答えに現れる`1/n!`は単体積分を直接計算した値ではなく、`Dirichlet(1,...,1)`の正規化定数の逆数として現れる。停止時刻の期待値は、その正規化定数の逆数を全次元で足したものとして`e`になる。
