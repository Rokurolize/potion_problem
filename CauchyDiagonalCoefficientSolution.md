# Cauchy対角係数解

この解では、生存prefixの列を二変数母関数の対角係数として拾う。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

`0<=x<=1`で

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

を考える。長さ`n`の項は、`n`回の一口追加積分で定数`1`を送ったものだから

```text
P(S_n<=x)=x^n/n!.
```

この階乗割りをCauchy係数として書けば

```text
x^n/n! = [z^n] exp(xz).
```

したがって全prefix質量は対角係数を全て足す操作

```text
F(x)=sum_{n>=0} [z^n] exp(xz)
```

である。`exp(xz)`の係数はちょうど`x^n/n!`なので

```text
F(x)=exp(x).
```

境界`1`で

```text
E[tau]=F(1)=e.
```

ここでは、単体体積の列をCauchy係数公式で指数関数の対角として読み、境界値を最後に評価する。
