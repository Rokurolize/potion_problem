# 境界jet伝播解

この解では、停止時刻の分布を求めない。残り容量`x`の問題を有限段で止め、その境界`x=0`での導関数だけを順に伝播させる。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止前prefixを数えれば

```text
tau = sum_{n>=0} 1_{S_n <= 1}
```

なので、残り容量`x`での有限段の期待報酬を

```text
H_N(x) = E[sum_{n=0}^N 1_{U_1+...+U_n <= x}]
```

と置く。求めたい値は`N`を無限大にした`H_N(1)`である。

## 有限段で閉じる

`H_0(x)=1`である。`N+1`段まで見ると、空prefixの報酬`1`を受け取ったあと、最初の一口`u`が`0<=u<=x`なら残り容量`x-u`で`N`段分の問題が残る。よって

```text
H_{N+1}(x) = 1 + int_0^x H_N(x-u) du
           = 1 + int_0^x H_N(t) dt.        (1)
```

ここでは無限級数の微分をしていない。すべて有限段の多項式に対する等式である。

## 境界のjetだけを見る

(1)を微分すると

```text
H_{N+1}'(x) = H_N(x).
```

したがって境界`x=0`で

```text
H_{N+1}(0) = 1,
H_{N+1}'(0) = H_N(0) = 1,
H_{N+1}''(0) = H_N'(0),
...
```

となる。帰納的に、`0<=k<=N`では

```text
H_N^{(k)}(0) = 1
```

であり、`k>N`では導関数は`0`である。つまり有限段の関数そのものが

```text
H_N(x) = sum_{k=0}^N x^k / k!
```

になる。

これは単体体積を各次元で積分して得たのではない。容量`0`の境界にある値`1`が、一口追加の積分作用で一つ上の導関数へ押し出される、という境界jetの伝播だけを使っている。

## 無限段へ戻す

各`x`で`H_N(x)`は`N`とともに単調に増える。さらに

```text
sum_{n=0}^N 1_{S_n <= x} ↑ sum_{n>=0} 1_{S_n <= x}
```

なので、単調収束により

```text
E[sum_{n>=0} 1_{S_n <= x}]
= lim_{N->infty} H_N(x)
= sum_{k=0}^infty x^k / k!
= exp(x).
```

元の問題では`x=1`だから

```text
E[tau] = exp(1) = e.
```

## この見方の芯

この証明は、`P(tau=n)`も`P(S_n<=1)`も先に出さない。有限段に切ってから、空prefixの境界値`1`が

```text
値 -> 1階導関数 -> 2階導関数 -> ...
```

と一段ずつ伝わることだけを見る。

一様性は、(1)の積分核が定数`1`になる点に現れる。独立性は、最初の一口の後も同じ有限段問題`H_N`が残る点に現れる。答えの`e`は、境界`0`から容量`1`まで伝播した全jetを足し戻した値である。
