# 終点流束密度解

この解では、何段目まで生きるかを直接数えず、停止前prefixの終点が区間内の各点へどれだけ流れ込むかを密度として見る。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

と置く。停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので、求めたい値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 原点以外の終点密度を見る

`n >= 1`について、`S_n`が`x`付近にある密度を

```text
q_n(x) dx
```

と書く。ただし`0 < x < 1`だけを見る。`S_n=x`へ到達するには、直前の終点`y`が`0<=y<=x`にあり、最後の一口が`x-y`であればよい。一様分布の密度は`1`なので、

```text
q_{n+1}(x) = int_0^x q_n(y) dy
```

である。

ここで`q_1(x)=1`である。これは、原点`S_0=0`から一口で`x`へ来る流束の密度が一様に`1`だからである。

## 全ての流束を重ねる

停止前prefixのうち、原点以外に現れるものの終点密度を

```text
q(x) = sum_{n >= 1} q_n(x)
```

と置く。非負項の和なので、上の式をそのまま足せる。すると

```text
q(x)
= q_1(x) + sum_{n >= 1} q_{n+1}(x)
= 1 + int_0^x sum_{n >= 1} q_n(y) dy
= 1 + int_0^x q(y) dy.
```

したがって

```text
q'(x) = q(x),
q(0) = 1.
```

よって

```text
q(x) = exp(x).
```

## 質量に戻す

停止前prefix全体の期待個数は、必ず存在する空prefix`S_0=0`の`1`と、`(0,1]`内に流れ込む全終点密度の質量の和である。

```text
E[tau]
= 1 + int_0^1 q(x) dx
= 1 + int_0^1 exp(x) dx
= e.
```

この証明では、`P(S_n<=1)`を各`n`で計算しない。各段の終点密度を「左から流れ込む一様な流束」として重ねると、総流束密度そのものが自分の原始関数で再生し、最後にその面積が期待停止前prefix数になる。
