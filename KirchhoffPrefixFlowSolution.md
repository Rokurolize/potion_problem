# Kirchhoff prefix流解

この解では、停止前prefix総量を、境界容量を流れる電流として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`以下にあるprefixの総量を

```text
F(x)=sum_{n>=0}P(S_n<=x)
```

と置く。境界を`dx`だけ広げると、既存prefixの各点から新しい一口が薄い層へ流れ込む。一様密度が`1`なので、総電流はその時点の総量`F(x)`に等しい。

Kirchhoff型の収支は

```text
dF = F(x) dx.
```

したがって

```text
F'(x)=F(x),  F(0)=1.
```

よって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

この証明では、各長さの体積を先に計算しない。境界を横切るprefix流の収支だけで指数方程式を得る。
