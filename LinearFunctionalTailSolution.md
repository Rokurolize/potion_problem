# linear functional tail解

この解では、停止判定を線形汎関数のレベル集合で表す。

長さ`n`のprefix`x`に対して

```text
L(x)=x_1+...+x_n
```

と置く。停止前条件は

```text
L(x)<=1,  x_i>=0
```

である。つまり、正の直交象限を線形汎関数`L`のレベル1以下で切ったものが停止前領域である。

`L(x)=r`の断面は倍率`r`の`n-1`単体で、体積密度は

```text
r^{n-1}/(n-1)!
```

である。したがって

```text
P(tau>n)=int_0^1 r^{n-1}/(n-1)! dr=1/n!.
```

尾和を取って

```text
E[tau]=sum_{n>=0}1/n!=e.
```

停止時刻は、prefixに線形汎関数`L`をかけた値が1を超える最初の時刻として読める。
