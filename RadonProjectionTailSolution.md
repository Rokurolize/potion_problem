# Radon projection tail解

この解では、立方体の密度を対角方向へ射影し、尾和をその射影質量として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。`n`個の増分の同時密度は`[0,1]^n`上で`1`である。和

```text
q(u)=u_1+...+u_n
```

への射影密度を`r_n(s)`と書くと、`0<=s<=1`では上限`u_i<=1`は効かないので

```text
r_n(s)=s^{n-1}/(n-1)!   (n>=1)
```

である。したがって

```text
P(S_n<=1)=int_0^1 r_n(s) ds = 1/n!.
```

整数値変数の尾和から

```text
E[tau]=sum_{n>=0} P(S_n<=1)
      =sum_{n>=0} 1/n!
      =e.
```

Radon射影として見ると、必要なのは和が`1`以下の部分だけである。その範囲では立方体の切断がまだ正錐の単体そのものなので、射影密度が階乗で正規化される。
