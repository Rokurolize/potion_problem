# 関係選択prefix解

この解では、長さ`n`の飲用列を関係表の行として見て、停止前prefixを選択条件で取り出す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。長さ`n`の表

```text
R_n = { (u_1,...,u_n) : u_i >= 0 }
```

に対して、停止前prefixは選択条件

```text
u_1+...+u_n <= 1
```

で得られる部分関係である。累積列

```text
t_k=u_1+...+u_k
```

へ射影すると、Jacobianは`1`で、選択結果は

```text
0 <= t_1 <= ... <= t_n <= 1
```

という順序関係になる。この関係は立方体`[0,1]^n`の`n!`個の順序室の一つなので、質量は

```text
P(S_n<=1)=1/n!.
```

空prefixを含めて全ての長さをunion allすれば

```text
E[tau]=sum_{n>=0}1/n! = e.
```

この証明では、分布名を使わない。関係表から選択・射影を行うと、停止条件が順序室の体積に変わることだけを見る。
