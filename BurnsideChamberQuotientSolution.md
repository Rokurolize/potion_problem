# Burnside chamber quotient解

この解では、停止前領域を対称群作用の基本領域として見る。

停止前prefixの長さ`n`成分は、累積和座標で

```text
0<=t_1<=...<=t_n<=1
```

という順序chamberになる。対称群`S_n`は立方体`[0,1]^n`に座標置換で作用し、重なりは測度零の境界だけである。

Burnside的に言えば、ほとんど全ての点の軌道サイズは`n!`であり、商の基本領域の体積は

```text
vol([0,1]^n/S_n)=1/n!.
```

したがって停止前確率は

```text
P(U_1+...+U_n<=1)=1/n!.
```

ゆえに

```text
E[tau]=sum_{n>=0}1/n!=e.
```

ここでは、単体積分を直接計算せず、立方体を対称群で割った基本領域の測度として停止前prefixを読む。
