# 多重解像度境界流解

この解では、停止前prefixの増加を粗い境界流から細かい境界流へ順に解像して読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`以下にあるprefixの総量を`F(x)`とする。粗い解像度では、境界を少し右へ動かすと、既存prefixの総量に比例して新しいprefixが生まれる。解像度を上げても、一様分布の密度が定数`1`なので局所的な流量は変わらない。

したがって全てのスケールで同じ局所法則

```text
delta F = F delta x + o(delta x)
```

が見える。細かい極限で

```text
F'(x)=F(x),  F(0)=1.
```

よって

```text
F(1)=e.
```

空prefixを含む停止前prefix総量が`E[tau]`なので

```text
E[tau]=e.
```

この見方では、答えは特定の階層の計算ではなく、どのスケールでも同じ境界流量が見えるという不変性から出る。
