# Gibbs正規化解

この解では、停止前prefix全体を有限粒子のGibbs測度の正規化として読む。

長さ`n`のprefixを累積時刻

```text
0 <= t_1 <= ... <= t_n <= 1
```

へ移す。増分から累積時刻への変換はJacobian`1`であり、この順序付き領域の質量が

```text
P(S_n <= 1)
```

である。

ここで、区間`[0,1]`上に粒子を任意個置く非相互作用Gibbs系を考える。粒子同士の相互作用はなく、1粒子のエネルギーも`0`、活動度も`1`とする。すると、ラベル付き`n`粒子配置の重みは

```text
dt_1 ... dt_n
```

であり、同じ非ラベル配置にはほとんど至る所で`n!`通りのラベル付けがある。

したがって、非ラベル`n`粒子配置のGibbs重みは

```text
1/n!
```

である。これは、順序付きprefix領域の質量と同じである。

## 正規化定数だけを見る

全ての粒子数を許したGibbs分配関数は

```text
Z = sum_{n >= 0} 1/n!.
```

非相互作用なので、この正規化定数は1粒子空間の体積`1`の指数であり、

```text
Z = exp(1).
```

一方、停止時刻の尾和公式から

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

各項は上の非ラベル`n`粒子Gibbs重みなので、

```text
E[tau] = Z = e.
```

この証明では、各単体の体積を個別に計算しない。停止前prefixを有限粒子Gibbs系の非ラベル配置として読み、答えを全粒子数を許した正規化定数として取り出す。
