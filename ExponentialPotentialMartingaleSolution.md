# 指数ポテンシャルの任意停止解

この解では、停止時刻の分布を出さず、残り距離に置いたポテンシャルが1回ごとに平均で1だけ減ることを見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

残り距離を

```text
R_n = 1 - S_n
```

と書く。`R_n >= 0`の間はまだ境界を越えていない。

## ポテンシャルを置く

`r>=0`で

```text
V(r) = exp(r)
```

と置き、`r<0`では

```text
V(r) = 0
```

とする。すると`0<=r<=1`について、一様乱数`U`を1回足した後の平均ポテンシャルは

```text
E[V(r-U)]
= int_0^r exp(r-u) du
= exp(r) - 1
= V(r) - 1.
```

ここで`u>r`なら越境して`r-u<0`となり、`V(r-u)=0`なので積分には入らない。

つまり、停止前の状態では、1回飲むと指数ポテンシャルの条件付き期待値がちょうど`1`減る。

## マルチンゲール

停止済みなら`V(R_n)=0`として、過程

```text
M_n = V(R_{n /\ tau}) + (n /\ tau)
```

を考える。上の計算から、`tau`の前では`V`の平均減少が`1`であり、`tau`の後ではどちらの項も止まる。したがって`M_n`はマルチンゲールである。

初期値は

```text
M_0 = V(1) = e.
```

一方、停止時刻を挟んだ時刻`n/\tau`では

```text
M_n = V(R_{n /\ tau}) + (n /\tau).
```

任意停止定理から

```text
E[M_n] = e.
```

したがって

```text
E[n /\ tau] <= e.
```

特に`tau`は可積分であり、`n /\ tau -> tau`なので単調収束により

```text
E[tau] <= e.
```

同時に

```text
E[V(R_{n /\ tau})] = e - E[n /\ tau].
```

左辺は非負で、`n->infty`で停止後は`0`になる。よって再び単調収束と有界な非負項の消失から

```text
E[tau] = e.
```

より直接には、`M_n`の等式

```text
E[n /\ tau] + E[V(R_{n /\ tau})] = e
```

で`n->infty`を取ればよい。

## この見方の芯

この証明は、`P(S_n<=1)`も`P(tau=n)`も計算しない。残り距離`r`に置いた指数ポテンシャルが、一様な次の一口によって平均でぴったり1だけ減ることを使う。

停止までの回数は、指数ポテンシャルが消費する単位量そのものとして読める。初期ポテンシャルが`V(1)=e`なので、期待停止時刻も`e`になる。
