    戦略:
    - fwdDiff_iter_eq_sum_shiftを使用して有限差分との関係を確立
    - 符号パターンの変換: (-1)^k = (-1)^n * (-1)^(n-k)

    - 場合によってはinductionを使用

    戦略:
    - 後方向は簡単（既にコメントで説明済み）
    - 前方向にはInt.alternating_sum_range_chooseを活用
    - 0 < x < nでの正値性は、交代和の部分和の評価が必要
    - B-spline APIが存在しないため、第一原理からの証明が必要

    3. irwin_hall_continuous (line 262)

    目標: Continuous (irwin_hall_cdf n)

    戦略:
    - n = 0の場合は実際に不連続（Heaviside関数）
    - n > 0の場合:
      - continuousOn_floorを使用して区間連続性を証明
      - Continuous.ifまたはcontinuous_piecewiseを適用
      - 各整数点での連続性を個別に証明

    実装順序

    1. まずirwin_hall_sum_at_nから着手（最も具体的な恒等式）
    2. 次にirwin_hall_support（交代和の正値性）
    3. 最後にirwin_hall_continuous（区分連続性）

    注意事項

    - 一部のAPIは直接存在しないため、補助定理の証明が必要
    - 特に「x^nのn階差分=n!」は自前で証明する必要がある
    - n = 0の扱いに注意（連続性の主張を修正する可能性あり）
