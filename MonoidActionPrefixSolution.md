# モノイド作用prefix解

この解では、一口を足す操作を自由モノイドの作用として見て、境界内に留まる軌道の総量を数える。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。長さ`n`の語`(u_1,...,u_n)`は、初期点`0`に加法モノイドが作用して

```text
0 -> u_1 -> u_1+u_2 -> ... -> S_n
```

を作るものと見る。生存条件は軌道の終点が`1`以下であること、つまり

```text
u_i>=0,  u_1+...+u_n<=1
```

である。境界`x`までの全生存語の質量を`F(x)`と置くと、境界を`dx`広げたとき、新しい語は既存の生存語に一文字を足して薄層へ送ることで得られる。密度は`1`なので

```text
F'(x)=F(x),  F(0)=1.
```

したがって

```text
F(1)=e.
```

停止時刻の期待値は空語を含む生存語の総量なので

```text
E[tau]=e.
```

この見方では、`e`は自由モノイド作用の軌道を境界内で全て足した重みである。
