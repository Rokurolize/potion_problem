# policy evaluation prefix解

この解では、毎回「まだ飲む」という固定方策の評価として尾確率を計算する。

停止規則は変えない。prefixの合計が1以下なら継続可能で、1を超えた時点で停止する。長さ`n`まで生き残る固定方策の評価値を

```text
J_n(c)
```

とし、残容量`c`から始める。`n=0`では`J_0(c)=1`で、1回進めると

```text
J_n(c)=int_0^c J_{n-1}(c-u) du
```

である。

この再帰を解くと

```text
J_n(c)=c^n/n!
```

だから、初期残容量`1`では

```text
P(tau>n)=J_n(1)=1/n!.
```

したがって

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

固定方策の評価という形にしても、評価関数は停止前単体の体積を再帰的に計算しているだけである。
