# 脱出正規化解

この解では、停止前prefixを直接足すかわりに、「非負領域から最後に漏れる流量」が確率1になるように正規化する。

独立な一様乱数`U_1,U_2,...`について

```text
R_0 = 1,
R_n = 1 - (U_1 + ... + U_n),
tau = min { n >= 1 : R_n < 0 }
```

と置く。`R_n >= 0`の間だけ鎖は生きており、

```text
tau = sum_{n >= 0} 1_{R_n >= 0}.
```

残り容量が`r`のとき、次の一口で脱出する確率は

```text
P(U > r) = 1 - r.
```

である。

## 生存中に払う質量

初期状態`r`から始めたとき、停止前に訪れる全状態の期待測度を`G_r`と書く。すると、最終的には必ず一度だけ脱出するので、

```text
int_0^1 (1 - x) G_r(dx) = 1.        (1)
```

左辺は「各生存状態`x`で、その次に外へ漏れる確率」を全訪問にわたって足したものであり、右辺は実際に漏れる回数が1回であることを表す。

求めたいのは

```text
G_1([0,1]) = E[tau].
```

そこで、(1)だけではなく、初期位置を`r`として同じ量を動かして見る。

```text
F(r) = G_r([0,1])
```

とする。

## 脱出正規化から方程式を読む

残り容量`r`でまず現在の状態を1回数える。次に`U <= r`なら残り容量は`r-U`となり、そこから同じ問題が続く。したがって

```text
F(r) = 1 + int_0^r F(r-u) du
     = 1 + int_0^r F(s) ds.
```

この式は単なる再帰ではなく、脱出が最終的に一回だけ起きるように総占有量を正規化する式である。両辺を微分すると

```text
F'(r) = F(r),
F(0) = 1.
```

よって

```text
F(r) = exp(r).
```

特に

```text
E[tau] = F(1) = e.
```

## この見方の芯

この証明では、停止時刻の分布も、各段の単体体積も先に求めない。生きている全状態に「次で外へ漏れる確率」を課すと、その総漏れ量は必ず1になる。そこから、初期容量を動かした総占有量`F(r)`が自分自身を増加率にするため、指数関数が現れる。
