# adjoint boundary pairing解

この解では、停止前prefix和を随伴方程式との境界ペアリングで評価する。

容量`x`に対し

```text
F(x)=sum_{n>=0}P(U_1+...+U_n<=x)
```

とおく。尾和公式から元の期待値は`F(1)`である。生存確率の再帰を微分すると

```text
F'(x)-F(x)=0,  F(0)=1.
```

ここで随伴作用素は

```text
L^*psi=-psi'-psi
```

である。`psi(1)=1`、`L^*psi=0`を満たす随伴解は

```text
psi(x)=e^{1-x}.
```

Greenの恒等式を`LF=0`と`L^*psi=0`に適用すると、内部積分は消え、残るのは境界ペアリングだけである。

```text
F(1) psi(1) = F(0) psi(0) = e.
```

従って

```text
E[tau]=F(1)=e.
```

この証明では、期待値を直接解く代わりに、端点評価を随伴境界値問題へ移している。
