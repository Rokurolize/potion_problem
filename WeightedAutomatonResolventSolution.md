# 重み付きオートマトンレゾルベント解

この解では、停止前prefixを、容量を状態に持つ重み付きオートマトンの全受理経路として数える。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 容量オートマトン

状態を残り容量`r in [0,1]`とする。空語は必ず受理され、重み`1`を持つ。文字を一つ読むとは、一口`u`を選んで

```text
r -> r-u
```

へ移ることである。ただし`0 <= u <= r`のときだけ受理経路として残す。一様密度が`1`なので、この遷移の重みはLebesgue測度`du`である。

初期残量`r`から始めた全受理経路の総重みを

```text
M(r)
```

と置く。空語の寄与が`1`あり、非空経路は最初の文字を読んだ後にまた同じオートマトンへ戻るので

```text
M(r) = 1 + int_0^r M(r-u) du.
```

変数を` s = r-u `とすれば

```text
M(r) = 1 + int_0^r M(s) ds.
```

したがって

```text
M'(r) = M(r),
M(0) = 1.
```

よって

```text
M(r) = exp(r).
```

求める値は初期残量`1`での全受理経路の総重みなので

```text
E[tau] = M(1) = e.
```

この証明では、停止時刻の分布を求めない。飲み続けられる全prefixを、重み付きオートマトンの全受理経路としてまとめる。`e`は、そのオートマトンのレゾルベント重みを初期状態で評価した値である。
