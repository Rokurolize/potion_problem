# 留数係数抽出解

この解では、停止前の各長さを個別に積分しない。生存領域の指示関数を複素積分で表し、全ての長さを足したあと、最後に一つの留数だけを読む。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えれば

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

だから

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 生存領域を係数として読む

固定した`n`について、`S_n <= 1`なら各`U_i`は自動的に`0 <= U_i <= 1`を満たす。したがって一様分布の上端制約を忘れても、生存領域はちょうど正の単体

```text
u_i >= 0,
u_1 + ... + u_n <= 1
```

である。

この単体体積を、Heaviside関数のBromwich表示

```text
1_{x <= 1}
= (1 / 2 pi i) int_{c-i infinity}^{c+i infinity} exp(z(1 - x)) / z dz
```

で書く。ただし`c > 0`である。すると

```text
P(S_n <= 1)
= int_{u_i >= 0} 1_{u_1+...+u_n <= 1} du_1...du_n
```

は

```text
(1 / 2 pi i) int_{c-i infinity}^{c+i infinity}
  exp(z) / z
  prod_{i=1}^n int_0^infinity exp(-z u_i) du_i
dz.
```

各内側の積分は`1/z`なので

```text
P(S_n <= 1)
= (1 / 2 pi i) int exp(z) / z^(n+1) dz.
```

これは`exp(z)`の`z^n`係数を拾うCauchy係数公式そのもので、値は`1/n!`である。

## 長さ方向を先に足す

ここで大事なのは、`1/n!`を得てから級数を眺めるのではなく、上の係数抽出式の段階で全ての`n`を足すことだ。形式的には

```text
E[tau]
= sum_{n >= 0} (1 / 2 pi i) int exp(z) / z^(n+1) dz.
```

`|z| > 1`の円周上で足し合わせると

```text
sum_{n >= 0} 1 / z^(n+1) = 1 / (z - 1).
```

したがって

```text
E[tau]
= (1 / 2 pi i) int exp(z) / (z - 1) dz.
```

積分路を`0`と`1`を囲む大きな円に取り直せば、もう`0`の高階極の全列は消えている。残る極は`z = 1`だけである。

その留数は

```text
Res_{z=1} exp(z) / (z - 1) = exp(1).
```

よって

```text
E[tau] = e.
```

## この見方の芯

通常の解では、次元`n`ごとの単体体積`1/n!`を計算してから足す。この解では、単体体積を「`exp(z)`の係数」として表し、次元方向の和を先に実行して`1/(z-1)`へ圧縮する。

答えの`e`は、無数の単体の係数を足し合わせたあとに現れる、ただ一つの極`z=1`の留数である。停止時刻の期待値が、境界`1`を越えない全prefixの係数抽出をまとめた residue として現れる。
