# 終対象prefix解

この解では、停止前prefixを境界`1`へ向かう射の集まりとして見て、終対象での全射数を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`以下のprefix総量を

```text
F(x)=sum_{n>=0} P(S_n <= x)
```

と置く。境界`0`では空prefixだけがあり、

```text
F(0)=1.
```

`0 <= x <= 1`の範囲で終点を`x`から`x+dx`へ動かすと、どのprefixからも一様密度`1`で新しい一口を付け足せる。終対象へ向かう射の総量は、その時点の総量に比例して増えるので

```text
F'(x)=F(x).
```

したがって

```text
F(x)=e^x.
```

欲しい値は境界`1`を終対象として評価した総量なので

```text
E[tau]=F(1)=e.
```

ここでいう終対象は、抽象語を飾っただけではない。全ての生存prefixが境界`1`以下へ一意に収まるという評価先を固定し、そこへ流れ込む総量の増分だけを見る。
