# simplicial zeta measure解

この解では、停止前prefix全体を単体圏のzeta測度で数える。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。部分和座標

```text
t_k=S_k
```

へ移ると、`S_n<=1`は

```text
0<=t_1<=...<=t_n<=1
```

になる。Jacobianは`1`であるから、この領域の測度は区間`[0,1]`上の`n`単体の測度である。

単体圏で、全ての非退化でない`n`単体をzeta側に符号なしで足すと、測度は

```text
1/n!
```

になる。したがって停止前prefixの総zeta測度は

```text
sum_{n>=0} 1/n! = e.
```

一方、停止時刻は停止前prefixの個数なので、この総測度がそのまま

```text
E[tau]
```

である。

ここではMöbius反転や符号付きEuler標数は使わない。停止前の単体を全次数で符号なしに足すzeta測度だけが必要である。
