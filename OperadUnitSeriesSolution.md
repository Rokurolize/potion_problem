# オペラド単位級数解

この解では、停止前prefixを「一列に合成される一項演算」の全次数として見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 一列合成としてのprefix

`S_n <= 1`を累積時刻

```text
0 <= t_1 <= ... <= t_n <= 1
```

へ移す。これは、区間`[0,1]`上に`n`個の入力を一列に合成した順序付き配置である。増分から累積時刻への変換はJacobian`1`なので、この配置空間の体積が`P(S_n <= 1)`である。

1点を置く基本演算の重みを

```text
x = length([0,1]) = 1
```

とする。ラベル付きの全入力集合では`n!`通りの順序があるが、一列合成のprefixではそのうち時刻順に整列した一つだけを選ぶ。したがって`n`次成分の重みは

```text
x^n/n!.
```

これは、単位元から自由に一列合成を繰り返すオペラド的な単位級数

```text
1 + x + x^2/2! + x^3/3! + ...
```

である。境界`1`では`x=1`なので、停止前prefixの総量は

```text
E[tau]
  = sum_{n >= 0} 1/n!
  = e.
```

この証明では、各単体体積を積分で求めない。prefixを一列合成の全次数としてまとめ、時刻順に整列した一つの成分だけを取るための`1/n!`をオペラドの単位級数の係数として読む。`e`は、その単位級数を区間長`1`で評価した値である。
