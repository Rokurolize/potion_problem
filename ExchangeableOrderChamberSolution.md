# exchangeable order chamber解

この解では、累積和で得られる点配置の交換可能性から体積を読む。

長さ`n`まで停止しない条件は

```text
U_1+...+U_n<=1.
```

累積和

```text
T_k=U_1+...+U_k
```

に変換すると、ヤコビアンは`1`で、領域は

```text
0<=T_1<=...<=T_n<=1
```

になる。これは`[0,1]^n`を`n!`個の順序chamberへ割ったうちの一つである。Lebesgue測度は座標の置換で不変だから、各chamberの体積は等しい。

したがって

```text
P(U_1+...+U_n<=1)=1/n!.
```

尾和公式により

```text
E[tau]=sum_{n>=0}P(U_1+...+U_n<=1)=sum_{n>=0}1/n!=e.
```

この見方では、停止前prefixを順序付き点配置へ移し、交換可能性がfactorialを生む。
