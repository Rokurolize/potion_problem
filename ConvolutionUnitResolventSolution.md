# 畳み込み単位レゾルベント解

この解では、停止前prefix全体を、半直線上の畳み込み代数で単位密度を解くレゾルベントとして読む。

`H(x)`を

```text
H(x) = 1_{x >= 0}
```

とする。`0 <= x <= 1`だけを見る限り、一様分布の密度`1_{0 <= u <= 1}`は、半直線の単位密度`H`と同じ因果的な芽を持つ。つまり、合計が`1`以下であるprefixを測るには、各一口の上端`1`はまだ切断として働かない。

長さ`n`のprefix質量は、`n`個の一口密度を畳み込んだものを`[0,1]`で積分した量である。半直線の単位密度で書けば

```text
P(S_n <= 1) = int_0^1 H^{*n}(x) dx.
```

ここで`H^{*0}`は原点の単位質量と見る。

## レゾルベントを先に足す

全prefixを足した測度は

```text
R = delta_0 + H + H*H + H*H*H + ...
```

である。これは畳み込み代数のレゾルベント

```text
R = (delta_0 - H)^{-1}
```

にあたる。

密度部分を`r(x)`と見ると、`x > 0`で

```text
r(x) = 1 + int_0^x r(y) dy.
```

なぜなら、非空prefixは最後の一口の直前までのprefixと、最後に足す単位密度に分解できるからである。

したがって

```text
r'(x) = r(x),   r(0)=1
```

となり、

```text
r(x)=exp(x).
```

求める期待値は、原点の空prefixを含む全prefix質量を容量`1`まで読んだものなので

```text
E[tau] = r(1) = e.
```

この証明では、各畳み込み冪を個別に計算しない。半直線上の単位密度を先にレゾルベントとして総和し、その局所方程式だけを解く。`e`は畳み込み単位のレゾルベント核の境界値として現れる。
