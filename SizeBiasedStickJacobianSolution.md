# サイズバイアス棒折りJacobian解

この解では、生存単体を直接積分せず、残りの棒から次の一口を割合で取る座標に変換して、Jacobianだけを読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 残り棒から割合で取る

`S_n <= 1`の領域で、残り量を

```text
R_0 = 1,
R_i = 1 - U_1 - ... - U_i
```

と置く。各一口を、その時点で残っている棒の割合として

```text
V_i = U_i / R_{i-1}
```

と書く。すると

```text
0 <= V_i <= 1,
U_i = R_{i-1} V_i,
R_i = R_{i-1}(1 - V_i).
```

したがって

```text
U_1 = V_1,
U_2 = (1 - V_1)V_2,
U_3 = (1 - V_1)(1 - V_2)V_3,
...
```

となる。この変換は`[0,1]^n`から生存単体

```text
{ u_i >= 0, u_1+...+u_n <= 1 }
```

への一対一対応である。

## Jacobian

変換は下三角なので、Jacobianは対角成分の積である。`U_i`を`V_i`で微分すると、その時点の残り棒

```text
R_{i-1} = product_{j < i} (1 - V_j)
```

が出る。したがって

```text
dU_1 ... dU_n
  = product_{i=1}^n R_{i-1} dV_1 ... dV_n
  = product_{j=1}^{n-1} (1 - V_j)^{n-j} dV_1 ... dV_n.
```

よって

```text
P(S_n <= 1)
  = int_{[0,1]^n} product_{j=1}^{n-1} (1 - V_j)^{n-j} dV
  = product_{j=1}^{n-1} 1/(n-j+1)
  = 1/n!.
```

空prefixの項は`1`である。したがって

```text
E[tau]
  = sum_{n >= 0} 1/n!
  = e.
```

この証明では、単体の体積公式を使わない。生存条件を「残り棒から割合を取る」座標へ吸収し、すべての制約を単位立方体へ移す。階乗は、残り棒のJacobianが作る一変数積分の積として現れる。
