# 対称テンソル指数解

この解では、停止前prefixを対称テンソル代数の指数元としてまとめる。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。累積時刻

```text
t_k = S_k
```

へ移すと、`S_n <= 1`は

```text
0 <= t_1 <= ... <= t_n <= 1
```

という順序付き配置になる。Jacobianは`1`である。

`H=L^2([0,1])`とし、定数関数`1`を考える。対称テンソル代数では、指数元

```text
Exp(1)=sum_{n>=0} 1^{otimes n}/n!
```

の`n`次成分が、区間上の非相互作用な`n`点配置の質量を表す。順序付き代表を一つ選ぶと、係数はちょうど

```text
vol{0 <= t_1 <= ... <= t_n <= 1}=1/n!.
```

したがって停止前prefixの全質量は

```text
E[tau]=sum_{n>=0} 1/n! = exp(1)=e.
```

この証明では、各段の積分を別々に計算しない。prefix列全体を対称テンソル指数元の成分として束ね、指数元の正規化から全和を読む。
