# 微小境界出生解

この解では、長さごとの単体体積を計算しない。境界を`x`から`x+dx`へ少しだけ動かしたとき、停止前prefixの総質量がどれだけ増えるかだけを見る。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

しきい値を`x`にした問題を考え、

```text
M(x) = E[sum_{n>=0} 1_{S_n <= x}]
```

と置く。求める値は`M(1)`である。`x=0`では空prefixだけが存在するので

```text
M(0) = 1.
```

## 境界を少し押し出す

`x`を`x+dx`へ動かす。新しく数えられるprefixは、どこかの既存prefix`S_n=s<=x`から次の一口を足して、ちょうど薄い帯

```text
x < s + U_{n+1} <= x + dx
```

へ入ったものだけである。

既存prefixの位置が`s<=x`なら、この条件は

```text
x-s < U_{n+1} <= x+dx-s
```

である。ここで`0 <= x-s <= x <= 1`なので、一様分布の密度はこの区間で`1`であり、この確率は一次の項で

```text
dx + o(dx)
```

である。重要なのは、この一次係数が`s`に依存しないことだ。

したがって、境界を`dx`だけ広げたときの総質量の増分は

```text
M(x+dx) - M(x) = M(x) dx + o(dx).
```

よって

```text
M'(x) = M(x),   M(0)=1.
```

## 期待値

微分方程式から

```text
M(x) = exp(x).
```

したがって元の問題では

```text
E[tau] = M(1) = e.
```

## この見方の芯

この証明では、`P(S_n<=1)`を個別に評価しない。全ての停止前prefixをまとめた質量`M(x)`だけを見る。

境界を少しだけ右へ動かすと、すでに存在する各prefixが同じ一次率で新しいprefixを薄い帯へ送り込む。一様分布の密度が`1`であることが、そのまま

```text
増加率 = 現在量
```

という自己増殖の式になる。`e`は、この微小境界出生をしきい値`0`から`1`まで積み上げた値である。
