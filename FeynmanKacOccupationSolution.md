# Feynman-Kac占有報酬解

この解では、停止までの飲用回数を、殺されるMarkov連鎖の占有報酬として読む。

残り容量を状態にする。初期状態は`r=1`で、状態`r`にいるとき一様乱数`U`を飲む。`U <= r`なら次の状態は

```text
r-U
```

であり、`U > r`なら境界を越えて殺される。

`0 <= r <= 1`から始めたときの期待停止回数を

```text
v(r)
```

とする。求めたい値は`v(1)`である。

## 一回分の報酬を先に払う

状態`r`でまだ生きているなら、次の一口は必ず一回数える。そのあと生き残った場合だけ、残りの期待報酬が続く。したがって

```text
v(r) = 1 + int_0^r v(r-u) du.
```

変数を`x=r-u`に替えると

```text
v(r) = 1 + int_0^r v(x) dx.      (1)
```

これは、殺された遷移核

```text
K f(r) = int_0^r f(x) dx
```

に対するFeynman-Kac型の占有報酬方程式

```text
v = 1 + K v
```

である。

## resolventを微分で閉じる

(1)を微分すると

```text
v'(r) = v(r)
```

また`r=0`では、どんな正の一口でもただちに越えるので、数える一口はちょうど一回で

```text
v(0) = 1
```

である。よって

```text
v(r) = exp(r).
```

したがって

```text
E[tau] = v(1) = e.
```

この証明では、停止時刻の分布を先に求めない。殺される残量過程に「生きている状態で一回飲む」という単位報酬を載せ、その占有報酬のFeynman-Kac方程式を解く。`e`は殺された遷移核のresolventが定数報酬`1`を返す値として現れる。
