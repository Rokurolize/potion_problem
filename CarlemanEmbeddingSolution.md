# Carleman埋め込み解

この解では、prefix和の更新をモーメント列への線形作用として埋め込む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めた滞在数から

```text
E[tau]=sum_{n>=0} P(S_n<=1)
```

である。境界`x`までの総prefix質量を

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

と置く。

Carleman埋め込みでは、非線形な「境界を少し動かす」操作を、単項式列`1,x,x^2,...`上の線形な係数更新として見る。境界`x`から`x+h`へ広げると、新しく加わるprefixは、既存のprefixに最後の一口が長さ`h`の窓へ入るものだけである。一様性により一次の寄与は

```text
F(x) h
```

で、高次の重なりは`o(h)`に吸収される。したがって埋め込まれた無限行列の一次ブロックは

```text
F(x+h)=F(x)+h F(x)+o(h).
```

つまり

```text
F'(x)=F(x),  F(0)=1.
```

よって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

Carleman埋め込みの役割は、prefix追加の自己増殖を線形な係数流として取り出すことにある。境界`1`で読むべき第一成分が、そのまま期待停止時刻になる。
