# Stokes境界消去解

この解では、単体体積を反復積分で計算しない。生存prefix領域に微分形式を置き、Stokesの定理で境界のうち座標面を全部消し、残った斜面だけから一段低い単体を読む。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止前prefixを数えると

```text
tau = sum_{n>=0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n>=0} P(S_n <= 1).
```

`S_n<=1`の範囲では各`U_i`の密度は`1`であり、

```text
A_n = P(S_n <= 1)
    = vol { x_i >= 0, x_1+...+x_n <= 1 }
```

である。`A_0=1`とする。

## 体積形式を境界へ押す

`n>=1`で

```text
K_n = { x_i >= 0, x_1+...+x_n <= 1 }
```

と置く。`K_n`上の体積形式を

```text
dx_1 ... dx_n
```

と書く。

ここで`(n-1)`形式

```text
alpha_n
= (1/n) sum_{i=1}^n (-1)^{i-1} x_i
   dx_1 ... hat{dx_i} ... dx_n
```

を置く。直接微分すると

```text
d alpha_n = dx_1 ... dx_n.
```

したがって Stokes の定理より

```text
A_n = int_{K_n} d alpha_n = int_{partial K_n} alpha_n.
```

## 座標面は消える

境界`partial K_n`は、座標面

```text
x_i = 0
```

と斜面

```text
x_1+...+x_n = 1
```

からなる。

座標面`x_i=0`に`alpha_n`を制限すると、`x_i`を含む項は係数が`0`で消える。ほかの項は微分形式の中に`dx_i`を含むので、`x_i=0`の面へ引き戻すと消える。したがって座標面からの寄与はすべて`0`である。

残るのは斜面だけである。

## 斜面は一つ低い単体

斜面を

```text
(x_1,...,x_{n-1}) |-> (x_1,...,x_{n-1}, 1-x_1-...-x_{n-1})
```

で表す。これは

```text
x_1,...,x_{n-1} >= 0,
x_1+...+x_{n-1} <= 1
```

を動くので、台は`K_{n-1}`である。

この斜面へ`alpha_n`を引き戻すと、交代和の符号と`dx_n=-(dx_1+...+dx_{n-1})`の寄与が合わさって

```text
alpha_n |_{斜面}
= (1/n) dx_1 ... dx_{n-1}.
```

斜面上では`x_1+...+x_n=1`なので、係数の和がちょうど`1`になるからである。よって

```text
A_n = (1/n) A_{n-1}.        (1)
```

## 期待値へ戻す

`A_0=1`と(1)から

```text
A_n = 1/n!
```

である。したがって

```text
E[tau]
= sum_{n>=0} A_n
= sum_{n>=0} 1/n!
= e.
```

## この見方の芯

この証明では、`n`重積分も停止時刻の確率質量も求めない。体積形式を原始形式`alpha_n`として境界へ押し出し、座標面では引き戻しが全部消えることだけを見る。

一様性は、生存領域上の密度が体積形式そのものになる点に使われる。独立性は、各長さのprefix領域が同じ直交座標の積密度で測れる点に使われる。答えの`e`は、Stokesの境界消去が作る再帰`A_n=A_{n-1}/n`を全次数で足した値として現れる。
