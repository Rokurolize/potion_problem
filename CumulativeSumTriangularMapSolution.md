# cumulative sum triangular map解

この解では、独立な増分列を累積和列へ送る三角写像で、停止前領域を順序領域に変える。

独立一様乱数`U_1,U_2,...`について

```text
S_n=U_1+...+U_n,
tau=min { n>=1 : S_n>1 }
```

とする。長さ`n`まで停止しない条件は

```text
S_n<=1.
```

ここで三角写像

```text
T_n(u_1,...,u_n)=(s_1,...,s_n),
s_k=u_1+...+u_k
```

を考える。逆写像は

```text
u_1=s_1,  u_k=s_k-s_{k-1}
```

で、ヤコビアンは`1`である。さらに`u_i>=0`かつ`S_n<=1`はちょうど

```text
0<=s_1<=s_2<=...<=s_n<=1
```

に変わる。

従って

```text
P(S_n<=1)
=vol{0<=s_1<=...<=s_n<=1}.
```

右辺は、`n`個の点を`[0,1]`に置いたときの`n!`個の順序chamberのうち一つであり、体積は`1/n!`である。よって尾和公式から

```text
E[tau]
=sum_{n>=0}P(S_n<=1)
=sum_{n>=0}1/n!
=e.
```

この見方では、確率の問題を、増分座標と順序座標の三角写像による体積保存に落としている。
