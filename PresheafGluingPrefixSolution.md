# 前層貼り合わせprefix解

この解では、境界`x`までに生きているprefixの総量を、区間の前層の貼り合わせ量として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`以下のprefix総量を

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

と置けば、求める値は`F(1)`である。

境界を`x`から`x+dx`へ広げると、新しく貼り合わされる局所切片は、既存prefixに最後の一口が薄い層`(x,x+dx]`へ入るものだけである。一様密度が`1`なので、この新しい切片の総量は

```text
F(x) dx + o(dx)
```

である。したがって貼り合わせ量は

```text
F'(x)=F(x)
```

を満たす。`x=0`では空prefixだけがあるから

```text
F(0)=1.
```

よって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

前層という言葉で言えば、各境界区間にあるprefix切片を、境界を広げながら制限写像で貼り合わせている。貼り合わせの増分が自己増殖方程式になる。
