# minimax境界値解

この解では、停止前prefix総量を、敵も味方も選択肢を持たないminimax値として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。残り容量`r`での値を`W(r)`とする。もし一口の選択をプレイヤーと敵の手番として書けば、通常は

```text
value = reward + min max future value
```

の形になる。しかしこの問題では一口は一様乱数で固定され、どちらのプレイヤーも操作できない。したがってminimax作用は平均作用に退化し、

```text
W(r)=1+int_0^r W(r-u) du.
```

すなわち

```text
W'(r)=W(r),  W(0)=1.
```

解は

```text
W(r)=e^r.
```

初期残量は`1`なので

```text
E[tau]=W(1)=e.
```

ここではminimaxという言葉は、選択のないゲームへ退化していることを明示するためのものだ。退化後に残るのは、境界値の自己再生方程式だけである。
