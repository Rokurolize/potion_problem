# path圏nerve解

この解では、停止前prefixを、区間`[0,1]`上の到着時刻列が作るpath圏のnerveとして読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

累積時刻に変換すると、長さ`n`のprefixは

```text
0<t_1<...<t_n<1
```

という`n`単体になる。これは全順序集合`[0,1]`を圏と見たとき、`0`から`1`へ向かう非退化`n`単体、つまりnerveの`n`次元部分である。

`[0,1]^n`は`n!`個の順序室に分かれ、各室は同じ体積を持つため

```text
P(S_n<=1)=1/n!.
```

したがってnerve全体の体積的Euler和ではなく、全次元の正の質量和として

```text
E[tau]=sum_{n>=0}1/n!=e.
```

ここでは、停止前の履歴を順序集合の鎖として束ね、全ての鎖の測度を足すことで答えを得ている。
