# incidence algebraのzeta核解

この解では、媚薬問題を「区間内の有限鎖を全部数えるzeta核」として見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止時刻は生存しているprefixの個数なので、

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって、期待値は区間`[0,1]`に入る全prefixの総質量である。

## prefixは区間内の鎖である

`n >= 1`で

```text
t_i = S_i
```

と置く。Jacobianは`1`で、`S_n <= 1`は

```text
0 < t_1 < ... < t_n <= 1
```

という区間`[0,1]`内の長さ`n`の鎖になる。`n=0`は空鎖である。

そこで、区間`[a,b]`に入る全有限鎖の総質量を

```text
Z(a,b)
```

と書く。求めたいのは

```text
E[tau] = Z(0,1)
```

である。

## 最初の点で分解する

`[a,b]`内の有限鎖は、空鎖であるか、最初の点`t`を一つ持ち、その後ろに`[t,b]`内の有限鎖を持つ。

したがって

```text
Z(a,b) = 1 + int_a^b Z(t,b) dt.        (1)
```

これは連続な半順序集合`[a,b]`のincidence algebraで、zeta核を「空鎖または最初の点で切る」再帰で書いたものである。

平行移動不変性から

```text
Z(a,b) = h(b-a)
```

と置ける。(1)は

```text
h(x) = 1 + int_0^x h(y) dy
```

になる。よって

```text
h'(x) = h(x),
h(0) = 1.
```

したがって

```text
h(x) = exp(x).
```

最後に

```text
E[tau] = Z(0,1) = h(1) = exp(1) = e.
```

## 何を数えたか

この証明は、各`P(tau=n)`を個別に求めない。生存prefixを、区間内に置かれた有限鎖として一つの対象にまとめる。

全有限鎖の総質量`Z`は、空鎖と最初の点で分解されるため、zeta核の再帰

```text
Z = empty + point * Z
```

を満たす。その連続版が`h'=h`であり、答えの`e`は区間`[0,1]`のzeta核の値として出る。
