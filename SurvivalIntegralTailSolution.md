# 生存関数尾積分解

この解では、停止時刻の期待値を生存関数の尾和として読み、各尾を順序領域へ移す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。整数値の正の停止時刻なので

```text
E[tau]=sum_{n>=0} P(tau>n)
      =sum_{n>=0} P(S_n<=1).
```

ここで`P(tau>n)`を尾確率として直接見る。部分和座標` t_k=S_k `へ移すと

```text
{tau>n} = {0 <= t_1 <= ... <= t_n <= 1}
```

であり、Jacobianは`1`である。この順序領域は、立方体`[0,1]^n`の`n!`個の順序室の一つなので

```text
P(tau>n)=1/n!.
```

したがって

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

この証明の焦点は、密度や停止時刻分布ではなく、期待値を尾確率の和として先にほどく点である。各尾は順序室一つ分にすぎない。
