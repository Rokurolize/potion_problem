# Z変換生存列解

この解では、生存prefix列を境界容量つきのZ変換としてまとめる。

`0 <= r <= 1`に対して

```text
a_n(r) = P(X_1 + ... + X_n <= r)
```

と置く。ただし`a_0(r)=1`である。求めたい期待値は尾和公式により

```text
E[tau] = sum_{n >= 0} a_n(1)
```

である。

そこで

```text
G(r,z) = sum_{n >= 0} a_n(r) z^n
```

を考える。

## 一段進める再帰

`n >= 1`では、最後の一口を`u`とすると

```text
a_n(r) = int_0^r a_{n-1}(r-u) du.
```

したがって

```text
sum_{n >= 1} a_n(r) z^n
  = z int_0^r G(r-u,z) du
  = z int_0^r G(v,z) dv.
```

よって

```text
G(r,z) = 1 + z int_0^r G(v,z) dv.
```

微分すると

```text
partial_r G(r,z) = z G(r,z),   G(0,z)=1.
```

したがって

```text
G(r,z) = exp(z r).
```

期待値は`z=1, r=1`での値なので

```text
E[tau] = G(1,1) = e.
```

この証明では、停止時刻の確率質量を先に出さない。生存するprefix数をZ変換で一つの関数に束ね、容量方向の微分方程式だけで閉じる。`e`は生存列のZ変換を`z=1`で読む値として現れる。
