# cover nerve tail解

この解では、停止前prefix領域を座標順の開被覆のnerveで測る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。`S_n<=1`の領域は

```text
Delta_n={u_i>=0, u_1+...+u_n<=1}
```

である。部分和座標

```text
t_k=u_1+...+u_k
```

へ移すと

```text
0<=t_1<=...<=t_n<=1
```

となる。これは区間`[0,1]`の順序付き点配置で、座標順の良い被覆を取ればnerveは標準`n`単体になる。

その体積は

```text
1/n!
```

なので、整数値変数の尾和から

```text
E[tau]=sum_{n>=0} P(S_n<=1)
      =sum_{n>=0} 1/n!
      =e.
```

被覆のnerveは、停止前prefixが区間上の順序構造だけで決まることを可視化している。
