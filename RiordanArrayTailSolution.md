# Riordan array tail解

この解では、停止時刻の尾確率列をRiordan配列の一列として扱う。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。尾和から

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

容量`x`までの尾確率を

```text
r_n(x)=P(S_n<=x)
```

と書くと、

```text
r_n(x)=int_0^x r_{n-1}(t) dt,
r_0(x)=1.
```

この積分作用素を下三角のRiordan配列として見ると、定数列`1,0,0,...`を順に積分して得る列は

```text
1, x, x^2/2!, x^3/3!, ...
```

である。したがって

```text
r_n(1)=1/n!.
```

よって

```text
E[tau]=sum_{n>=0} r_n(1)=e.
```

Riordan配列の言葉では、停止前prefixの列は積分作用素の基本列であり、評価`x=1`が指数級数を返す。
