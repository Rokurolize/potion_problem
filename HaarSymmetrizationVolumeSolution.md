# Haar対称化体積解

この解では、順序付きprefix領域を対称群の一様平均で取り出す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。累積時刻

```text
t_k = S_k
```

へ移すと、Jacobianは`1`であり、生存条件`S_n <= 1`は

```text
0 <= t_1 <= ... <= t_n <= 1
```

になる。対称群`S_n`は立方体`[0,1]^n`に座標の置換で作用する。この有限群のHaar平均、すなわち一様平均で立方体を対称化すると、測度零の対角を除いて`n!`個の同じ大きさの順序室に分かれる。

したがって恒等順序室の体積は

```text
vol{0 <= t_1 <= ... <= t_n <= 1} = 1/n!.
```

よって

```text
P(S_n <= 1)=1/n!,
E[tau]=sum_{n>=0} 1/n! = e.
```

この証明では、単体積分を直接計算しない。対称群の一様平均により、停止前prefixが立方体の基本領域一つ分であることだけを読む。
