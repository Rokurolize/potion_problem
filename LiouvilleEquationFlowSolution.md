# Liouville方程式流解

この解では、prefix質量が一口ごとに右へ流れる様子をLiouville型の保存方程式として見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixも含める標準恒等式により

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

境界`x`までに存在できるprefixの総質量を

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

と置く。境界を`x`から`x+dx`へ動かすと、新しく入る質量は「すでに`x`以下にいるprefix」に最後の一口を長さ`dx`の窓へ落とすことで生まれる。独立一様性により、その流入率は現在の総質量そのものなので

```text
F(x+dx)-F(x)=F(x) dx + o(dx).
```

これは境界座標に沿ったLiouville型の流れ方程式

```text
dF/dx = F
```

である。境界`0`では空prefixだけが存在するから

```text
F(0)=1.
```

したがって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

ここで保存されているのは各固定長の確率質量ではなく、全prefix長を合算した占有質量である。そのため、境界を動かしたときの流入がそのまま自己増殖率になる。
