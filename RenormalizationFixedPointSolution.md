# 再正規化固定点解

独立一様乱数を `X_1, X_2, ...` とし、

```text
S_n = X_1 + ... + X_n
```

とおく。境界を `r` に動かした停止前prefix総量を

```text
F(r) = sum_{n >= 0} P(S_n <= r)
```

と定める。求めたい値は `F(1)` である。

空prefixがあるので

```text
F(0) = 1
```

である。

## 一段進めて残りを再正規化する

最初の一口を `x` とする。`x <= r` なら、残り容量は `r-x` であり、その後の全prefix総量は同じ関数 `F(r-x)` で測れる。`0 <= r <= 1` では、許される `x` の範囲で一様密度は常に `1` だから、

```text
F(r) = 1 + int_0^r F(r-x) dx.
```

変数を入れ替えると

```text
F(r) = 1 + int_0^r F(u) du.      (1)
```

これは「一口を剥がし、残り容量を同じスケールの問題として読み直す」再正規化である。

## 固定点を読む

右辺の作用素を

```text
(T F)(r) = 1 + int_0^r F(u) du
```

と書くと、停止前prefix総量は

```text
F = T F
```

を満たす固定点である。固定点方程式(1)を微分すると

```text
F'(r) = F(r),   F(0)=1
```

なので、

```text
F(r) = exp(r)
```

である。したがって

```text
E[tau] = F(1) = e
```

となる。

この証明では、停止時刻の分布や各次元の単体体積を先に求めない。一口を剥がした後に残る問題が、容量だけを変えた同じ問題として戻ることを固定点として読む。`e` は、この再正規化作用素の固定点の境界値である。
