# Betti number simplex解

この解では、停止前領域を可縮な単体として扱い、その体積だけを測る。

長さ`n`で止まっていない履歴は

```text
D_n={x_i>=0, x_1+...+x_n<=1}
```

である。`D_n`は標準`n`単体で、位相的には可縮である。したがってBetti数は

```text
b_0=1, b_k=0 (k>0)
```

となり、余分な穴や複雑な成分はない。確率として必要なのは、この単一成分のLebesgue体積である。

標準単体の体積は

```text
vol(D_n)=1/n!
```

だから

```text
P(tau>n)=1/n!.
```

尾和公式より

```text
E[tau]=sum_{n>=0}P(tau>n)=sum_{n>=0}1/n!=e.
```

停止前領域は各長さで一つの可縮な単体なので、位相的な複雑さはなく、体積だけが指数級数の係数を作る。
