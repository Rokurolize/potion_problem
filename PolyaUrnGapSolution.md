# Pólya壺隙間解

この解では、余白を含む隙間ベクトルをPólya壺の対称性として読み、次の生存確率を取り出す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。生存確率を

```text
p_n=P(S_n<=1)
```

と置く。

`S_{n-1}<=1`のもとで、これまでの`n-1`個の一口と余白`R=1-S_{n-1}`を合わせると、和が`1`である`n`個の隙間になる。対称なPólya壺の初期状態を全色1個ずつと見ると、次に余白色を引く予測確率は、対称性により

```text
1/n
```

である。

実際、次の一口`U_n`が生き残る条件付き確率は余白長の平均そのものだから

```text
P(S_n<=1 | S_{n-1}<=1)=1/n.
```

したがって

```text
p_n=p_{n-1}/n
```

となり、`p_0=1`から

```text
p_n=1/n!.
```

よって

```text
E[tau]=sum_{n>=0}p_n=e.
```

Pólya壺の見方は、余白を一つの色として扱い、次も生き残る確率を対称な予測確率として読むものである。
