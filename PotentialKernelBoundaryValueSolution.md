# potential kernel boundary value解

この解では、残り容量上のポテンシャル核の境界値として期待停止時刻を読む。

残り容量が`x`のとき、停止するまでに読めるprefix数の期待値を`V(x)`とする。空prefixを一つ数え、次の入力`u`が`0<=u<=x`なら容量`x-u`で続くので

```text
V(x)=1+int_0^x V(x-u) du.
```

積分作用素

```text
(Kf)(x)=int_0^x f(x-u)du
```

を使えば、これは

```text
V=(I-K)^{-1}1
```

というポテンシャル核の式である。両辺を微分すると

```text
V'(x)=V(x)
```

であり、境界値は空prefixだけが残る

```text
V(0)=1.
```

したがって

```text
V(x)=e^x.
```

初期容量は`1`なので

```text
E[tau]=V(1)=e.
```

ここでは停止時刻を直接足すのではなく、容量過程のポテンシャル核を解き、その境界値から答えを取り出している。
