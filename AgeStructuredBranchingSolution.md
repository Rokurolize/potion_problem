# 年齢付き分枝過程解

この解では、停止前prefixを「出生時刻の系譜」として読む。個体は生まれてから1単位時間のあいだ、時刻の密度`1`で子を生むとする。すると、ある個体の祖先からの出生間隔列は、ちょうど一様乱数の列と同じ形になる。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止前prefixを数えると

```text
tau = sum_{n>=0} 1_{S_n <= 1}.
```

したがって`E[tau]`は、全ての長さの停止前prefix領域の質量である。

## prefixを出生系譜にする

時刻`0`に根個体を1つ置く。各個体は、自分の出生後の年齢`u`が

```text
0 <= u <= 1
```

の範囲で、密度`du`に従って子を生む。ここでは確率過程としての揺らぎは要らない。必要なのは、期待子測度が区間`[0,1]`上のLebesgue測度であることだけである。

根からある個体までの出生間隔を

```text
u_1, u_2, ..., u_n
```

と書くと、その個体の出生時刻は

```text
u_1 + ... + u_n.
```

時刻`x`までに生まれた全個体の期待総数を

```text
B(x)
```

と置く。すると、`B(1)`は

```text
sum_{n>=0} volume { 0<=u_i<=1, u_1+...+u_n <= 1 }
```

であり、これはそのまま`E[tau]`である。

## 年齢構造から再帰を読む

時刻`x`までの個体を数える。根個体がまず1つある。根以外の個体は、最後の親が時刻`s`までに生まれていて、その子が残り時間`x-s`以内に生まれたものである。

期待子測度の密度は`1`なので、親の出生時刻が`s<=x`なら、その親が時刻`x`までに生む子の期待数は

```text
x - s
```

ではなく、密度で見ると薄い時刻`t`への流入が常に`1`である。したがって、時刻`t`に生まれる期待密度は、それ以前に存在した全個体の期待数`B(t)`である。

これを累積形で書けば

```text
B(x) = 1 + int_0^x B(t) dt.        (1)
```

`x<=1`だけを見ているので、各個体の出生可能年齢の上限`1`はまだ切れない。どの親も、未来の薄い時刻へ密度`1`で子を送れる。

## 期待値

(1)から

```text
B'(x) = B(x),
B(0) = 1.
```

よって

```text
B(x) = exp(x).
```

求める値は

```text
E[tau] = B(1) = e.
```

## この見方の芯

一様乱数列を、単なる点列ではなく「親から子への出生間隔」として読む。停止前であることは、系譜上の出生時刻が時刻`1`以前であることに変わる。

その期待総数は、深さごとの単体体積を計算しなくても、年齢付き分枝過程の平均出生方程式

```text
B = 1 + int B
```

で閉じる。`e`は、時刻`1`までに生まれる期待系譜数として現れる。
