# 境界を動かして微分する解

この解では、境界を`1`に固定せず、まず境界を`x`として考える。最後に`x=1`を代入する。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n
```

とし、

```text
tau_x = min { n >= 1 : S_n > x }
```

と置く。求めたいのは`E[tau_1]`である。

`0 <= x <= 1`に対して

```text
F(x) = E[tau_x]
```

と書く。

## 生きているprefixを数える

停止時刻は、まだ境界を超えていないprefixの個数である。

```text
tau_x = sum_{n >= 0} 1_{S_n <= x}.
```

したがって

```text
F(x) = sum_{n >= 0} P(S_n <= x).        (1)
```

ここで`S_0=0`なので

```text
F(0) = 1.
```

## 境界を少しだけ右へ動かす

`x`から`x+dx`へ境界を動かしたとき、期待値はどれだけ増えるかを見る。

増える分は、これまで`x`を超えていたが、`x+dx`までは超えていないprefixである。つまり、どれかの`S_n`が薄い層

```text
x < S_n <= x + dx
```

に入る寄与である。

`n >= 1`のprefixが`x`付近に来るには、その一つ前のprefixがどこか`y`にいて、次の一口で`x-y`付近を引けばよい。`U`は一様なので、密度は`1`である。

したがって、薄い層への期待流入密度は、境界`x`までに存在しうる停止前prefixの期待個数そのものになる。

```text
F'(x) = F(x).        (2)
```

この等式の意味は単純である。境界をほんの少し右へ動かすと、既に境界の左にある各prefixが、同じ幅だけ「次の一口で新しい層へ届く機会」を持つ。期待される機会の総数が`F(x)`である。

より式で書くと、(1)から

```text
F'(x)
= sum_{n >= 1} density of S_n at x.
```

一方、`S_n = S_{n-1} + U_n`で、`U_n`の密度は`1`だから

```text
density of S_n at x
= P(S_{n-1} <= x) の密度積分
= int_0^x density of S_{n-1} at y dy
```

を全`n`で足すと、右辺はちょうど`sum_{n >= 0} P(S_n <= x)`、すなわち`F(x)`になる。

## 初期値だけで終わる

境界関数`F`は

```text
F'(x) = F(x),
F(0) = 1
```

を満たす。よって

```text
F(x) = exp(x).
```

したがって

```text
E[tau_1] = F(1) = exp(1) = e.
```

## 見方の要点

この解では、`P(tau=n)`を求めない。各`n`の体積を最後に足すのでもない。境界を少し動かしたとき、期待停止回数がどれだけ増えるかを見る。

境界の左側にあるprefixの期待総数が、そのまま新しい薄い層へ流れ込む期待密度になる。そのため、期待値そのものが自分自身を微分したものになり、`e`が現れる。
