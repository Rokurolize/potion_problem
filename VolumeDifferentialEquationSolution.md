# 体積微分方程式解

`S_n = X_1 + ... + X_n` とする。`S_n <= 1` の範囲では一様密度は1なので、

```text
P(S_n <= 1) = A_n(1)
```

ただし

```text
A_n(t) = vol{x_i >= 0, x_1 + ... + x_n <= t}
```

である。

この体積を直接積分せず、`t` を少し増やしたときに増える薄い層だけを見る。`K_n(t) = {x_i >= 0, sum x_i <= t}` と書くと、

```text
K_n(t + h) \ K_n(t)
```

は、斜面 `sum x_i = t` を厚さ `h` だけ押し出したものになる。最後の座標を

```text
x_n = t - (x_1 + ... + x_{n-1})
```

で決めると、斜面の有効な影は

```text
{x_1, ..., x_{n-1} >= 0, x_1 + ... + x_{n-1} <= t}
```

であり、これはちょうど `K_{n-1}(t)` である。厚み方向のヤコビアンはこの座標選択で1なので、

```text
A_n'(t) = A_{n-1}(t)
```

を得る。初期値は

```text
A_0(t) = 1,  A_n(0) = 0  (n >= 1)
```

である。したがって母関数

```text
F(t) = sum_{n >= 0} A_n(t)
```

は

```text
F'(t) = sum_{n >= 1} A_n'(t)
      = sum_{n >= 1} A_{n-1}(t)
      = F(t)
```

かつ `F(0) = 1` を満たす。よって

```text
F(t) = exp(t)
```

である。

到着数は

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので、単調収束により

```text
E[tau]
  = sum_{n >= 0} P(S_n <= 1)
  = sum_{n >= 0} A_n(1)
  = F(1)
  = e
```

となる。各 `A_n(1)=1/n!` を個別に覚える必要はない。次元を1つ落とす薄層の増分だけで、総体積の母関数そのものが指数関数になる。
