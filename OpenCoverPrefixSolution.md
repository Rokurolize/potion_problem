# 開被覆prefix解

この解では、停止前prefixの空間を境界容量による開被覆で読み、局所的な増分を貼り合わせる。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`以下のprefix総量を

```text
F(x)=sum_{n>=0}P(S_n<=x)
```

と置く。区間`[0,1]`を小さな開区間で被覆する。各開区間で境界を`dx`だけ動かすと、既存prefixの総量`F(x)`が一様密度`1`で薄い層へ次のprefixを送るので

```text
F(x+dx)-F(x)=F(x)dx+o(dx).
```

開被覆の重なりで補正は出ない。局所法則が同じなので、貼り合わせた大域方程式は

```text
F'(x)=F(x),  F(0)=1.
```

したがって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

この証明では、各長さの単体を個別に積分しない。境界近傍の開被覆で見える同じ薄層法則を貼り合わせるだけである。
