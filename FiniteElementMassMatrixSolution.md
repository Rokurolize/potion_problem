# 有限要素質量行列解

この解では、生存単体の体積を一次有限要素の質量行列の極限として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`までの総prefix質量を

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

と置く。

区間`[0,1]`を細かい有限要素に分け、境界を一要素ずつ右へ進める。質量行列で見ると、新しい境界要素に流入する質量は、既存prefixの総質量に要素幅を掛けたものになる。メッシュ幅を`h`とすれば

```text
F(x+h)-F(x)=h F(x)+o(h).
```

極限で

```text
F'(x)=F(x),  F(0)=1.
```

したがって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

有限要素の言葉では、質量行列の局所寄与が一様密度のため定数になり、境界方向の全体方程式が指数関数へ退化している。
