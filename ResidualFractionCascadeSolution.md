# 残量割合カスケード解

この解では、一口の絶対量`U_i`ではなく、その時点で残っている距離のうち何割を飲んだかを見る。すると停止前履歴の条件は毎段で同じ単位区間になり、複雑さはJacobianだけに集まる。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えれば

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 残量に対する割合へ変える

残り距離を

```text
R_0 = 1,
R_i = 1 - S_i
```

と書く。停止前に長さ`n`のprefixを持つという条件は

```text
R_i >= 0   (0 <= i <= n)
```

である。

各段で、実際に飲んだ量を残り距離の割合として

```text
V_i = U_i / R_{i-1}
```

と置く。停止前prefixでは`0<=V_i<=1`であり、

```text
R_i = R_{i-1}(1 - V_i).
```

したがって、停止前履歴全体は、`V_i`たちの単位立方体へ移る。境界`S_n<=1`はもう複雑な斜めの単体ではなく、各段で同じ条件`0<=V_i<=1`になる。

## Jacobianを木として足す

初期残量を一般の`r`にしたとき、そこから伸びる全停止前履歴のLebesgue総質量を

```text
H(r)
```

と書く。節点自身も停止前prefixなので、まず`1`を数える。

次に、残量`r`から割合`v`だけ飲むと、実量は

```text
u = r v
```

であり、微小量は

```text
du = r dv.
```

その後の残量は

```text
r(1-v).
```

よって、この節点の下にぶら下がる全履歴質量は

```text
int_0^1 r H(r(1-v)) dv.
```

変数を

```text
t = r(1-v)
```

に戻すと、`r dv = -dt`なので

```text
H(r) = 1 + int_0^r H(t) dt.        (1)
```

これは、各深さの体積を計算した式ではない。停止前履歴の木を、残量割合の座標で一段だけほどき、Jacobianがちょうど次の残量方向の積分に変わることを使っている。

## 初期残量1を読む

(1)から

```text
H'(r) = H(r),
H(0) = 1.
```

したがって

```text
H(r) = exp(r).
```

初期残量は`1`なので、停止前履歴全体の総質量は

```text
H(1) = e.
```

この総質量は、深さ`n`ごとの停止前prefix領域の体積

```text
P(S_n <= 1)
```

を全て足したものだから

```text
E[tau] = H(1) = e.
```

## この見方の芯

この証明では、斜めの単体を直接測らない。各一口を「残り距離の割合」に直すと、停止前である条件は毎回ただの`0<=V_i<=1`になる。

残る情報は、割合座標から実量座標へ戻すJacobianだけである。そのJacobianが一段下の残量積分に変わるため、停止前履歴木の総量は`H=1+\int H`という自己相似な式で閉じる。`e`は、単体体積の列ではなく、残量割合カスケードの総Jacobianとして現れる。
