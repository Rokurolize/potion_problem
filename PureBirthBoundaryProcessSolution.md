# 純出生境界過程解

この解では、境界`x`を時間のように動かし、生きているprefixの期待個数を純出生過程として見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`以下で生きているprefixの総期待数を

```text
M(x)=sum_{n>=0} P(S_n<=x)
```

と置く。求める値は`M(1)`である。

境界を`x`から`x+dx`へ広げる。既存の生存prefixそれぞれは、最後に一様な一口を足して、その値が新しい薄い層`(x,x+dx]`へ入る場合にだけ新しいprefixを一つ生む。一様密度は`1`なので、各prefixの出生率は`dx`である。

したがって期待値の一次変化は

```text
M(x+dx)-M(x)=M(x) dx+o(dx).
```

つまり

```text
M'(x)=M(x).
```

境界`0`では空prefixだけがあるから

```text
M(0)=1.
```

よって

```text
M(x)=e^x,
E[tau]=M(1)=e.
```

ここでの「出生」は物理的な時間発展ではなく、境界を広げたときに許されるprefixが増えることを表す。停止時刻の期待値は、境界`1`まで育てた純出生過程の平均個体数である。
