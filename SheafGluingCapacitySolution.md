# sheaf gluing capacity解

この解では、残り容量ごとの停止前prefix総量を、区間上の局所データを貼り合わせる量として見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。残り容量が`x`のとき、停止前prefixの期待総数を`G(x)`と書く。空prefixを一つ持ち、次の増分`u`が`0<=u<=x`なら残り容量は`x-u`である。したがって局所データは

```text
G(x)=1+int_0^x G(x-u) du.
```

これは小区間ごとの残り容量データを貼り合わせる条件であり、変数を替えれば

```text
G(x)=1+int_0^x G(t) dt.
```

よって

```text
G'(x)=G(x),  G(0)=1,
```

なので

```text
G(x)=e^x.
```

最初の容量は`1`だから

```text
E[tau]=G(1)=e.
```

sheafという言葉で大げさにしているが、核は「残り容量で見れば局所再帰がそのまま全体へ貼り合わさる」という一点である。
