# Dirichlet隙間正規化解

この解では、飲んだ量そのものではなく、「境界`1`までの隙間を含めた分割」として停止前のprefixを数える。

独立な一様乱数を`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

と置く。`n`回後も止まっていないとは

```text
S_n <= 1
```

である。

停止時刻は、生存しているprefixの個数だから

```text
E[tau] = sum_{n>=0} P(S_n <= 1).        (1)
```

## 余白を一つ足す

`S_n <= 1`のとき、最後に残った余白を

```text
G_{n+1} = 1 - S_n
```

と置く。また

```text
G_i = U_i      (1 <= i <= n)
```

と置けば、

```text
G_1 + ... + G_n + G_{n+1} = 1,
G_i >= 0.
```

つまり、生存している`n`口分のデータは、`n+1`個の非負な隙間で区間`[0,1]`を割った点である。

逆に、`n+1`個の非負な隙間で和が`1`なら、最初の`n`個を飲んだ量とすれば必ず`S_n<=1`になる。したがって`P(S_n<=1)`は、この`n+1`成分の隙間単体を、最後の成分を消して見たときの体積である。

## Dirichlet(1,...,1)の正規化を使う

`n+1`成分のDirichlet分布

```text
Dirichlet(1,...,1)
```

は、この隙間単体上で一様である。その密度は定数で、全質量を`1`にするための正規化定数は

```text
n!
```

である。なぜなら、密度を`n!`にすると単体全体の質量が`1`になるからである。

ここで元の`P(S_n<=1)`は、同じ単体上の「正規化していないLebesgue体積」である。したがって

```text
P(S_n <= 1) = 1 / n!.
```

これは、積分を毎回計算したのではない。隙間を足すとDirichlet分布の台そのものになり、その正規化定数がただちに生存確率を与える。

## 平均回数

(1)より

```text
E[tau]
= sum_{n>=0} 1/n!
= exp(1)
= e.
```

## この見方の芯

停止前の`n`口は、`n`個の増分だけでなく、最後の余白を含めた`n+1`個の隙間として見るのが自然である。すると生存条件は不等式ではなく「隙間の和が`1`」という正規化条件になる。

`e`は、各段の隙間単体をDirichlet分布へ正規化するときに現れる階乗定数を、全ての段で足し戻した値である。
