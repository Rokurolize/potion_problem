# Irwin-Hall left tail解

この解では、和`S_n`の分布全体ではなく、停止に必要な左端だけを見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n=U_1+...+U_n,
tau=min { n>=1 : S_n>1 }
```

とする。尾和公式より

```text
E[tau]=sum_{n>=0}P(S_n<=1).
```

`S_n`はIrwin-Hall分布を持つ。一般の密度は折れ曲がるが、`0<=x<=1`の左端では上限制約`u_i<=1`がまだ効かず、正の直交象限での畳み込みだけになる。従って`n>=1`で

```text
f_{S_n}(x)=x^{n-1}/(n-1)!  (0<=x<=1).
```

よって

```text
P(S_n<=1)=int_0^1 x^{n-1}/(n-1)! dx=1/n!.
```

したがって

```text
E[tau]=sum_{n>=0}1/n!=e.
```

この証明ではIrwin-Hall分布の全形を使わない。停止境界が`1`なので、ちょうど畳み込み密度の左端だけで全てが決まる。
