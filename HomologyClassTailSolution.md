# homology class tail解

この解では、停止前単体の基本ホモロジー類を積分領域として使う。

長さ`n`で停止前にある領域は標準単体

```text
D_n={x_i>=0, x_1+...+x_n<=1}
```

である。これは向きを入れると相対ホモロジー

```text
H_n(D_n, boundary D_n)
```

の基本類を持つ。この基本類に体積形式

```text
dx_1...dx_n
```

を積分すると、停止前確率が得られる。

```text
P(tau>n)=int_{[D_n]} dx_1...dx_n=vol(D_n)=1/n!.
```

よって尾和公式から

```text
E[tau]=sum_{n>=0}P(tau>n)=sum_{n>=0}1/n!=e.
```

確率は、停止前単体の基本類に一様密度の体積形式を評価したものとして表せる。
