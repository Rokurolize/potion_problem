# 境界成長バランス解

この解では、閾値`1`を一度固定せず、境界を`x`まで動かしたときに蓄積する生存prefixの総量を見る。

独立な一様乱数`U_1,U_2,...`に対して

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。`0 <= x <= 1`について

```text
M(x) = sum_{n >= 0} P(S_n <= x)
```

を置く。求める値は

```text
E[tau] = M(1)
```

である。これは、`tau`が`S_n <= 1`であるprefixの個数だからである。

## 境界を少し右へ動かす

`x`から`x+h`へ境界を動かしたとき、新しく数えられるprefixは

```text
x < S_n <= x+h
```

に入るものだけである。

`n >= 1`について、その直前の部分和を`S_{n-1}=y`と見る。`0 <= x <= 1`かつ小さい`h>0`では、`y <= x`なら次の一様増分が

```text
x-y < U_n <= x+h-y
```

に入る幅はちょうど`h`である。ここで一様分布の密度は`1`であり、`x+h-y <= 1`も成り立つ。したがって、直前のprefixが`y <= x`にある各質量は、境界の薄い層へ率`1`で流れ込む。

よって、薄層に新しく入る期待個数は

```text
M(x+h) - M(x) = h M(x) + o(h).
```

したがって

```text
M'(x) = M(x)        (0 <= x < 1)
```

である。

## 初期条件

境界が`0`のときに数えられるのは空prefixだけである。

```text
M(0) = P(S_0 <= 0) = 1.
```

ゆえに

```text
M(x) = exp(x)
```

となり、

```text
E[tau] = M(1) = e.
```

この証明は、各`P(S_n <= 1)`を個別に計算しない。境界を動かしたとき、そこまでに蓄積されたprefix質量`M(x)`そのものが次の薄層への流入率になる、という局所バランスだけで指数関数を読む。
