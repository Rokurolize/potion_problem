# cellular filtration volume解

この解では、停止前領域を容量で濾過されるセルの列として見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。容量`x`までで生き残る長さ`n`のprefix領域を

```text
C_n(x)={u_i>=0, u_1+...+u_n<=x}
```

と書く。これは`x`で濾過された`n`セルである。体積を`c_n(x)`とすると、境界を`dx`だけ動かしたときに増える薄い層の断面は一段低いセルだから

```text
c_0(x)=1,
c_n'(x)=c_{n-1}(x),
c_n(0)=0  (n>=1).
```

よって

```text
c_n(x)=x^n/n!.
```

求める期待値は容量`1`での全セル体積の和であり、

```text
E[tau]=sum_{n>=0} c_n(1)
      =sum_{n>=0} 1/n!
      =e.
```

この証明は、停止境界を越える瞬間ではなく、境界内に残っている全ての濾過セルを数える。
