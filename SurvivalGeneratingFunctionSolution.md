# 生存確率母関数解

この解では、境界`x`と個数マーカー`z`を入れた母関数を作り、生存確率列をまとめて閉じる。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止時刻は、境界をまだ越えていないprefixの個数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

よって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 個数を印づける

`0 <= x <= 1`で

```text
a_n(x) = P(S_n <= x)
```

とおく。`a_0(x)=1`である。

一様性と独立性から、`n >= 0`について

```text
a_{n+1}(x) = int_0^x a_n(x-u) du
            = int_0^x a_n(t) dt.
```

ここで母関数

```text
F(z,x) = sum_{n >= 0} z^n a_n(x)
```

を考える。`z`は「何個の乱数を使ったprefixか」を数える印である。

## 母関数が満たす方程式

項別に微分すると

```text
partial_x F(z,x)
= sum_{n >= 1} z^n a_n'(x)
= sum_{n >= 1} z^n a_{n-1}(x)
= z F(z,x).
```

初期条件は

```text
F(z,0) = 1.
```

したがって

```text
F(z,x) = exp(zx).
```

求めたい期待値は、個数の印を消して`z=1`、境界を`x=1`にしたものである。

```text
E[tau]
= sum_{n >= 0} P(S_n <= 1)
= F(1,1)
= exp(1)
= e.
```

## 何がまとまっているか

この証明は、`P(S_n<=1)`を個別に計算しない。生存prefixの全列を`F(z,x)`に入れ、境界`x`を動かすと個数マーカー`z`が1つ前の列を引き上げることを見る。

`z`を残しておくと、途中で

```text
F(z,1) = exp(z)
```

まで分かる。つまり、期待値に必要な`z=1`の値だけでなく、生存prefix列の指数母関数そのものがこの問題の構造として現れている。
