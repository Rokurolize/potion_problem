# Lagrange乗数鞍点解

この解では、各単体体積を直接求めず、和制約をLagrange乗数で押さえた正規化から全prefix質量を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めると

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

長さ`n`の生存領域は、`u_i>=0`かつ`sum u_i<=x`である。和制約にLagrange乗数`lambda`を入れると、正錐上の指数正規化は

```text
int_{u_i>=0} exp(-lambda sum u_i) du_1...du_n = lambda^{-n}.
```

この正規化は、境界容量を動かしたときの鞍点条件が唯一の法線`sum u_i=x`だけを見ることを表している。全次数を足した総量

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

では、境界を`x`から`x+h`へ動かすと鞍点法線に沿う薄層が既存prefix総量を幅`h+o(h)`で受け取る。したがって

```text
F(x+h)-F(x)=hF(x)+o(h).
```

よって

```text
F'(x)=F(x),
F(0)=1.
```

したがって

```text
F(x)=exp(x),
E[tau]=F(1)=e.
```

この証明では、個々の単体体積を使わない。和制約のLagrange法線だけが全次数の境界増分を支配することを見る。`e`は、その鞍点正規化を境界`1`で読んだ値である。
