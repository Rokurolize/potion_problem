# ランダムラベル順序解

この解では、`n`次元単体の体積を積分で計算しない。生存prefixを、ランダムに貼られたラベルが偶然に時刻順と一致する事象として読む。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止前prefixを数えれば

```text
tau = sum_{n>=0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n>=0} P(S_n <= 1).
```

ここで`P(S_n<=1)`を別の確率として見る。

## 累積時刻を補助点にする

`S_n<=1`は、累積時刻

```text
t_1 = U_1,
t_2 = U_1 + U_2,
...
t_n = U_1 + ... + U_n
```

が

```text
0 < t_1 < t_2 < ... < t_n <= 1
```

を満たすことと同じである。変換

```text
(U_1,...,U_n) <-> (t_1,...,t_n)
```

のJacobianは`1`なので、`P(S_n<=1)`はこの順序領域のLebesgue体積である。

ここまでは普通の単体表示だが、ここではその体積を積分で出さない。

## ラベルが昇順になる確率

補助的に、区間`[0,1]`へ独立な一様点

```text
T_1,...,T_n
```

を置く。これらはほとんど確実に相異なる。交換可能性により、`n!`個の大小順序

```text
T_{\sigma(1)} < T_{\sigma(2)} < ... < T_{\sigma(n)}
```

はすべて同じ確率を持つ。

全ての順序の確率を足すと`1`なので、特定の順序

```text
T_1 < T_2 < ... < T_n
```

が起こる確率は

```text
1 / n!.
```

しかしこの事象の確率は、まさに領域

```text
0 < t_1 < t_2 < ... < t_n <= 1
```

のLebesgue体積である。したがって

```text
P(S_n <= 1) = 1 / n!.
```

`n=0`では空prefixが1つあるので同じく`1/0! = 1`である。

## 足し戻す

以上より

```text
E[tau]
= sum_{n>=0} P(S_n <= 1)
= sum_{n>=0} 1/n!
= e.
```

## この見方の芯

この証明は、Irwin-Hall分布や停止時刻の確率質量を求めない。累積時刻列を、独立に散らした`n`個の点にラベル`1,...,n`が貼られている状況として見る。

生存しているという条件は、点の場所そのものの珍しさではない。ラベル順が空間順と完全に一致する、ただ一つの順序を引いたということである。一様性は補助点の交換可能性を作り、独立性は全てのラベル順を等確率にする。答えの`e`は、その「正しい順序は`n!`通りのうち1つ」を全ての長さで足した値として現れる。
