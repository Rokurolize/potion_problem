# Janossy活動度微分解

この解では、停止前prefixをPoisson過程のJanossy密度に活動度を入れて読む。

長さ`n`の生存prefixは、累積時刻

```text
0 <= t_1 <= ... <= t_n <= 1
```

へ移すと、区間`[0,1]`上の順序付き`n`点配置である。増分から累積時刻への変換はJacobian`1`なので、この順序付き領域の体積が

```text
P(S_n <= 1)
```

である。

## Janossy密度に活動度を入れる

単位強度Poisson過程の、順序付き`n`点配置に対するJanossy密度は

```text
exp(-1)
```

である。ここで点数を記録する活動度`z`を入れると、全配置の確率総量は

```text
sum_{n >= 0} exp(-1) z^n / n!.
```

これは

```text
exp(-1) exp(z)
```

である。確率測度としては`z=1`で`1`に戻る。

元の停止前prefix測度は、同じ順序付き配置空間から共通因子`exp(-1)`を外したものなので、活動度付き総量は

```text
exp(z).
```

期待停止回数は活動度を`z=1`にした全prefix総量である。したがって

```text
E[tau] = exp(1) = e.
```

この証明では、各単体を個別に積分しない。Janossy密度に活動度を入れ、全階層に共通する空白正規化だけを外すことで、停止前prefixの総量を一度に読む。
