# Adversarial review

Objection 1: This might be a Poisson costume for the same factorial tail.

Response: It does reuse the factorial tail, but the final proof step is a
normalization argument.  That is a genuine explanatory compression rather than
bulk renaming.

Objection 2: Introducing a Poisson variable might suggest a coupling that is
not proved.

Response: No coupling is claimed.  The proof uses only the mass function
`P(N(1)=n)=e^{-1}/n!` as an external probability distribution.

Objection 3: The simplex integral is still present.

Response: It is present only to establish the original tail probability.  The
distinct contribution is how the infinite sum is evaluated.
