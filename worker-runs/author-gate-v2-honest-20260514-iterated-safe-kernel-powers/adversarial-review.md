# Adversarial review

Objection 1: This is close to the Volterra resolvent candidate.

Response: It is close, so only one such candidate is accepted in this batch.
The distinction is that this proof validates each tail as a kernel power before
summing.

Objection 2: The kernel power computation could just be simplex volume.

Response: The same numbers appear, but the induction is on the one-step
transition of the stopped process.  The proof object is probabilistic.

Objection 3: The novelty might be too modest for the target reaction.

Response: The dossier acknowledges the risk.  The candidate passes because it
is precise, compact, and useful as an audit-friendly survival-kernel proof.
