# Yule系譜で読む媚薬問題

この解では、媚薬の停止時刻を、純出生過程の個体数として読む。

各回の増分を独立な一様乱数`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

`n`回飲んだ後も止まっていない条件は

```text
S_n <= 1.
```

したがって、停止時刻は生きているprefixの個数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

期待値を取ると

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).        (1)
```

ここまでは停止時刻の一般事実である。

## prefixを系譜の出生時刻にする

`S_n <= 1`のもとで、

```text
t_i = S_i
```

と置くと、

```text
0 < t_1 < t_2 < ... < t_n <= 1
```

である。逆に、こうした増加列があれば

```text
U_1 = t_1,
U_2 = t_2 - t_1,
...
U_n = t_n - t_{n-1}
```

で、もとの増分に戻る。差を取るだけなので体積は変わらない。

したがって(1)の右辺は、すべての長さの増加列の体積を足したものになる。

```text
E[tau]
= sum_{n >= 0} int_{0<t_1<...<t_n<=1} dt_1 ... dt_n.        (2)
```

ここから先で、これを別の対象として読む。

## Yule過程

時刻`0`に個体が1ついるとする。各個体は、単位率で子を1つ産む。産まれた子も同じ規則で子を産む。これがYule純出生過程である。

`N_t`を時刻`t`に生きている個体数とする。

ある個体を1つ選び、その祖先からその個体までの出生時刻を

```text
0 < t_1 < ... < t_n <= t
```

と書く。Yule過程では、各枝の出生率が`1`なので、この1本の系譜がこの小さな領域

```text
dt_1 ... dt_n
```

に入る寄与はそのまま

```text
dt_1 ... dt_n
```

である。

従って、時刻`1`の個体数の期待値は、全世代`n`と全出生時刻配置を足した総量である。

```text
E[N_1]
= sum_{n >= 0} int_{0<t_1<...<t_n<=1} dt_1 ... dt_n.        (3)
```

(2)と(3)を比べると、

```text
E[tau] = E[N_1].
```

つまり、媚薬を飲む平均回数は、時刻1のYule過程の平均個体数である。

## 個体数の平均だけを解く

Yule過程では、時刻`t`に個体が`N_t`個いると、全体の出生率は`N_t`である。短い時間`dt`の間に、期待個体数は

```text
E[N_t] * dt
```

だけ増える。よって

```text
d/dt E[N_t] = E[N_t],
E[N_0] = 1.
```

したがって

```text
E[N_t] = exp(t).
```

特に

```text
E[tau] = E[N_1] = exp(1) = e.
```

## 見方の要点

この解では、媚薬の一口一口を、一本の系譜の出生間隔として読む。停止前のprefixは、Yule過程で時刻1までに存在しうる個体の祖先列と同じ配置を持つ。

そのため、求める期待値は「停止時刻の分布」ではなく、「枝分かれ集団の平均個体数」になる。`e`は級数を足し切った名前としてではなく、単位率で自己増殖する平均個体数として現れる。
