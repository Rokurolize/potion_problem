# Hilbert space simplex解

この解では、Euclidean空間内の正錐を合計超平面で切って停止前領域を得る。

`R^n`に通常の内積を入れ、ベクトル

```text
u=(1,...,1)
```

を考える。長さ`n`で停止前にある条件は

```text
<u,x><=1,  x_i>=0
```

である。これはHilbert空間内の正錐を1本の半空間で切った標準単体である。

この単体は頂点

```text
0,e_1,...,e_n
```

を持つので、体積は

```text
1/n!
```

である。したがって

```text
P(tau>n)=1/n!,
E[tau]=sum_{n>=0}1/n!=e.
```

内積で見ると、停止境界は法線`u`を持つ超平面であり、その下にある正錐部分の体積が尾確率になる。
