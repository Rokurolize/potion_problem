# quantum shuffle退化解

この解では、停止前prefixを量子shuffle代数の一文字列として見て、交換係数を1に戻す退化で答えを読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めれば

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

累積時刻へ移すと、生存prefixは区間`[0,1]`上の順序付き点列である。一文字`dt`の反復積分として全prefixを束ねると、shuffle指数が現れる。

量子shuffleでは交差にq重みが付くが、この問題は一文字で、しかも最後は可換な退化`q=1`を読む。したがって全次数和は通常のshuffle指数

```text
sum_{n>=0} int_{0<=t_1<=...<=t_n<=1} dt_1...dt_n = exp(1)
```

へ戻る。よって

```text
E[tau]=e.
```

この証明では、各単体体積を個別に計算しない。停止前prefixを量子shuffle指数に入れ、可換退化で通常の指数へ戻ることを見る。`e`は、その退化値である。
