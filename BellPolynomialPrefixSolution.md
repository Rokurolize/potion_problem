# Bell polynomial prefix解

この解では、停止前prefixをBell多項式のうち単一ブロックだけが残る場合として見る。

長さ`n`の停止前条件は

```text
x_1+...+x_n<=1
```

である。ここで合計値`r`を一つのブロック重みと考えると、`n`個のラベル付き要素が一つの連続ブロックに入る重み密度は

```text
r^{n-1}/(n-1)!
```

になる。これは完全Bell多項式の展開から、多ブロック成分を消して一ブロック成分だけを見ているのと同じ階乗正規化である。

この一ブロック重みを`0<=r<=1`で積分すると

```text
P(tau>n)=int_0^1 r^{n-1}/(n-1)! dr=1/n!.
```

したがって

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

停止前prefixは、分割全体ではなく「合計値という一つのブロック」に潰したBell型の重みとして数えられる。
