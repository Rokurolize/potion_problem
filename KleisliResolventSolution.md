# Kleisliレゾルベント解

この解では、一口飲む操作を確率カーネルのKleisli射として見て、停止前prefixの期待個数をそのレゾルベントとして読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 一口カーネル

残り容量を状態`r`とする。`0 <= r <= 1`で、一口飲んでまだ生きている場合だけを残す劣確率カーネルを

```text
K(r, ds) = 1_{0 <= s <= r} ds
```

と書く。ここで`s`は新しい残り容量であり、`s = r-U`である。一様密度が`1`なのでこの形になる。

停止前prefixを一段延ばすことは、このカーネルをKleisli合成することに等しい。したがって、初期残量`r`から見た停止前prefixの期待個数は

```text
R 1(r) = (I + K + K^2 + K^3 + ... )1(r)
```

である。求める値は`R1(1)`である。

## Kleisliレゾルベント方程式

`m(r)=R1(r)`と置くと

```text
m(r) = 1 + int_0^r m(s) ds.
```

右辺の`1`は空prefixであり、積分項は「一口飲んでまだ生きている」Kleisli射を一回通った後の残り問題である。

微分して

```text
m'(r) = m(r),
m(0) = 1.
```

よって

```text
m(r) = exp(r).
```

したがって

```text
E[tau] = m(1) = e.
```

この証明では、停止時刻の分布を先に求めない。一口の劣確率カーネルをKleisli射として合成し、その全反復をレゾルベントとして足す。`e`は、初期残量`1`でのKleisliレゾルベントの値である。
