# value function simplex解

この解では、残容量`c`に対する価値関数が単体体積そのものになることを使う。

長さ`n`まで停止しない価値を

```text
V_n(c)=P(x_1+...+x_n<=c)
```

と定義する。`0<=c<=1`では、これは倍率`c`の標準単体の体積である。

```text
V_n(c)=vol{x_i>=0, x_1+...+x_n<=c}=c^n/n!.
```

元の問題は`c=1`から始まるので

```text
P(tau>n)=V_n(1)=1/n!.
```

尾和を取ると

```text
E[tau]=sum_{n>=0}V_n(1)
      =sum_{n>=0}1/n!
      =e.
```

価値関数を先に一般の残容量で書くと、停止前確率は単体体積のスケーリング則として一行で出る。
