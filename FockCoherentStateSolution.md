# Fock空間のcoherent state解

この解では、媚薬問題の生存prefix全体を、対称Fock空間の指数ベクトルとして読む。

独立一様乱数`U_1,U_2,...`に対して

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止時刻は生存しているprefixの個数だから、

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 順序付きprefixを対称化する

`n >= 1`について累積和

```text
t_i = S_i
```

を取ると、Jacobianは`1`であり、`S_n <= 1`は順序単体

```text
0 < t_1 < ... < t_n <= 1
```

になる。よって

```text
P(S_n <= 1)
= int_{0<t_1<...<t_n<=1} dt_1 ... dt_n.
```

ここで順序を忘れる。単位立方体`[0,1]^n`は、対角線を除いて`n!`個の順序領域に分かれる。したがって

```text
int_{0<t_1<...<t_n<=1} dt_1 ... dt_n
= (1 / n!) * int_{[0,1]^n} 1 dt_1 ... dt_n.
```

これは対称テンソルの規格化係数そのものである。

## 指数ベクトルのノルムとして見る

`H = L^2([0,1])`とし、`1`を恒等的に`1`である関数とする。対称Fock空間の指数ベクトルを

```text
Exp(1) = direct_sum_{n >= 0} 1^{tensor n} / sqrt(n!)
```

と書く。ここで`n=0`成分はスカラー`1`である。

そのノルム二乗は

```text
||Exp(1)||^2
= sum_{n >= 0} ||1||_{L^2([0,1])}^{2n} / n!
= sum_{n >= 0} 1 / n!.
```

一方、上の順序単体の計算により、この各項はちょうど`P(S_n <= 1)`である。つまり

```text
E[tau]
= ||Exp(1)||^2.
```

`||1||^2 = int_0^1 1 dx = 1`なので、指数ベクトルの基本公式から

```text
||Exp(1)||^2
= exp(||1||^2)
= exp(1)
= e.
```

## 何が見えているか

飲用列の生存prefixは、順序付き時刻列である。対称Fock空間へ移すと、順序を忘れた`n`点配置を`1/sqrt(n!)`で規格化した成分になる。

期待値は、全次数に散らばったこれらの成分のノルム二乗であり、答えの`e`はcoherent stateのノルム公式として出る。
