# cumulative distribution integral解

この解では、部分和の累積分布関数を直接積分して停止前確率を出す。

`S_n=X_1+...+X_n`とする。`0<=s<=1`の範囲では、`S_n<=s`となる領域は

```text
x_i>=0, x_1+...+x_n<=s
```

であり、標準単体を倍率`s`で拡大したものだから

```text
P(S_n<=s)=s^n/n!.
```

停止前条件は`S_n<=1`なので

```text
P(tau>n)=P(S_n<=1)=1/n!.
```

したがって

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

密度を畳み込む代わりに、累積分布関数そのものを単体の拡大体積として見ると、`s=1`で指数級数の係数がそのまま現れる。
