# 分位点室体積解

この解では、累積時刻を分位点関数の値として見て、分位点室の体積を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。部分和`S_k`を

```text
q_k = S_k
```

と書けば、停止前条件は

```text
0 <= q_1 <= ... <= q_n <= 1
```

である。増分から分位点列への変換は下三角で対角が全て`1`なので、Jacobianは`1`である。

立方体`[0,1]^n`は、測度零の境界を除けば全ての順序型に分かれる。分位点列はそのうち昇順の室一つを選んだもので、全ての室は置換対称性で同じ体積を持つ。室の数は`n!`だから

```text
vol{0 <= q_1 <= ... <= q_n <= 1}=1/n!.
```

よって

```text
E[tau]=sum_{n>=0}P(S_n<=1)
      =sum_{n>=0}1/n!
      =e.
```

分位点という言い方では、停止前prefixは経験分布の逆関数を有限個のジャンプで記録したものになる。答えは、その分位点室の基本体積である。
