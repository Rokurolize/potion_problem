# 乗法残量Jacobian解

この解では、停止前prefixの単体を、各段で「残り距離の何割を飲んだか」という単位立方体へほどく。答えの`e`は、斜めの単体体積ではなく、残量が段ごとに掛け算で縮むJacobianの総和として出る。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 割合座標へ移す

長さ`n`の停止前prefixを考える。残り距離を

```text
R_0 = 1,
R_i = 1 - S_i
```

とし、各一口をその直前の残り距離に対する割合

```text
V_i = U_i / R_{i-1}
```

で表す。停止前であれば

```text
0 <= V_i <= 1
```

であり、

```text
R_i = R_{i-1}(1 - V_i).
```

したがって

```text
R_i = prod_{j=1}^i (1 - V_j),
U_i = V_i prod_{j=1}^{i-1} (1 - V_j).
```

つまり、停止前の斜め領域

```text
u_1 + ... + u_n <= 1
```

は、割合座標では単位立方体`[0,1]^n`になる。

## Jacobianが階段状に掛かる

変換`(V_1,...,V_n) -> (U_1,...,U_n)`は下三角である。対角成分は

```text
partial U_i / partial V_i = R_{i-1}
= prod_{j=1}^{i-1} (1 - V_j).
```

よってJacobianは

```text
prod_{i=1}^n R_{i-1}
= prod_{i=1}^n prod_{j=1}^{i-1} (1 - V_j)
= prod_{j=1}^{n-1} (1 - V_j)^{n-j}.
```

したがって

```text
P(S_n <= 1)
= int_{[0,1]^n} prod_{j=1}^{n-1} (1 - v_j)^{n-j} dv_1...dv_n.
```

この積分は完全に分離している。

```text
P(S_n <= 1)
= prod_{j=1}^{n-1} int_0^1 (1 - v)^{n-j} dv
= prod_{j=1}^{n-1} 1/(n-j+1)
= 1/n!.
```

`n=0`では空prefixなので寄与は`1`である。

## 期待値

以上より

```text
E[tau]
= sum_{n >= 0} P(S_n <= 1)
= sum_{n >= 0} 1/n!
= e.
```

## この見方の芯

通常の単体体積計算では、`u_1+...+u_n<=1`をそのまま測る。この証明では、各段の残量を基準にして履歴を単位立方体へ引き伸ばす。

すると停止条件は全ての座標で同じ`0<=V_i<=1`になり、複雑さはJacobianだけに残る。そのJacobianは、早い段の割合ほど後の残量すべてに影響するため、階段状の積

```text
prod_{j=1}^{n-1} (1 - V_j)^{n-j}
```

になる。`e`は、この乗法的な残量縮小の総Jacobianを全ての深さで足した値である。
