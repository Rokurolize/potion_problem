# Banach norm tail解

この解では、prefixを`l^1`ノルム空間の正錐上の点として見る。

長さ`n`のprefixを

```text
x=(x_1,...,x_n)
```

とし、正錐`x_i>=0`上で

```text
||x||_1=x_1+...+x_n
```

を考える。停止前条件は

```text
||x||_1<=1
```

である。したがって停止前領域は、`l^1`単位球を正錐で切った標準`n`単体である。

独立一様分布の密度は1なので

```text
P(tau>n)=vol{x_i>=0, ||x||_1<=1}=1/n!.
```

尾和公式から

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

停止は、prefixベクトルの`l^1`ノルムが1を超える瞬間として読める。
