# 遅延Laplace極解

`m(t)`を、境界を`t`に置いたときの停止時刻の期待値とする。つまり

```text
m(t) = E[ min { n >= 1 : U_1 + ... + U_n > t } ].
```

`0 <= t <= 1`では、空prefixを1個数えたあと、最初の一口が`x <= t`なら同じ問題が残り`t - x`で続く。したがって

```text
m(t) = 1 + int_0^t m(t - x) dx      (0 <= t <= 1)
```

である。ここまでは更新方程式だが、ここではこの式を半直線全体の更新方程式の「時刻1までの前半」としてLaplace側から見る。

密度`1_{0 <= x <= 1}`のLaplace変換は

```text
phi(s) = int_0^1 exp(-sx) dx = (1 - exp(-s)) / s
```

である。`m`は更新測度の分布関数なので、そのLaplace変換は1回積分された幾何級数で

```text
M(s)
  = int_0^infty exp(-st) m(t) dt
  = (1/s) * 1 / (1 - phi(s))
  = 1 / (s - 1 + exp(-s)).
```

分母の`exp(-s)`は、時間領域では「1だけ遅れて戻る」項である。実際、方程式を時間領域へ戻すと、全半直線では

```text
m'(t) = m(t) - m(t - 1) * 1_{t >= 1},      m(0) = 1
```

となる。これは上端1を持つ一様分布の右端から、1単位後にだけ遅延波が返ってくるという意味である。

しかし求めたい値はちょうど`t = 1`までである。`0 <= t < 1`では遅延項`m(t - 1) * 1_{t >= 1}`が存在しない。したがって、この区間で見えるLaplace側の有効な分母は

```text
s - 1
```

だけであり、時間領域では

```text
m'(t) = m(t),      m(0) = 1
```

となる。ゆえに

```text
m(t) = exp(t)      (0 <= t <= 1)
```

である。求める値は境界`t = 1`の値だから

```text
E[tau] = m(1) = e
```

である。

見方を変えると、長さ1の窓の中では一様分布の右端から返ってくる遅延波がまだ到着しない。Laplace側では分母に遅延項が見えているのに、時刻1までの局所解ではまだ働けない。だから最初に残る極`s = 1`だけが答えを決める。
