# 遅延Laplace極解

`u(t)` を、残り時間が `t` のときに数える到着数の期待値とする。最初の一回を数え、その待ち時間が `x <= t` なら同じ問題が残り時間 `t - x` で続くので、

```text
u(t) = 1 + int_0^t u(t - x) dx      (0 <= t <= 1)
```

である。ここまでは更新方程式だが、ここではLaplace側から見る。

密度 `1_{0 <= x <= 1}` のLaplace変換は

```text
phi(s) = int_0^1 exp(-sx) dx = (1 - exp(-s)) / s
```

である。更新測度のLaplace変換は畳み込みの幾何級数だから

```text
U_hat(s) = 1 / (1 - phi(s))
         = s / (s - 1 + exp(-s))
```

となる。

この式だけを見ると遅延項 `exp(-s)` が混ざっている。しかし、求めたいのは `t = 1` までの値である。逆Laplace変換では `exp(-s)` は時間を1だけ遅らせる作用であり、`0 <= t <= 1` の区間ではまだ一度も戻ってこない。したがってこの区間では

```text
U_hat(s)
```

の有効部分は

```text
s / (s - 1)
```

と同じである。時間領域ではこれは

```text
u(t) = exp(t)      (0 <= t <= 1)
```

を意味する。

よって

```text
E[tau] = u(1) = e
```

である。

見方を変えると、長さ1の窓の中では一様分布の右端から返ってくる遅延波がまだ到着しない。Laplace側ではその遅延項を時刻1まで無視でき、残る極 `s = 1` だけが答えを決める。
