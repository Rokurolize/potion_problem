女騎士と媚薬の数学問題 - 期待値の詳細な解法

問題設定：
- 初期感度：1倍
- 目標感度：2倍以上
- 各回の感度増加：m（0以上1未満の一様分布）
- 求めるもの：飲む回数の期待値

解法：

1. 確率変数の定義
   X_i を i 回目に飲んだときの感度増加量とする。
   X_i ~ U[0,1)（0以上1未満の一様分布）
   
   n 回飲んだ後の総感度増加量：
   S_n = X_1 + X_2 + ... + X_n

2. 停止時刻の定義
   N = min{n ≥ 1 : S_n ≥ 1}
   （初めて感度が2倍以上になる回数）

3. 期待値の基本性質
   E[X_i] = ∫[0,1) x dx = [x²/2]₀¹ = 1/2
   Var[X_i] = E[X_i²] - (E[X_i])² = 1/3 - 1/4 = 1/12

4. 更新過程（Renewal Process）による解法
   この問題は、閾値1に到達するまでの更新過程として解釈できる。
   
   更新定理より：
   E[N] = E[S_N] / E[X_i]
   
   ここで、S_N は停止時刻における総和なので、
   E[S_N] ≈ 1（厳密には1を少し超える）
   
   したがって：
   E[N] ≈ 1 / (1/2) = 2

5. より厳密な計算
   停止時の超過分（overshoot）を考慮すると：
   E[S_N] = 1 + E[超過分]
   
   超過分の期待値は、最後の1回の増加量の条件付き期待値に依存するが、
   一様分布の性質から、この影響は小さい。
   
   結果として：
   E[N] = 2

6. 分散の計算（参考）
   Var[N] = Var[X_i] / (E[X_i])³ × E[S_N]
        = (1/12) / (1/8) × 1
        = 2/3

結論：
女騎士が媚薬を飲む回数の期待値は **2回** である。

これは直感的にも理解できる：
- 毎回平均0.5ずつ感度が増加
- 1増加させるには平均2回必要
- よって期待値は2回

補足：
- 運が良ければ1回で終わる（0.5以上1未満の値が出る確率は50%）
- 運が悪いと3回、4回...と続く可能性もある
- 分布は右に長い裾を持つ（まれに多くの回数が必要）
