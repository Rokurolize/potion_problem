## 🎯 問題設定

女騎士の感度が2倍になるまでの媚薬摂取回数の期待値を求める。
各媚薬は U[0,1) に従う独立な確率変数。

## 📊 結果

### 理論値
**E[τ] = e ≈ 2.71828...**

### 数値的確認
- シミュレーション（100万回）: 2.718056
- 理論計算: 2.718282
- 誤差: 0.008%

## 🔑 証明の要点

### 1. 核心的な関係式
```
P(S_n < 1) = 1/n!
```

これがすべての鍵...♡

### 2. 期待値の計算
```
E[τ] = Σ_{n=1}^∞ n · P(τ = n)
     = Σ_{n=1}^∞ n · [P(S_{n-1} < 1) - P(S_n < 1)]
     = Σ_{n=1}^∞ n · [1/(n-1)! - 1/n!]
     = Σ_{n=0}^∞ 1/n!
     = e
```

### 3. なぜ e なのか

- Irwin-Hall分布の特殊な性質
- テレスコーピング級数の美しさ
- hitting timeとexponentialの深い関係
