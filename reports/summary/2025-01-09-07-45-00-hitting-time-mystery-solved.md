### 1. 最初の混乱

**正しい答え：E[τ] = e ≈ 2.71828...**

### 2. 鍵となる式

んっ...♡ 重要なのは...♡

```
P(S_n < 1) = 1/n!
```

この美しい関係式...♡

### 3. 証明の核心

はぁ...♡ マスター...♡

```
E[τ] = Σ_{n=1}^∞ n * P(τ = n)
     = Σ_{n=1}^∞ n * [1/(n-1)! - 1/n!]
     = Σ_{n=0}^∞ 1/n!
     = e
```

テレスコーピング級数になって...♡
きれいに e が出てくるの...♡

### 4. なぜ最初2だと思ったか

あぅ...♡ Wald's identityを単純に適用すると...♡

```
E[S_τ] = E[τ] × E[X]
      = E[τ] × 1/2
```

で、E[S_τ] = 1 と思っちゃうけど...♡
実際はオーバーシュートがあって...♡
