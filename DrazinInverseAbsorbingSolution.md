# Drazin逆吸収解

この解では、境界`1`を吸収壁と見て、非吸収部分の基本作用素をDrazin逆の有限容量として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。吸収前の状態空間を`[0,1]`とし、非吸収遷移作用素を

```text
(Tf)(s)=int_0^{1-s} f(s+u) du
```

と置く。これは「次の一口を足してもまだ`1`以下に残る」部分だけを数える作用素である。

空prefixから出発したとき、吸収前に訪れるprefix数の期待値は基本作用素

```text
N = I + T + T^2 + ...
```

を定数関数`1`へ作用させ、`s=0`で評価したものだから

```text
E[tau]=(N1)(0).
```

`G(s)=(N1)(s)`と置くと、基本方程式は

```text
G(s)=1+int_0^{1-s} G(s+u) du.
```

変数`x=1-s`で`F(x)=G(1-x)`と書き直せば

```text
F(x)=1+int_0^x F(y) dy.
```

よって

```text
F'(x)=F(x),  F(0)=1,
F(1)=e.
```

したがって

```text
E[tau]=e.
```

有限吸収連鎖なら`N=(I-T)^{-1}`で済む。この問題では連続状態なので、吸収固有空間を除いた基本作用素、つまりDrazin逆に相当する部分を積分方程式として実体化している。
