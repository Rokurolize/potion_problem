# 条件付き生存率積解

この解では、`P(S_n<=1)`を単体体積として直接出さず、「ここまで生き残ったあと、次も生き残る確率」の積として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefix数を数えると

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

ここで

```text
p_n=P(S_n<=1)
```

と置く。`S_{n-1}<=1`で条件付けたとき、点

```text
(U_1,...,U_{n-1}, R)
```

を

```text
R=1-S_{n-1}
```

で補うと、`n`個の非負成分の和が`1`である単体上の一様な点になる。対称性から、余り`R`の平均は各成分と同じで

```text
E[R | S_{n-1}<=1] = 1/n.
```

次の一口`U_n`は一様で独立なので、条件付きで次も生き残る確率は残り長さそのものの平均である。

```text
P(S_n<=1 | S_{n-1}<=1)
= E[ P(U_n<=R | U_1,...,U_{n-1}) | S_{n-1}<=1]
= E[R | S_{n-1}<=1]
= 1/n.
```

したがって

```text
p_n = p_{n-1}/n,  p_0=1,
```

なので

```text
p_n=1/n!.
```

よって

```text
E[tau]=sum_{n>=0} p_n
      =sum_{n>=0} 1/n!
      =e.
```

この見方では、各次元の体積を毎回計算しない。生存しているprefixは、余りを含めると対称な割り分けになり、その対称性が次の生存率を`1/n`に決める。`e`は、その条件付き生存率の連続積が作る階乗の和として現れる。
