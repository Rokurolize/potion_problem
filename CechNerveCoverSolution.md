# Cech nerve cover解

この解では、停止前単体を小さい直方体で被覆し、そのnerveの極限として体積を読む。

長さ`n`で停止していない履歴は

```text
Delta_n={x_i>=0, x_1+...+x_n<=1}
```

である。刻み幅を`h=1/m`とし、格子点

```text
k_1+...+k_n<=m
```

に対応する小直方体で`Delta_n`を内側から被覆する。この被覆の交わり関係は、合計次数`m`以下の格子点からできる単体的なnerveで記録される。

格子点の個数は

```text
binom(m+n,n)
```

で、各小直方体の体積は`h^n=m^{-n}`である。境界の誤差は`O(1/m)`で消えるため、

```text
P(tau>n)=vol(Delta_n)
       =lim_{m->infty} binom(m+n,n)/m^n
       =1/n!.
```

したがって尾和公式から

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

停止前履歴を直接積分する代わりに、有限被覆のnerveで数えてから連続極限へ戻すと、階乗分母が格子単体の漸近として現れる。
