# 順序多面体の線形拡張解

独立一様乱数を `X_1, X_2, ...` とし、

```text
S_n = X_1 + ... + X_n
```

とおく。停止前prefixの個数として

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので、

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

`n >= 1`で部分和

```text
t_i = S_i
```

へ移る。差分を取れば `X_i = t_i - t_{i-1}` に戻るので、ヤコビアンは `1` である。条件 `S_n <= 1` は

```text
0 <= t_1 <= t_2 <= ... <= t_n <= 1
```

になる。

## 全順序の順序多面体

この領域は、全順序

```text
1 < 2 < ... < n
```

の順序多面体

```text
O(P_n) = { 0 <= t_1 <= ... <= t_n <= 1 }
```

そのものである。

一般に、有限poset `P` の順序多面体は、各線形拡張ごとに単位立方体 `[0,1]^n` の中の標準単体を一つ持つ。したがって

```text
vol O(P) = e(P) / n!
```

である。ただし `e(P)` は `P` の線形拡張の個数である。

今回の `P_n` はすでに全順序なので、線形拡張は一つしかない。よって

```text
P(S_n <= 1)
= vol O(P_n)
= 1 / n!
```

である。

したがって

```text
E[tau]
= sum_{n >= 0} 1 / n!
= e
```

となる。

この解では、単体の体積を積分で計算しない。停止前prefixは「時刻が増える」という全順序制約を持つ点配置であり、その配置空間の体積は順序多面体の線形拡張数で決まる。`e` は、各長さで線形拡張が一つだけ残る全順序の順序多面体を全次数で足した値として現れる。
