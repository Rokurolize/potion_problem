# 大正準理想気体解

この解では、停止前prefix全体を、区間`[0,1]`に置いた一次元理想気体の全粒子数分配関数として読む。

独立な一様乱数を`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

と置く。`S_0=0`も含めれば、

```text
tau = sum_{n>=0} 1_{S_n <= 1}
```

である。したがって

```text
E[tau] = sum_{n>=0} P(S_n <= 1).        (1)
```

## 粒子配置へ変換する

`S_n <= 1`の上では、到着時刻

```text
t_i = S_i
```

が

```text
0 <= t_1 <= ... <= t_n <= 1
```

を満たす。増分から到着時刻への変換のJacobianは`1`である。

よって`P(S_n<=1)`は、区間`[0,1]`に`n`個の粒子を順序付きで置く配置空間の体積である。

ここで粒子を区別しないことにする。区別された粒子の配置空間`[0,1]^n`の体積は`1`であり、ほとんど至る所で同じ非順序配置は`n!`通りのラベル付けを持つ。したがって、非区別粒子`n`個の配置空間体積は

```text
1/n!
```

である。

## 全粒子数を一度に足す

停止前prefixの期待総数は、全ての粒子数の配置空間体積を足したものになる。

```text
E[tau] = sum_{n>=0} 1/n!.        (2)
```

これは、活動度`z=1`、体積`1`の理想気体の大正準分配関数である。

粒子同士の相互作用がないので、分配関数は1粒子体積の指数関数になる。

```text
Xi(z) = exp(z * 1).
```

したがって

```text
E[tau] = Xi(1) = exp(1) = e.
```

## この見方の芯

この証明は、停止時刻の分布を求めない。生存している全prefixを、粒子数が固定されていない非相互作用粒子系としてまとめて数える。

`e`は、各粒子数の単体体積を後から足した偶然ではない。非相互作用配置の大正準分配関数、つまり「全ての粒子数を同時に許す正規化」がそのまま平均停止回数になっている。
