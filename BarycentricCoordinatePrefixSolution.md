# barycentric coordinate prefix解

この解では、停止前prefixを標準単体の重心座標として読む。

長さ`n`まで停止しない条件は

```text
u_i>=0,  u_1+...+u_n<=1.
```

余り

```text
u_{n+1}=1-(u_1+...+u_n)
```

を加えると、これは

```text
u_1,...,u_{n+1}>=0,  u_1+...+u_{n+1}=1
```

という`n`次元標準単体の重心座標になる。`u_{n+1}`を落とす射影はヤコビアン`1`で、射影された領域の体積は

```text
1/n!
```

である。従って

```text
P(U_1+...+U_n<=1)=1/n!.
```

尾和公式より

```text
E[tau]=sum_{n>=0}P(U_1+...+U_n<=1)=sum_{n>=0}1/n!=e.
```

停止前条件は「余り」も含めた重心座標が非負である、という単体の基本条件そのものである。
