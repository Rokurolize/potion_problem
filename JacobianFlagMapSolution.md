# Jacobian旗写像解

この解では、増分列から累積時刻列への三角写像を旗写像として見て、そのJacobianだけで生存領域を数える。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。長さ`n`の停止前prefixに対して

```text
Phi_n(u_1,...,u_n)=(s_1,...,s_n),
s_k=u_1+...+u_k
```

を考える。これは部分和の旗

```text
0 <= s_1 <= ... <= s_n <= 1
```

を作る写像である。Jacobian行列は下三角で対角成分が全て`1`なので

```text
|det D Phi_n|=1.
```

したがって

```text
P(S_n <= 1)
 = vol{0 <= s_1 <= ... <= s_n <= 1}.
```

区間`[0,1]^n`は、測度零の重なりを除いて`n!`個の順序領域に分かれ、それぞれの体積は同じである。よって

```text
P(S_n <= 1)=1/n!.
```

空prefixを含めて足すと

```text
E[tau]=sum_{n>=0} P(S_n <= 1)
      =sum_{n>=0} 1/n!
      =e.
```

この見方の焦点は、停止条件を斜めの不等式から旗の順序条件へ移すことにある。確率分布名は不要で、三角Jacobianと対称分割だけで終わる。
