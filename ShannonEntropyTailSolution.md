# Shannon entropy tail解

この解では、停止前領域を一様測度の情報量で見る。

長さ`n`で止まらない履歴の集合は

```text
D_n={x_i>=0, x_1+...+x_n<=1}
```

である。元の同時分布は単位立方体上の一様分布なので、任意の可測集合の確率はそのLebesgue体積である。したがって、停止前事象の自己情報量は

```text
-log P(tau>n)=-log vol(D_n)
```

で測られる。

標準単体の体積は

```text
vol(D_n)=1/n!
```

だから、確率そのものは

```text
P(tau>n)=exp(-log n!)=1/n!.
```

尾和公式により

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

情報量の言葉では、停止前事象の符号長が`log n!`であり、その指数を戻すと指数級数の係数が現れる。
