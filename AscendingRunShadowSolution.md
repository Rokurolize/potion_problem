# 初期上昇ランの影として解く

この解では、媚薬の足し算をいったん別の問題へ移す。別の問題とは、独立な一様乱数列が最初に下降するまでの長さである。

元の問題では、独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。`n`回飲んだ後も止まっていないことは

```text
S_n <= 1
```

である。

## もう一つの実験

別に、独立一様乱数`V_1,V_2,...`を用意する。初めからどこまで厳密に上昇しているかを

```text
R = max { n >= 0 : V_1 < V_2 < ... < V_n }
```

と定める。空の条件は真なので、少なくとも`R >= 0`である。

このとき

```text
P(R >= n) = P(V_1 < V_2 < ... < V_n).
```

`n`個の独立な連続一様乱数の大小順序はすべて同じ確率で起こる。並び方は

```text
n!
```

通りあり、そのうち上昇順は1通りだけである。したがって

```text
P(R >= n) = 1 / n!.        (1)
```

## 媚薬側の同じ尾

媚薬側で`tau > n`とは、`n`回後も止まっていないこと、つまり

```text
S_n <= 1
```

である。ここで

```text
t_i = S_i
```

と置くと、

```text
0 < t_1 < t_2 < ... < t_n <= 1.
```

逆に、このような上昇列から

```text
U_1 = t_1,
U_2 = t_2 - t_1,
...
U_n = t_n - t_{n-1}
```

で増分へ戻れる。変換は差を取る三角変換で、体積を変えない。

したがって

```text
P(tau > n)
= volume { 0 < t_1 < ... < t_n <= 1 }
= 1 / n!.        (2)
```

(1)と(2)から

```text
P(tau > n) = P(R >= n)
```

である。つまり

```text
tau - 1
```

と

```text
R
```

は同じ尾分布を持つ。

## 上昇ランの平均を数える

非負整数値の量は、尾確率を足すと平均になる。よって

```text
E[tau]
= sum_{n >= 0} P(tau > n)
= sum_{n >= 0} P(R >= n).
```

右辺は`R+1`の期待値である。つまり、媚薬を飲む平均回数は

```text
initial increasing run length + 1
```

の平均と同じである。

あとは(1)を足すだけでよい。

```text
E[tau]
= sum_{n >= 0} 1 / n!
= e.
```

## 何が化けたか

元の問題は「足し算が1を超えるまで」だった。ところが、部分和

```text
S_1, S_2, ..., S_n
```

へ変数を取り替えると、条件は「`n`個の点が最初から上昇順に並んでいる」へ化ける。

そのため、媚薬の停止時刻は、別の一様乱数列の初期上昇ランと同じ尾を持つ。答えの`e`は、足し算の分布計算ではなく、ランダムな順列がどこまで最初から昇順でいるかの平均として現れる。
