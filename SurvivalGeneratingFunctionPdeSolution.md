# survival generating function PDE解

この解では、生存確率を次数方向に並べた生成関数が満たすPDEから一気に読む。

初期容量を`x`にして

```text
p_n(x)=P(U_1+...+U_n<=x)
```

とおく。ただし`p_0(x)=1`である。`0<=x<=1`では、最後の変数で切ると

```text
p_n(x)=int_0^x p_{n-1}(x-u) du
```

だから

```text
p_n'(x)=p_{n-1}(x),  p_n(0)=0  (n>=1).
```

ここで生存確率の生成関数

```text
G(x,z)=sum_{n>=0} p_n(x) z^n
```

を作ると、上の微分関係は

```text
partial_x G(x,z)=z G(x,z),  G(0,z)=1
```

に束ねられる。従って

```text
G(x,z)=exp(zx).
```

元の停止時刻については尾和公式で

```text
E[tau]=sum_{n>=0}P(U_1+...+U_n<=1)=G(1,1)=e.
```

この見方では、各単体の体積を個別に計算せず、生存確率列が容量方向の一次PDEに閉じることを使って指数関数を取り出している。
