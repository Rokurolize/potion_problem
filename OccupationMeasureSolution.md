# 停止前prefixの占有測度で解く

この解では、何回目で止まるかの分布を求めない。かわりに、停止するまでに部分和が区間`[0,1]`の各場所をどれだけ訪れるかを数える。

独立な一様乱数を`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止前に現れるprefixは

```text
S_0, S_1, ..., S_{tau-1}
```

である。したがって

```text
tau = number of stopped-before-crossing prefixes.
```

## 占有測度

`[0,1]`の部分集合`B`に対し、

```text
mu(B) = E[ #{ n : 0 <= n < tau, S_n in B } ]
```

と定める。これは、停止前prefixが`B`を訪れる期待回数である。

求めたい量は全体の質量

```text
E[tau] = mu([0,1]).
```

最初のprefixは必ず`S_0=0`にいる。したがって`mu`は`0`に質量`1`を持つ。

問題は、`0`以外の場所での密度である。

## 密度の自己再生式

`0 < x < 1`を見る。停止前prefixがちょうど`x`付近に現れる方法は二つに分かれる。

まず、`S_0=0`から一口で`x`付近へ来る寄与がある。一様乱数なので、この寄与の密度は`1`である。

次に、どこかの停止前prefixが`y`にあり、そこから次の一口で`x`付近へ来る寄与がある。これは`0 <= y < x`の全てから来る。一様乱数の密度はやはり`1`なので、もし`mu`が`(0,1)`上で密度`g`を持つなら、

```text
g(x) = 1 + int_0^x g(y) dy.        (1)
```

ここで`1`は原点からの一歩、積分はそれ以後の全prefixからの一歩である。

この式は、停止時刻の分布ではなく、停止前prefix全体の期待訪問密度を直接記述している。

## 期待訪問密度を解く

(1)を微分すると

```text
g'(x) = g(x).
```

また(1)から

```text
g(0) = 1.
```

よって

```text
g(x) = exp(x).
```

したがって、停止前prefixの期待総数は

```text
E[tau]
= mu([0,1])
= 1 + int_0^1 g(x) dx.
```

右辺の`1`は必ず存在する`S_0=0`の分である。そこで

```text
E[tau]
= 1 + int_0^1 exp(x) dx
= 1 + (exp(1) - 1)
= exp(1)
= e.
```

## この解の見方

普通は、`n`回後に生きている確率を各`n`で数える。ここでは時間方向の数え上げをやめ、空間方向に見る。

停止前の全prefixが区間上に作る期待密度は、自分自身を一様ステップで右へ送ったものに、原点からの最初の一歩を足したものになる。その自己再生式が

```text
g = 1 + int g
```

であり、その密度が`exp(x)`になる。答えの`e`は、停止前prefix全体が`[0,1]`に残す期待質量として現れる。
