# Kan拡張prefix解

この解では、長さごとのprefix質量を、境界方向への左Kan拡張としてまとめて読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`までの停止前prefix総量を

```text
F(x)=sum_{n>=0} P(S_n <= x)
```

と置く。`0 <= x <= 1`では、一口追加の密度は定数`1`なので、境界を`dx`だけ広げたとき既存prefix全体が次の段へ左Kan拡張される量は

```text
dF = F(x) dx.
```

したがって

```text
F'(x)=F(x),  F(0)=1.
```

この微分方程式は、空prefixを始対象として、境界方向に自由に一口を付け足す左Kan拡張の普遍性が、ただの指数生成に退化した形である。よって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

この見方では、各`n`の体積を個別に出さない。境界を動かす関手が、既存prefix全体を一段先へ自由に延長することだけを使う。
