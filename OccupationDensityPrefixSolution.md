# occupation density prefix解

この解では、停止前prefixを容量区間`[0,1]`上の占有密度として数える。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。時刻`n`の部分和が`x`以下にいる確率を

```text
A_n(x)=P(S_n<=x)
```

と書く。`0<=x<=1`では最後の増分を積分して

```text
A_n(x)=int_0^x A_{n-1}(x-u) du,
A_0(x)=1.
```

よって帰納的に

```text
A_n(x)=x^n/n!.
```

停止前prefixの期待個数は、各時刻の占有確率の総和である。

```text
E[tau]=sum_{n>=0} P(S_n<=1)
      =sum_{n>=0} A_n(1)
      =sum_{n>=0} 1/n!
      =e.
```

「占有密度」という名前を付けているが、必要なのは難しい過程論ではない。停止境界の内側にいる時刻を一つずつ数え、その占有質量を足すだけである。
