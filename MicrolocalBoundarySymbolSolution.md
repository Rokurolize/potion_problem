# microlocal境界記号解

この解では、一口追加核を境界近くの局所記号として見て、停止前prefixの全量を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`以下のprefix総量を

```text
F(x)=sum_{n>=0} P(S_n<=x)
```

と置く。`0<=x<=1`では、一様密度の右端は境界近くでは見えず、一口追加核は半直線上の定数密度の局所記号だけを持つ。

境界を`x`から`x+dx`へ動かすと、新しく入るprefixは、既存prefixに最後の一口が薄い境界層へ入るものだけである。局所記号が定数`1`なので、その主記号寄与は

```text
F(x) dx
```

である。したがって

```text
F'(x)=F(x),  F(0)=1.
```

よって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

microlocalな言い方をすれば、この問題では境界記号が完全に平坦で、低次の補正が消える。その主記号だけが指数関数を作る。
