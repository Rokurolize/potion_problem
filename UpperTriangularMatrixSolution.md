# 上三角行列指数解

この解では、停止前prefixの有限切断を上三角行列の指数として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`までの長さ別質量を

```text
a_n(x)=P(S_n <= x)
```

と置く。`0 <= x <= 1`では、一様密度の右端は効かないので

```text
a_0(x)=1,
a_n'(x)=a_{n-1}(x)  (n>=1),
a_n(0)=0  (n>=1).
```

有限次元で見ると、ベクトル`a=(a_0,...,a_N)`は、下隣へ送るnilpotent行列`J`に対して

```text
a'(x)=J a(x),  a(0)=e_0
```

を満たす。したがって

```text
a(x)=exp(xJ)e_0.
```

成分は

```text
a_n(x)=x^n/n!
```

であり、有限切断を上へ伸ばすと

```text
E[tau]=sum_{n>=0}a_n(1)=sum_{n>=0}1/n!=e.
```

ここでは単体体積を直接積分しない。長さを一つ上げる操作をnilpotentな上三角行列として束ね、その行列指数の成分を読む。
