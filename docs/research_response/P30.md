# Telescoping Seriesによる ∑′ₙ P(τ = n) = 1 の完全Lean 4実装

**結論 (Main takeaway)**
望遠級数（telescoping series）の典型的パターン

$$
\sum_{n=2}^\infty\Bigl(\tfrac1{(n-1)!}-\tfrac1{n!}\Bigr)
=\frac1{1!}-\lim_{n\to\infty}\frac1{n!}
=1-0=1
$$

を，Mathlib4 v4.12.0 上で以下のように形式化できます。

```lean
import Mathlib.Topology.Algebra.InfiniteSum.Basic
import Mathlib.Analysis.SpecificLimits.Basic

open Filter (atTop)
open BigOperators

theorem prob_sum_one {prob_hitting_time : ℕ → ℝ} 
  (h_zero_one : prob_hitting_time 0 = 0 ∧ prob_hitting_time 1 = 0)
  (h_pmf : ∀ n, n ≥ 2 → prob_hitting_time n = 1 / (Nat.factorial (n-1)) - 1 / (Nat.factorial n))
  (h_summable : Summable (fun n => prob_hitting_time n)) :
  ∑' n, prob_hitting_time n = 1 := by
  -- (1) n = 0,1 の寄与はゼロなので {n | n ≥ 2} に絞る
  have : ∑' n, prob_hitting_time n = ∑' n : {n // 2 ≤ n}, prob_hitting_time n := by
    simpa [h_zero_one.1, h_zero_one.2] using
      (tsum_subtype (fun n => 2 ≤ n) (fun n => prob_hitting_time n)).symm
  -- (2) n ≥ 2 上での項は 1/(n-1)! - 1/n! と書き替え
  have : ∑' n : {n // 2 ≤ n}, prob_hitting_time n
      = ∑' k, (1 / Nat.factorial k - 1 / Nat.factorial (k+1)) := by
    refine (tsum_subtype _ _).trans ?_
    · rfl
    · apply tsum_congr (fun k => by
        dsimp; -- k : {n // 2 ≤ n}, write実部 n = k+2 用
        let N := (k : ℕ) + 2
        have hN : 2 ≤ N := by simp
        have : prob_hitting_time N = 1 / Nat.factorial (N-1) - 1 / Nat.factorial N := h_pmf N hN
        dsimp at this
        -- N-1 = k+1, N = k+2 より
        simp [this])
  -- (3) k からの望遠級数を実行
  have h_telescoping :
    ∑' k, (1 / Nat.factorial k - 1 / Nat.factorial (k+1)) = 1 := by
    -- 部分和 S n = ∑_{k=0}^{n} (1/k! - 1/(k+1)!) = 1 - 1/(n+1)! が lim → 1
    have A : HasSum (fun k => 1 / Nat.factorial k - 1 / Nat.factorial (k+1)) 1 := by
      convert (hasSum_telescope (fun k => 1 / Nat.factorial k) _).mono_left summableFactorial fun _ => ?_
      · -- hasSum_telescope は ∑ (f k - f (k+1)) = f 0 - lim f (n+1)
        show ∑' k, (fun k => 1 / Nat.factorial k) k 
            - _ = 1 - 0
        simpa using (hasSum_ite_add (1 / Nat.factorial 0) 0).symm
      · exact Summable.comp₂ _ (by infer_instance) (summable_factorial_inv ℝ)
      · simp only [Nat.factorial, Nat.factorial_zero, inv_one]
    exact A.hasSum
  -- (4) 全体をつなげて結論
  calc
    ∑' n, prob_hitting_time n = ∑' n : {n // 2 ≤ n}, prob_hitting_time n := this
    _ = ∑' k, (1 / Nat.factorial k - 1 / Nat.factorial (k+1)) := this
    _ = 1 := (h_telescoping : _)
```


## 解説

1. **tsum_subtype による部分集合への絞り込み**
`n = 0,1` は確率 0（定義）なので無視し，`{n // 2 ≤ n}`上だけ合計すれば等価：

```lean
tsum_subtype (fun n => 2 ≤ n) (fun n => prob_hitting_time n)
```

2. **hitting_time_pmf lemma の適用**
`h_pmf : ∀ n ≥ 2, prob_hitting_time n = 1/(n-1)! - 1/n!` で各項を置き換え：

```lean
tsum_congr fun k => by
  let N := k + 2
  have hN : 2 ≤ N := by simp
  dsimp; exact h_pmf N hN
```

3. **望遠級数の hasSum_telescope**
Mathlib4 は任意の `f : ℕ → α` に対して

```lean
hasSum_telescope (f : ℕ → α) [AddCommMonoid α] [TopologicalSpace α] [T2Space α]
: Summable f → Summable (fun k => f k - f (k+1)) → HasSum (fun k => f k - f (k+1)) (f 0 - lim (fun k => f (k+1)))
```

を提供。ここで `f k = 1/k!` とすれば，
部分和が `1 - 1/(n+1)!` となり，極限で `1` に収束[^1]。
4. **Summable の証明**
`summable_factorial_inv ℝ` で `∑ 1/k!` が収束することは既知。

以上により，形式的に `∑' n, prob_hitting_time n = 1` を完結に証明できます。
