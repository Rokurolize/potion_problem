1. `Real.summable_pow_div_factorial c` gives us that ∑ c^n / n! converges

## Priority

**HIGH** - This is a key mathematical foundation for factorial convergence theory in the formal proof.

```lean
import Mathlib.Tactic.Basic
import Mathlib.Analysis.SpecialFunctions.Pow
import Mathlib.Topology.Metric.Controlled
import Mathlib.Topology.Metric.Basic
import Mathlib.Data.Real.Summable

open Filter

lemma factorial_dominates_exponential {c : ℝ} (hc : c > 1) :
  ∀ᶠ n in atTop, (n.factorial : ℝ) > c ^ n := by
  -- 1. Summability gives limit zero
  have h_summable : Summable (fun n => c ^ n / n.factorial) :=
    Real.summable_pow_div_factorial c
  have h_tendsto : Tendsto (fun n => c ^ n / n.factorial) atTop (𝓝 0) :=
    h_summable.tendsto_cofinite_zero

  -- 2. From the metric-space characterization of tendsto, for ε = 1
  have h_eventually : ∀ᶠ n in atTop, |c ^ n / n.factorial - 0| < 1 :=
    (Metric.tendsto_nhds.1 h_tendsto) 1 (by norm_num)

  -- 3. But c^n / n! ≥ 0, so |c^n / n!| = c^n / n!, hence c^n / n! < 1
  filter_upwards [h_eventually] with n hn
  have hn' : c ^ n / n.factorial < 1 := by
    simpa only [sub_zero, Real.norm_eq_abs, abs_of_nonneg (pow_div_nonneg _ (Nat.factorial_pos _).le)]
      using hn

  -- 4. Finally divide_lt_one gives n! > c^n
  simpa only [not_lt] using (div_lt_one (Nat.cast_pos.2 (Nat.factorial_pos n))).1 hn'
```

Explanation of the key API uses:

1. **Metric.tendsto_nhds**:
`Metric.tendsto_nhds.1 : Tendsto f l (𝓝 a) → ∀ ε > 0, ∀ᶠ x in l, dist (f x) a < ε`.
We apply it to `h_tendsto : Tendsto _ atTop (𝓝 0)` and `ε = 1` to get `∀ᶠ n, dist (c^n / n! - 0) < 1`.
2. **div_lt_one**:
`div_lt_one (hb : 0 < b) : a / b < 1 ↔ a < b`.
After establishing `c ^ n / n.factorial < 1` and noting `n.factorial > 0`, we use `div_lt_one` to conclude `c ^ n < n.factorial`.

Imports required:

```lean
import Mathlib.Data.Real.Summable          -- for Real.summable_pow_div_factorial
import Mathlib.Topology.Metric.Basic       -- for Metric.tendsto_nhds
import Mathlib.Algebra.Order.Field.Basic   -- for div_lt_one
```

Alternative approach: one could avoid metric-space APIs by working within `Filter.Eventually` directly, using `Filter.Tendsto.eventually` once it is added to Mathlib4; or by using the fact `h_summable.tendsto_cofinite_zero.eventually` if such a lemma becomes available. However, the metric-space lemma is the current v4.12.0 way to convert a `Tendsto … (𝓝 0)` into an `eventually` inequality for real-valued sequences.
