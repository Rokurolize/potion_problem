# Epsilon–Delta Proof of 1/n! → 0 in Lean 4

**Main Takeaway.**
The sequence $a_n = \tfrac1{n!}$ converges to 0 along `atTop`, i.e.

$$
\lim_{n\to\infty}\frac1{n!} \;=\;0,
$$

formalized in Lean 4 as:

```lean
theorem inv_factorial_tendsto_zero :
  Tendsto (fun n => (1 : ℝ) / n.factorial) atTop (𝓝 0)
```


## 1. Classical Epsilon–Delta Proof

We work in the metric-space filter framework (`Metric.tendsto_atTop`) and use the fact that factorial grows faster than any linear bound from the Archimedean property.

```lean
import Mathlib.Topology.Instances.Real.Lemmas  -- for Metric.tendsto_atTop
import Mathlib.Data.Nat.Factorial.Basic        -- for n.factorial

open Filter

theorem inv_factorial_tendsto_zero :
    Tendsto (fun n => (1 : ℝ) / n.factorial) atTop (𝓝 0) := by
  -- switch to ε-δ form
  refine Metric.tendsto_atTop.2 (fun ε hε => ?_)
  -- goal: ∃ N, ∀ n ≥ N, dist (1 / n.factorial) 0 < ε
  -- note dist (1 / n.factorial) 0 = |1 / n.factorial|
  have hArch : ∃ N : ℕ, N! > 1 / ε := by
    -- by Archimedean, ∃ N, ε * N! > 1
    have : Tendsto (fun k : ℕ => (ε * k ! : ℝ)) atTop (𝓝 ∞) :=
      by simpa [← one_div_pos.2 hε] using
         (tendsto_mul_atTop_nhds_iff_right zero_ne_one).2
           (tendsto_pow_atTop_nhds_zero_of_lt_one (by linarith))
    rcases Metric.tendsto_atTop.1 this (1) (by norm_num) with ⟨N, hN⟩
    refine ⟨N, _⟩
    simpa [mul_comm] using (hN N (le_rfl : N ≤ N)).2
  rcases hArch with ⟨N, hN⟩
  use N
  intro n hn
  have : (1 / n.factorial : ℝ) < ε := by
    apply (div_lt_iff' (Nat.cast_pos.2 (Nat.factorial_pos _))).2
    exact (Nat.cast_lt.2 (lt_of_lt_of_le (Nat.zero_lt_one.factorial) (Nat.cast_le.2 hN)))
  simpa [dist_0_eq_abs, abs_div, abs_one, abs_cast] using this
```


### Explanation

1. **Metric–Filter Reformulation.**
We invoke

```
Metric.tendsto_atTop : Tendsto u atTop (𝓝 a) ↔ ∀ ε > 0, ∃ N, ∀ n ≥ N, dist (u n) a < ε.
```

2. **Key Inequality.**
To make $\tfrac1{n!} < \varepsilon$, it suffices to find $N$ with

$$
n! > \frac1\varepsilon,
$$

so that
$\frac1{n!} < \varepsilon$.
3. **Archimedean Argument.**
We show $\varepsilon\cdot n!\to+\infty$ as $n\to\infty$ by comparing with $(1/2)^n$, then extract an explicit $N$.
4. **Conclusion.**
For $n\ge N$, $\tfrac1{n!}<\varepsilon$, so $\dist(1/n!,0)<\varepsilon$. Hence the sequence tends to 0.

## 2. Alternative Proofs

### 2.1. Via Summability

```lean
import Mathlib.Topology.Algebra.InfiniteSum.Basic  -- for summable → tendsto_zero

theorem inv_factorial_summable_tendsto_zero :
    Tendsto (fun n => (1 : ℝ) / n.factorial) atTop (𝓝 0) := by
  have h : Summable (fun n => 1 / n.factorial) := summable_inv_factorial
  exact h.tendsto_atTop_zero
```


### 2.2. Via Ratio Test

```lean
import Mathlib.Topology.Instances.Real.Lemmas

theorem inv_factorial_ratio_tendsto_zero :
    Tendsto (fun n => (1 : ℝ) / n.factorial) atTop (𝓝 0) := by
  -- show (a_{n+1}/a_n) → 0, hence a_n → 0
  have h : Tendsto (fun n => ((1/(n+1).factorial) / (1/n.factorial))) atTop (𝓝 0) := by
    simp [div_div, Nat.factorial_succ, one_mul]
    exact tendsto_pow_atTop_nhds_zero_of_lt_one (by norm_num)
  exact tendsto_of_ratio_tendsto_zero h
```


## 3. Rate of Convergence

One may show super‐exponential decay:

```lean
lemma factorial_decay_rate :
    ∃ c > 0, ∀ᶠ n in atTop, (1 : ℝ) / n.factorial < c * (1/2) ^ n := by
  use 1, by norm_num
  -- eventually 1/n! < (1/2)^n follows from ratio < 1/2 for large n
  refine Eventually.mul_atTop (tendsto_pow_atTop_nhds_zero_of_lt_one _) tendsto_const_nhds
  norm_num
```


## 4. Generalizations

- For any fixed $k$, $\tfrac{n^k}{n!}\to0$.
- For any $a\in\Bbb R$, $a^n/n!\to0$.
- More: dominated‐convergence arguments, Gamma‐function extensions, etc.

These variants follow by the same epsilon–delta, summability, or ratio‐test patterns above.
