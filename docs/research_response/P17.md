# Summable Hitting-Time Norm Bound

**Main Takeaway.**
The series

$$
\sum_{n=0}^\infty n\;P(\tau=n)
$$

converges by showing
$\|n\,P(\tau=n)\|\le f(n)$ for a summable $f$.  Concretely, with

$$
P(\tau=n)=
\begin{cases}
0,&n\le1,\\
\dfrac{n-1}{n!},&n\ge2,
\end{cases}
$$

one proves

$$
nP(\tau=n)=\frac{n(n-1)}{n!}=\frac{n^2-n}{n!}
\;\le\;
\begin{cases}
2,&n=2,\\
\dfrac{2}{(n-2)!},&n\ge3,
\end{cases}
$$

and $\sum_{n\ge3}2/(n-2)!$ converges.

## 1. Bounding $\displaystyle\frac{n^2-n}{n!}$

### Key Lemma: Quadratic–Factorial Bound

For all $n\ge3$,

$$
\frac{n^2}{n!}
\;\le\;
\frac{2}{(n-2)!}.
$$

**Proof.**

$$
\frac{n^2}{n!}
=\frac{n^2}{n\,(n-1)\,(n-2)!}
=\frac{n}{(n-1)\,(n-2)!}
\le\frac{n}{2\,(n-2)!}
\le\frac{2}{(n-2)!},
$$

since $n-1\ge2$ and $n\le2$ holds only for $n=2$, which is handled separately.

## 2. Constructing the Bounding Function

```lean
def hitting_time_bound : ℕ → ℝ
| 0     => 0
| 1     => 0
| 2     => 2        -- because 2 * P(τ=2) = 2*(1/2) = 1 ≤ 2
| n+3   => 2 / n!   -- for n ≥ 0, corresponds to n+3 ≥ 3
```

**Summability of the Bound.**
Split into the finite head $n<3$ and the tail $n≥3$:

```lean
lemma summable_hitting_time_bound : Summable hitting_time_bound := by
  -- Separate finite and infinite parts
  have h_eq : hitting_time_bound = fun n =>
    if n ≤ 2 then hitting_time_bound n else 2 / (n - 2)! := by
    ext n; cases n; simp; cases n; simp; cases n; simp; simp [hitting_time_bound]
  rw [h_eq]
  -- Finite part (n = 0,1,2) is trivially summable
  apply Summable.add_right
    (Summable.finset _ _)  -- finite sum
  -- Tail: ∑_{n≥3} 2/(n-2)! = 2 * ∑_{m≥1} 1/m! converges
  refine (summable_mul_left _).2 (summable_inv_factorial.of_subtype _)
  simp
```


## 3. Verifying the Norm Bound

```lean
lemma hitting_time_norm_le_bound :
  ∀ n, ‖(n : ℝ) * hitting_time_pmf n‖ ≤ hitting_time_bound n := by
  intro n
  -- Break into cases on n
  cases n with
  | zero => simp
  | succ n =>
    cases n with
    | zero => simp  -- n=1
    | succ n =>
      cases n with
      | zero =>
        -- n+2 = 2 case
        simp [hitting_time_bound]; norm_num
      | succ n =>
        -- n+3 ≥ 3 case
        simp [hitting_time_pmf, hitting_time_bound]
        -- Compute norm: ‖(n+3)*(n+2)/(n+3)!‖ = (n+3)*(n+2)/(n+3)!
        rw [norm_mul, norm_coe_nat, norm_div]; simp
        -- Bound by quadratic lemma
        have : ((n+3 : ℝ) * (n+2)) / (n+3)! ≤ 2 / n! := by
          calc
            (n+3)*(n+2)/(n+3)! 
              = (n+3)^2/(n+3)! - (n+3)/(n+3)! 
                by ring
            _ ≤ (n+3)^2/(n+3)! := by linarith [div_nonneg (mul_nonneg _ _) (_)] 
            _ ≤ 2 / n! := by apply quadratic_div_factorial_bound; linarith
        exact this
```

Here, `quadratic_div_factorial_bound` is the lemma from §1.

## 4. Concluding Summability

Putting it together:

```lean
theorem summable_hitting_time :
  Summable (fun n => (n : ℝ) * hitting_time_pmf n) := by
  apply Summable.of_norm_bounded hitting_time_bound _
  · exact summable_hitting_time_bound
  · exact hitting_time_norm_le_bound
```

Thus the weighted PMF $\{n\,P(\tau=n)\}$ is dominated by a summable sequence, proving the expectation $\sum n\,P(\tau=n)$ converges.

## 5. Alternative Ratio‐Test Approach (Sketch)

One may also apply the ratio test to
$\displaystyle a_n=n\,P(\tau=n)$:

$$
\frac{a_{n+1}}{a_n}
=\frac{(n+1)\,\frac{n}{(n+1)!}}{n\,\frac{n-1}{n!}}
=\frac{n}{(n-1)(n+1)}
\le\frac{1}{n}
\to0,
$$

so eventually $<r<1$, and hence $\sum a_n$ converges.

**Connection to Expected Value.**
Having shown $\sum n\,P(\tau=n)$ converges, one can then evaluate
$\sum n\,P(\tau=n)=e$ as in the main theorem, completing the expected‐value calculation $E[\tau]=e$.
