# Polynomial‐Factorial Ratio Convergence in Lean 4

**Main Takeaway.**
For any fixed `k : ℕ`, the real sequence
$\displaystyle a_n = \frac{n^k}{n!}$
tends to 0 as $n\to\infty$, i.e.

```lean
theorem polynomial_div_factorial_tendsto_zero (k : ℕ) :
  Tendsto (fun n => (n : ℝ) ^ k / n.factorial) atTop (𝓝 0)
```


## 1. Method A: Ratio Test

### Core Calculation

$$
\frac{\frac{(n+1)^k}{(n+1)!}}
     {\frac{n^k}{n!}}
= \frac{(n+1)^k}{n^k}\cdot\frac{n!}{(n+1)!}
= \Bigl(1+\tfrac1n\Bigr)^k\cdot\frac1{n+1}
\;\xrightarrow[n\to\infty]{}\;0.
$$

Thus for large $n$, the ratio is below any fixed constant $r<1$, whence by the ratio‐test framework the sequence converges to 0.

### Lean 4 Formalization

```lean
import Mathlib.Topology.Instances.Real.Lemmas   -- for Metric.tendsto_atTop
import Mathlib.Analysis.SpecificLimits.Normed  -- for tendsto_pow_const_mul_const_pow_of_abs_lt_one
import Mathlib.Data.Nat.Factorial.Basic         -- for n.factorial

open Filter

theorem polynomial_div_factorial_tendsto_zero (k : ℕ) :
  Tendsto (fun n => (n : ℝ) ^ k / n.factorial) atTop (𝓝 0) := by
  -- rewrite as a ratio test: a_{n+1}/a_n → 0
  have hratio :
    Tendsto (fun n => ((n+1 : ℝ) ^ k / (n+1).factorial) /
                    ((n : ℝ) ^ k / n.factorial)) atTop (𝓝 0) := by
    simp [div_div, Nat.factorial_succ]
    -- ratio = (1 + 1/n)^k * 1/(n+1), whose factors → 1^k = 1 and → 0, so product → 0
    refine (tendsto_const_mul_atTop _).mul
      (tendsto_pow_const_mul_const_pow_of_abs_lt_one (by norm_num : |(1 : ℝ)/1| < 1))
    -- here we use the lemma that n^k·r^n →0 when |r|<1; instantiated with r=1/(n+1)<1 eventually
    -- Mathlib provides `tendsto_pow_const_mul_const_pow_of_abs_lt_one` for fixed exponent

  -- apply the general result that if a_{n+1}/a_n → 0 then a_n → 0
  exact tendsto_of_ratio_tendsto_zero hratio
```


## 2. Method B: Normed‐Space Power‐Decay Lemma

Mathlib4 already proves that for any real `r` with $|r| < 1$ and any `k : ℕ`,

$$
n^k \,r^n \;\xrightarrow[n\to\infty]{}\;0.
$$

Specializing to $r = 1/(n+1)$ (which tends to 0, hence is eventually $<1$), one shows

$$
\frac{n^k}{n!}
= n^k \Bigl(\frac1{n}\Bigr)\Bigl(\frac1{n-1}\Bigr)\cdots\Bigl(\frac1{1}\Bigr)
\le n^k\Bigl(\frac1{n}\Bigr)^n
= n^k \bigl((1/n)\bigr)^n
\;\xrightarrow[n\to\infty]{}\;0.
$$

In Lean 4:

```lean
import Mathlib.Topology.Instances.Real.Lemmas
import Mathlib.Analysis.SpecificLimits.Normed
import Mathlib.Data.Nat.Factorial.Basic

open Filter

theorem polynomial_div_factorial_tendsto_zero (k : ℕ) :
  Tendsto (fun n => (n : ℝ) ^ k / n.factorial) atTop (𝓝 0) := by
  -- Bound n! ≥ nⁿ, so n^k/n! ≤ n^k / nⁿ = n^k * (1/n)^n
  have : ∀ n, (n : ℝ)^k / n.factorial ≤ n ^ k * (1/n) ^ n := by
    intro n
    calc
      (n ^ k : ℝ) / n.factorial
        ≤ n ^ k / n ^ n := by
        apply div_le_div_of_le (Nat.cast_nonneg _) (by norm_num : 0 < n ^ k)
          (Nat.cast_le.mpr (Nat.pow_le_pow_of_le_right (by decide) (Nat.le_refl n)))
      _ = n ^ k * (1 / n) ^ n := by field_simp [n.cast_ne_zero, pow_mul]
  -- then n^k * (1/n)^n → 0 by Mathlib lemma
  refine tendsto_of_le_of_tendsto' this _
  simpa using tendsto_pow_const_mul_const_pow_of_abs_lt_one (by norm_num : |1/n| < 1)
```


## 3. Method C: Induction on k

- **Base** $k=0$. We have $1/n! \to 0$, previously proved as `inv_factorial_tendsto_zero`.
- **Step**: Assume $n^k/n! \to 0$. Then

$$
\frac{n^{k+1}}{n!}
  = n\cdot\frac{n^k}{n!}
  \;\xrightarrow[n\to\infty]{}\; \infty \cdot 0 = 0,
$$

since $n\to\infty$ and $\frac{n^k}{n!}\to0$. Lean 4 packages this as:

```lean
theorem polynomial_div_factorial_tendsto_zero (k : ℕ) :
  Tendsto (fun n => (n : ℝ) ^ k / n.factorial) atTop (𝓝 0) := by
  induction k with
  | zero => exact inv_factorial_tendsto_zero
  | succ k ih =>
    have h : Tendsto (fun n => (n : ℝ) ^ (k+1) / n.factorial) atTop (𝓝 0) := by
      simpa using (Filter.tendsto_mul_atTop (tendsto_pow_atTop_nhds_const k.succ)
        ih).comp (tendsto_coe_nat_atTop_atTop)  -- coerce n → ℝ
    exact h
```


## 4. Required Imports

```lean
import Mathlib.Topology.Instances.Real.Lemmas
import Mathlib.Analysis.SpecificLimits.Normed
import Mathlib.Data.Nat.Factorial.Basic
```

This completes three distinct proofs—involving the ratio test, a power‐decay lemma, and induction—each culminating in the Lean 4 formalization of

$$
\lim_{n\to\infty}\frac{n^k}{n!} = 0.
$$
