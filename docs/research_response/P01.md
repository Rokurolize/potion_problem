# Volume of the Standard Simplex via Measure Theory in Lean 4

**Main Result.**
In Lean 4 with Mathlib4, for each natural number `n`, the Lebesgue volume of the standard `n`-simplex

```lean
stdSimplex ℝ (Fin n.succ) = { f : Fin (n + 1) → ℝ | (∀ i, 0 ≤ f i) ∧ ∑ i, f i = 1 }
```

equals `1 / n.factorial`.

## 1. Measure‐Theoretic Foundation

1. **Lebesgue Measure on ℝⁿ**
Mathlib4 provides the *volume* on a finite‐dimensional real vector space via additive Haar measure:

```lean
import Mathlib.MeasureTheory.Measure.Haar.OfBasis

-- In particular, on `ℝⁿ = EuclideanSpace (Fin n) ℝ`:
instance : MeasureSpace (EuclideanSpace (Fin n) ℝ) :=
  MeasureTheory.MeasureSpaceOfInnerProductSpace ℝ _

@[simp]
theorem volume_pi_eq_addHaar (n : ℕ) :
  (MeasureTheory.volume : Measure (Fin n → ℝ)) =
    (stdOrthonormalBasis ℝ (Fin n)).toBasis.addHaar := ...
```

2. **Product Measure and Fubini–Tonelli**
To integrate over ℝⁿ as an iterated integral, import Fubini’s theorem:

```lean
import Mathlib.MeasureTheory.Integral.Prod

open MeasureTheory

theorem volume_eq_iteratedIntegral {n : ℕ} :
  ∀ f, Integrable f volume →
  (∫ x, f x ∂volume) =
    ∫ x₁, ∫ x₂, … ∫ xₙ, f (x₁, …, xₙ) ∂volume ∂ … ∂volume := ...
```

3. **Measurability of the Standard Simplex**
The standard simplex is a measurable set since it is a finite intersection of half‐spaces:

## 2. Volume Calculation Methods

### Method A: Direct Iterated Integration

1. **Set up the integral.**
Identify the simplex in coordinates `(x₁,…,xₙ)` with

$$
Δ^n = \{\, (x₁,…,xₙ) : xᵢ ≥ 0,\; x₁+⋯+xₙ ≤ 1 \},
$$

and note its volume equals

$$
\int_{x₁=0}^1 \int_{x₂=0}^{1 - x₁} ⋯
    \int_{xₙ=0}^{1 - x₁ - … - xₙ₋₁} dxₙ … dx₂ dx₁.
$$
2. **Lean 4 proof outline.**

```lean
import Mathlib.MeasureTheory.Integral.Prod
import Mathlib.MeasureTheory.Integral.IntervalIntegral

open MeasureTheory
open IntervalIntegral

theorem volume_stdSimplex_direct (n : ℕ) :
  MeasureTheory.volume (stdSimplex ℝ (Fin n.succ)) = 1 / n.factorial := by
  -- Change to iterated integral over `Icc 0 1` and use `volume_eq_iteratedIntegral`
  have : volume = (volume.pi n).restrict _  -- product measure
  -- Use Fubini to convert to iterated `∫ x in 0..1, …`
  simp [stdSimplex, Fintype.sum_fin, Fin.sum_univ_cast_succ]
  -- Integrate step by step using `integral_comp`, `integral_id`, and `integral_pow`
  exact by
    simp [integral_id, integral_comp _ (· ^ ·)]
    -- finish by induction on n, showing ∫₀¹ xⁿ dx = 1 / (n + 1)
```

3. **Key one‐dimensional lemma.**

```lean
import Mathlib.MeasureTheory.Integral.IntervalIntegral

open IntervalIntegral

theorem integral_pow (n : ℕ) :
  ∫ x in (0 : ℝ) .. 1, x ^ n = 1 / (n + 1) := by
  induction n with
  | zero => simp
  | succ n ih =>
    have h : ∫ x in 0..1, x ^ (n + 1) = (1 ^ (n + 2) - 0) / (n + 2) := integral_pow_succ _ _
    simp [h, one_pow, sub_zero, ih]
```


### Method B: Linear Transformation

1. **Parametrize simplex by affine map.**
The standard simplex is the image of the unit cube `[1]^n` under the map

$$
A : (x₁,…,xₙ) ↦ \bigl(x₁,\, x₂(1 - x₁),\, ⋯,\, xₙ\bigl(1 - x₁ - ⋯ - xₙ₋₁)\bigr).
$$
2. **Jacobian determinant = `1 / n!`**
One shows that `volume (A '' [1]^n) = |det A| * volume [1]^n`, and `det A = 1 / n!`.
3. **Lean 4 imports and proof sketch.**

```lean
import Mathlib.MeasureTheory.Measure.Haar.OfBasis
import Mathlib.MeasureTheory.Integral.Prod
import Mathlib.MeasureTheory.Measure.Lebesgue.Basic

open MeasureTheory

theorem volume_stdSimplex_linear (n : ℕ) :
  volume (stdSimplex ℝ (Fin n.succ)) = 1 / n.factorial := by
  -- Define `A` as a `MeasurableEquiv` from `ℝ^n` to the simplex
  have hA : MeasurableEquiv _ _ := … 
  calc
    volume (stdSimplex ℝ (Fin n.succ))
      = volume (A '' (Icc 0 1).prod …) := by simp [stdSimplex, hA]
    _ = |det A.toLinearMap| * volume (Icc 0 1).prod … := by
      apply volume_map_linearMap; simp [hA]
    _ = 1 / n.factorial := by norm_num
```


## 3. Lean 4 Implementation

```lean
import Mathlib.MeasureTheory.Measure.Lebesgue.Basic
import Mathlib.MeasureTheory.Integral.Prod
import Mathlib.MeasureTheory.Integral.IntervalIntegral
import Mathlib.MeasureTheory.Measure.Haar.OfBasis
import Mathlib.Analysis.Convex.Basic

open MeasureTheory

theorem volume_standard_simplex (n : ℕ) :
  volume (stdSimplex ℝ (Fin n.succ)) = 1 / n.factorial := by
  -- Direct method via iterated integrals
  have : volume = (volume.pi n).restrict _ := rfl
  simp [stdSimplex, Fintype.sum_fin, Fin.sum_univ_cast_succ]
  rw [volume_eq_iteratedIntegral, integral_pow]
```
