# Infinite Telescoping Series Convergence to First Term in Lean 4

**Main Takeaway.**
If a real‐valued sequence `a : ℕ → ℝ` tends to 0 and the series of its consecutive differences
$\sum_{n=0}^\infty\bigl(a_n - a_{n+1}\bigr)$ converges, then its sum equals the initial term $a_0$. Formally:

```lean
theorem telescoping_series_sum {a : ℕ → ℝ}
  (h_tendsto : Tendsto a atTop (𝓝 0))
  (h_summable : Summable (fun n => a n - a (n + 1))) :
  ∑' n, (a n - a (n + 1)) = a 0
```


## 1. Proof Outline

1. **Summable ↔ HasSum ↔ Partial‐sum Tendsto.**
From `h_summable : Summable f`, obtain

$$
\texttt{hasSum }f\;L \quad\text{with}\quad L = \sum' n, f n,
$$

and the equivalence

$$
\texttt{HasSum }f\;L \;\longleftrightarrow\;
  \texttt{Tendsto (fun s, ∑ i in s, f i)}\;\texttt{atTop}\;\bigl(\𝓝\,L\bigr)
$$

via `hasSum_iff_tendsto_sum_nat`.
2. **Explicit Partial Sum Formula.**
A standard finite telescoping lemma shows for each `n`

$$
S_n \;=\;\sum_{i=0}^n\!(a_i - a_{i+1})
  = a_0 - a_{n+1}.
$$
3. **Pass to the Limit.**
As $n\to\infty$, $a_{n+1}\to0$ by `h_tendsto`.  Hence

$$
S_n \;\longrightarrow\; a_0 - 0 = a_0.
$$

Uniqueness of the limit in `𝓝` gives the desired equality of sums.

## 2. Complete Lean 4 Implementation

```lean
import Mathlib.Topology.Algebra.InfiniteSum.Basic  -- HasSum, Summable, tsum, hasSum_iff_tendsto_sum_nat
import Mathlib.Data.Real.Basic                    -- ℝ
import Mathlib.Tactic.Ring                        -- ring simplicities
import Mathlib.Tactic.Simp                        -- simp

open Filter

theorem telescoping_series_sum {a : ℕ → ℝ}
  (h_tendsto : Tendsto a atTop (𝓝 0))
  (h_summable : Summable (fun n => a n - a (n + 1))) :
  ∑' n, (a n - a (n + 1)) = a 0 := by
  -- Convert summability into HasSum with the limit L
  obtain ⟨L, h_hasSum⟩ := h_summable.hasSum
  -- Reformulate `HasSum f L` as a limit of partial sums
  have h_lim : Tendsto (fun n => ∑ i in Finset.range (n + 1), (a i - a (i + 1)))
                     atTop (𝓝 L) :=
    by simpa only [Finset.sum_range] using
      (hasSum_iff_tendsto_sum_nat h_summable).1 h_hasSum
  -- Compute partial sums explicitly
  have h_partial : ∀ n,
    ∑ i in Finset.range (n + 1), (a i - a (i + 1)) = a 0 - a (n + 1) := by
    intro n
    -- use the finite telescoping lemma
    rw [telescoping_series_partial_sum a 0 n (zero_le n)]
    simp
  -- Rewrite h_lim in terms of a₀ - a_{n+1}, then pass to the limit
  have : Tendsto (fun n => a 0 - a (n + 1)) atTop (𝓝 L) := by
    simpa using h_lim.comp (tendsto_id'.sub (h_tendsto.comp (tendsto_add_atTop_nat 1)))
  -- But `a 0 - a (n+1)` tends to `a 0 - 0 = a0`
  have : Tendsto (fun n => a 0 - a (n + 1)) atTop (𝓝 (a 0 - 0)) := by
    simpa only [sub_zero] using tendsto_const_nhds.sub (h_tendsto.comp (tendsto_add_atTop_nat 1))
  -- Uniqueness of limits in a neighborhood filter yields L = a 0
  simpa [sub_zero] using tendsto_nhds_unique this this
  -- Conclude that `L = a 0` and hence `tsum f = a 0`
```

## 4. Explanation of Key Steps

- **`h_summable.hasSum`** turns `Summable f` into an explicit `HasSum f L`.
- **`hasSum_iff_tendsto_sum_nat`** relates `HasSum f L` to the limit of the finite‐sum function on `Finset.range (n + 1)`.
- **`telescoping_series_partial_sum`** is the standard finite telescoping lemma:

```lean
theorem telescoping_series_partial_sum {α : Type*} [AddCommGroup α]
  (a : ℕ → α) (m n : ℕ) (hmn : m ≤ n) :
  ∑ i in Finset.range (n - m + 1), (a (m + i) - a (m + i + 1)) = a m - a (n + 1)
```

- **`tendsto_const_nhds.sub`** and filter composition manage the subtraction in the limit.
- **`tendsto_nhds_unique`** enforces that two convergent sequences in `𝓝` must share their limits.

This completes the formal proof in Lean 4 that any convergent telescoping series of the form $\sum(a_n - a_{n+1})$ sums to its first term $a_0$.
