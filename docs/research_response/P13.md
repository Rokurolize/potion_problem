# Series Reindexing by Constant Shift in Lean 4

**Main Takeaway.**
For any summable sequence `f : ℕ → ℝ` and integer shift `a : ℕ`, shifting the index preserves the infinite sum:

```lean
theorem reindex_series_shift {f : ℕ → ℝ} (a : ℕ)
  (h_sum : Summable f) :
  ∑' n : {n // n ≥ a}, f n = ∑' k : ℕ, f (k + a)
```


## 1. Direct Bijection Approach

Define the explicit bijection between `ℕ` and the subtype of naturals ≥ `a`:

```lean
import Mathlib.Topology.Algebra.InfiniteSum.Basic  -- for Equiv.tsum_eq, Summable

/-- Shift bijection: k ↦ k + a -/
def shift_bij (a : ℕ) : ℕ ≃ {n // n ≥ a} where
  toFun k      := ⟨k + a, Nat.le_add_right _ _⟩
  invFun n     := n.val - a
  left_inv k   := by simp [Nat.sub_add_cancel (Nat.le_add_right a k)]
  right_inv n  := by
    ext
    simp [Nat.add_sub_cancel' n.prop]
```

Apply `Equiv.tsum_eq` to reindex the sum:

```lean
theorem reindex_series_shift {f : ℕ → ℝ} (a : ℕ)
    (h_sum : Summable f) :
  ∑' n : {n // n ≥ a}, f n = ∑' k : ℕ, f (k + a) := by
  -- rewrite the right‐hand `tsum` to use the bijection explicitly
  have : ∑' k : ℕ, f (k + a) = ∑' k : ℕ, f (shift_bij a k).val := by
    congr 1; funext; rfl
  rw [this]
  -- now reindex by the inverse equivalence
  exact (Equiv.tsum_eq (shift_bij a).symm h_sum).symm
```


## 2. Subtype Summation Method

Use `tsum_subtype` to express the sum over the subtype as a conditional sum over `ℕ`, then shift:

```lean
import Mathlib.Topology.Algebra.InfiniteSum.Basic

theorem reindex_series_shift_v2 {f : ℕ → ℝ} (a : ℕ)
    (h_sum : Summable f) :
  ∑' n : {n // n ≥ a}, f n = ∑' k : ℕ, f (k + a) := by
  -- 1) Convert subtype‐sum to full ℕ with `ite`
  have A : ∑' n : {n // n ≥ a}, f n
         = ∑' n : ℕ, if h : n ≥ a then f n else 0 := by
    exact tsum_subtype _ fun _ _ => by simp
  rw [A]
  -- 2) `∑ n, if n≥a then f n else 0 = ∑ k, f (k+a)`
  have B : ∑' n : ℕ, if n ≥ a then f n else 0
         = ∑' k : ℕ, f (k + a) := by
    -- reindex by noticing `n ≥ a ↔ ∃ k, n = k + a`
    calc ∑' n, ite (n ≥ a) (f n) 0
        = ∑' k : ℕ, ite (k + a ≥ a) (f (k + a)) 0 := by
      exact reindex_series_general (AddEquiv.addRight a) (h_sum.comp_injective (add_left_injective a))
    _ = ∑' k, f (k + a) := by
      simp only [if_true, Nat.le_add_right]
  exact B
```

Here `reindex_series_general` is the general lemma:

```lean
theorem reindex_series_general {α β} [Countable α] [Countable β]
    {g : β → ℝ} (φ : α ≃ β) (hg : Summable g) :
  ∑' a, g (φ a) = ∑' b, g b :=
Equiv.tsum_eq φ hg
```


## 3. Splitting Approach

Split the full series at `a` into the finite head plus the tail, then extract the tail:

```lean
import Mathlib.Topology.Algebra.InfiniteSum.Basic

theorem reindex_series_shift_v3 {f : ℕ → ℝ} (a : ℕ)
    (h_sum : Summable f) :
  ∑' n : {n // n ≥ a}, f n = ∑' k : ℕ, f (k + a) := by
  -- split ℕ-sum into range a + remainder
  have split_full :
    ∑' n : ℕ, f n
    = ∑ n in Finset.range a, f n + ∑' n : {n // n ≥ a}, f n := by
    exact (tsum_add_tsum_compl h_sum).symm
  -- split again shifting the remainder
  have split_shift :
    ∑' n : ℕ, f n
    = ∑ n in Finset.range a, f n + ∑' k : ℕ, f (k + a) := by
    -- apply first identity to the shifted function `λ k, f (k+a)`
    have h_shift := (h_sum.comp_injective (add_left_injective a))
    exact (tsum_add_tsum_compl h_shift).trans (congrArg2 (· + ·) rfl (Equiv.tsum_eq (shift_bij a).symm h_sum))
  -- cancel the common head `∑ range a, f n`
  simpa only [add_left_inj] using split_full.trans split_shift.symm
```

## 5. Examples

1. **Geometric series tail.**

```lean
example {r : ℝ} (hr : |r| < 1) (a : ℕ) :
  ∑' n : {n // n ≥ a}, r ^ n = r ^ a / (1 - r) := by
  rw [reindex_series_shift a (summable_pow_mul_const _ hr), pow_add, ← tsum_mul_right, tsum_geometric_of_abs_lt_1 hr]
```

2. **Factorial tail.**

```lean
example :
  ∑' n : {n // n ≥ 2}, 1 / n.factorial = exp 1 - 2 := by
  rw [reindex_series_shift 2 (summable_inv_factorial), tsum_inv_factorial, sub_sub_self, sub_zero]
```

3. **Harmonic series tail diverges.**

```lean
example (a : ℕ) (ha : a > 0) :
  ¬Summable (fun n : {n // n ≥ a} => (1 : ℝ) / n.val) := by
  simpa [reindex_series_shift a] using mt (summable_comp_injective _) (by simpa using summable_nat_add_iff ha).1
```


These three methods—**bijection**, **subtype**, and **splitting**—all rigorously establish that shifting the index by a constant preserves the sum of a summable series in Lean 4.
