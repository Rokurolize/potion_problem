# Lower Bound on the Factorial via Stirling’s Approximation

**Main Result.**
For every natural number $n\ge1$,

$$
n!\;\ge\;\Bigl(\tfrac{n}{e}\Bigr)^n.
$$

## 1. Proof via Integral Comparison (Stirling’s Inequality)

1. **Logarithmic Formulation.**
Observe

$$
\ln(n!) \;=\;\sum_{k=1}^n\ln k.
$$
2. **Integral Bound.**
For a monotone increasing function $f(x)=\ln x$, the right‐Riemann sum over $[1,n]$ satisfies

$$
\sum_{k=1}^n\ln k
\;\ge\;\int_{1}^{n}\ln x\,dx
=\bigl[x\ln x - x\bigr]_{1}^{n}
=n\ln n - n +1.
$$
3. **Exponentiation.**
Exponentiating both sides (noting that the exponential is strictly increasing) gives

$$
n!
\;=\;\exp\bigl(\ln(n!)\bigr)
\;\ge\;\exp\bigl(n\ln n - n +1\bigr)
=e^{1}\,(n/e)^n
=e\,(n/e)^n.
$$
4. **Refinement.**
Since $e>1$, we in particular have

$$
n!\;\ge\;(n/e)^n,
$$

as desired.

## 2. Lean 4 Formalization

```lean
import Mathlib.Data.Nat.Factorial.Basic
import Mathlib.Analysis.SpecialFunctions.Log.Basic

open Real

theorem factorial_ge_n_div_e_pow_n (n : ℕ) (hn : 1 ≤ n) :
    (n.factorial : ℝ) ≥ (n / exp 1) ^ n := by
  -- Take natural logs of both sides:
  -- suffices to show ln(n!) ≥ n * ln(n / e).
  have hlog :
    log (n.factorial : ℝ) ≥ n * log (n / exp 1) := by
    -- log(n!) = ∑_{k=1}^n log k
    have : log (n.factorial : ℝ) = ∑ k in Finset.range (n + 1), log k := by
      -- use `log_prod` from mathlib4
      simp [log_prod (by simp [hn.le_succ])]
    -- bound sum by integral
    calc
      ∑ k in Finset.range (n + 1), log k
        ≥ ∑ k in Finset.Icc 1 n, ∫ x in (k : ℝ)..(k + 1), log x := by
      · refine Finset.sum_ge_sum (by simp) fun k hk =>?_
        -- each term log k ≥ ∫_k^{k+1} log x dx
        calc
          log k = ∫ x in (k : ℝ)..(k + 1), log k   by simp
          _    ≥ ∫ x in (k : ℝ)..(k + 1), log x   by mono; exact monotone_log
      -- sum the integrals
      _ = ∫ x in (1 : ℝ)..(n + 1), log x := by
        simp [integral_piecewise]
      _ = (n + 1) * log (n + 1) - n := by
        -- compute ∫_1^{n+1} log x dx = [x log x - x]_1^{n+1}
        simp only [integral_log, sub_add_cancel, log_one, mul_one, sub_zero]
      _ ≥ n * log (n / exp 1) := by
        -- algebraic rearrangement using log division
        have : (n + 1) * log (n + 1) - n = n * log n - n + log (n + 1) + (n + 1 - n) * (log (1 + 1 / n))
          by ring
        simp [this, log_div, log_exp, sub_mul, mul_comm, add_comm]
  -- exponentiate to conclude
  simpa [exp_log (by positivity)] using exp_le_exp hlog
```
