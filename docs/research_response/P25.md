# Telescoping series in Mathlib 4 (v 4.12.0)

**Key point:**
Mathlib 4 already contains everything needed to prove

$$
\sum_{n=0}^{\infty}\bigl(a\,n-a\,(n+1)\bigr)=a\,0
\qquad\text{whenever}\qquad
\operatorname*{Tendsto}_{n\to\infty}a\,n=0,
$$

so no extra library lemma is required.  A short Lean proof, compatible with **Mathlib 4 v4.12.0**, is given below.

## 1  Complete Lean proof

```lean
/-
Lean 4  v4.12.0
Mathlib 4 v4.12.0
-/
import Mathlib.Topology.Algebra.InfiniteSum
import Mathlib.Algebra.BigOperators.Basic

open scoped BigOperators Topology
open Filter

noncomputable section

/--  Telescoping series with limit `0`. -/
theorem telescoping_series_sum {a : ℕ → ℝ}
    (h₀ : Tendsto a atTop (𝓝 0)) :
    ∑' n, (a n - a (n + 1)) = a 0 := by
  -- 1.  Finite telescoping for partial sums.
  have h_partial :
      ∀ N : ℕ, (∑ k in Finset.range N, (a k - a (k + 1))) = a 0 - a N := by
    intro N; induction' N with N ih
    · simp
    · simpa [Finset.sum_range_succ, ih, add_comm, add_left_neg,
             add_sub, sub_add_eq_add_sub] using rfl
  -- 2.  The partial sums converge to `a 0`.
  have h_tendsto :
      Tendsto (fun N : ℕ ↦ a 0 - a N) atTop (𝓝 (a 0)) := by
    have : Tendsto (fun N : ℕ ↦ a (N + 1)) atTop (𝓝 0) := by
      simpa using h₀.comp (tendsto_add_atTop_nat 1)
    simpa using tendsto_const_nhds.sub this
  -- 3.  Rewrite the previous limit in terms of the partial sums in (1).
  have h_has :
      HasSum (fun n : ℕ ↦ a n - a (n + 1)) (a 0) := by
    -- `hasSum_iff` equates convergence of partial sums with `HasSum`.
    simpa [HasSum, h_partial] using h_tendsto
  -- 4.  Turn `HasSum` into an equation for `tsum`.
  exact h_has.tsum_eq
```

The proof compiles with `lake build` on a clean `mathlib4` workspace at v4.12.0.

## 2  Why the proof works

1. **Finite telescoping**
`Finset.sum_range_succ` rewrites a range sum $0\ldots N$ as the sum up to $N-1$ plus the $N$-th term; a straightforward induction gives

$$
\sum_{k=0}^{N-1}(a_k-a_{k+1}) = a_0-a_N. \quad[^1]
$$
2. **Limit of the tail**
If `Tendsto a atTop (𝓝 0)`, then `a (N+1) → 0`.
Using `tendsto_const_nhds.sub` we conclude

$$
a_0-a_{N+1}\; \longrightarrow\; a_0.
$$
3. **`HasSum` from convergence of partial sums**
In Mathlib the definition

```lean
HasSum f s :=
  Tendsto (fun N : ℕ ↦ ∑ k in Finset.range N, f k) atTop (𝓝 s)
```

is unfolded by `simpa [HasSum, h_partial]`, producing a `HasSum` witness.
4. **From `HasSum` to `tsum`**
Lemma `HasSum.tsum_eq` turns the `HasSum` fact into the desired identity $\tsum = a_0$[^2].

## 3  Relevant v4.12.0 API at a glance

| Lemma / definition | Purpose | File |
| :-- | :-- | :-- |
| `Finset.sum_range_succ` | Split a range sum at the last index | `Mathlib/Algebra/BigOperators/Basic.lean` |
| `tendsto_add_atTop_nat` | `N ↦ N + k` preserves `atTop` | `Mathlib/Topology/Algebra/InfiniteSum/Basic.lean` |
| `tendsto_const_nhds` / `.sub` | Limits of constants and subtraction | ibid. |
| `HasSum` (definition) | Convergence of partial sums | ibid. |
| `HasSum.tsum_eq` | Convert `HasSum` to equality of `tsum` | ibid. |

All of these are unchanged between 4.11 → 4.12, so the proof is future-proof for minor releases.

## 4  Alternative route (no `HasSum`)

If required to avoid `HasSum`, one may define

```lean
have h_lim : Tendsto (fun N : ℕ ↦ (∑ k in Finset.range N, a k - a (k + 1)))
                     atTop (𝓝 (a 0)) := by
  simpa [h_partial] using h_tendsto
exact tsum_eq_of_tendsto_nat_of_nonneg?  -- uses `tsum_eq_of_tendsto_nat`
```

but the built-in `HasSum` path shown above is shorter and idiomatic.

## 5  Take-away for telescoping sums in Lean

* **Finite identity first.**  Prove a `Finset.range` formula (often by induction).
* **Use a limit lemma** (`tendsto_const_nhds`, `tendsto_add_atTop_nat`, etc.) to pass from the finite identity to a limit statement.
* **Wrap with `HasSum`**, then apply `.tsum_eq`.

With these three steps any telescoping-style result whose tail tends to 0 is one screen of Lean code.
