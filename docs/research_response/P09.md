# Factorial Telescoping Sum with Indicator Functions in Lean 4

**Main Takeaway.**
The infinite sum

```lean
∑' n : ℕ, (if n ≥ 2 then (1 : ℝ)/(n-1).factorial - 1/n.factorial else 0)
```

converges to 1. In Lean 4 with Mathlib4, we formalize:

```lean
theorem factorial_telescoping_sum_one :
  ∑' n : ℕ, (if n ≥ 2 then (1 : ℝ)/(n-1).factorial - 1/n.factorial else 0) = 1
```


## 1. Indicator Function Decomposition

Define

```lean
f : ℕ → ℝ := fun n =>
  if n ≥ 2 then 1/(n-1).factorial - 1/n.factorial else 0
```

Then `f n = 0` for `n = 0` or `1`, and `f n = 1/(n-1)! - 1/n!` for `n ≥ 2`. Its support lies in `⋃’ n, {n | n ≥ 2}`. One shows:

```lean
have support_subset : Function.support f ⊆ {n | n ≥ 2} := by
  intro n hn
  simp [Function.mem_support, f] at hn; split_ifs at hn with h
  · exact h
  · contradiction
```


## 2. Restricting the Infinite Sum to the Subtype

By absolute convergence of the series (see Section 3), we may restrict to the subtype `{n // n ≥ 2}`:

```lean
import Mathlib.Topology.Algebra.InfiniteSum.Basic

theorem factorial_telescoping_sum_one :
    ∑' n, f n = ∑' n : {n // n ≥ 2}, f n := by
  -- use `tsum_subtype_eq_of_support_subset` [^17]
  rw [← tsum_subtype_eq_of_support_subset (fun n => f n) support_subset]
```


## 3. Summability and Absolute Convergence

We show the series of absolute values converges by comparison with `∑ 1/(n-1)!`, which is a tail of `exp(1)`:

```lean
import Mathlib.Analysis.Normed.Group.InfiniteSum

lemma summable_factorial_diff :
  Summable (fun n => if n ≥ 2 then f n else 0) := by
  apply Summable.of_norm_bounded (fun n => if n ≥ 2 then 1/(n-1).factorial else 0) ?_; 
  -- majorant is summable since `∑_{k≥1} 1/k!` converges to `e-1`
  exact (summable_inv_factorial.of_subtype (by simp)).add_right_const _
  intro n; split_ifs with h₂
  · -- Case `n ≥ 2`: `‖1/(n-1)! - 1/n!‖ ≤ 1/(n-1)! + 1/n!`
    simp [norm_sub, norm_div]; linarith
  · simp
```


## 4. Concluding the Sum on the Subtype

Having restricted and established summability, invoke the pre‐proved lemma:

```lean
import Mathlib.Topology.Algebra.InfiniteSum.Basic

theorem factorial_telescoping_series_eq_one :
  ∑' n : {n // n ≥ 2}, (1/((n : ℕ)-1).factorial - 1/(n : ℕ).factorial) = 1 :=
by
  -- Partial sums telescope: S_N = 1 - 1/N!
  have : HasSum (fun n : {n // n ≥ 2} =>
    1/((n : ℕ)-1).factorial - 1/(n : ℕ).factorial) 1 := by
    have partial : ∀ N (hN : N ≥ 2),
      ∑ i in Finset.range (N - 1),
        (1/(i+1).factorial - 1/(i+2).factorial) = 1 - 1/(N : ℕ).factorial := by
      intro N hN; simp [Finset.sum_range_succ', sub_eq_add_neg]
    refine (hasSum_subtype_iff_support _ _).2 partial
  exact this.summable.tsum_eq
```

Then combining both steps yields the main theorem:

```lean
theorem factorial_telescoping_sum_one :
  ∑' n : ℕ, f n = 1 := by
  -- restrict to `n ≥ 2`
  rw [← tsum_subtype_eq_of_support_subset (fun n => f n) support_subset]
  -- apply the subtype sum
  exact (factorial_telescoping_series_eq_one).symm
```


## 5. Alternative Direct Calculation

A more `"calc"`‐style proof without subtypes:

```lean
theorem factorial_telescoping_sum_one' :
  ∑' n : ℕ, f n = 1 := by
  calc
    ∑' n, f n
        = 0 + 0 + ∑' n : {n // n ≥ 2}, f n := by
      -- split off `n=0,1`
      rw [tsum_eq_zero_add (by simp : f 0 = 0), tsum_eq_zero_add (by simp : f 1 = 0)]
    _ = 0 + 0 + 1 := by rw [factorial_telescoping_series_eq_one]
    _ = 1 := by norm_num
```

**Why Indicator Methods Matter.**
Using `ite` let us treat the early vanishing terms uniformly, leverage Mathlib4’s `tsum_subtype_eq_of_support_subset` to restrict sums, and hinge on absolute‐convergence results for rigorous justification of support‐based reindexing.
