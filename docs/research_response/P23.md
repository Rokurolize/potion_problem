# inv_factorial_ratio_tendsto_zero

**Main statement.**

```lean
import Mathlib.Data.Nat.Factorial.Cast
import Mathlib.Analysis.SpecificLimits.Basic

lemma inv_factorial_ratio_tendsto_zero :
  Tendsto (fun n => ((1 : ℝ) / (n + 1).factorial) / (1 / n.factorial)) atTop (nhds 0) := by
  -- 1. Reduce the ratio to `1/(n+1)`
  have : (fun n => ((1 : ℝ) / (n + 1).factorial) / (1 / n.factorial)) = fun n => 1 / (n + 1) := by
    ext n
    -- use the cast lemma `Nat.cast_factorial` and the factorial succ lemma
    simp [Nat.cast_factorial, Nat.factorial_succ]
    ring
  -- 2. Rewrite and apply the standard limit `tendsto_one_div_add_atTop_nhds_zero_nat`
  rw [this]
  exact tendsto_one_div_add_atTop_nhds_zero_nat (1 : ℝ)
```


## Explanation

1. **Factorial handling in ℝ**
    - `Nat.cast_factorial (n + 1)` gives
`((n + 1).factorial : ℝ) = Polynomial.eval 1 (ascPochhammer ℝ (n + 1))`,
which `simp` uses to move the factorial into a multiplicative context.
    - `Nat.factorial_succ n` rewrites `(n + 1).factorial = (n + 1) * n.factorial`.
    - `ring` then closes the algebraic step, yielding

$$
\frac{1/(n+1)!}{1/n!}
  = \frac{n!}{(n+1)!}
  = \frac{n!}{(n+1)\,n!}
  = \frac{1}{n+1}.
$$
2. **Limit theorem**
    - Mathlib provides

```lean
theorem tendsto_one_div_add_atTop_nhds_zero_nat :
  Tendsto (fun n : ℕ => 1 / (n + 1)) atTop (nhds 0)
```

    - After rewriting our original sequence to `fun n => 1/(n+1)`, we apply this lemma directly.

## Imports Required

```lean
import Mathlib.Data.Nat.Factorial.Cast           -- for `Nat.cast_factorial`
import Mathlib.Analysis.SpecificLimits.Basic    -- for `tendsto_one_div_add_atTop_nhds_zero_nat`
```


## Alternative Approaches

- **Direct ratio lemma.** One could also prove

```lean
lemma factorial_ratio (n : ℕ) :
  ((1 : ℝ) / (n + 1).factorial) / (1 / n.factorial) = 1 / (n + 1) := by
  simp [Nat.cast_factorial, Nat.factorial_succ]
```

and then use it in the main proof via `simp` and `rw`.
- **Using `funext`.** Instead of `ext n`, one can write:

```lean
have h_eq : _ = _ := by
  funext n; simp [Nat.cast_factorial, Nat.factorial_succ]; ring
```

achieving the same result.

This completes a clean v4.12.0 proof that the ratio of consecutive inverse factorials tends to 0.
