# Series Reindexing via $$
n-2
$$ Bijection in Lean 4

**Main Takeaway.**
For any nonnegative sequence $f:\mathbb N\to\mathbb R$, reindexing by the shift $n\mapsto n-2$ gives

$$
\sum_{n=2}^\infty f(n-2)\;=\;\sum_{k=0}^\infty f(k).
$$

In Lean 4 with Mathlib4, this follows immediately from the general lemma `Equiv.tsum_eq` and the explicit bijection

$$
\mathrm{shift} : \mathbb N\;\simeq\;\{\,n\mid 2\le n\},
$$

defined by $k\mapsto k+2$.

```lean
import Mathlib.Topology.Algebra.InfiniteSum.Basic  -- ≃-reindexing API

open Equiv Filter

/-- The bijection \(\mathbb N \simeq \{n ∈ ℕ \mid 2 ≤ n\}\) by \(k ↦ k+2\). -/
def n_minus_two_bij : ℕ ≃ {n // 2 ≤ n} where
  toFun    k := ⟨k+2, Nat.le_add_right _ _⟩
  invFun   n := n.val - 2
  left_inv k := by
    simp only [Subtype.coe_mk]; exact Nat.add_sub_cancel _ _
  right_inv n := by
    ext; simp [Nat.sub_add_cancel n.prop]

@[simp] lemma n_minus_two_bij_symm_toFun (k : ℕ) :
  n_minus_two_bij.symm.toFun ⟨k+2, Nat.le_add_right _ _⟩ = k := by
  -- Characterizes the inverse on the image
  dsimp; simp [n_minus_two_bij]

/-- Reindexing the sum over `n ≥ 2` by `n ↦ n - 2`. -/
theorem reindex_series_n_minus_two {f : ℕ → ℝ} (h_nonneg : ∀ n, 0 ≤ f n) :
    ∑' n : {n // 2 ≤ n}, f (n.val - 2) = ∑' k : ℕ, f k := by
  -- 1) Rewrite the left-hand tsum to use the inverse of our bijection
  have h1 : ∑' n : {n // 2 ≤ n}, f (n.val - 2)
         = ∑' n : {n // 2 ≤ n}, f (n_minus_two_bij.invFun n) := by
    -- by definition `n_minus_two_bij.invFun n = n.val - 2`
    congr 1
  rw [h1]
  -- 2) Apply the general reindexing lemma along the equivalence
  exact Equiv.tsum_eq n_minus_two_bij.symm (by
    -- Summability follows from nonnegativity and standard facts,
    -- but since both sides converge to the same ENNReal sum, we cast:
    apply (summable_coe.2 _).comp_injective
    -- `summable_coe` converts `Summable f` to `Summable (fun n => (f n : ℝ≥0∞))`,
    -- which holds by `h_nonneg`.
    refine ENNReal.summable_of_nonneg_of_le (fun n => by exact_mod_cast h_nonneg n) _
    -- The inequality `f ≤ f` is trivial:
    intro n; rfl
  )

/-! ## Concrete Examples -/

example : ∑' n : {n // 2 ≤ n}, (1 : ℝ) / (n.val - 2)!.factorial
       = ∑' k : ℕ, (1 : ℝ)/k.factorial := by
  apply reindex_series_n_minus_two
  intro _; exact (div_nonneg zero_le_one (Nat.cast_nonneg _))

example {x : ℝ} (hx : 0 ≤ x) (h : |x| < 1) :
  ∑' n : {n // 2 ≤ n}, x^(n.val - 2) = 1/(1 - x) := by
  -- Reindex to the standard geometric series
  rw [reindex_series_n_minus_two (fun _ => pow_nonneg hx _)]
  simpa using tsum_geometric_of_abs_lt_1 hx h

example : ∑' n : {n // 2 ≤ n}, (n.val - 2 : ℝ) / 2^(n.val - 2) = 4 := by
  rw [reindex_series_n_minus_two (fun _ => div_nonneg (Nat.cast_nonneg _) (pow_pos (by norm_num : 0<2) _))]
  -- We use the known formula ∑ k/2^k = 4
  exact (tsum_div_pow_eq_of_lt_1 (by norm_num : (2 : ℝ)⁻¹ < 1)).symm
```

**Explanation of Key Steps.**

1. **Bijection `n_minus_two_bij`.**
    - `toFun k = k+2` ensures every `n ≥ 2` is hit.
    - `invFun n = n.val - 2` inverts the shift.
    - `left_inv` and `right_inv` follow from `Nat.add_sub_cancel` on indices satisfying `n ≥ 2`.
2. **General Reindexing with `Equiv.tsum_eq`.**
    - `Equiv.tsum_eq` states that for any equivalence `φ : α ≃ β` and `Summable g`,
$\sum' a : α, g(φ\,a) = \sum' b : β, g\,b$.
    - We apply it to `φ = n_minus_two_bij.symm` and `g = fun n => f (n.val - 2)`.
3. **Nonnegativity and Summability.**
    - We assume `f` is nonnegative so that both sides converge in `ℝ≥0∞`.
    - `ENNReal.summable_of_nonneg_of_le` then shows summability in the extended‐nonnegative sense.
    - Casting back to `ℝ` via `summable_coe` justifies the application of `Equiv.tsum_eq`.

This completes a succinct, robust formal proof in Lean 4 that
$\sum_{n=2}^\infty f(n-2) = \sum_{k=0}^\infty f(k)$.
