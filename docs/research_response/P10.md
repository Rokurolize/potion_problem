# Summability of the Factorial-Difference Series in Lean 4

**Main Statement.**
In Mathlib4, the series

```lean
∑' n, if n ≥ 2 then (1 : ℝ) / (n - 1).factorial - 1 / n.factorial else 0
```

is absolutely convergent, i.e.

```lean
lemma summable_factorial_diff :
  Summable (fun n : ℕ =>
    if n ≥ 2 then (1 : ℝ) / (n - 1).factorial - 1 / n.factorial else 0)
```


## 1. Absolute-Convergence via Direct Comparison

We show

$$
\bigl|(1/(n-1)!)-1/n!\bigr|\;=\;(n-1)/n!
\;\le\;1/(n-1)!,
$$

and use that $\sum 1/(n-1)!$ converges (it is a tail of the exponential series).

```lean
import Mathlib.Analysis.Normed.Group.InfiniteSum  -- for Summable.of_norm_bounded
import Mathlib.Data.Nat.Factorial.Basic            -- for factorial lemmas

theorem summable_factorial_diff :
  Summable (fun n : ℕ =>
    if n ≥ 2 then (1 : ℝ) / (n - 1).factorial - 1 / n.factorial else 0) := by
  -- bound by b_n = if n ≥ 2 then 1/(n-1)! else 0
  have h_bound : ∀ n,
    ‖if n ≥ 2 then (1 : ℝ)/(n - 1).factorial - 1/n.factorial else 0‖
      ≤ if n ≥ 2 then 1/(n - 1).factorial else 0 := by
    intro n
    split_ifs with h₂
    · -- case n ≥ 2
      simp only [norm_sub, norm_div, norm_div, norm_one, norm_cast]
      -- simplify |1/(n-1)! - 1/n!| = (n-1)/n!
      have : ‖(1 : ℝ)/(n-1).factorial - 1/n.factorial‖ = (n-1)/n.factorial := by
        field_simp [Nat.cast_ne_zero.mpr (Nat.factorial_pos (n-1)).ne']
        simp [abs_sub, abs_div]
      rw [this]
      -- now (n-1)/n! ≤ 1/(n-1)! by n! = n·(n-1)! ≥ n·(n-1)! for n≥2
      calc (n-1)/n.factorial
          ≤ (n-1)/((n-1)! * (n-1 + 1)) := by
        simpa [Nat.succ_sub_one, mul_comm] using
          (div_le_div_right (Nat.cast_pos.mpr (Nat.factorial_pos (n-1))))
            (by rw [factorial_succ]; exact le_rfl)
      _ = 1/(n-1)! := by field_simp [Nat.cast_succ, Nat.cast_mul]
    · -- case n < 2
      simp
  -- apply comparison test
  apply Summable.of_norm_bounded (fun n =>
    if n ≥ 2 then 1/(n - 1).factorial else 0) h_bound
  -- the majorant series is summable since ∑_{n≥2} 1/(n-1)! = ∑_{k≥1} 1/k! < ∞
  simpa only [cast_add, Finset.range, Nat.succ_sub_one, if_pos] using
    (summable_inv_factorial.add_right_const _).of_add_left
```


## 2. Telescoping-Series Decomposition

Alternatively, decompose

$$
a_n =  
\begin{cases}
1/(n-1)! - 1/n!, & n≥2,\\
0, &\text{else},
\end{cases}
$$

as the difference of two exponential‐tail series:

```lean
import Mathlib.Data.Nat.Factorial.Basic
import Mathlib.Analysis.Normed.Group.InfiniteSum

theorem summable_factorial_diff : Summable (fun n => if n ≥ 2 then _ else 0) := by
  -- decompose into S₁ - S₂
  have decomp :
    (fun n => if n ≥ 2 then 1/(n-1).factorial - 1/n.factorial else 0) =
    (fun n => if n ≥ 1 then 1/n.factorial else 0) -
    (fun n => if n ≥ 2 then 1/n.factorial else 0) := by
    ext n; split_ifs; simp_all [zero_sub, sub_zero]
  -- each component is a tail of 1/n!; hence summable
  have sum1 : Summable (fun n => if n ≥ 1 then 1/n.factorial else 0) :=
    (summable_inv_factorial).of_subtype (by simp)
  have sum2 : Summable (fun n => if n ≥ 2 then 1/n.factorial else 0) :=
    (summable_inv_factorial).of_subtype (by simp)
  -- difference of summable series is summable
  simpa using (sum1.sub sum2).congr decomp.symm
```


## 3. Telescoping via Monotonicity

Recognize the partial sums telescope because

$$
\sum_{n=2}^N \bigl(\tfrac1{(n-1)!}-\tfrac1{n!}\bigr)
=1 - \tfrac1{N!},
$$

and $1-1/N!$ converges as $N\to\infty$.  In Lean:

```lean
import Mathlib.Data.Nat.Factorial.Basic
import Mathlib.Analysis.Normed.Group.InfiniteSum

theorem summable_factorial_diff : Summable (fun n => if n ≥ 2 then _ else 0) := by
  -- show partial sum S N = 1 - 1/N!
  have h_sum : HasSum (fun n => if n ≥ 2 then 1/(n-1).factorial - 1/n.factorial else 0) 1 := by
    simp only [hasSum_iff_tendsto_sum_nat]; refine ⟨_, _⟩
    · ext N; simp [Finset.sum_range_succ', factorial_succ]
    · simpa using tendsto_const_nhds.sub (tendsto_inv_atTop_nhds_zero.comp tendsto_factorial_atTop)
  exact h_sum.summable
```

All three approaches yield the desired `Summable` instance.
