# Hitting Time PMF Sums to Unity

**Main Takeaway.**
For the hitting time

$$
\tau \;=\;\min\{n:\;S_n\ge1\},
$$

where $S_n$ is the sum of $n$ i.i.d.\ $\mathrm{Uniform}[0,1)$ random variables, the probability mass function

$$
P(\tau=n)
\;=\;
\begin{cases}
0, & n\le1,\\
P(S_{n-1}<1)-P(S_n<1), & n\ge2
\end{cases}
$$

satisfies

$$
\sum_{n=0}^\infty P(\tau=n)\;=\;1.
$$

Equivalently, in Lean 4:

```lean
theorem hitting_time_pmf_sum_one :
  ∑' n : ℕ, (if n ≤ 1 then 0 else 1/(n-1)! - 1/n!) = 1
```


## 1. Probabilistic Interpretation

1. **Definition of the PMF.**
For each $n$:

$$
P(\tau=n)
=
\begin{cases}
0, & n\le1,\\
P(S_{n-1}<1)\;-\;P(S_n<1), & n\ge2.
\end{cases}
$$
2. **Irwin–Hall formula.**
One shows (by induction or known result) that

$$
P(S_k<1)\;=\;\frac{1}{k!}
\quad\text{for each }k\ge0.
$$

Hence for $n\ge2$,

$$
P(\tau=n)
= \frac{1}{(n-1)!}\;-\;\frac{1}{n!}.
$$

## 2. Direct Telescoping Proof

Observe that

$$
\sum_{n=0}^\infty P(\tau=n)
=\sum_{n=2}^\infty\Bigl(\tfrac1{(n-1)!}-\tfrac1{n!}\Bigr).
$$

A standard finite‐telescoping argument gives for each $N\ge2$:

$$
\sum_{n=2}^N\Bigl(\tfrac1{(n-1)!}-\tfrac1{n!}\Bigr)
=1 - \tfrac1{N!}.
$$

Since $\tfrac1{N!}\to0$ as $N\to\infty$, the infinite sum equals $1$.

### Lean 4 Proof

```lean
import Mathlib.Topology.Algebra.InfiniteSum.Basic  -- hasSum, tsum
import Mathlib.Tactic.Ring
import Mathlib.Tactic.Simp

open Filter

theorem hitting_time_pmf_sum_one :
  ∑' n : ℕ, (if n ≤ 1 then 0 else 1/(n-1)! - 1/n!) = 1 := by
  -- Step 1: Restrict to `n ≥ 2` since terms for `n=0,1` vanish
  have support_subset :
    Function.support (fun n => if n ≤ 1 then 0 else 1/(n-1)! - 1/n!)
      ⊆ {n | 2 ≤ n} := by
    intro n hn
    simp [Function.mem_support] at hn
    split_ifs at hn with h
    · exact le_of_not_le h
    · contradiction
  -- Convert tsum over ℕ to tsum over the subtype `{n // 2 ≤ n}`
  have eq_subtype :
    ∑' n : ℕ, (if n ≤ 1 then 0 else 1/(n-1)! - 1/n!)
      = ∑' k : {n // 2 ≤ n}, (1/((k : ℕ)-1)! - 1/(k : ℕ)!) :=
    by simpa using tsum_subtype_eq_of_support_subset _ support_subset
  -- Step 2: Apply the telescoping series result on the subtype
  have key :
    ∑' k : {n // 2 ≤ n}, (1/((k : ℕ)-1)! - 1/(k : ℕ)!) = 1 := by
    -- Compute partial sums: for each N ≥ 2,
    --   ∑_{k=2}^N (1/(k-1)! - 1/k!) = 1 - 1/N!
    have partial_sum : ∀ N (hN : 2 ≤ N),
      ∑ i in Finset.range (N-1), (1/(i+1)! - 1/(i+2)!) = 1 - 1/N! := by
      intro N hN
      -- `range (N-1)` indexes i = 0..N-2 so (i+2) runs 2..N
      simp [Finset.sum_range_succ', sub_eq_add_neg]
      -- telescoping over i
      abel
    -- Lift `partial_sum` to `HasSum` on the subtype
    refine HasSum.subtype_range _ partial_sum tendsto_const_nhds
    exact λ k _ => by simp
  -- Conclude the result
  simpa using eq_subtype.trans key
```


## 3. Generating Function Method

Define the probability generating function

$$
G(s)=\sum_{n=0}^\infty P(\tau=n)\,s^n
=\sum_{n=2}^\infty\Bigl(\tfrac1{(n-1)!}-\tfrac1{n!}\Bigr)s^n.
$$

At $s=1$, one recovers

$$
G(1)
=\sum_{n=2}^\infty\Bigl(\tfrac1{(n-1)!}-\tfrac1{n!}\Bigr)
=1.
$$

## 4. Measure‐Theoretic Approach

Define a counting‐measure‐based probability on $\ℕ$:

```lean
def hitting_time_measure : Measure ℕ :=
  Measure.sum (fun n => if n ≤ 1 then 0 else (1/((n-1)!)-1/n!) • Measure.dirac n)
```

One shows

```lean
lemma hitting_time_is_prob_measure :
  IsProbabilityMeasure hitting_time_measure :=
by
  -- Total mass = ∑ P(τ=n) = 1 by the telescoping result
  have : hitting_time_measure univ
        = ∑' n, (if n ≤ 1 then 0 else 1/(n-1)! - 1/n!) := by simp [hitting_time_measure]
  simp [this, hitting_time_pmf_sum_one]
```


## 5. Renewal‐Theory Perspective

In renewal theory, $\tau$ is the first‐passage or renewal time above level 1.  The renewal theorem ensures $\tau<\infty$ a.s.\ and that its PMF normalizes to 1.

**Conclusion.**
All approaches confirm the PMF for the hitting time $\tau$ of the uniform‐sum process sums to 1, completing the proof of `hitting_time_pmf_sum_one`.
