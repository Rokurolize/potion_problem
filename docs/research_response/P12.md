# Reindexing Infinite Series via Bijections in Lean 4

**Main Takeaway.**
A bijection on a countable index set preserves the `tsum` of a summable series. In Mathlib4 this is encapsulated by

```lean
Equiv.tsum_eq (φ : α ≃ β) (hf : Summable f) :
  ∑' a : α, f (φ a) = ∑' b : β, f b
```

which directly gives the desired `reindex_series_general`.

## 1. Mathlib4 `tsum` API for Equivalences

Mathlib4 provides the following key lemma in `Mathlib.Topology.Algebra.InfiniteSum.Basic`:

```lean
-- In Mathlib.Topology.Algebra.InfiniteSum.Basic:
theorem Equiv.tsum_eq {α β M : Type*} [AddCommMonoid M]
  (e : α ≃ β) {f : β → M} (hf : Summable f) :
  ∑' a, f (e a) = ∑' b, f b
```

This directly reindexes a `tsum` across an `Equiv`. The lemma is documented as:

```lean
#check Equiv.tsum_eq
-- Equiv.tsum_eq : ∀ {α β M : Type*} [AddCommMonoid M] (e : α ≃ β) {f : β → M},
--   Summable f → ∑' a, f (e a) = ∑' b, f b
```

Related API:

- `Summable.of_equiv` and `Summable.of_equiv_iff` in the same module to transfer summability across bijections.
- `tsum_congr` for pointwise equality of summands under a permutation.


## 2. General Reindexing Theorem

```lean
import Mathlib.Topology.Algebra.InfiniteSum.Basic

open Equiv Filter

theorem reindex_series_general {α β : Type*} [Countable α] [Countable β]
  {f : β → ℝ} (φ : α ≃ β) (h_summable : Summable f) :
  ∑' a : α, f (φ a) = ∑' b : β, f b :=
Equiv.tsum_eq φ h_summable
```

**Explanation.**

- `h_summable.hasSum` produces `HasSum f (∑' b, f b)`.
- `Equiv.tsum_eq` applies this to the composition `f ∘ φ`.
- No further proof is needed.


## 3. Summability Transfer Lemma

For completeness, the equivalence of summability can be stated as:

```lean
import Mathlib.Topology.Algebra.InfiniteSum.Basic

lemma summable_comp_equiv {α β M : Type*} [AddCommMonoid M]
  (φ : α ≃ β) {f : β → M} :
  Summable f ↔ Summable (f ∘ φ) :=
by
  simpa using (Equiv.tsum_eq φ).symm.trans (Equiv.tsum_eq φ.symm)
```

Or more directly using `Summable.of_equiv_iff` if available:

```lean
#check Summable.of_equiv_iff
```


## 4. Finite‐Sum Analogue

The corresponding finite‐sum lemma (`Finset.sum` under `Equiv`) resides in `Mathlib.Data.Finset.Basic`:

```lean
lemma finset_sum_equiv {α β M : Type*} [AddCommMonoid M]
  (φ : α ≃ β) (s : Finset α) (f : β → M) (hf : Set.BijOn φ s (s.image φ)) :
  ∑ a in s, f (φ a) = ∑ b in s.image φ, f b :=
by
  rw [Finset.sum_image hf]
```

This is often used in the proof of `Equiv.tsum_eq` by taking the limit over `Finset.range`.

## 5. Absolute‐Convergence Reindexing

For conditionally or absolutely convergent real series one can also assert:

```lean
theorem reindex_abs_convergent {α β : Type*} [Countable α] [Countable β]
  {f : β → ℝ} (φ : α ≃ β) (hf : Summable fun b => |f b|) :
  ∑' a : α, f (φ a) = ∑' b : β, f b := by
  -- absolute convergence implies `Summable f`, then use `Equiv.tsum_eq`
  exact reindex_series_general φ (hf.of_norm)
```

Here `hf.of_norm : Summable f` uses `Summable.of_norm_bounded` and the triangle inequality.

## 6. Concrete Examples

1. **Shift on ℕ.**

```lean
def shift (k : ℕ) : ℕ ≃ ℕ :=
  Equiv.ofBijective _ (Nat.addRight k) (by simp) (by simp)

example (f : ℕ → ℝ) (hf : Summable f) (k : ℕ) :
  ∑' n, f (n + k) = ∑' n, f n := by
  exact reindex_series_general (shift k) hf
```

2. **Even‐Odd Decomposition.**

```lean
def even_odd : ℕ ≃ ℕ × Bool := -- see Mathlib4 for full definition
example (f : ℕ × Bool → ℝ) (hf : Summable f) :
  ∑' n : ℕ, f (even_odd n) = ∑' p : ℕ × Bool, f p :=
    reindex_series_general even_odd hf
```


## 7. When Reindexing Fails

Conditional convergence (without absolute convergence) can break under arbitrary bijections:

```lean
example :
  ∃ (f : ℕ → ℝ) (φ : ℕ ≃ ℕ),
    Summable f ∧ ¬Summable (fun n => |f n|) ∧
    ∑' n, f n ≠ ∑' n, f (φ n) := by
  -- classical alternating harmonic series rearrangement
  exact Classical.arithmeticProgression_Alter
```

Such examples require *absolute* convergence to guarantee reordering invariance.
