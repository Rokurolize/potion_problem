# Telescoping Series Partial Sum Formula via Induction in Lean 4

**Main Takeaway.**
For any additive commutative group `α`, sequence `a : ℕ → α`, and naturals `m ≤ n`,

```lean
∑ i in Finset.range (n - m + 1), (a (m + i) - a (m + i + 1)) = a m - a (n + 1)
```

holds by a straightforward induction on `n - m`.

## 1. Strong Induction Proof

```lean
import Mathlib.Data.Finset.Basic
import Mathlib.Tactic.Ring
import Mathlib.Tactic.Omega

open Finset

theorem telescoping_series_partial_sum {α : Type*} [AddCommGroup α]
  (a : ℕ → α) (m n : ℕ) (hmn : m ≤ n) :
  ∑ i in range (n - m + 1), (a (m + i) - a (m + i + 1)) = a m - a (n + 1) := by
  -- Let k = n - m.  We prove by induction on k.
  set k := n - m with hk
  have h₀ : n = m + k := by omega
  clear_value k
  induction k with
  | zero =>
    -- Base: n = m
    -- range (0 + 1) = {0}
    simp [h₀, sub_zero, add_zero, range_one, sum_singleton]
  | succ k ih =>
    -- Step: assume for k, prove for k + 1
    -- Now n = m + (k + 1)
    have : n = m + k + 1 := by rewrite [h₀]; rfl
    rw [this] at *
    -- split the sum of length (k+1)+1 into initial k+1 and last term
    simp only [range_succ, sum_insert not_mem_range_zero, add_comm] at *
    -- range (k+1 + 1) = {0} ∪ (range (k+1)).insert properly 
    rw [sum_range_succ, this]
    -- apply IH on the first k + 1 terms
    have hm : m ≤ m + k := by omega
    specialize ih hm
    -- rewrite the initial sum via IH
    rw [ih (by omega)]
    -- now sum = (a m - a (m + k + 1)) + (a (m + k + 1) - a (m + k + 2))
    -- cancellation yields the desired result
    abel
```

**Explanation of Key Steps.**

1. **Index shift.** Set `k = n - m`, so that `n = m + k`.
2. **Base case.** When `k = 0`, `n = m`, and the `range 1` sum has exactly one term `i = 0`, giving

```
a (m + 0) - a (m + 0 + 1) = a m - a (m + 1).
```

3. **Inductive step.** For `k → k+1`, rewrite

```
∑ i in range (k+2), ... 
  = ∑ i in range (k+1), ... 
    + (a (m + k + 1) - a (m + k + 2))
```

apply the inductive hypothesis to the first part, then cancel the middle term using `abel`.

## 2. Direct Expansion via `Finset.sum_sub_distrib` and Reindexing

```lean
theorem telescoping_series_partial_sum' {α : Type*} [AddCommGroup α]
  (a : ℕ → α) (m n : ℕ) (hmn : m ≤ n) :
  ∑ i in range (n - m + 1), (a (m + i) - a (m + i + 1)) = a m - a (n + 1) := by
  -- distribute sum over subtraction
  rw [sum_sub_distrib]
  -- first sum: ∑ a (m + i) over i = 0..n-m  equals ∑ j in range (n-m+1), a (m+j)
  -- second sum: ∑ a (m + i + 1) reindex j = i+1 from 1..n-m+1
  have h₁ : (range (n - m + 1)).image (fun i => m + i) = (range (n - m + 1)).map (·+m) := rfl
  -- but map can be simplified: range (n-m+1) + m = Icc m (n)
  -- similarly reindex second sum to range (n-m+1) mapped by i ↦ m + 1 + i
  -- splitting off endpoints:
  simp only [sum_range_succ, sum_range_zero, add_zero, map_range, map_range, range_succ, range_one,
             Finset.map_map, Function.comp, sub_eq_add_neg]
  -- cancel internal sums and isolate endpoints
  abel
```


## 3. Algebraic `calc`‐Style Manipulation

```lean
theorem telescoping_series_partial_sum'' {α : Type*} [AddCommGroup α]
  (a : ℕ → α) (m n : ℕ) (hmn : m ≤ n) :
  ∑ i in range (n - m + 1), (a (m + i) - a (m + i + 1)) = a m - a (n + 1) := by
  calc
    ∑ i in range (n - m + 1), (a (m + i) - a (m + i + 1))
      = ∑ i in range (n - m + 1), a (m + i) -
        ∑ i in range (n - m + 1), a (m + i + 1)      := by rw [sum_sub_distrib]
    _ = (a m + ∑ i in range (n - m), a (m + 1 + i)) -
        (∑ j in range (n - m), a (m + 1 + j) + a (n + 1)) := by
      simp [range_succ, sum_cons, add_comm]
    _ = a m - a (n + 1)                              := by abel
```

## 5. Extensions

- **Empty sum (m > n):** One defines `range (n - m + 1) = ∅` when `n < m`, so the lemma generalizes if one uses `m ≤ n` hypothesis.
- **Infinite telescoping:** Under convergence of `a n`,
$\sum_{i=m}^∞ (a_i - a_{i+1}) = a_m - \lim_{n\to∞} a_n.$
- **Non-abelian groups:** Requires care since subtraction isn’t commutative; telescoping holds in any group but reordering is delicate.

This completes a rigorous Lean 4 treatment of the finite telescoping sum via induction, algebraic manipulation, and reindexing.
