# Complete Lean 4 Proofs for Hitting Time Expectation

**Main Takeaway:**
E[τ] = e follows by showing

1. ∑ₙ P(τ = n) = 1 via a telescoping series,
2. ∑ₙ n·P(τ = n) = ∑_{k=0}^∞ 1/k! = e by reindexing,
3. thus E[τ] = e.

## 1. Imports and Setup

```lean
open Probability HittingTime
```


## 2. Component 1: ∑′ₙ P(τ = n) = 1

```lean
theorem prob_sum_one : ∑' n, prob_hitting_time n = 1 := by
  -- P(τ = 0) = P(τ = 1) = 0; for n ≥ 2,
  -- prob_hitting_time n = 1/(n-1)! - 1/n!  (telescoping)
  have h : ∀ n, prob_hitting_time n = if n < 2 then 0 else 1/(n-1)! - 1/n! :=
    HittingTime.hitting_time_pmf_formula
  -- Extract ∑_{n≥2} (1/(n-1)! - 1/n!) = 1 by `hasSum` telescoping
  calc
    ∑' n, prob_hitting_time n
        = ∑' n, if n < 2 then 0 else 1/(n-1)! - 1/n! := by simp [h]
    _ = ∑' n in Finset.range 2, 0 + ∑' n, (1/(n-1)! - 1/n!) := by
      -- split off the finite 0 part
      rw [tsum_ite_eq_left]
      · simp
      · apply summable_of_nonneg; intros _; simp; exact sub_nonneg.2 (by norm_num)
    _ = ∑' k, (1/(k)! - 1/(k+1)!) := by
      -- reindex: n = k+1
      let e : ℕ ≃ ℕ where
        toFun k := k+1
        invFun n := n-1
        left_inv := by rintro k; rfl
        right_inv := by rintro n; cases n <; · contradiction; rfl
      have : (fun k => 1/(k-1)! - 1/k!) ∘ e = fun k => 1/k! - 1/(k+1)! := by
        funext k; dsimp [e]; cases k; simp
      exact (tsum_comp_eq this (summable_of_sub _ _)).symm
    _ = (1/0! - lim (fun k => 1/(k+1)!)) := by exact hasSum_tsub_tsum telescoping_hasSum
    _ = 1 - 0 := by simp [lim_invFactorial_zero]
    _ = 1 := by simp
```


## 3. Component 2: Reindexing ∑′_{n≥2} 1/(n-2)! = ∑′ₖ 1/k!

```lean
lemma reindex_series :
    ∑' n : {n // n ≥ 2}, (1 : ℝ) / (n.val - 2).factorial =
      ∑' k, 1 / k.factorial := by
  -- Use `tsum_subtype` and `tsum_comp_eq`
  let e : {n // n ≥ 2} ≃ ℕ where
    toFun n := n.val - 2
    invFun k := ⟨k + 2, by simp⟩
    left_inv := by rintro ⟨n, hn⟩; simp [Nat.sub_add_cancel (Nat.le_of_succ_le_succ hn)]
    right_inv := by rintro k; rfl
  have h : (fun n => 1/(n.val - 2)!) = fun n => (1 / n!) ∘ e := by
    funext n; simp
  exact tsum_comp_eq h (summable_invFactorial.summable)
```


## 4. Component 3: Summability of ∑ n·P(τ=n)

```lean
lemma summable_hitting_time : Summable (fun n => n * prob_hitting_time n) := by
  -- For n ≥ 2: n·P(τ=n) = n·((n-1)/n!) = 1/(n-2)!
  have h : ∀ n, n * prob_hitting_time n = if n < 2 then 0 else 1/(n-2)! :=
    HittingTime.hitting_time_telescoping_property
  simp only [h]
  -- Summable sum of 1/(n-2)! after splitting off finitely many zeros
  refine summable_add_of_summable_right _ summable_zero
  simpa using (summable_invFactorial.summable.comp (fun n => if n < 2 then 0 else n-2) _)
```


## 5. Component 4: Main Theorem E[τ] = e

```lean
theorem uniform_sum_hitting_time_expectation :
    expected_hitting_time = exp 1 := by
  -- E[τ] = ∑ n·P(τ=n)
  have A : expected_hitting_time = ∑' n, n * prob_hitting_time n := rfl
  -- Summable by Component 3
  have B : Summable fun n => n * prob_hitting_time n := summable_hitting_time
  -- ∑ n·P(τ=n) = ∑_{k≥0} 1/k! by telescope + reindex
  calc
    ∑' n, n * prob_hitting_time n
      = ∑' n, if n < 2 then 0 else 1/(n-2)! := by simp [h]
    _ = ∑' k, 1 / k! := reindex_series
    _ = exp 1 := by exact exp_one_eq_tsum_invFactorial
  -- Conclude
  simp [A]
```


## 6. Mathematical Explanation

- **Telescoping PMF Sum:**
For n≥2, P(τ=n)=1/(n-1)!−1/n!. Summing yields 1/1!−limₙ1/n!=1.
- **Reindexing:**
∑_{n≥2}1/(n-2)! with k=n−2 gives ∑_{k≥0}1/k!=e.
- **Weighted Series:**
n·P(τ=n)=1/(n-2)! telescopes similarly to the exponential series.


## 7. Integration Strategy

1. **Independent Lemmas:** Prove `prob_sum_one`, `reindex_series`, `summable_hitting_time` separately.
2. **Main Result:** Chain these with the standard `exp_one_eq_tsum_invFactorial`.
3. **Order:** PMF sum → reindex → summability → main result.

## 9. Alternative Approaches

- Use `hasSum` directly with partial-sum limits instead of `tsum_…` lemmas.
- Prove telescoping via explicit `hasSum_sub` and `hasSum_tsub` lemmas.
- Handle subtype sums with `tsum_subtype'` rather than explicit equiv.

**Conclusion:**
This completes the formal Lean 4 proof that the expected uniform hitting time equals Euler’s number e, showcasing telescoping sums, summability, and series reindexing in `mathlib4`.
