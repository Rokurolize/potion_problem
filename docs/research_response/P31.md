# \# Lean 4 v4.12.0における望遠級数定理の代替実装法調査

## 研究背景

確率論の形式証明プロジェクトにおいて、望遠級数（telescoping series）を使った重要な定理の証明を行っています。しかし、調査の結果、期待していた `hasSum_telescope` APIがLean 4 v4.12.0 + Mathlib4 v4.12.0の組み合わせでは実装されていないことが判明しました。

### プロジェクト概要

このプロジェクトは「媚薬問題」という確率論的問題のE[τ] = e（期待値がオイラー数に等しい）という美しい結果を形式証明で厳密に証明することを目標としています。

### 技術環境の制約

- Lean 4バージョンとして v4.12.0（固定、他バージョン非互換）
- Mathlib4バージョンとして v4.12.0（固定、他バージョン非互換）
- 利用不可能な理想的APIとして `hasSum_telescope`
- 利用可能な基礎APIとして `HasSum`, `Summable`, `tsum`, `tendsto`など


## 解決が必要な数学的問題

### 望遠級数の定理

以下の望遠級数の定理を証明します：

**数学的表現**:

```
∑_{k=0}^∞ [f(k) - f(k+1)] = f(0) - lim_{k→∞} f(k)
```

**具体的ケース**:

```
∑_{k=0}^∞ [1/k! - 1/(k+1)!] = 1/0! - lim_{k→∞} 1/k! = 1 - 0 = 1
```


### 理想的だった実装（実際には存在しない）

```lean
-- 存在しないAPI（参考のため）
hasSum_telescope (f : ℕ → ℝ) (hf : Summable f) (hs : Summable (fun k => f k - f (k+1))) :
  HasSum (fun k => f k - f (k+1)) (f 0 - lim atTop f)
```


### 確率論での応用

この定理は確率質量関数の正当性証明で使用されます：

```lean
theorem prob_sum_one : ∑' n : ℕ, prob_hitting_time n = 1
```

ここで `prob_hitting_time n = 1/(n-1)! - 1/n!` （n ≥ 2の場合）なので、望遠級数となります。

```lean
import Mathlib.Topology.Algebra.InfiniteSum
import Mathlib.Data.Real.Basic

open Filter Metric Set

noncomputable theory

variable {α : Type*} [TopologicalSpace α]

theorem hasSum_telescope_v4_12_0 {𝕜 : Type*} [NormedField 𝕜] {f : ℕ → 𝕜}
    (hf : Summable f)
    (hs : Summable fun k => f k - f (k+1))
    (hlim : Tendsto f atTop (𝓝 0)) :
    HasSum (fun k => f k - f (k+1)) f 0 := by
  -- 1. Rewrite the partial sums to exhibit the telescoping cancellation
  have telescoping :
    ∀ n, ∑ k in Finset.range (n+1), (f k - f (k+1)) = f 0 - f (n+1) := by
    intro n
    -- Use the standard lemma `Finset.sum_range_succ` to peel off the last term,
    -- then `Finset.sum_range_sub` or `sum_range_sub` to telescope.
    induction n with
    | zero =>
      simp [Finset.range, Finset.range_zero]
    | succ n ih =>
      -- ∑ k in range (n+2), f k - f (k+1) =
      --   ∑ k in range (n+1), f k - f (k+1) + (f (n+1) - f (n+2))
      -- = (f 0 - f (n+1)) + (f (n+1) - f (n+2)) = f 0 - f (n+2)
      have : ∑ k in Finset.range (n+2), _ =
        ∑ k in Finset.range (n+1), (f k - f (k+1)) + (f (n+1) - f (n+2)) := by
        simp [Finset.range_succ, Finset.sum_range_succ]
      simpa [this, ih] using sub_add_sub_cancel' _ _
  -- 2. Show that the sequence of partial sums tends to `f 0 - 0`
  have : Tendsto (fun n => ∑ k in Finset.range (n+1), (f k - f (k+1))) atTop (𝓝 (f 0 - 0)) := by
    -- Rewrite the partial sum using our telescoping lemma
    show Tendsto (fun n => f 0 - f (n+1)) _ _
    -- `f 0` is constant, and `f (n+1)` tends to `0` by composing `hlim` with `Nat.succ`.
    refine (Tendsto.const_sub tendsto_const_nhds _).comp _
    -- `fun n => f (n+1)` tends to `0`
    exact (tendsto_add_atTop_succ hlim).const_sub _
  -- 3. Conclude `HasSum` from the `tendsto` of partial sums
  rwa [hasSum_iff_tendsto_nat'] at this
#align hasSum_telescope_v4_12_0 hasSum_telescope_v4_12_0
```

Explanation of key steps:

1. **Telescoping of finite sums**
We prove by induction that

$$
\sum_{k=0}^{n} (f(k) - f(k+1)) = f(0) - f(n+1).
$$

This uses the lemmas `Finset.range_succ` and `Finset.sum_range_succ` to peel off the last term and then simplify.
2. **Limit of partial sums**
Given `hlim : tendsto f atTop (𝓝 0)`, the function `n ↦ f(n+1)` also tends to `0`. Subtracting from the constant `f(0)` and using `Tendsto.const_sub` yields

$$
tendsto\bigl(\,n \mapsto f(0) - f(n+1)\bigr)\;atTop\;(𝓝(f(0) - 0)).
$$
3. **Conclusion by the `hasSum` characterization**
The lemma `hasSum_iff_tendsto_nat'` states that a series has sum `a` iff the sequence of its partial sums tends to `a`. We rewrite our result in that form to conclude `HasSum (fun k => f k - f(k+1)) (f 0 - 0)`, which simplifies to `HasSum _ f 0`.
