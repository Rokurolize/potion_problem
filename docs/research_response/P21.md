## Priority

**HIGH** - Core to factorial dominance proof in formal verification

```lean
import Mathlib.Order.Filter.Basic  -- for Filter.Tendsto.eventually
import Mathlib.Topology.Basic      -- for nhds and atTop

example {c : ℝ} (hc : c > 1)
  (h_summable : Summable (fun n : ℕ => c ^ n / n.factorial)) :
  ∀ᶠ n in atTop, c ^ n / n.factorial < 1 := by
-- 1. From summability we get `Tendsto … atTop (𝓝 0)`
have h_tendsto :
  Tendsto (fun n => c ^ n / n.factorial) atTop (𝓝 0) :=
  by
    -- `Nat.cofinite_eq_atTop` says `atTop = cofinite`,
    -- and `tendsto_cofinite_zero` is from `Summable.tendsto_cofinite_zero`.
    rw [← Nat.cofinite_eq_atTop]
    exact h_summable.tendsto_cofinite_zero

-- 2. Use the general lemma
--    `Filter.Tendsto.eventually`
--    which has type
--      ∀ {α β : Type _} {f : α → β} {l : Filter α} {b : β},
--        Tendsto f l (𝓝 b) →
--        ∀ {ε : ℝ}, 0 < ε →
--        ∀ᶠ a in l, ‖f a - b‖ < ε
--    specialized to `b = 0` and `ε = 1`, noting `‖x - 0‖ = |x|`.
have h_eventual : ∀ᶠ n in atTop, |c ^ n / n.factorial| < 1 :=
  h_tendsto.eventually (one_pos : 0 < (1 : ℝ))

-- 3. Since `c ^ n / n.factorial` is eventually nonnegative for large `n`
--    (indeed strictly positive), we can drop the absolute value:
filter_upwards [h_eventual] with n h using
  (abs_lt.mp h).1   -- yields `c ^ n / n.factorial < 1`

-- QED
```

Explanation of the key pieces:

1. **Tendsto from Summable**
We rewrite `atTop` as `Nat.cofinite` and apply `Summable.tendsto_cofinite_zero` to get

```
Tendsto (fun n => c ^ n / n.factorial) atTop (𝓝 0)
```

2. **Filter.Tendsto.eventually**
In Mathlib4 (v4.12.0), the lemma

```
filter.tendsto.eventually
  {α β : Type _} {f : α → β} {l : Filter α} {b : β}
  (h : Tendsto f l (𝓝 b)) {ε : ℝ} (hε : 0 < ε) :
  ∀ᶠ a in l, ‖f a - b‖ < ε
```

lives in `Mathlib.Order.Filter.Basic`.  We apply it to `h_tendsto`, `ε = 1` and `one_pos`.
3. **Dropping the absolute value**
Since each term `c ^ n / n.factorial` is strictly positive, `|c ^ n / n.factorial| = c ^ n / n.factorial`, so from

```
∀ᶠ n in atTop, |c ^ n / n.factorial| < 1
```

we derive

```
∀ᶠ n in atTop, c ^ n / n.factorial < 1
```

by `filter_upwards` and `abs_lt.mp`.
