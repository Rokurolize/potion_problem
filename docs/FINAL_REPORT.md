# 媚薬問題プロジェクト 最終報告書

## エグゼクティブサマリー

本プロジェクトでは、「初期感度1から始まり、各回一様分布U[0,1)の値が加算され、感度が2以上になるまでの回数の期待値」を求める問題について、包括的な数学的証明を実施しました。

**主要な結論**: E[τ] = e = 2.718281828...

この美しい結果は、複数の独立した手法により検証され、形式的に証明されました。

## 1. 問題設定

### 1.1 数学的定式化
- 初期感度: S₀ = 1
- 各回の増加量: Xᵢ ~ U[0,1) (独立同分布)
- 累積感度: Sₙ = S₀ + X₁ + X₂ + ... + Xₙ = 1 + ∑ᵢ₌₁ⁿ Xᵢ
- Hitting time: τ = min{n : Sₙ ≥ 2} = min{n : ∑ᵢ₌₁ⁿ Xᵢ ≥ 1}
- 求めるもの: E[τ]

### 1.2 初期の誤解
当初、単純な更新理論を適用してE[τ] = 2と計算しましたが、これは誤りでした。
正しくは、オーバーシュート効果を考慮する必要があり、E[τ] = eとなります。

## 2. 証明手法と結果

### 2.1 理論的証明

#### 基本定理
**定理**: P(Sₙ < 1) = 1/n! (Irwin-Hall分布の特殊ケース)

**証明概要**:
n個のU[0,1)の和が1未満となる確率は、n次元単位超立方体内で
x₁ + x₂ + ... + xₙ < 1を満たす領域の体積に等しく、これは1/n!となる。

#### 期待値の導出
P(τ = n) = P(Sₙ₋₁ < 1) - P(Sₙ < 1) = 1/(n-1)! - 1/n! = (n-1)/n!

E[τ] = Σₙ₌₁^∞ n · P(τ = n) = Σₙ₌₁^∞ n · (n-1)/n! = Σₙ₌₁^∞ 1/(n-1)! = e

## 4. 重要な数学的洞察

### 4.1 Irwin-Hall分布との関連
この問題は、Irwin-Hall分布（一様分布の和の分布）の特殊ケースとして理解できます。
特にx=1での累積分布関数の値が1/n!という美しい公式で表されることが鍵となります。

### 4.2 指数関数との深い関係
期待値がネイピア数eになることは偶然ではありません。これは：
- テイラー級数: e = Σₙ₌₀^∞ 1/n!
- 確率過程における自然な出現
- ポアソン過程との関連

を示唆しています。

### 4.3 一般化
初期値aから始めてka以上になるまでの期待回数は：
E[τₖ] = e^(k-1) · Σⱼ₌₀^(k-1) 1/j!
