# compact operator simplex解

この解では、停止前領域を積分作用素が保存する単体部分として取り出す。

長さ`n`のprefix空間上で、関数を停止前領域に制限する作用素

```text
(K_n f)(x)=1_{x_1+...+x_n<=1} f(x)
```

を考える。単位立方体上の一様測度に対して、この作用素が定数関数`1`から取り出す総質量は

```text
int K_n 1 dx
 =int_{x_i>=0, x_1+...+x_n<=1}1 dx.
```

右辺は標準`n`単体の体積なので

```text
P(tau>n)=1/n!.
```

したがって

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

停止前へ制限する作用素を長さごとに見ると、各段階で残る質量が単体体積として計算される。
