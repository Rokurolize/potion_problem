# 生成作用素の真空期待値解

独立一様乱数を `X_1, X_2, ...` とし、

```text
S_n = X_1 + ... + X_n
```

とおく。停止前prefixの個数として

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

だから、

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

ここで、区間 `[0,1]` 上の順序付き点列を作る生成作用素 `C` を考える。関数 `f` に対して

```text
(Cf)(t) = int_0^t f(s) ds
```

と定める。これは「終端時刻 `t` の手前に、最後の点を一つ生成する」操作である。

空prefixを真空ベクトル `1` と見ると、`C^n 1` は長さ `n` の順序付き点列の総密度である。実際、

```text
(C^n 1)(t)
= int_{0 <= u_1 <= ... <= u_n <= t} du_1 ... du_n
```

であり、累積時刻 `u_i = S_i` への変数変換のヤコビアンは `1` なので、

```text
P(S_n <= 1) = (C^n 1)(1)
```

である。

## 全ての生成を一度に足す

停止前prefix全体は、真空から任意回だけ生成したものなので、

```text
G = (I + C + C^2 + ...) 1
```

と書ける。これは形式的には

```text
G = (I - C)^{-1} 1
```

であり、同値に

```text
G = 1 + C G
```

を満たす。

したがって

```text
G(t) = 1 + int_0^t G(s) ds
```

である。両辺を微分すると

```text
G'(t) = G(t),   G(0) = 1
```

なので、

```text
G(t) = exp(t)
```

を得る。

求める期待値は、全ての生成履歴を境界 `1` で評価した真空期待値だから、

```text
E[tau]
= sum_{n >= 0} (C^n 1)(1)
= G(1)
= e
```

である。

この証明では、長さごとの単体体積を先に求めない。停止前prefixを「空状態から点を一つずつ生成する作用素の幾何級数」として束ねる。`e` は、生成作用素のレゾルベントを真空ベクトルに作用させたときの境界値として現れる。
