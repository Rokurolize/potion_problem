# 最小不動点解

この解では、答えを分布からではなく、期待値そのものが満たす最小不動点として決める。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

残り容量が`x`のとき、これから数えられる生存prefixの期待個数を

```text
H(x)
```

と書く。求める値は`H(1)`である。`x=0`でも、空prefixだけはまだ生きているので

```text
H(0) = 1
```

である。

## 期待値の自己参照

残り容量`x`から見ると、まず空prefixが1個ある。次に最初の飲量を`u`とする。`u <= x`なら残り容量は`x-u`になり、そこから先の期待個数は`H(x-u)`である。`u > x`なら追加の生存prefixはない。

`0 <= x <= 1`では`U_1`の密度が`1`なので、

```text
H(x) = 1 + int_0^x H(x-u) du.
```

つまり

```text
H(x) = 1 + int_0^x H(t) dt.
```

ここで重要なのは、この式をただの微分方程式として見るだけではないことである。`H`は非負関数であり、右辺作用素

```text
(Phi f)(x) = 1 + int_0^x f(t) dt
```

の最小非負不動点である。

## Picard反復で最小解を作る

`H_0(x)=1`から始めて

```text
H_{k+1}(x) = 1 + int_0^x H_k(t) dt
```

と反復する。このとき単調に

```text
H_0(x) <= H_1(x) <= H_2(x) <= ...
```

であり、各`H_k`は「高々`k`回まで飲んだ生存prefixの期待個数」を表す。したがって極限が本来の`H`である。

計算すると

```text
H_0(x) = 1,
H_1(x) = 1 + x,
H_2(x) = 1 + x + x^2/2!,
...
H_k(x) = sum_{n=0}^k x^n/n!.
```

よって最小不動点は

```text
H(x) = sum_{n>=0} x^n/n! = exp(x).
```

したがって

```text
E[tau] = H(1) = e.
```

## この解の芯

この証明は、`P(tau=n)`を求めない。`tau`を生存prefixの期待個数として直接扱い、その期待値が自然な単調作用素の最小不動点であることを使う。

一様性は積分核を`1`にし、独立性は残り容量`x-u`から同じ関数`H`を再利用できることに現れる。
