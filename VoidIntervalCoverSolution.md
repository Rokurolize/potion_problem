# 空白区間被覆解

この解では、停止前prefixを点列として数えるかわりに、その右側に残る空白区間まで含めて見る。`e`は、Poisson過程で区間全体が「点列と最後の空白」に分解されるとき、空白確率を外す係数として現れる。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## prefixの右側の空白を付ける

`S_n <= 1`であるprefixに対し、累積時刻

```text
0 < S_1 < ... < S_n <= 1
```

を記録する。さらに、最後の点`S_n`から境界`1`までの区間

```text
(S_n, 1]
```

を「右空白」と呼ぶ。`n=0`では点列は空で、右空白は区間全体`(0,1]`である。

停止前prefixの測度は、各`n`についてこの点列のLebesgue測度

```text
dS_1 ... dS_n
```

を足したものである。ここまでは元の停止時刻をそのまま数えている。

## Poisson過程では右空白に同じ重みが付く

区間`[0,1]`上の強度`1`のPoisson過程を考える。点がちょうど

```text
0 < t_1 < ... < t_n <= 1
```

にあり、かつ右空白`(t_n,1]`に点がない、という密度を順序付き点列で書く。

Poisson過程では、指定した`n`点の密度が`dt_1...dt_n`で、さらに右側に点がない確率が

```text
exp(-(1-t_n))
```

である。一方、左側`[0,t_n]`の点数はすでに指定した`n`点で尽くされているので、全区間で見るJanossy密度は

```text
exp(-1) dt_1...dt_n
```

になる。右空白の長さに依存する因子と、左側で指定した点数の因子が合わさって、結局どの`n`、どの配置にも同じ`exp(-1)`が掛かる。

つまりPoisson確率測度で「有限点列とその右空白が区間`[0,1]`を被覆する」全事象を数えると、停止前prefix測度はその共通因子`exp(-1)`を外したものになっている。

## 全確率から戻す

Poisson過程の標本は、必ず一意に

```text
区間内の昇順点列 + 最後の点から1までの右空白
```

へ分解される。点がない場合も、空の点列と右空白`(0,1]`として含まれる。

したがって、全ての`n`と全ての点列を足したPoisson側の総質量は`1`である。停止前prefix測度は各配置でその`exp(-1)`倍を外したものなので、

```text
E[tau] = exp(1) * 1 = e.
```

## この見方の芯

単体体積を直接足すのではない。停止前prefixに「境界までの最後の空白」を貼り付けると、Poisson過程の全標本空間と同じ分解になる。Poisson側だけが、区間`[0,1]`全体に点が出る確率密度として一律の空白因子`exp(-1)`を払っている。

答えの`e`は、その空白因子を剥がして、確率測度から停止前prefixの生の測度へ戻すための倍率である。
