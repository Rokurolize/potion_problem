# 殺されるジャンプ過程のPoisson方程式解

この解では、累積量を状態とするジャンプ過程を考え、停止までの期待ステップ数をPoisson方程式で求める。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

状態`x`から始める。つまり、すでに累積量が`x`であり、次に一様乱数を足していく。`x > 1`に出たら吸収されたものとする。

`0 <= x <= 1`で、吸収までに必要な追加回数の期待値を

```text
u(x)
```

とおく。求める値は`u(0)`である。

## 一歩解析

状態`x`から一度飲むと、次の状態は`x+U`である。この1回は必ず数える。もし`x+U > 1`ならそこで終わり、`x+U <= 1`なら残りの期待値は`u(x+U)`である。

したがって

```text
u(x) = 1 + int_0^{1-x} u(x+t) dt.
```

境界の外側では`u(y)=0`、`y>1`である。

これは、区間`[0,1]`で動き、境界を越えたら殺されるジャンプ過程のPoisson方程式である。生成作用素を

```text
(Lf)(x) = int_0^1 (f(x+t)-f(x)) dt
```

と書き、`f(y)=0`を`y>1`で課すと、上の式は

```text
Lu(x) = -1
```

に等しい。

## 残り距離へ反転する

残り距離を

```text
r = 1 - x
```

とし、

```text
v(r) = u(1-r)
```

とおく。すると

```text
v(r) = 1 + int_0^r v(r-t) dt
     = 1 + int_0^r v(s) ds.
```

両辺を微分して

```text
v'(r) = v(r),
v(0) = 1.
```

よって

```text
v(r) = exp(r).
```

したがって初期状態`x=0`、すなわち`r=1`では

```text
E[tau] = u(0) = v(1) = e.
```

## 見方

この証明では、`tau`の分布を求めない。累積量を状態にして、停止までの期待追加回数を直接境界値問題にする。

一様性は生成作用素の積分核を定数にし、独立性は一歩後も同じ関数`u`で未来を評価できることに現れる。答えの`e`は、殺される一様ジャンプ過程のPoisson方程式を残り距離で反転したときの指数解である。
