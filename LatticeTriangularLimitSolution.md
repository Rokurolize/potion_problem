# 格子三角行列極限解

この解では、連続の積分方程式を直接解かない。まず境界`1`を`N`個の小区間に切り、有限個の残り容量だけを持つ三角行列問題に落とす。その有限問題の答えが`(1+1/N)^N`になり、最後に`e`へ極限を取る。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は停止前prefix数なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 有限格子版を作る

`N`を固定し、連続な増分の代わりに、幅`1/N`の小片を一つ選ぶ格子増分を考える。残り容量が`k/N`ある状態を`k`と書き、次に使う小片数`j`を

```text
j = 1,2,...,k
```

から同じ重み`1/N`で選べるものとして数える。容量を越える選択は吸収され、停止前prefixには入らない。

この有限問題の期待停止前prefix数を`h_N(k)`とすると、`h_N(0)=1`であり、`k >= 1`では

```text
h_N(k) = 1 + (1/N) sum_{i=0}^{k-1} h_N(i).
```

右辺は今より小さい状態だけを使う。したがってこれは下三角行列

```text
I - A_N
```

を前進代入で解く問題である。ここで`A_N`は、下側に全部`1/N`を置いたVolterra型の有限行列である。

## 三角行列は二項係数を吐く

上の再帰から差分を取ると

```text
h_N(k+1) - h_N(k) = (1/N) h_N(k).
```

よって

```text
h_N(k+1) = (1 + 1/N) h_N(k).
```

初期値`h_N(0)=1`だから

```text
h_N(k) = (1 + 1/N)^k.
```

特に境界`1`に対応する`k=N`では

```text
h_N(N) = (1 + 1/N)^N.
```

これは有限格子上の三角行列逆行列の最後の成分である。

## 連続問題へ戻す

この格子問題は、連続一様増分の積分作用素を下三角Riemann和で近似したものになっている。連続版の値関数`h(x)`は

```text
h(x) = 1 + int_0^x h(x-u) du
```

を満たし、格子版はこの式の左Riemann和

```text
h_N(k) = 1 + (1/N) sum_{i=0}^{k-1} h_N(i)
```

そのものである。単調で有界なVolterra再帰なので、格子幅`1/N`を0へ送ると、`h_N(N)`は連続問題の`h(1)`へ収束する。

したがって

```text
E[tau] = h(1)
       = lim_{N -> infinity} h_N(N)
       = lim_{N -> infinity} (1 + 1/N)^N
       = e.
```

## この見方の芯

この証明では、`P(S_n<=1)=1/n!`を先に出さない。停止問題を有限格子の吸収報酬問題へ押し込み、下三角行列の前進代入だけで`(1+1/N)^N`を得る。

`e`は階乗級数の和としてではなく、連続容量問題を有限容量問題で解いたときの三角行列逆行列の極限値として現れる。
