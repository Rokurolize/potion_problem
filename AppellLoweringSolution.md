# Appell降下列解

この解では、各長さの生存確率を個別に積分しない。境界`x`を動かしたときの生存体積列が、微分で一段下がるAppell型の列になっていることだけを見る。

独立な一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

`S_0=0`を含めれば

```text
tau = sum_{n>=0} 1_{S_n <= 1}.
```

境界を`x`にした生存体積を

```text
A_n(x) = P(S_n <= x)        (0 <= x <= 1)
```

と置く。ただし`A_0(x)=1`である。求めたい値は

```text
sum_{n>=0} A_n(1)
```

である。

## 一段下がる微分

`n>=1`では、`0<=x<=1`の範囲で上限制約`U_i<=1`は生存領域内では自動的に満たされる。したがって

```text
A_n(x)
= volume { u_i>=0, u_1+...+u_n <= x }.
```

境界`x`を少し増やすと、新しく増えるのは

```text
u_1+...+u_n = x
```

付近の薄い層である。最後の変数`u_n`を

```text
u_n = x - (u_1+...+u_{n-1})
```

で決めると、薄い層の断面はそのまま

```text
{ u_1,...,u_{n-1} >= 0,
  u_1+...+u_{n-1} <= x }
```

になる。よって

```text
A_n'(x) = A_{n-1}(x).        (1)
```

また`n>=1`では、境界`x=0`で正の長さのprefixは体積を持たないので

```text
A_n(0) = 0.                 (2)
```

つまり、列`A_0,A_1,A_2,...`は

```text
D A_n = A_{n-1},   A_0=1,   A_n(0)=0
```

という「微分で添字が一つ下がる列」である。

## 列をまとめて閉じる

普通はここから`A_n(x)=x^n/n!`を帰納で出す。ここでは列全体を

```text
G(x,z) = sum_{n>=0} A_n(x) z^n
```

にまとめる。

(1)より

```text
partial_x G(x,z)
= sum_{n>=1} A_n'(x) z^n
= z sum_{n>=0} A_n(x) z^n
= z G(x,z).
```

(2)より

```text
G(0,z)=1.
```

したがって

```text
G(x,z)=exp(zx).
```

期待値で必要なのは、段数を数える印`z`を最後に`1`へ戻した値である。

```text
E[tau]
= sum_{n>=0} A_n(1)
= G(1,1)
= exp(1)
= e.
```

## この見方の芯

この証明は、停止時刻の分布も単体体積公式も先に求めない。境界を動かしたとき、生存体積列が微分作用素の下で

```text
A_n -> A_{n-1}
```

と降りていくことだけを見る。

一様性は、生存領域の密度が`1`で、薄層断面の質量がそのまま一段短いprefix体積になる点に使われる。独立性は、各座標が同じ積密度で測られ、段数を足しても同じ降下則が保たれる点に使われる。答えの`e`は、このAppell型の降下列を生成関数で一括して読んだ境界値として現れる。
