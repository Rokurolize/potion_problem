# 更新密度解

この解では、媚薬問題を更新過程として見る。

各回の増分を独立な一様乱数`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

`S_n`は、長さ`1`以下の区間に入っている更新時刻である。停止時刻`tau`は、時刻`1`を超える最初の更新番号なので、

```text
tau = 1 + #{ n >= 1 : S_n <= 1 }.
```

したがって、`1`以下に落ちる正の更新時刻の期待個数が分かればよい。

## 更新密度を書く

`f_n(x)`を`S_n`の密度とする。`0 < x < 1`では、一様分布の密度が`1`なので、

```text
f_1(x) = 1,
f_{n+1}(x) = int_0^x f_n(y) dy.
```

正の更新時刻全体の密度を

```text
u(x) = sum_{n >= 1} f_n(x)
```

と置く。`0 < x < 1`では上の再帰を足し合わせて

```text
u(x)
= 1 + int_0^x u(y) dy.
```

両辺を微分すると

```text
u'(x) = u(x),    u(0) = 1.
```

よって

```text
u(x) = exp(x)        (0 < x < 1).
```

## 平均個数を積分する

区間`(0,1]`に入る正の更新時刻の期待個数は、更新密度の積分である。

```text
E[ #{ n >= 1 : S_n <= 1 } ]
= int_0^1 u(x) dx
= int_0^1 exp(x) dx
= e - 1.
```

最後に、`tau`は初期位置`S_0=0`から数えて、正の更新時刻の個数に、境界を超える最後の一口を加えたものだから、

```text
E[tau]
= 1 + (e - 1)
= e.
```

## この見方の要点

個々の確率`P(tau=n)`は計算しない。代わりに、時刻`1`までに現れる更新点の密度をまとめて数える。

一様増分では、時刻`1`より手前の畳み込み再帰がそのまま

```text
u = 1 + integral(u)
```

になり、更新密度が`exp(x)`と決まる。平均飲用回数は、その密度の面積に最後の一口を足したものとして`e`になる。
