# Malliavin発散証明書解

この解では、停止前prefixの体積を一つずつ求めない。各一様ジャンプを同じ微小量だけ押す摂動を考え、その発散恒等式が境界に残す量を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めて停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 全ジャンプを同じ方向へ押す

`0 <= x <= 1`に対して

```text
F(x) = sum_{n >= 0} P(S_n <= x)
```

を置く。境界`x`までの全prefix質量である。

境界を`x`から`x+h`へ動かすことは、各prefixについて、最後に一つ一様ジャンプを追加し、その追加ジャンプが薄い帯

```text
(x-S_n, x+h-S_n]
```

に入るかを見ることと同じである。`S_n <= x`なら、この帯の長さは`h+o(h)`である。一様密度は定数`1`なので、Malliavin型の言い方では「追加ジャンプ方向の発散」は境界帯に定数密度の流量を残すだけになる。

したがって、全ての生存prefixに同じ発散証明書を足すと、内部の摂動は互いに余分な重みを作らず、境界に

```text
F(x+h) - F(x) = h F(x) + o(h)
```

だけが残る。

これは

```text
F'(x) = F(x)
```

である。初期値は空prefixだけなので

```text
F(0) = 1.
```

よって

```text
F(x) = exp(x).
```

求める値は

```text
E[tau] = F(1) = e.
```

この証明では、`P(S_n<=1)`を先に計算しない。追加ジャンプ方向の発散恒等式が、薄い境界帯で既存prefix総量をそのまま再生することだけを使う。`e`は、その発散証明書を境界`0`から`1`まで積んだ値である。
