# 放射流束単体解

`S_n = X_1 + ... + X_n` とする。`S_n <= 1` の範囲では一様密度は1なので、

```text
P(S_n <= 1) = A_n(1)
```

と書ける。ただし

```text
A_n(t) = vol{x_i >= 0, x_1 + ... + x_n <= t}
```

である。

ここで `A_n(t)` を積分で直接計算せず、領域

```text
K_n(t) = {x_i >= 0, x_1 + ... + x_n <= t}
```

に放射ベクトル場

```text
R(x) = x / n
```

を入れる。このベクトル場の発散は

```text
div R = 1
```

なので、発散定理より

```text
A_n(t) = int_{K_n(t)} 1 dx = int_{partial K_n(t)} R・nu dS
```

である。

境界のうち座標面 `x_i = 0` では、外向き法線は `-e_i` であり、そこでは `x_i = 0` だから

```text
R・nu = -x_i/n = 0
```

となる。したがって流束は斜面

```text
x_1 + ... + x_n = t
```

からしか出ない。

この斜面を最初の `n - 1` 座標に射影すると

```text
{x_1, ..., x_{n-1} >= 0, x_1 + ... + x_{n-1} <= t}
```

そのものになる。斜面上では `x_1 + ... + x_n = t` なので、法線方向の成分と面積要素の傾きが打ち消し合い、流束は

```text
A_n(t) = (t / n) A_{n-1}(t)
```

になる。初期値は `A_0(t) = 1` であるから、

```text
A_n(1) = 1/n!
```

を得る。

到着数は

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので、単調収束により

```text
E[tau]
  = sum_{n >= 0} P(S_n <= 1)
  = sum_{n >= 0} A_n(1)
  = sum_{n >= 0} 1/n!
  = e
```

となる。単体体積の公式を借りるのではなく、座標面では流れが消え、斜面だけが次元を1つ下げるという発散定理の再帰で `1/n!` が出る。
