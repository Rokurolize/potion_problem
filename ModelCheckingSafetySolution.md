# model checking safety解

この解では、`S_n<=1`を安全性としてモデル検査風に読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。深さ`n`まで安全性が破られない事象は

```text
S_n<=1
```

である。従って停止時刻は、安全性がまだ真である深さを数えたものとして

```text
tau = sum_{n>=0} 1_{S_n<=1}
```

と書ける。

固定した`n`について、安全性を満たす入力列の集合は

```text
{(u_1,...,u_n): u_i>=0, u_1+...+u_n<=1}
```

で、これは標準単体だから体積は`1/n!`である。よって

```text
E[tau] = sum_{n>=0} P(S_n<=1)
       = sum_{n>=0} 1/n!
       = e.
```

モデル検査の言葉では、各深さで安全性を満たす実行prefixの測度を合計している。
