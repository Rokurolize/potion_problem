# prefix DAG path measure解

この解では、停止前prefixを容量を頂点とする連続DAGの道として数える。

残り容量を`x`とする。入力`u<=x`を読むことは、頂点`x`から`x-u`へ下る有向辺を一つ選ぶことに等しい。したがって長さ`n`の停止前prefixの総質量は、この連続DAGで`1`から任意の非負頂点へ下る長さ`n`の道の測度である。

長さ`n`の道の質量を`A_n(x)`と書くと

```text
A_0(x)=1,
A_{n+1}(x)=int_0^x A_n(y)dy.
```

これは、最後に到達する残り容量`y`で道を切っているだけである。帰納的に

```text
A_n(x)=x^n/n!.
```

元の停止時刻は、空prefixを含む停止前prefix数なので

```text
E[tau]=sum_{n>=0}A_n(1)=sum_{n>=0}1/n!=e.
```

この見方では、乱数列を直接積分する代わりに、容量DAG上の全有向道の測度として期待値を読む。
