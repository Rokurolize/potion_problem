# 更新報酬Palm解

この解では、停止前prefixを更新過程の報酬点として数え、Palm的に一つの報酬点から全体を戻す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。更新時刻`S_n`が区間`[0,1]`にある回数は、空更新時刻`S_0=0`を含めて`tau`である。したがって

```text
E[tau]=E[sum_{n>=0} 1_{S_n<=1}].
```

更新報酬測度

```text
U(A)=E[sum_{n>=0} 1_{S_n in A}]
```

を置くと、求める値は`U([0,1])`である。`[0,1]`内では一様増分の右端制約がまだ効かず、更新密度は

```text
u(x)=1+int_0^x u(y) dy
```

を満たす。微分して

```text
u'(x)=u(x),  u(0)=1
```

だから

```text
u(x)=e^x.
```

ここで`U([0,1])`は原子`0`と密度部分を合わせたprefix総量として

```text
E[tau]=e.
```

Palm的に見ると、区間内の典型的な更新点を一つ選び、その左側の履歴を同じ更新測度で戻すことが、上の再生方程式そのものである。
