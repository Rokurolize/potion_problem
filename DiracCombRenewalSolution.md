# Dirac comb renewal解

この解では、停止前prefixの質量を、原点の原子を持つ更新測度として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n=U_1+...+U_n,
tau=min { n>=1 : S_n>1 }
```

とする。停止前であることは`S_n<=1`であり、尾和公式から

```text
E[tau]=sum_{n>=0} P(S_n<=1)
```

である。

`[0,1]`上のLebesgue測度を`mu`とし、原点の質量を`delta_0`とする。更新測度を

```text
R = delta_0 + mu + mu*mu + mu*mu*mu + ...
```

と定める。すると

```text
R([0,1]) = sum_{n>=0} P(S_n<=1) = E[tau].
```

ここで`0<=x<=1`に限れば、`mu^{*n}`の密度は上限`1`の切れ目にまだ当たらず

```text
d(mu^{*n})/dx = x^{n-1}/(n-1)!  (n>=1)
```

である。したがって

```text
R([0,1])
=1+int_0^1 sum_{n>=1} x^{n-1}/(n-1)! dx
=1+int_0^1 e^x dx
=e.
```

この見方では、各`n`の単体をばらばらに足すかわりに、全prefixを一つのDirac comb付き更新測度に束ね、その区間質量として期待停止時刻を読んでいる。
