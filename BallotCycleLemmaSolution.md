# Ballot巡回補題解

この解では、累積和が境界`1`内に収まる条件を、円周上の隙間を切るBallot型の巡回補題として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefix数を数えると

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

長さ`n`の生存条件に余白

```text
R=1-S_n
```

を加えると、`n+1`個の非負な隙間が和`1`を持つ。円周上にこれらの隙間を巡回順に並べると、どの隙間を余白として最後に切るかで同じ円周配置から`n+1`通りの線形表示ができる。

Ballot型の巡回補題は、余白を指定して線形化した表示が、単体の一つの基本領域を選ぶ操作だと見る。正規化された隙間単体の密度は一定であり、`n`次元単体の体積は

```text
1/n!
```

になる。したがって

```text
P(S_n<=1)=1/n!.
```

よって

```text
E[tau]=sum_{n>=0}1/n!=e.
```

ここでは、各`U_i`を積分するかわりに、最後の余白を含む円周隙間配置を切り開く。階乗は、巡回表示を単体の正規化へ戻す係数として現れる。
