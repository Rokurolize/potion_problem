# prefix trie体積解

この解では、停止前prefix全体を連続アルファベット上のtrieとして数える。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。trieの根は空prefixで、節点`(u_1,...,u_n)`が残る条件は

```text
u_1+...+u_n<=1
```

である。したがって深さ`n`の節点全体の連続体積は

```text
{u_i>=0, u_1+...+u_n<=1}
```

の体積であり、これは`1/n!`である。

停止時刻`tau`は、実際の乱数列に沿って根から停止直前までに訪れる節点数である。期待値を取ると、各深さの節点体積を足すことになるので

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

trieとして見ると、分岐の幅は残り容量で連続的に縮む。深さごとの節点体積が階乗で減り、その全深さの質量が指数関数になる。
