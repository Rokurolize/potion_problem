# 陰計算評価汎関数解

この解では、単体体積の列を陰計算の評価汎関数で一度に読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。標準的な停止時刻恒等式より

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

`P(S_n<=x)`は、`n`次元単体

```text
u_1+...+u_n <= x,  0<=u_i<=1
```

の体積である。`0<=x<=1`では上限`u_i<=1`は効かないので

```text
P(S_n<=x)=x^n/n!.
```

ここで評価汎関数`L`を

```text
L(x^n)=1/n!
```

と定めると、境界`1`での総prefix数は

```text
E[tau]=sum_{n>=0} L(1^n).
```

これは指数型母関数の評価そのもので、

```text
sum_{n>=0} 1/n! = e.
```

したがって

```text
E[tau]=e.
```

陰計算の言葉では、単体体積を各次数で積分する代わりに、単項式列を階乗で割る評価汎関数へ送っている。停止時刻の期待値は、その評価汎関数が幾何級数型のprefix列を指数級数へ変える値である。
