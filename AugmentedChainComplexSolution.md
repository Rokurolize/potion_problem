# augmented chain complex解

この解では、停止前prefixを標準単体の増大する鎖として数える。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。`tau>n`は`S_n<=1`と同値なので、

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

固定した`n`について、停止前prefix全体は

```text
Delta_n={u_i>=0, u_1+...+u_n<=1}
```

である。これは標準`n`単体であり、その体積は拡大鎖複体の正規化で

```text
vol(Delta_n)=1/n!
```

となる。具体的には、境界演算で一段落としたときに現れるfacetの体積が前段の単体体積で、`vol(Delta_0)=1`から帰納的に決まる。

したがって

```text
E[tau]=sum_{n>=0} vol(Delta_n)
      =sum_{n>=0} 1/n!
      =e.
```

鎖複体として見ると、停止時刻は一つの次数を選ぶ量ではなく、全次数の停止前単体の体積を足す量である。
