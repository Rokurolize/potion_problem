# 軌道商階乗解

この解では、停止前prefixを立方体の点配置の軌道商として数え、階乗を軌道の大きさとして読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。`t_k=S_k`と置くと、`S_n<=1`は順序付き点列

```text
0 <= t_1 <= ... <= t_n <= 1
```

の領域であり、変換のJacobianは`1`である。

区間内のラベル付き`n`点は立方体`[0,1]^n`で表される。対角を除けば、対称群`S_n`の各軌道は`n!`個のラベル付けを持つ。停止前prefixは、その軌道から昇順の代表を一つだけ選んだものなので、ラベル付き立方体の全体積`1`を軌道サイズで割ればよい。

```text
P(S_n <= 1)=1/n!.
```

空prefixを含めて全ての長さを足すと

```text
E[tau]=sum_{n>=0} P(S_n <= 1)
      =sum_{n>=0} 1/n!
      =e.
```

この見方では、`1/n!`は積分の答えではなく、ラベル付き配置から順序代表へ降りる軌道商の係数である。
