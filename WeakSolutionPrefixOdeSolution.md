# weak solution prefix ODE解

この解では、停止前prefix和を弱解として同定する。

容量`x`での停止前prefix期待数を

```text
F(x)=sum_{n>=0}P(U_1+...+U_n<=x)
```

とする。`F(1)`が元の`E[tau]`である。各項について

```text
P(U_1+...+U_n<=x)=int_0^x P(U_1+...+U_{n-1}<=x-u)du
```

だから、分布の意味で

```text
F'=F
```

が成り立つ。境界では空prefixだけが残り

```text
F(0)=1.
```

弱解の一意性により、この方程式の解は

```text
F(x)=e^x.
```

したがって

```text
E[tau]=F(1)=e.
```

滑らかさを仮定して各単体を積分するのではなく、尾和全体を一つの弱解として閉じるのがこの見方である。
