# Bell多項式分割解

この解では、停止前prefix全体を分割多項式としてまとめ、非相互作用のためBell多項式が指数へ潰れることを見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めて

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

累積時刻`0<=t_1<=...<=t_n<=1`へ移すと、長さ`n`のprefixは区間`[0,1]`上の順序付き配置である。配置を集合分割の指数公式で見ると、相互作用のない一点ブロックだけが連結成分として残る。

したがってBell多項式で書く全配置分配関数は

```text
Z(z)=exp(z int_0^1 dt)=exp(z).
```

欲しい値は活動度`z=1`で

```text
E[tau]=Z(1)=e.
```

この証明では、各`n`の単体体積を先に計算しない。配置分割のBell多項式が一点ブロックだけで指数化することを見る。`e`は、その分割多項式の値である。
