# wavelet低域prefix解

この解では、prefix総量の境界方向の低域成分だけを取り出す。高周波の細かな分割は、境界`1`までの総量には寄与しない。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`までの総量

```text
F(x)=sum_{n>=0}P(S_n<=x)
```

を見る。境界を微小に動かしたときの増分は、局所密度の平均、つまり低域成分だけで決まる。一様分布ではこの低域成分が定数`1`であり、wavelet的な詳細成分は境界層の主項に現れない。

したがって

```text
F(x+dx)-F(x)=F(x)dx+o(dx),
```

すなわち

```text
F'(x)=F(x),  F(0)=1.
```

解は

```text
F(x)=e^x
```

であり、

```text
E[tau]=F(1)=e.
```

この証明では、分布全体を細かく復元しない。必要なのは境界流の低域成分だけで、それが定数であることが指数関数を強制する。
