# 随伴テスト関数で消す解

この解では、生存確率列そのものを解かない。かわりに、renewal方程式にちょうど合うテスト関数を当て、期待値を一発で抜き出す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

`S_0=0`も含めれば、停止時刻は境界`1`以下にあるprefixの個数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって、部分和全体のrenewal測度

```text
R = sum_{n >= 0} law(S_n)
```

を使うと

```text
E[tau] = R([0,1]).
```

## renewal方程式

一様分布の測度を`mu(du)=1_{0<=u<=1}du`と書く。すると

```text
R = delta_0 + R * mu.
```

これを区間`[0,1]`上だけで使う。`x<=1`に入るには、直前のrenewal点`y`が`0<=y<=x`にあり、次の増分が`x-y`までであればよい。

分布関数

```text
F(x) = R([0,x])
```

は、`0<=x<=1`で

```text
F(x) = 1 + int_0^x F(t) dt
```

を満たす。ここまではrenewal方程式を積分形で書いただけである。

## 解かずに当てる

普通はここから微分して`F'=F`を解く。ここでは逆に、未知の`F`に対してテスト関数

```text
phi(x) = exp(1-x)
```

を当てる。

`phi'(x)=-phi(x)`であり、`phi(1)=1`である。積分形のrenewal方程式を微分の弱形式として見ると、`0`にある空prefixの原子を含めて

```text
dF = delta_0 + F(x) dx
```

である。そこで`phi`を掛けて`[0,1]`で積分すると

```text
int_0^1 phi(x) dF(x)
= phi(0) + int_0^1 phi(x) F(x) dx.
```

左辺を部分積分する。`F`は右連続な増加関数で、`0`に原子`1`を持つので、

```text
int_0^1 phi dF
= phi(1)F(1) - phi(0)F(0-) - int_0^1 F(x) phi'(x) dx.
```

ここで`F(0-)=0`、`phi(1)=1`、`phi'=-phi`だから

```text
int_0^1 phi dF
= F(1) + int_0^1 F(x) phi(x) dx.
```

これが右辺

```text
phi(0) + int_0^1 phi(x) F(x) dx
```

に等しいので、両辺の積分項が消えて

```text
F(1) = phi(0) = e.
```

すなわち

```text
E[tau] = R([0,1]) = F(1) = e.
```

## この見方の芯

この証明は、`F(x)=e^x`を先に求めない。renewal方程式の随伴側で

```text
phi'=-phi,  phi(1)=1
```

を満たす関数を選ぶと、内部の積分が完全に相殺され、初期境界の値だけが残る。

答えの`e`は、未知のrenewal質量を解いた値ではなく、終端値`phi(1)=1`から境界`0`へ戻した随伴テスト関数の値`phi(0)`として現れる。
