# marked species integral解

この解では、停止前prefixを、容量`1`の区間に順序付きの印を置く種として扱う。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前の長さ`n` prefixは、部分和

```text
t_k=S_k
```

を使うと

```text
0<=t_1<=...<=t_n<=1
```

という「区間上の順序付き印」になる。増分から部分和への変換は下三角で行列式`1`なので、重みは変わらない。

順序付き印を一度ラベルなしの`n`点集合として見ると、`[0,1]^n`の点配置を`n!`通りの順序に分けた一室が上の領域である。したがってその重みは

```text
1/n!.
```

空印を含めて全ての停止前prefixを足すと

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

この見方の利点は、停止前prefixを「列」ではなく「区間上の印付き構造」として数える点にある。指数関数は計算の結果ではなく、有限集合種の総指数母関数として出てくる。
