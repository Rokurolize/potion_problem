# weighted tree partition解

この解では、停止前履歴を深さ付きの連続分岐木の分配関数として読む。

容量`x`の節点からは、次の残り容量`y`を`0<=y<=x`で連続的に選んで子へ進む。深さ`n`まで進める枝の総重みを`Z_n(x)`とすると

```text
Z_0(x)=1,
Z_{n+1}(x)=int_0^x Z_n(y)dy.
```

これは、長さ`n+1`の停止前prefixを最後の残り容量で分解している。従って

```text
Z_n(x)=x^n/n!.
```

停止時刻`tau`は、容量`1`から見て存在する停止前prefixの深さ方向の総数であるから

```text
E[tau]=sum_{n>=0}Z_n(1).
```

よって

```text
E[tau]=sum_{n>=0}1/n!=e.
```

この証明では、停止前prefixを連続分岐木の分配関数にまとめ、深さ方向の全重みとして`e`を得る。
