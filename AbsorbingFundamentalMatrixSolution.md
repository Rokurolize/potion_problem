# 吸収基本行列解

この解では、連続状態の吸収過程を細かい格子で近似し、基本行列の極限を取る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。区間`[0,1]`を幅`h`の格子に分け、`1`を超えたら吸収される有限Markov連鎖で近似する。非吸収状態間の遷移行列を`Q_h`とすると、吸収前の期待訪問数は基本行列

```text
N_h = I + Q_h + Q_h^2 + ...
```

で与えられる。初期点`0`から全非吸収状態への総期待訪問数は、格子近似の期待停止時刻である。

格子幅を`h -> 0`とすると、`Q_h`の和は積分作用素

```text
(Tf)(s)=int_0^{1-s} f(s+u) du
```

へ収束する。したがって基本行列の極限`N=(I-T)^{-1}`について

```text
G(s)=(N1)(s)
```

は

```text
G(s)=1+int_0^{1-s} G(s+u) du
```

を満たす。`F(x)=G(1-x)`と置けば

```text
F(x)=1+int_0^x F(y) dy.
```

ゆえに

```text
F'(x)=F(x),  F(0)=1,
F(1)=e.
```

初期状態は`s=0`なので

```text
E[tau]=G(0)=F(1)=e.
```

有限吸収連鎖の基本行列というよく知られた公式を、この停止時刻へ連続極限として持ち込むだけで答えが出る。
