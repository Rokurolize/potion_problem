# Poisson順序統計量解

この解では、一様乱数の累積和が作る単体を、Poisson過程の到着時刻の条件付き分布として読む。

独立一様乱数`U_1,U_2,...`に対し、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止時刻は、まだ境界`1`を越えていないprefixの個数で数えられる。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

よって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 生存prefixをPoisson時計へ移す

標準Poisson過程`N(t)`を考える。`N(1)=n`で条件づけると、区間`[0,1]`内の到着時刻

```text
0 < T_1 < ... < T_n < 1
```

は、`n`個の独立一様点を小さい順に並べた順序統計量と同じ分布を持つ。したがってその同時密度は単体

```text
0 < t_1 < ... < t_n < 1
```

上で定数`n!`である。

一方、元の問題で`S_n <= 1`となる条件は、変数変換

```text
t_i = S_i
```

により

```text
0 < t_1 < ... < t_n <= 1
```

という同じ単体になる。Jacobianは`1`なので、

```text
P(S_n <= 1)
= volume { 0 < t_1 < ... < t_n <= 1 }.
```

この体積は、Poisson時計の条件付き到着時刻密度を積分して`1`になる事実から即座に読める。

```text
n! * volume { 0 < t_1 < ... < t_n <= 1 } = 1.
```

したがって

```text
P(S_n <= 1) = 1 / n!.
```

## 期待値

これを生存prefixの和に戻すと、

```text
E[tau]
= sum_{n >= 0} P(S_n <= 1)
= sum_{n >= 0} 1 / n!
= e.
```

## この読み替えの要点

`Poisson(1)`の確率質量を直接使うのではなく、条件付き到着時刻の配置を使う。Poisson過程で「`n`回起きた後、その`n`個がどこに置かれるか」を見ると、元の累積和の生存条件がそのまま順序統計量の単体になる。

つまり答えの`e`は、Poisson分布の公式を借りてきたものではなく、「一様点を並べると単体密度が`n!`になる」という時計の対称性から出る。
