# 転送半群境界解

この解では、境界容量を動かすときの停止前prefix総量を、転送半群のスカラー観測量として読む。

容量`r`までで生きている全prefixの総質量を

```text
M(r) = sum_{n >= 0} P(S_n <= r)
```

と置く。求めたい値は`M(1)`である。空prefixがあるので

```text
M(0)=1
```

である。

## 境界を進める転送

境界を`r`から`r+h`へ動かす。新しく入るprefixは、古いprefixの末尾に一口を足して、終点が薄い層

```text
(r, r+h]
```

へ入ったものである。

古いprefixの終点を`y <= r`とする。このprefixから薄い層へ送れる一口の長さは

```text
r-y < u <= r+h-y
```

であり、`h`が十分小さければこの区間の長さは`h`である。一様密度は`1`なので、各既存prefixは薄い境界層へ質量`h + o(h)`を送る。

したがって

```text
M(r+h) - M(r) = h M(r) + o(h).
```

これは、境界方向の転送半群が全prefix総量という観測量に対して生成子`1`として働く、という式である。極限を取れば

```text
M'(r)=M(r).
```

初期条件`M(0)=1`から

```text
M(r)=exp(r).
```

よって

```text
E[tau]=M(1)=e.
```

この証明では、各段の体積を計算しない。境界を少し進める転送作用を見て、全prefix総量という観測量がその作用で自己増殖率`1`を持つことだけを使う。
