# Poisson到着逆像解

この解では、一様乱数の和をそのまま足す代わりに、区間内のPoisson到着列を「逆向き」に見て停止前prefixを数える。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 到着時刻から増分へ戻す

強度`1`のPoisson過程を区間`[0,1]`で見る。点が`n`個あり、その到着時刻が

```text
0 <= T_1 <= ... <= T_n <= 1
```

であるとする。この到着列から

```text
U_1 = T_1,
U_2 = T_2 - T_1,
...
U_n = T_n - T_{n-1}
```

と戻せば、部分和は

```text
S_k = T_k
```

になる。したがって、`n`個の到着列はそのまま`S_n<=1`である長さ`n`の停止前prefixに対応する。変換のJacobianは三角行列で対角が`1`なので、体積を変えない。

## Poisson確率から空白因子を外す

Poisson過程で、`[0,1]`に点が`n`個あり、順序付き到着時刻が上の微小領域に入る確率密度は

```text
exp(-1) dT_1 ... dT_n
```

である。これは、到着時刻の体積に、区間全体に対する空白因子`exp(-1)`が掛かったものだ。

一方、元の停止前prefixの測度では、この空白因子は付かない。つまり、深さ`n`の停止前prefix全体の体積は、Poisson到着列の確率質量から共通因子`exp(-1)`を外したものになっている。

全ての`n`を足すと、Poisson過程の全配置確率の和は`1`だから

```text
exp(-1) * sum_{n >= 0} P(S_n <= 1) = 1.
```

したがって

```text
E[tau]
= sum_{n >= 0} P(S_n <= 1)
= exp(1)
= e.
```

この証明では、`P(S_n<=1)=1/n!`を個別に計算しない。Poisson到着列から一様増分列へ逆に戻すことで、停止前prefix全体が「Poisson配置確率から空白因子だけを外した総質量」だと分かる。
