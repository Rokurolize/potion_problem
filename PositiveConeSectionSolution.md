# 正錐断面解

待ち時間を `X_1, X_2, ...` とする。各 `X_i` は `[0, 1]` 上の一様分布だが、`X_1 + ... + X_n <= 1` を調べる範囲では密度は1なので、問題は正錐の体積計算に落ちる。

`n` 次元正錐

```text
C_n = {x in R^n | x_i >= 0}
```

に高さ関数

```text
h(x) = x_1 + ... + x_n
```

を入れる。すると

```text
P(X_1 + ... + X_n <= 1)
```

は、正錐のうち `h(x) <= 1` で切った部分

```text
K_n = {x_i >= 0, h(x) <= 1}
```

の体積である。

この体積を直接測る代わりに、正錐全体のラプラス体積を測る。

```text
int_{C_n} exp(-h(x)) dx
  = prod_{i=1}^n int_0^infty exp(-x_i) dx_i
  = 1
```

一方、`h` のレベルで分解すると、`h(x) = s` の断面の余次元1体積密度は `s^{n-1}/(n-1)!` でなければならない。なぜなら

```text
int_0^infty exp(-s) s^{n-1}/(n-1)! ds = 1
```

だからである。

したがって高さ1以下の部分の体積は

```text
vol(K_n)
  = int_0^1 s^{n-1}/(n-1)! ds
  = 1/n!
```

となる。`n = 0` では空和が0なので寄与は1である。

到着数を

```text
tau = sum_{n >= 0} 1_{X_1 + ... + X_n <= 1}
```

と書けば、期待値は

```text
E[tau]
  = sum_{n >= 0} P(X_1 + ... + X_n <= 1)
  = sum_{n >= 0} 1/n!
  = e
```

正錐を高さで切った断面密度がガンマ積分そのものになり、その累積体積が各項 `1/n!` を与える。停止時刻の分布を作らず、錐のレベル断面だけで答えが出る。
