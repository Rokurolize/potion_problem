# cardinal B-spline先頭片解

この解では、`S_n = U_1 + ... + U_n`の分布を全体として求めない。畳み込み密度がcardinal B-splineになることだけを使い、境界`1`までに見える最初の多項式片だけを読む。

独立一様乱数`U_1,U_2,...`について

```text
tau = min { n >= 1 : S_n > 1 }
```

とする。正整数値の停止時刻なので

```text
E[tau] = sum_{n >= 0} P(tau > n)
       = sum_{n >= 0} P(S_n <= 1).
```

## 畳み込み密度の先頭片

`n >= 1`について、`S_n`の密度は区間`[0,n]`に台を持つcardinal B-splineである。これは一様密度

```text
1_{[0,1]}(x)
```

を`n`回畳み込んだものである。

ただし、欲しいのは密度全体ではない。`P(S_n <= 1)`では`0 <= x <= 1`しか見ない。この範囲では、一様密度の右端`1`による折り返しや有限差分の後続項はまだ現れない。したがって畳み込みは、半直線上の定数密度`1_{x>=0}`を`n`回積分した先頭片と同じである。

そこで密度の先頭片を`b_n(x)`と書くと

```text
b_1(x) = 1              (0 <= x <= 1),
b_{n+1}(x) = int_0^x b_n(y) dy.
```

よって帰納的に

```text
b_n(x) = x^{n-1}/(n-1)!  (0 <= x <= 1).
```

したがって

```text
P(S_n <= 1)
  = int_0^1 b_n(x) dx
  = 1/n!.
```

空prefixの項は`P(S_0 <= 1)=1`であり、これは`1/0!`である。

## 期待値

以上から

```text
E[tau]
  = sum_{n >= 0} 1/n!
  = e.
```

この証明では、Irwin-Hall分布の全区間の折れ線多項式を作らない。境界がちょうど`1`であるため、cardinal B-splineの先頭片だけが必要になる。`e`は、先頭片を作る「一回積分」の反復が階乗を作り、その全次数を足すことで現れる。
