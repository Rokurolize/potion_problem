# Magnus崩壊解

この解では、停止前prefixの全和を時間順序指数として見たうえで、生成子が可換なためMagnus展開が最初の項だけに崩れることを使う。

容量を`r`まで動かしたときの全prefix質量を`M(r)`とする。空prefixを含めるので

```text
M(0) = 1
```

である。

境界を`r`から`r+dr`へ少し動かす。新しく入るprefixは、既存のprefixの末尾に、幅`dr`の一口を足したものとして数えられる。一様密度は`1`なので、新しい質量は

```text
dM = M(r) dr
```

である。つまり

```text
dM/dr = A(r) M(r),   A(r)=1.
```

これは時間依存線形方程式だが、ここでは生成子`A(r)`がただのスカラー`1`である。

## Magnus展開

一般の時間依存生成子なら、解は

```text
M(r) = exp(Omega(r)) M(0)
```

と書け、`Omega`はMagnus展開

```text
Omega(r)
  = int_0^r A(s) ds
    - 1/2 int_{0<s2<s1<r} [A(s1), A(s2)] ds2 ds1
    + ...
```

で与えられる。

しかし今は全ての`A(s)`が同じスカラー`1`なので、交換子は全部

```text
[A(s1), A(s2)] = 0
```

で消える。したがって

```text
Omega(r) = int_0^r 1 ds = r.
```

よって

```text
M(r) = exp(r).
```

求める値は

```text
E[tau] = M(1) = e.
```

この証明では、Dyson級数の各項を最後まで展開しない。停止前prefixを境界方向の線形発展としてまとめ、さらに生成子が可換なため時間順序の補正が全て消えることだけを見る。`e`はMagnus展開が一次項に崩壊した指数として現れる。
