# 停止線分解の相殺解

この解では、停止前prefixの総量を直接解かず、各prefixから出る「生存する枝」と「停止する枝」を同じ恒等式で分解して相殺する。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。`A_n={S_n<=1}`と書けば

```text
tau = sum_{n >= 0} 1_{A_n}
```

なので

```text
E[tau] = sum_{n >= 0} P(A_n).
```

## ひとつのprefixに証明書を置く

停止前prefixの位置を`s<=1`とし、残り距離を

```text
r = 1 - s
```

と書く。このprefixに

```text
Phi(r) = exp(r)
```

という重みを置く。

次の一口`u`が`0<=u<=r`なら、まだ生存して次のprefixへ進む。`u>r`なら、そこで停止線を越える。ここで使う恒等式は

```text
Phi(r)
= 1 + int_0^r Phi(r-u) du.        (1)
```

実際、

```text
int_0^r exp(r-u) du = exp(r) - 1
```

だからである。

(1)は、prefix一個の重みを「そのprefix自身を数える`1`」と「生存して次へ渡る全ての枝の重み」に分けている。停止する枝は右辺に残らない。境界を越えた先では以後のprefixが存在しないからである。

## 全ての深さで足して相殺する

深さ`n`でまだ生きているprefix全体について、`Phi(1-S_n)`を足した量を

```text
B_n = E[ Phi(1-S_n) 1_{A_n} ]
```

と置く。深さ`n`の各prefixに(1)を適用して、全てのprefix測度で積分すると

```text
B_n = P(A_n) + B_{n+1}.        (2)
```

右辺の`B_{n+1}`は、深さ`n`のprefixから生存枝`0<=u<=1-S_n`だけを伸ばした全質量である。これはちょうど、深さ`n+1`でまだ生きているprefix全体と同じものになる。

したがって(2)を`n=0`から`N-1`まで足すと、中間の`B_1,...,B_{N-1}`が全部消えて

```text
sum_{n=0}^{N-1} P(A_n) = B_0 - B_N.
```

初期状態では`S_0=0`なので

```text
B_0 = Phi(1) = e.
```

また`0<=B_N<=e P(A_N)`であり、左辺の部分和は`e`以下に抑えられる。従って`P(A_N)->0`で、`B_N->0`である。

よって`N`を無限大へ送れば

```text
E[tau]
= sum_{n>=0} P(A_n)
= e.
```

## この見方の芯

この証明では、`P(S_n<=1)`も`P(tau=n)`も計算しない。停止前prefixの木に指数重みを置き、各節点で

```text
親の重み = その節点を数える1 + 生存枝へ渡す重み
```

という局所分解をするだけである。

全深さで足すと、生存枝へ渡した重みは次の深さの親重みとして完全に相殺される。最後に残るのは初期重み`exp(1)`だけで、それが平均停止前prefix数になる。
