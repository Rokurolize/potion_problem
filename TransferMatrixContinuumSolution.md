# transfer matrix continuum解

この解では、有限状態の転送行列を連続容量へ極限した作用素として扱う。

残り容量`x`から一手進む続行作用素を

```text
(Tf)(x)=int_0^x f(y)dy
```

とする。これは、入力を一つ読んで停止しなかったときの残り容量`y`を足し上げる連続版の転送行列である。

長さ`n`まで停止しないprefixの質量は

```text
T^n1(1)
```

である。転送作用素を反復すると

```text
T^n1(x)=int_{0<=y_n<=...<=y_1<=x}dy_n...dy_1=x^n/n!.
```

したがって尾和公式より

```text
E[tau]=sum_{n>=0}T^n1(1)=sum_{n>=0}1/n!=e.
```

ここでは、問題を連続状態の転送行列の全経路和として解いている。
