# エントロピー障壁体積解

この解では、生存単体の体積を、正の錐に入れたエントロピー型障壁の正規化から読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。期待値は

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

長さ`n`について、条件`S_n<=1`は正の錐

```text
u_i>=0
```

を半径

```text
r=u_1+...+u_n
```

で`r<=1`に切ったものである。割合`p_i=u_i/r`に移ると、エントロピー障壁

```text
-sum_i log p_i
```

を持つ標準単体の一様正規化が現れる。

指数重み付きの正の錐全体では

```text
int_{u_i>=0} exp(-(u_1+...+u_n)) du_1...du_n = 1.
```

これを半径と割合に分けると

```text
1 = A_n int_0^infty exp(-r) r^{n-1} dr
  = A_n (n-1)!.
```

したがって割合断面の正規化は`A_n=1/(n-1)!`であり、`r<=1`まで積むと

```text
P(S_n<=1)=1/n!.
```

よって

```text
E[tau]=sum_{n>=0}1/n!=e.
```

この証明では、正錐のエントロピー障壁が作る半径・割合分解で、単体体積の階乗を正規化定数として取り出している。
