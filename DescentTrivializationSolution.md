# descent自明化解

この解では、停止前prefixの局所データが境界`1`まで自明に貼り合わさることを使う。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`以下のprefix総量を

```text
F(x)=sum_{n>=0} P(S_n <= x)
```

と置く。境界を薄い区間`[x,x+dx]`で覆うと、そこへ新しく入るprefixは既存prefixに一口を足したものだけである。一様密度が定数`1`なので、この局所貼り合わせの遷移関数は常に同じで、

```text
F(x+dx)-F(x)=F(x)dx+o(dx).
```

つまりdescentデータは非自明なねじれを持たず、

```text
F'(x)=F(x),  F(0)=1
```

に自明化する。よって

```text
F(x)=e^x,
E[tau]=F(1)=e.
```

この証明では、全区間の分布を先に作らない。小区間ごとのprefix生成則が同じで、貼り合わせに補正が出ないため、指数関数がそのまま大域解になる。
