# キュムラント母関数解

この解では、停止前prefixの個数を活動度で重みづけし、そのキュムラント母関数を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めれば

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

境界`x`までのprefixに活動度`z`を付けて

```text
Z(x,z)=sum_{n>=0} z^n P(S_n<=x)
```

を置く。既存prefixが薄い境界層へ次のprefixを生む幅は、一様密度`1`により`dx`である。活動度は一段増えるたびに`z`を掛けるので

```text
partial_x Z(x,z)=z Z(x,z),
Z(0,z)=1.
```

キュムラント母関数を

```text
K(x,z)=log Z(x,z)
```

とすると

```text
partial_x K(x,z)=z,
K(0,z)=0.
```

したがって

```text
K(x,z)=zx,
Z(x,z)=exp(zx).
```

欲しい期待値は、活動度で重み付けた停止前prefix数を`z=1`で読むことなので

```text
E[tau]=Z(1,1)=e.
```

この証明では、各`P(S_n<=1)`を先に出さない。活動度つきprefix分配関数のキュムラントが、境界方向に線形に伸びることだけを見る。`e`は、そのキュムラント母関数を指数へ戻した値である。
