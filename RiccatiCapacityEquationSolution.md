# Riccati capacity equation解

この解では、残容量`c`に対する生存容量関数の微分方程式を使う。

長さ`n`まで停止しない容量関数を

```text
u_n(c)=P(X_1+...+X_n<=c)
```

と置く。`0<=c<=1`では

```text
u_0(c)=1,
u_n'(c)=u_{n-1}(c),
u_n(0)=0  (n>=1)
```

が成り立つ。これは階段状のRiccati型容量方程式の線形化と見なせる。

この再帰的微分方程式を解くと

```text
u_n(c)=c^n/n!
```

である。元の問題では`c=1`だから

```text
P(tau>n)=u_n(1)=1/n!.
```

尾和を取れば

```text
E[tau]=sum_{n>=0}1/n!=e.
```

残容量に対する微分方程式として書くと、階乗分母は積分を一段ずつ重ねることから出る。
