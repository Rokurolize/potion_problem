# Bernstein作用素極限解

この解では、連続な容量問題をBernstein型の有限状態作用素で近似し、その極限から`e`を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めて

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

境界`[0,1]`を`N`分割し、容量`k/N`での停止前prefix総量を`B_k`とする。有限状態では、一口追加の作用素を「下側の格子点を平均して一段進む」Bernstein型の正作用素で近似する。境界を一格子だけ動かすと、各既存prefixは幅`1/N`で新しいprefixを作るので

```text
B_{k+1}=B_k+(1/N)B_k+o(1/N)
```

となる。したがって

```text
B_N -> (1+1/N)^N
```

の形で連続極限に近づく。極限では

```text
lim_{N->infty} B_N = e.
```

これは連続方程式

```text
F'(x)=F(x),
F(0)=1
```

のBernstein作用素近似である。初期容量`1`での値が求める期待値なので

```text
E[tau]=e.
```

この証明では、各次元の単体体積を先に出さない。容量方向を有限状態のBernstein型正作用素で近似し、その極限が指数関数の境界値へ収束することを見る。`e`は、この正作用素極限の値である。
