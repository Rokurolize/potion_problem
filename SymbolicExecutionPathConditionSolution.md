# symbolic execution path condition解

この解では、停止しない実行を記号実行のpath conditionとして数える。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。長さ`n`まで停止しない実行のpath conditionは、各`u_i`が`[0,1]`にあることに加えて

```text
u_1+...+u_n<=1
```

である。この条件が充足可能であるだけでなく、確率質量としてどれだけあるかを見ると、対応する領域は`n`次元単体であり体積は

```text
1/n!
```

である。

停止時刻は「充足可能な停止前pathを何段読めるか」の和として

```text
tau = sum_{n>=0} 1_{S_n<=1}
```

と表される。よって

```text
E[tau] = sum_{n>=0} P(S_n<=1)
       = sum_{n>=0} 1/n!
       = e.
```

記号実行として見ると、各深さのpath conditionの体積を足し上げるだけで指数級数が現れる。
