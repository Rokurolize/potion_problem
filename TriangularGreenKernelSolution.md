# 三角Green核解

この解では、停止前prefixを段数ごとに計算せず、区間`[0,1]`上の三角積分作用素のGreen核としてまとめて読む。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 一歩進む作用素

区間`[0,1]`上の関数に

```text
(Kf)(x) = int_0^x f(y) dy
```

を作用させる。これは、終点が`y`にある停止前prefixに一様な一口を足して、終点`x`以下へ送る作用素である。

空prefixは原点に一つある。そこから`n`回一口を足してなお`x`以下にいる総量は

```text
K^n 1 (x)
```

で表せる。実際、`K`を一回掛けるたびに、最後の一口をどこから来たかで積分しているだけである。

したがって、境界`x`までに存在する全停止前prefixの総量は

```text
G(x) = sum_{n >= 0} K^n 1 (x).
```

求める値は

```text
E[tau] = G(1)
```

である。

## Green方程式

上の級数は形式的には

```text
G = 1 + K G
```

を満たす。ここで`K`は上三角ではなく、過去`y<=x`だけを見るVolterra型の作用素なので、この方程式は初期値だけで一意に決まる。

式を`x`で書くと

```text
G(x) = 1 + int_0^x G(y) dy.
```

よって

```text
G'(x) = G(x),
G(0) = 1.
```

したがって

```text
G(x) = exp(x).
```

結局

```text
E[tau] = G(1) = e.
```

この証明の芯は、停止前prefixの全段を`1 + K + K^2 + ...`というGreen核にまとめる点にある。`K`が三角作用素なので、逆作用素`(I-K)^{-1}`の核は指数関数になり、境界`1`での総質量がそのまま`e`として現れる。
