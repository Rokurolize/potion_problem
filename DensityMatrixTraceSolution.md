# 密度行列trace解

この解では、停止前prefixの全次数を対角密度行列として束ね、そのtraceを読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。長さ`n`の生存領域の質量を

```text
a_n=P(S_n<=1)
```

と置く。累積時刻へ変換すれば生存領域は順序室

```text
0<=t_1<=...<=t_n<=1
```

であり、体積は`1/n!`である。

形式的に、全次数の対角作用素

```text
rho = diag(a_0,a_1,a_2,...)
```

を作ると、停止前prefix総量はそのtraceである。

```text
Tr(rho)=sum_{n>=0}a_n=sum_{n>=0}1/n!=e.
```

```text
E[tau]=e.
```

この見方では、各次数を別々に眺めず、全次数の対角成分をtraceで一括して足す。
