# Green核全質量解

この解では、停止までに訪れる累積和の期待測度をGreen核として見て、その全質量を求める。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

まだ境界`1`を越えていない累積和だけを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって`E[tau]`は、殺される前の累積和過程が区間`[0,1]`に置く期待訪問回数である。

## Green測度

`S_0=0`を含めて、期待訪問測度`G`を

```text
G(B) = E[sum_{n >= 0} 1_{S_n in B, S_n <= 1}]
```

で定める。求める期待値は

```text
E[tau] = G([0,1]).
```

`G`は、点`0`の原子`delta_0`と、1回以上ジャンプした後の密度部分を持つ。密度を`g(x)`と書くと、`0 < x <= 1`で

```text
g(x) = 1 + int_0^x g(y) dy.
```

理由は次の通りである。位置`x`に初めて到達する1歩前の位置を`y`とすると、`0 <= y < x`でなければならない。ジャンプ幅`x-y`は`[0,1]`上一様なので、密度は`1`である。さらに、`0`から直接`x`へ飛ぶ寄与が`1`ある。

## Green核を閉じる

上の式を微分すると

```text
g'(x) = g(x),
g(0+) = 1.
```

したがって

```text
g(x) = exp(x).
```

Green測度の全質量は、原子`0`の質量`1`と密度部分の質量を足したものなので

```text
G([0,1])
= 1 + int_0^1 exp(x) dx
= e.
```

よって

```text
E[tau] = e.
```

## 見方

この証明は、停止時刻の分布も、各`P(S_n<=1)`も先に求めない。殺される前の累積和過程がどれだけ区間内を訪れるかを期待測度としてまとめ、そのGreen核の全質量を読む。

一様性は遷移密度を`1`にし、独立性は訪問測度が自分自身を同じ核で再生することに現れる。
