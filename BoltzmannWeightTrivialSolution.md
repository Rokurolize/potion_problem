# Boltzmann重み自明解

この解では、停止前prefixの各状態にBoltzmann重みを付けるが、この問題では全ての重みが`1`に退化することを見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。prefix状態のエネルギーを

```text
E(state)=0
```

と置けば、温度`1`でのBoltzmann重みは

```text
exp(-E(state))=1.
```

したがって境界`x`以下の全状態の分配関数は

```text
Z(x)=sum_{n>=0}P(S_n<=x).
```

一様密度により、境界を動かしたときの増分は

```text
Z'(x)=Z(x),  Z(0)=1.
```

ゆえに

```text
Z(x)=e^x.
```

求める期待停止時刻は、空prefixを含む生存状態の総Boltzmann重みなので

```text
E[tau]=Z(1)=e.
```

この見方では、Boltzmann重みを導入しても非自明なエネルギーは現れない。重みが全て`1`へ潰れるため、残るのは状態数の指数生成だけである。
