# Riesz representer capacity解

この解では、容量`1`での期待prefix数を、評価汎関数のRiesz表現として読む。

容量`x`から始めた期待prefix数を

```text
F(x)=sum_{n>=0}P(U_1+...+U_n<=x)
```

とおく。元の答えは`F(1)`である。最後の入力で切ると、各生存確率列は微分で一段前へずれるので

```text
F'(x)=F(x),  F(0)=1.
```

したがって、任意の滑らかな試験関数`phi`について

```text
int_0^1 (F'-F) phi dx = 0
```

であり、境界値`F(0)=1`を持つ弱解として一意に決まる。評価汎関数`f -> f(1)`のこの解上での値は、Riesz的に言えば方程式のGreen表現で端点に現れる代表元の値である。

初期値問題を解けば

```text
F(x)=e^x.
```

よって

```text
E[tau]=F(1)=e.
```

ここでは停止前prefix和を関数空間上の一意な表現元として捉え、端点評価から答えを読む。
