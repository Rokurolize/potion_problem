# Riordan配列prefix解

この解では、生存prefix列をRiordan配列の一列として読み、列和で期待値を取り出す。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めると

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

`a_n(x)=P(S_n<=x)`と置く。`0<=x<=1`では最後の一口を外せるので

```text
a_0(x)=1,
a_n'(x)=a_{n-1}(x).
```

これは、列を一つ下げるRiordan型の生成規則である。列和

```text
R(x)=sum_{n>=0} a_n(x)
```

を取ると

```text
R'(x)=R(x),
R(0)=1.
```

したがって

```text
R(x)=exp(x),
E[tau]=R(1)=e.
```

この証明では、各`a_n(1)`を先に計算しない。生存prefix列がRiordan配列の基本列として一段シフトし、その列和が指数関数になることだけを見る。`e`は、その列和の境界値である。
