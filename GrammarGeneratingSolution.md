# 文法生成関数解

この解では、停止前prefixを生成する文法を作り、その生成関数を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。非終端記号`A_x`を「残り境界`x`以内でまだ生成できるprefix」とする。空語は常に生成でき、さらに`0<=u<=x`の一口を生成すると残りは`x-u`になる。総生成量を`F(x)`と書けば

```text
F(x)=1+int_0^x F(x-u) du.
```

変数を替えると

```text
F(x)=1+int_0^x F(v) dv.
```

したがって

```text
F'(x)=F(x),  F(0)=1,
F(x)=e^x.
```

欲しい期待値は境界`1`で生成できる空語を含むprefix総量だから

```text
E[tau]=F(1)=e.
```

この証明では、各長さを先に列挙しない。文法の再帰式がそのまま生成関数の積分方程式になり、指数関数で閉じる。
