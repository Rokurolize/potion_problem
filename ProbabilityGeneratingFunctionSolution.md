# 確率母関数解

この解では、期待値だけを直接狙わず、停止時刻`tau`の確率母関数を先に閉じる。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

`tau`は正の整数値を取るので、`0 <= z <= 1`について

```text
E[z^tau]
= 1 + (z - 1) * sum_{k >= 0} z^k P(tau > k).        (1)
```

これは

```text
1 - z^tau = (1 - z) * (1 + z + ... + z^{tau-1})
```

を期待値にしただけである。

## 尾確率だけを入れる

`tau > k`は、`k`回飲んだ後もまだ境界`1`を越えていないという意味である。したがって

```text
P(tau > k) = P(S_k <= 1).
```

`k=0`ではこれは`1`である。`k >= 1`では累積和

```text
t_i = S_i
```

へ変数変換すると、Jacobianは`1`で、

```text
S_k <= 1
```

は順序単体

```text
0 < t_1 < ... < t_k <= 1
```

になる。よって

```text
P(S_k <= 1)
= int_{0<t_1<...<t_k<=1} dt_1 ... dt_k
= 1 / k!.
```

したがって

```text
sum_{k >= 0} z^k P(tau > k)
= sum_{k >= 0} z^k / k!
= exp(z).
```

(1)に代入して

```text
E[z^tau] = 1 + (z - 1) exp(z).        (2)
```

## 期待値は微分で出る

確率母関数を微分すると

```text
d/dz E[z^tau]
= d/dz { 1 + (z - 1) exp(z) }
= z exp(z).
```

したがって

```text
E[tau]
= (d/dz E[z^tau]) at z=1
= exp(1)
= e.
```

## 何が強いか

この方法は、`P(tau=n)`を個別に出さない。かわりに、正整数値の停止時刻に共通する尾和恒等式を使い、尾確率`P(tau>k)`だけを一つの指数関数へまとめる。

最後に得ているのは期待値だけではなく、

```text
E[z^tau] = 1 + (z - 1) exp(z)
```

という分布を符号化した式である。期待値`e`は、その式の`z=1`での傾きとして現れる。
