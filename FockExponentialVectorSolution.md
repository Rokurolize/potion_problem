# Fock指数ベクトル解

この解では、停止前prefixを長さごとに足すかわりに、順序付き点列を対称Fock空間の指数ベクトルとしてまとめる。答えの`e`は、指数ベクトルの正規化定数として出る。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

だから

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 順序付きprefixを対称化する

`n >= 1`で

```text
t_i = S_i
```

とおくと、Jacobianは`1`であり、`S_n <= 1`は

```text
0 < t_1 < ... < t_n < 1
```

と同じである。したがって`P(S_n <= 1)`は、区間`[0,1]`上の順序付き`n`点列のLebesgue質量である。

ここで順序を忘れ、`[0,1]^n`上の対称関数として見る。各非衝突配置には`n!`個の順序があるので、

```text
int_{0<t_1<...<t_n<1} dt_1...dt_n
= (1/n!) int_{[0,1]^n} 1 dt_1...dt_n.
```

右辺の`1/n!`は計算の副産物ではなく、順序付きprefixを対称テンソルの`n`粒子成分へ移すときのFock正規化である。

## 指数ベクトルとして一度に読む

`H = L^2([0,1])`とし、`f(t)=1`を考える。対称Fock空間の指数ベクトルは

```text
Exp(f) = (1, f, f^{otimes 2}/2!, f^{otimes 3}/3!, ...)
```

である。全ての停止前prefixを数える測度は、まさにこの各粒子数成分の全質量を足したものになっている。

実際、

```text
mass_n
= (1/n!) int_{[0,1]^n} f(t_1)...f(t_n) dt_1...dt_n
= (1/n!) <f,1>^n.
```

ここでは`<f,1>=int_0^1 1 dt=1`なので、

```text
mass_n = 1/n!.
```

しかし各`mass_n`を個別に評価する必要はない。指数ベクトルの全質量汎関数は

```text
Gamma(f)
= sum_{n >= 0} (1/n!) <f,1>^n
= exp(<f,1>).
```

これはFock空間で「独立な粒子を任意個生成する」指数法則そのものである。

## 境界`1`へ戻す

いま`f=1_{[0,1]}`なので

```text
<f,1> = 1.
```

したがって停止前prefix全体の質量は

```text
E[tau] = Gamma(1_{[0,1]}) = exp(1) = e.
```

## この見方の芯

一様乱数の累積和は、停止前である限り区間内の順序付き点列である。Fock側へ移すと、順序を持つ履歴は対称な粒子配置の指数ベクトルへ変わり、長さ方向の総和は指数ベクトルの正規化に吸収される。

`e`は、`1/n!`を後から足した数ではなく、停止前履歴を「任意個の同一粒子を生成した指数ベクトル」として見たときの全質量である。
