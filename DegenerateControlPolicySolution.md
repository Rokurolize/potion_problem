# 退化制御政策解

この解では、停止前prefixの期待個数を、選択肢のない制御問題の価値関数として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。残り容量`r`から始めたときの、空prefixを含む生存prefix総量を`V(r)`と置く。制御問題として見ると、各時点で政策は存在せず、次の一口`u`が一様密度で選ばれるだけである。したがってBellman型の価値方程式は

```text
V(r)=1+int_0^r V(r-u) du.
```

変数を替えると

```text
V(r)=1+int_0^r V(v) dv.
```

よって

```text
V'(r)=V(r),  V(0)=1.
```

したがって

```text
V(r)=e^r,
E[tau]=V(1)=e.
```

この証明では、最適化は退化している。政策選択がないため、制御問題はそのまま停止前prefixの価値方程式に潰れる。
