# tensor exponential prefix解

この解では、停止前履歴をテンソル代数の指数ベクトルとして読む。

長さ`n`まで停止しない条件は

```text
U_1+...+U_n<=1.
```

累積和座標

```text
t_k=U_1+...+U_k
```

へ移ると、これは順序単体

```text
0<=t_1<=...<=t_n<=1
```

になる。従って停止前prefixの長さ`n`成分の質量は

```text
int_{0<=t_1<=...<=t_n<=1} dt_1...dt_n.
```

テンソル代数で`v`を区間`[0,1]`の1次成分、`v^{\otimes n}`を順序付きに積分した成分と見ると、直線路のテンソル指数は

```text
exp_{\otimes}(v)=sum_{n>=0} v^{\otimes n}/n!.
```

上の順序単体積分はちょうどその次数`n`成分の係数なので

```text
P(S_n<=1)=1/n!.
```

尾和公式により

```text
E[tau]=sum_{n>=0}P(S_n<=1)=sum_{n>=0}1/n!=e.
```

停止前prefix全体は、累積和で直線路のテンソル指数に写り、期待値はその全次数のスカラー和になる。
