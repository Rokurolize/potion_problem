# static analysis prefix解

この解では、停止前prefixを静的解析で到達可能と判定される状態列として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。状態は残り容量`x`で、入力`u`を読むと`x-u`へ遷移する。ただし`u<=x`でなければ停止する。長さ`n`のprefixが到達可能である条件は

```text
u_1+...+u_n<=1.
```

したがって深さ`n`で静的解析が到達可能と判定する状態列の体積は

```text
{u_i>=0, u_1+...+u_n<=1}
```

の体積、すなわち`1/n!`である。停止時刻は到達可能prefixの数なので

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

静的解析として見ると、実行ごとの停止を追うのではなく、容量制約を満たす全prefix状態の到達可能体積を足している。
