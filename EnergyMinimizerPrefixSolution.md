# energy minimizer prefix解

この解では、停止前prefix和を、微分方程式のエネルギー最小解として特徴づける。

容量`x`での期待停止時刻を

```text
F(x)=sum_{n>=0}P(U_1+...+U_n<=x)
```

とおく。各項を微分すると一つ前の項へずれるので

```text
F'(x)=F(x),  F(0)=1.
```

この方程式は、関数`f`に対する残差エネルギー

```text
E(f)=int_0^1 (f'(x)-f(x))^2 dx + (f(0)-1)^2
```

の最小値`0`を達成する唯一の関数としても特徴づけられる。残差が0で境界値が1なら

```text
f(x)=e^x.
```

従って、停止前prefix和そのものがこのエネルギーを消す唯一の解であり、

```text
E[tau]=F(1)=e.
```

ここでは、各単体体積を計算するのではなく、prefix和全体が満たす変分的な証明書で答えを固定している。
