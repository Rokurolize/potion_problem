# Poisson配置測度の正規化解

この解では、停止前に現れるprefixを、確率分布ではなく「Poisson配置の正規化前の測度」として数える。

独立な一様乱数を`U_1,U_2,...`とし、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

と置く。`n`回後も止まっていない条件は

```text
S_n <= 1
```

である。

停止時刻は、生きているprefixの個数なので

```text
E[tau] = sum_{n>=0} P(S_n <= 1).        (1)
```

## prefixを配置に変える

`S_n <= 1`の上では

```text
0 <= S_1 <= S_2 <= ... <= S_n <= 1
```

である。増分から到着時刻への変換

```text
(U_1,...,U_n) <-> (S_1,...,S_n)
```

のJacobianは`1`である。

したがって、`P(S_n<=1)`は、区間`[0,1]`に`n`個の順序付き点を置く単体

```text
0 <= t_1 <= ... <= t_n <= 1
```

のLebesgue体積である。まだこの体積を`1/n!`とは計算しない。

## Poisson配置の正規化だけを見る

強度`1`のPoisson過程を`[0,1]`上で見る。点の個数が`n`で、かつ点の位置が

```text
0 <= T_1 <= ... <= T_n <= 1
```

にある確率密度は、順序付き単体上で

```text
exp(-1)
```

で一定である。これは、Poisson過程の有限次元密度が

```text
exp(-1) dt_1 ... dt_n
```

になるというだけである。

一方、元の停止前prefixが持つ測度は、同じ順序付き単体上の

```text
dt_1 ... dt_n
```

である。つまり元のprefix測度は、Poisson配置測度から消滅因子`exp(-1)`を外したものになっている。

## 全質量を読む

Poisson配置測度は確率測度なので、全ての点数`n`について足した総質量は`1`である。

元のprefix測度は、その各階層でPoisson配置測度の`exp(1)`倍である。したがって全階層を足した総質量は

```text
exp(1) * 1 = e.
```

しかしこの全階層の総質量は、まさに(1)の右辺である。よって

```text
E[tau] = e.
```

## この見方の芯

この証明は、単体体積を個別に計算しない。停止前の全prefixを「Poisson過程の配置空間」と同じ台に置き、違いが全階層でただ一つの正規化定数`exp(-1)`だけであることを使う。

`e`は級数を足した結果ではなく、確率測度に戻すためにPoisson過程が払っていた消滅因子の逆数として現れる。
