# 正準集団境界解

この解では、停止前prefix全体を温度`1`の正準集団として見て、境界`1`での分配関数を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。長さ`n`の生存prefixを状態とし、エネルギーを全て`0`、活動度を`1`とする。境界`x`以下の分配関数は

```text
Z(x)=sum_{n>=0} P(S_n<=x).
```

境界を`dx`だけ広げると、既存の各状態から一つ次の状態が薄層へ入る。一様密度が`1`で、Boltzmann重みも全て`1`なので

```text
Z(x+dx)-Z(x)=Z(x)dx+o(dx).
```

したがって

```text
Z'(x)=Z(x),  Z(0)=1.
```

よって

```text
Z(x)=e^x,
E[tau]=Z(1)=e.
```

この見方では、`e`は正準集団の正規化定数である。ただし相互作用もエネルギー差も無いため、分配関数は一次の境界流で完全に決まる。
