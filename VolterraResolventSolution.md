# Volterraレゾルベント解

この解では、停止時刻の分布を先に出さず、残り容量から見た「これから数えられる生存prefixの期待値」を再生方程式にする。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。求めたい量は

```text
E[tau] = E[sum_{n >= 0} 1_{S_n <= 1}]
```

である。ここで`n=0`の項は、まだ何も飲んでいない時点の生存prefixである。

## 残り容量から関数を作る

`x >= 0`に対して

```text
F(x) = E[sum_{n >= 0} 1_{U_1+...+U_n <= x}]
```

とおく。求める答えは`F(1)`である。

最初の項`n=0`が必ず`1`を寄与する。さらに、最初の乱数を`u`としたとき、`u <= x`なら残り容量は`x-u`になり、その後に期待される生存prefix数は`F(x-u)`である。`u > x`ならそこで越えるので追加寄与はない。

`U_1`は`[0,1]`上一様で、いま必要なのは`0 <= x <= 1`だけだから、

```text
F(x) = 1 + int_0^x F(x-u) du.
```

変数を入れ替えれば

```text
F(x) = 1 + int_0^x F(v) dv.
```

これは畳み込み核`1`のVolterra方程式である。

## レゾルベントで閉じる

両辺を微分すると

```text
F'(x) = F(x),
F(0) = 1.
```

したがって

```text
F(x) = exp(x).
```

求める期待値は

```text
E[tau] = F(1) = e.
```

## どこで一様性を使ったか

一様性は、最初の飲量が`du`の幅に入る確率密度を`1`として、残り容量`x-u`へそのまま移せるところにだけ現れる。独立性は、その後の過程が新しい残り容量から再び同じ`F`で表せるところに現れる。

この見方では、`1/n!`の単体体積を明示的に足し上げない。停止時刻を「生存prefixの期待個数」として受け止め、その期待値そのものがVolterra方程式のレゾルベントになっていることを使う。
