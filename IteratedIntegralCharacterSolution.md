# 反復積分の文字解

この解では、生存prefixを「区間`[0,1]`上の反復積分の全次数」としてまとめる。

独立一様乱数`U_1,U_2,...`を取り、

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

生存しているprefixを数えれば

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

したがって

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

ここで各項を別々に計算せず、全体を反復積分として読む。

## 累積和は順序単体を作る

`n >= 1`で

```text
t_i = S_i
```

と置くと、Jacobianは`1`であり、`S_n <= 1`は

```text
0 < t_1 < t_2 < ... < t_n < 1
```

と同じである。端点の違いは連続分布では期待値に影響しない。

よって

```text
P(S_n <= 1)
= int_{0<t_1<...<t_n<1} dt_1 ... dt_n.
```

この右辺は、区間`[0,1]`上の1形式`dt`を`n`回反復積分したものである。

## 全次数を一度に積む

形式的に

```text
I(x) = sum_{n >= 0} int_{0<t_1<...<t_n<x} dt_1 ... dt_n
```

と置く。空の反復積分を`1`と数える。

`I(x)`を`x`で少し伸ばすと、新しく増える鎖は、最後の点を`x`付近に持ち、その手前には任意の鎖を持つものだけである。したがって

```text
I'(x) = I(x),
I(0) = 1.
```

よって

```text
I(x) = exp(x).
```

`x=1`を入れると

```text
E[tau]
= I(1)
= exp(1)
= e.
```

## 何を使ったか

この証明の中心は、確率分布を先に閉じることではない。生存prefixを順序付きの時刻列へ送り、すべての長さの順序列を「反復積分の文字」として足す。

1文字の反復積分の全和は、積分区間を伸ばすと自分自身を係数にして増える。そこから微分方程式`I'=I`が出て、期待値はその解の`I(1)`として`e`になる。
