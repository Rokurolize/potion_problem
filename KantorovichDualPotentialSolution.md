# Kantorovich双対ポテンシャル解

この解では、停止前prefixを直接数えず、各prefixが次のprefixへ送る質量輸送を双対ポテンシャルで評価する。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。空prefixを含めて

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

残り容量`r`から見た停止前prefixの総量を`H(r)`と書く。直接には

```text
H(r)=1+int_0^r H(r-u) du
```

だが、ここではこの式を輸送の双対側で読む。

一つのprefixは、許される一口`u in [0,r]`へLebesgue測度で子prefixを送る。この親子輸送に対し、双対ポテンシャル`phi`を

```text
phi'(r)=phi(r),
phi(0)=1
```

で決める。すると任意の親容量`r`について

```text
phi(r)-1 = int_0^r phi(v) dv.
```

右辺は、親から子へ送られる全ての残り容量`v=r-u`を双対ポテンシャルで評価した値である。つまり`phi`は、親自身の単位質量と子への輸送評価が釣り合うKantorovich双対証明書になっている。

この釣り合いは、停止前履歴木の各節点で同じ形に繰り返される。したがって、全prefix質量をどの深さで切っても、双対ポテンシャルの境界差がちょうど残りの子孫輸送を支払う。単調極限を取ると

```text
H(r)=phi(r).
```

上の微分方程式から

```text
phi(r)=exp(r).
```

初期残量は`1`なので

```text
E[tau]=H(1)=phi(1)=e.
```

この証明では、各段の体積を計算しない。親子の質量輸送に対して、子孫全体の価値を支払う双対ポテンシャルを作る。`e`は、そのKantorovich双対ポテンシャルを初期容量`1`で評価した値である。
