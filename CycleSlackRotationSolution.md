# 余白巡回回転解

この解では、最後に残る余白を一成分として足し、`n+1`個の成分の巡回回転で階乗係数を見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefix数を数えれば

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

`S_n<=1`の領域では、余白

```text
R=1-S_n
```

を加えると

```text
U_1+...+U_n+R=1,  U_i>=0, R>=0
```

となる。つまり、生存している長さ`n`のprefixは、`n+1`個の非負成分で`1`を分ける単体の、最後の成分を余白として印付けしたものと同じである。

この`n+1`成分はDirichlet`(1,...,1)`の台で対称であり、余白を最後に置くことは巡回回転で選んだ一つの表示にすぎない。正規化されたDirichlet測度では全体の質量が`1`で、その一定密度は

```text
n!
```

でなければならない。したがって、正規化前の生存prefix領域の体積は

```text
P(S_n<=1)=1/n!.
```

よって

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

この見方では、`n`次元単体をそのまま積分せず、余白を含む対称な`n+1`分割として見る。階乗は、余白の位置を固定した表示をDirichlet正規化へ戻す係数として現れる。
