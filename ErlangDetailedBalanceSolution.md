# Erlang詳細釣合正規化解

この解では、生存prefixの各長さの質量を、出生死亡過程の詳細釣合と同じ比で読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。求める値は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

`n`口でまだ生きているprefixの質量を

```text
w_n = P(S_n <= 1)
```

とおく。`w_0=1`である。

## 削除と挿入の釣合

`S_n <= 1`のprefixを累積時刻

```text
0 <= T_1 <= ... <= T_n <= 1
```

として見る。これは`[0,1]`上の順序付き`n`点配置である。

順序付き`n`点配置から点を一つ選んで削除する方法は`n`通りある。したがって、印付き削除側の総質量は

```text
n w_n
```

である。

逆に、順序付き`n-1`点配置へ点を一つ挿入して順序付き`n`点配置へ戻すには、区間`[0,1]`のどこかに点を一つ置けばよい。既存の点列はその挿入位置を自動的に順序へ戻す。挿入できる全長は常に

```text
1
```

である。したがって挿入側の総質量は

```text
w_{n-1}.
```

削除してから戻す対応と、挿入してから消す対応は同じ印付き配置空間を測っているので

```text
n w_n = w_{n-1}.
```

これは、出生率`1`、死亡率`n`のErlang型出生死亡過程の詳細釣合

```text
pi_{n-1} * 1 = pi_n * n
```

と同じ式である。したがって

```text
w_n = 1/n!.
```

## 正規化定数

よって停止時刻の期待値は

```text
E[tau]
  = sum_{n >= 0} w_n
  = sum_{n >= 0} 1/n!
  = e.
```

この証明では、`n`次元単体の体積を積分で求めない。生存prefixの全長さを、到着率`1`、離脱率`n`の出生死亡過程の非正規化定常重みとして読む。`e`はその詳細釣合重みの正規化定数として現れる。
