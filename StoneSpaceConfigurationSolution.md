# Stone空間配置解

この解では、有限配置のBoolean情報をStone空間の点として見て、順序代表の総量を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。長さ`n`の停止前prefixは、累積時刻

```text
t_k=S_k
```

を使うと、区間`[0,1]`上の昇順有限配置

```text
0<=t_1<=...<=t_n<=1
```

である。有限集合の採用・非採用というBoolean代数のStone空間として見ると、ラベル付き配置の各軌道から昇順代表を一つ選んでいることになる。

対角を除けば`n`点のラベル付き配置は`n!`個の順序代表を持ち、昇順代表の質量は

```text
1/n!.
```

したがって

```text
P(S_n<=1)=1/n!,
E[tau]=sum_{n>=0}1/n!=e.
```

この証明では、Stone双対性の大きな機械は不要である。Boolean的な有限配置を順序代表へ落とすときの階乗だけが働く。
