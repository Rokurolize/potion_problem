# 生存確率微分鎖解

この解では、停止時刻の期待値を生存確率の和として置き、その和が自分自身を微分として再生することを見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }.
```

停止時刻は、まだ越えていないprefixの数である。

```text
tau = sum_{n >= 0} 1_{S_n <= 1}.
```

よって、非負項の和として

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 境界を1からxへ動かす

`0 <= x <= 1`について

```text
a_n(x) = P(S_n <= x)
```

とおく。`a_0(x)=1`である。

`n+1`個の和が`x`以下であることは、最初の飲量を`u`として、残りの`n`個が`x-u`以下であることと同じである。`U_1`は`[0,1]`上一様であり、ここでは`0 <= x <= 1`なので

```text
a_{n+1}(x) = int_0^x a_n(x-u) du
            = int_0^x a_n(t) dt.
```

したがって

```text
a_{n+1}'(x) = a_n(x).
```

生存確率列は、微分すると1つ前に戻る鎖になっている。

## 和を微分する

求めたい量の境界版を

```text
A(x) = sum_{n >= 0} a_n(x)
```

とおく。各`a_n`は非負で、`0 <= x <= 1`では

```text
a_n(x) <= x^n/n!
```

が帰納的に従うため、級数は一様に収束し、項別微分できる。よって

```text
A'(x)
= sum_{n >= 1} a_n'(x)
= sum_{n >= 1} a_{n-1}(x)
= A(x).
```

また

```text
A(0) = a_0(0) = 1
```

である。したがって

```text
A(x) = exp(x).
```

求める期待値は

```text
E[tau] = A(1) = e.
```

## この証明の見方

この証明は、`a_n(1)=1/n!`を最初に求めない。境界`x`を動かしたときに、生存確率列が

```text
a_0 <- a_1 <- a_2 <- ...
```

という微分鎖になることを使う。答えの`e`は、全生存確率をまとめた関数`A`が自分自身の導関数になるところから出る。
