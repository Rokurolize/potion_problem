# multinomial coefficient limit解

この解では、停止前単体の体積を多項係数の極限として取り出す。

長さ`n`でまだ止まっていない条件は

```text
x_1+...+x_n<=1
```

である。これに余り

```text
x_{n+1}=1-(x_1+...+x_n)
```

を加えると、`n+1`個の非負量の和が1になる標準単体に入る。

刻み幅を`1/m`にして、整数

```text
k_1+...+k_{n+1}=m
```

を数える。この個数は

```text
binom(m+n,n)
```

である。各格子点が代表する小領域の大きさはおよそ`m^{-n}`なので、

```text
P(tau>n)=vol{x_i>=0, x_1+...+x_n<=1}
       =lim_{m->infty} binom(m+n,n)/m^n
       =1/n!.
```

したがって尾和公式から

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

停止前履歴を、余りを含む多項分割の連続極限として数えると、階乗分母が多項係数の最高次項として現れる。
