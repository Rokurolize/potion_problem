# 指数レース隙間解

この解では、停止前prefixの領域を直接積分せず、独立な指数時計を正規化したときにできる隙間ベクトルとして読む。`1/n!`は単体体積ではなく、指数レースを確率ベクトルへ押し下げるときの正規化定数として現れる。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えると

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 生存prefixを隙間にする

`S_n <= 1`の領域では

```text
u_1 + ... + u_n <= 1,
u_i >= 0.
```

最後の余白を

```text
u_{n+1} = 1 - (u_1 + ... + u_n)
```

と加えると、

```text
u_1 + ... + u_{n+1} = 1,
u_i >= 0
```

という`n+1`個の隙間ベクトルになる。したがって`P(S_n <= 1)`は、この隙間単体の生のLebesgue質量である。

## 指数レースが同じ隙間を作る

独立な指数分布`E_1,...,E_{n+1}`を取り、

```text
T = E_1 + ... + E_{n+1},
P_i = E_i / T
```

とおく。指数分布の同時密度は

```text
exp(-(e_1+...+e_{n+1})) de_1...de_{n+1}
```

である。変数変換

```text
e_i = t p_i,
p_1 + ... + p_{n+1}=1
```

を使うと、Jacobianは`t^n`であり、

```text
exp(-t) t^n dt dp_1...dp_n
```

になる。`t`を積分すると

```text
int_0^infty exp(-t) t^n dt = n!
```

だから、比率ベクトル`(P_1,...,P_{n+1})`は隙間単体上で密度`n!`を持つ。

つまり、指数レースでできる比率は確率分布であり、その台は停止前prefixに余白を足した隙間単体そのものだ。確率密度が`n!`で全質量`1`になるので、生の隙間単体の質量は

```text
P(S_n <= 1) = 1/n!.
```

## 期待値

これを停止前prefixの総数へ戻すと、

```text
E[tau]
= sum_{n >= 0} P(S_n <= 1)
= sum_{n >= 0} 1/n!
= e.
```

## この見方の芯

停止前の`n`口に余白を足すと、`n+1`個の競合する指数時計の「勝ち方の比率」と同じ空間になる。指数レースの総時間`T`を捨てると、残る比率の密度は一様で、その正規化定数が`n!`になる。

答えの`e`は、単体体積を個別に計算して足したものではなく、指数時計の総時間を積分して消したあとに残る隙間比率の正規化定数を、全ての口数で足し戻した値である。
