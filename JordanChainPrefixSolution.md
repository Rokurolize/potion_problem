# Jordan鎖prefix解

この解では、prefixの長さ方向を一本のJordan鎖として扱う。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`で長さ`n`のprefixが生存する質量を

```text
a_n(x)=P(S_n <= x)
```

と置く。`0<=x<=1`では

```text
a_0(x)=1,
a_n'(x)=a_{n-1}(x).
```

これは

```text
a_0 -> a_1 -> a_2 -> ...
```

という長さ方向のJordan鎖で、境界微分が鎖を一段戻すというだけである。初期条件は空prefix

```text
a_0(0)=1,  a_n(0)=0 (n>=1)
```

なので、鎖に沿って積分すると

```text
a_n(x)=x^n/n!.
```

したがって

```text
E[tau]=sum_{n>=0}a_n(1)=e.
```

この証明の焦点は、確率分布ではなく長さ方向の線形代数である。境界を動かすとJordan鎖が一段ずつ起動し、指数級数が現れる。
