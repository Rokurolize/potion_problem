# Reeb graph capacity解

この解では、和写像のReeb graphを容量軸そのものとして使う。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。固定した`n`で

```text
q(u_1,...,u_n)=u_1+...+u_n
```

を考える。停止前領域は`q<=1`であり、`0<=q<=1`の範囲では各level setは正錐内の単体切片である。Reeb graphはこの範囲では容量軸`[0,1]`に潰れる。

level `s`の切片体積は

```text
s^{n-1}/(n-1)!
```

なので

```text
P(S_n<=1)=int_0^1 s^{n-1}/(n-1)! ds = 1/n!.
```

空prefixを含めて尾和を取れば

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

Reeb graphとして見る利点は、停止条件が高次元領域ではなく容量軸上の区間`[0,1]`として読める点である。
