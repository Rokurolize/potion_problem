# 原始関数閉包解

この解では、各段の生存確率を別々に計算せず、「一段進むこと」が原始関数を取る操作そのものだと見る。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

と置く。停止前prefixを数えれば

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 閾値`x`で各段を包む

`0 <= x <= 1`について

```text
A_n(x) = P(S_n <= x)
```

と置く。ただし`A_0(x)=1`である。`n+1`段目で`S_{n+1}<=x`となるには、直前の和が`t`以下のどこかにあり、最後の一口が残りの幅に入ればよい。密度で書けば

```text
A_{n+1}(x) = int_0^x A_n(t) dt.
```

これは単なる畳み込み計算ではなく、「一段増やす」と「`0`から`x`まで原始関数を取る」が同じ操作である、という閉包である。

## 全段を足すと微分で閉じる

全段の総量を

```text
A(x) = sum_{n >= 0} A_n(x)
```

とする。`0 <= x <= 1`では各項は非負なので、上の再帰をそのまま足してよい。すると

```text
A(x)
= A_0(x) + sum_{n >= 0} A_{n+1}(x)
= 1 + int_0^x sum_{n >= 0} A_n(t) dt
= 1 + int_0^x A(t) dt.
```

したがって

```text
A'(x) = A(x),
A(0) = 1.
```

よって

```text
A(x) = exp(x)
```

であり、求める値は

```text
E[tau] = A(1) = e.
```

である。

この証明の芯は、`P(S_n<=1)=1/n!`を先に取り出さない点にある。生存prefixを一段延ばす操作を原始関数として束ねると、全段の和が自分自身の原始関数で閉じ、指数関数が正規化条件として出てくる。
