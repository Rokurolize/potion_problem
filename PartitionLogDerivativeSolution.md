# 分配関数対数微分解

この解では、活動度つき分配関数の対数微分を使い、停止前prefix総量を境界方向の成長率として読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`以下の停止前prefixを活動度`z`でまとめて

```text
Z(x,z)=sum_{n>=0} z^n P(S_n <= x)
```

と置く。一様分布では、境界を`dx`だけ進めると既存prefixの総量`Z(x,z)`が次の一口を幅`dx`だけ受け取り、活動度を一つ増やす。よって

```text
partial_x Z(x,z)=z Z(x,z),
Z(0,z)=1.
```

対数微分で書けば

```text
partial_x log Z(x,z)=z.
```

したがって

```text
log Z(x,z)=zx,
Z(x,z)=exp(zx).
```

活動度を`1`に戻し、境界を`1`に置くと

```text
E[tau]=sum_{n>=0} P(S_n <= 1)=Z(1,1)=e.
```

ここでは級数の各係数を先に計算しない。分配関数の対数微分が定数`z`であることから、全体が指数関数でなければならないと読む。
