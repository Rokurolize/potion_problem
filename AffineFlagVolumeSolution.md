# affine flag体積解

この解では、増分座標をaffine flag座標へ移し、停止前prefixを旗単体として測る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。affine flag座標

```text
t_k=u_1+...+u_k
```

を取る。この変換は下三角で対角が全て`1`なので、Jacobianは`1`である。停止前条件は

```text
0<=t_1<=...<=t_n<=1
```

という旗単体になる。

立方体`[0,1]^n`は、測度零の境界を除いて`n!`個の順序型の旗単体に分かれる。したがって一つの旗単体の体積は

```text
1/n!.
```

よって

```text
P(S_n<=1)=1/n!,
E[tau]=sum_{n>=0}1/n!=e.
```

この見方では、斜めの不等式を扱わない。affine flagへ移すことで、停止条件が順序室一つ分になる。
