# urn process tail解

この解では、残り容量を壺の中の連続質量として扱い、尾確率を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。壺には最初に質量`1`が入っている。各回に大きさ`U_i`を取り出し、壺の質量が負になったところで停止する。`n`回後にも負になっていない条件は

```text
S_n<=1.
```

したがって尾和公式から

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

`n`回分の取り出し量がまだ壺に収まる領域は

```text
{u_i>=0, u_1+...+u_n<=1}
```

であり、体積は`1/n!`である。ゆえに

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

壺過程として見ると、必要なのは色の組成ではなく、連続質量が非負で残っているprefixの全体積である。
