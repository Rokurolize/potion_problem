# 調和ポテンシャル証明書解

独立一様乱数を `X_1, X_2, ...` とし、残り距離を

```text
R_n = 1 - (X_1 + ... + X_n)
```

とおく。`R_n >= 0` の間だけprefixは生存している。

残り距離 `r` に対して

```text
V(r) = exp(r)
```

というポテンシャルを置く。ただし境界を越えた後のポテンシャルは `0` と見る。

## 1回飲むと平均で1だけ燃える

`0 <= r <= 1` のとき、一口 `U` を飲んだ後に残るポテンシャルの条件付き期待値は

```text
E[ V(r-U) 1_{U <= r} ]
= int_0^r exp(r-u) du
= exp(r) - 1.
```

したがって、生存状態では

```text
V(r) - E[ V(r-U) 1_{U <= r} ] = 1
```

である。

つまり、停止前に一回飲むたびに、指数ポテンシャルは条件付き期待値としてちょうど `1` だけ減る。

## 停止回数をポテンシャル消費として読む

停止時刻を

```text
tau = min { n >= 1 : R_n < 0 }
```

とする。上の等式を、停止するまでの各時刻で足すと、望ましくは

```text
E[tau] = V(1) - E[V(R_tau) 1_{R_tau >= 0}]
```

となる。停止時には `R_tau < 0` なので右端のポテンシャルは `0` である。よって

```text
E[tau] = V(1) = exp(1) = e.
```

厳密には、`tau` を `tau ∧ N` で止めてから同じ等式を足し、非負性と単調収束で `N -> infinity` とすればよい。

この解では、停止時刻の分布も単体体積も使わない。残り距離に置いた `exp(r)` が、一様な一口に対して「平均で1だけ燃える」調和ポテンシャル証明書になっていることだけを見る。期待停止回数は、初期ポテンシャル `exp(1)` の消費量として `e` になる。
