# exponential tilt normalization解

この解では、停止前prefix空間に共通の指数因子を掛けて確率測度に正規化する。

長さ`n`の停止前領域を

```text
D_n={u_i>=0, u_1+...+u_n<=1}
```

とする。元の停止時刻について

```text
E[tau]=sum_{n>=0} vol(D_n)
```

である。

各`D_n`上に同じ重み`e^{-1}`を掛けた測度を考える。累積和

```text
t_k=u_1+...+u_k
```

へ送ると、`D_n`は

```text
0<=t_1<=...<=t_n<=1
```

へヤコビアン`1`で移る。これは、強度`1`のPoisson過程で、`[0,1]`内にちょうど`n`個の点があるときのJanossy密度

```text
e^{-1} dt_1...dt_n
```

そのものである。

したがって、全ての`n`を足した指数傾斜測度はPoisson過程の全確率になり

```text
sum_{n>=0} e^{-1} vol(D_n)=1.
```

よって

```text
E[tau]=sum_{n>=0} vol(D_n)=e.
```

この証明では、停止前prefix全体が指数因子`e^{-1}`でちょうど確率に正規化されることを使い、期待停止時刻を正規化定数として読む。
