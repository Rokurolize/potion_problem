# 指数レース生存解

この解では、一様乱数の和を直接見るかわりに、各一口を指数時計の中での勝ち残り時間として読む。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixの個数は

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので、求める量は

```text
E[tau] = sum_{n >= 0} P(S_n <= 1).
```

## 一様増分を指数時計の勝利条件にする

強度`1`のPoisson過程を時刻区間`[0,1]`で見る。点が`n`個あるとき、その順序統計量は

```text
0 <= T_1 <= ... <= T_n <= 1
```

の上で一定密度を持つ。

一方、`S_n <= 1`の条件下では、部分和列

```text
0 <= S_1 <= ... <= S_n <= 1
```

が同じ順序付き領域を走る。増分から部分和への変換のJacobianは`1`である。

つまり、停止前prefixを数えることは、`[0,1]`の中に現れたPoisson点列を、Poisson確率の消滅因子を外して数えることに等しい。

## レースの空白因子

Poisson過程で`[0,1]`内の配置を確率として見ると、点列の密度には常に

```text
exp(-1)
```

が付く。これは「区間全体の総時計が、配置を確率測度にするために払う空白因子」である。

停止前prefixの測度では、この空白因子は付いていない。したがって、Poisson配置確率測度を停止前prefix測度に戻すには、全階層を一括して

```text
exp(1)
```

倍すればよい。

Poisson配置確率測度の全質量は`1`なので、停止前prefix測度の全質量は

```text
exp(1) * 1 = e.
```

この全質量はまさに

```text
sum_{n >= 0} P(S_n <= 1) = E[tau]
```

である。よって

```text
E[tau] = e.
```

この証明では、`P(S_n <= 1)=1/n!`を先に計算しない。指数時計の配置を確率測度へ正規化するときに現れる空白因子を外すと、停止前prefix全体の質量がそのまま`e`になる。
