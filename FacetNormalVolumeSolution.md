# facet normal volume解

この解では、停止前単体の体積を斜面facetの法線方向から再帰的に求める。

容量`x`での長さ`n`停止前領域を

```text
D_n(x)={u_i>=0, u_1+...+u_n<=x}
```

とし、その体積を`A_n(x)`とする。斜面facet

```text
u_1+...+u_n=x
```

の法線方向に`x`を少し動かすと、体積の増分はそのfacetの射影体積で決まる。最後の座標を消去する射影により

```text
A_n'(x)=A_{n-1}(x).
```

また`A_0(x)=1`である。帰納的に

```text
A_n(x)=x^n/n!.
```

元の問題では

```text
P(U_1+...+U_n<=1)=A_n(1)=1/n!.
```

尾和公式から

```text
E[tau]=sum_{n>=0}1/n!=e.
```

この証明では、体積を全体で積分するのではなく、停止境界facetが動くときの法線方向の増分だけを使う。
