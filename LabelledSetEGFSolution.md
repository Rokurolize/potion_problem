# ラベル付きSET指数型母関数解

この解では、停止前prefixをラベル付き有限集合の指数型母関数として読む。

長さ`n`の生存prefixは、累積時刻

```text
0 <= t_1 <= ... <= t_n <= 1
```

に移すと、区間`[0,1]`上の順序付き`n`点配置である。増分から累積時刻への変換はJacobian`1`である。

一方、ラベル付き`n`点配置を先に考えると、空間は

```text
[0,1]^n
```

で体積`1`である。対角集合は測度`0`なので、ほとんど全ての配置は`n!`通りのラベル順を持つ。その中で、時刻順がprefix順と一致するものは一つだけである。したがって順序付きprefix領域の体積は

```text
1/n!
```

である。

## SETの指数型母関数

この`1/n!`は、ラベル付き有限集合`SET`の指数型母関数の係数そのものである。1点の重みが区間の体積

```text
int_0^1 1 dx = 1
```

であるとき、全有限集合の指数型母関数は

```text
sum_{n >= 0} 1^n / n! = exp(1).
```

尾和公式から

```text
E[tau] = sum_{n >= 0} P(S_n <= 1),
```

各項が上の`SET`係数なので

```text
E[tau] = exp(1) = e.
```

この証明では、単体体積を積分で計算しない。生存prefixをラベル付き配置の中で時刻順とラベル順が一致する一つの成分として読み、全長さの和を`SET`の指数型母関数に委ねる。
