# occupancy prefix box解

この解では、停止前prefixを容量箱に順に入る占有列として数える。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。容量`1`の箱に、長さ`U_i`の荷物を順に入れる。長さ`n`のprefixが箱の中に収まる条件は

```text
U_1+...+U_n<=1
```

である。したがって深さ`n`まで占有可能な荷物列の体積は

```text
{u_i>=0, u_1+...+u_n<=1}
```

の体積で、

```text
1/n!
```

になる。

停止時刻は、箱からあふれる直前までに収まったprefixの数、空prefixを含めた個数である。よって

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

占有問題として見ると、乱数の列を直接追うよりも、「箱に収まる全prefix列の体積」を足すだけでよい。
