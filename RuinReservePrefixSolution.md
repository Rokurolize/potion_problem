# ruin reserve prefix解

この解では、初期準備金`1`が請求で破産するまでのprefix数として問題を見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。`U_i`を請求額、`1`を初期準備金と考えると、`n`件目まで破産していない条件は

```text
S_n<=1.
```

停止時刻は破産前に観測されるprefixの個数なので、尾和公式により

```text
E[tau]=sum_{n>=0} P(S_n<=1).
```

固定した`n`では、請求額列が準備金内に収まる領域は

```text
{u_i>=0, u_1+...+u_n<=1}
```

であり、体積は`1/n!`である。したがって

```text
E[tau]=sum_{n>=0} 1/n! = e.
```

破産時の超過額は不要である。破産前に準備金内へ収まる請求prefixの全体積だけを足せばよい。
