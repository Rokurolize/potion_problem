# continued fraction tail解

この解では、有限地平の尾和を連分数的な後ろ向き再帰の極限として見る。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。容量`x`で残り`N`段まで見る停止前prefix総量を`F_N(x)`と書く。

```text
F_0(x)=1,
F_{N+1}(x)=1+int_0^x F_N(t) dt.
```

帰納的に

```text
F_N(x)=sum_{k=0}^{N} x^k/k!.
```

有限段の後ろ向き再帰を繰り返すと、各段で積分作用素が一つ重なる。これは有限連分数を打ち切りながら極限を取るのと同じ役割を持ち、極限`F`は

```text
F(x)=1+int_0^x F(t) dt.
```

したがって

```text
F'(x)=F(x),  F(0)=1,
```

より

```text
F(x)=e^x.
```

最初の容量は`1`なので

```text
E[tau]=F(1)=e.
```

この見方では無限和を先に書かない。有限地平の後ろ向き再帰を延ばし、その極限方程式として指数関数を得る。
