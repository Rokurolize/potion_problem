# Dirac質量境界評価解

この解では、境界`1`での値をDirac質量による評価汎関数として扱い、停止前prefixの全和を読む。

独立一様乱数`U_1,U_2,...`について

```text
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。境界`x`までのprefix総量を

```text
F(x)=sum_{n>=0} P(S_n <= x)
```

と置く。`0 <= x <= 1`では、次の一口の密度は区間内で定数`1`なので

```text
F'(x)=F(x),  F(0)=1.
```

いま境界評価を

```text
delta_1(F)=F(1)
```

と書く。微分方程式の解は、初期Dirac質量`delta_0`を境界方向に指数的に押し出したものとして

```text
F(x)=exp(x)
```

である。したがって評価汎関数を境界`1`に当てると

```text
E[tau]=delta_1(F)=F(1)=e.
```

この証明では、確率分布の全形を求めない。停止前prefix測度が境界方向に自己増殖し、最後にDirac評価で`x=1`を読むだけである。
