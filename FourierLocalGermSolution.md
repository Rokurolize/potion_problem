# Fourier局所芽解

独立一様乱数を `X_1, X_2, ...` とし、

```text
S_n = X_1 + ... + X_n
```

とおく。停止前prefixの個数として

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので、

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

`S_n <= 1` の範囲では各 `X_i` の上端制約 `X_i <= 1` は自動的に満たされる。したがって、時刻 `0 <= x <= 1` に見える畳み込み密度は、半直線上の定数密度

```text
1_{x >= 0}
```

の局所芽だけで決まる。

## Fourier側で局所芽を見る

半直線のHeaviside密度 `H(x)=1_{x>=0}` のFourier-Laplace的な記号は、形式的に

```text
1 / (i xi)
```

である。畳み込みは記号の積になるので、`n` 回畳み込んだ局所芽は

```text
1 / (i xi)^n
```

に対応する。

これを逆変換すると、`x >= 0` で

```text
H^{*n}(x) = x^{n-1} / (n-1)!      (n >= 1)
```

となる。つまり、`S_n` の密度は `0 <= x <= 1` では

```text
x^{n-1} / (n-1)!
```

である。

したがって

```text
P(S_n <= 1)
= int_0^1 x^{n-1} / (n-1)! dx
= 1 / n!
```

であり、空prefix `n=0` の寄与 `1` を加えると

```text
E[tau]
= sum_{n >= 0} 1 / n!
= e
```

である。

この証明では、Irwin-Hall分布全体を求めない。Fourier側では、境界 `1` までに見えるのは一様密度の右端ではなく、半直線定数密度の局所芽だけである。その局所芽の畳み込み記号が階乗係数を強制し、総和として `e` が現れる。
