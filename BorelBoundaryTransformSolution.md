# Borel境界変換解

この解では、各`n`について単体体積を計算してから足すのではなく、停止前prefixの全列を一つのBorel変換として持ち上げる。境界`1`は、その変換を`z=1`で評価する操作として現れる。

独立な一様乱数`U_1,U_2,...`について

```text
S_0 = 0,
S_n = U_1 + ... + U_n,
tau = min { n >= 1 : S_n > 1 }
```

とする。停止前prefixを数えれば

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

なので

```text
E[tau] = sum_{n >= 0} P(S_n <= 1)
```

である。

## 境界を動かして一つの関数にする

境界を`x >= 0`へ動かし、

```text
A(x) = sum_{n >= 0} P(S_n <= x)
```

とおく。ただし`P(S_0 <= x)=1`である。

`0 <= x <= 1`では、`S_n <= x`なら各`U_i`は自動的に`0 <= U_i <= 1`を満たす。したがって

```text
P(S_n <= x)
```

は正の単体

```text
u_1 + ... + u_n <= x
```

の体積であり、

```text
P(S_n <= x) = x^n / n!
```

となる。これは個別の答えを足しただけに見えるが、ここでは別の読み方をする。

## prefix列のBorel変換

列`1,1,1,...`の通常の母関数は発散境界を持つが、停止前prefixでは次元`n`の単体体積が自然に`n!`を割る。つまり境界`x`までの全prefix質量は、定数列のBorel変換

```text
B(x) = sum_{n >= 0} x^n / n!
```

そのものになる。

この関数は、微分すると一つ次元を落とす。

```text
B'(x)
= sum_{n >= 1} x^(n-1) / (n-1)!
= B(x),
```

かつ`B(0)=1`である。したがって

```text
B(x) = exp(x).
```

いま欲しい境界は`x=1`なので

```text
E[tau] = A(1) = B(1) = exp(1) = e.
```

## なぜこれは元の停止時刻を外していないか

ここで数えているのは、各長さ`n`について`S_n <= 1`であるprefixだけである。これはまさに

```text
tau = sum_{n >= 0} 1_{S_n <= 1}
```

で数えられる停止前prefixの個数であり、超過後の項は一つも入らない。

`e`は「単体体積を足すと指数関数になる」という計算結果ではなく、停止前prefix列をBorel変換に通し、境界`1`で評価した値として現れる。乱数の上限`1`と停止境界`1`が一致しているため、境界内の各prefixは上端制約を忘れて正の単体だけで読める。
