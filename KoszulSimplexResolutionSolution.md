# Koszul simplex resolution解

この解では、停止前単体を一次式

```text
x_1+...+x_n<=1
```

で切られた正の直交象限として扱う。

一次式

```text
l=x_1+...+x_n
```

に沿って次数を測ると、`l=r`の層は`n-1`次元単体になる。Koszul複体が一次式の制約を解消するときと同じように、自由な`n`個の座標から1本の線形制約を取り出すと、層の体積密度は

```text
r^{n-1}/(n-1)!
```

になる。

停止前領域は`0<=r<=1`の層を全部足したものだから

```text
P(tau>n)=int_0^1 r^{n-1}/(n-1)! dr=1/n!.
```

尾和公式により

```text
E[tau]=sum_{n>=0}P(tau>n)
      =sum_{n>=0}1/n!
      =e.
```

線形制約を1本だけ持つKoszul的な解消として見ると、停止前確率は制約層の体積密度を積分するだけになる。
